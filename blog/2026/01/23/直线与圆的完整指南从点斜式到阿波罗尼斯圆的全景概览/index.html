直线与圆的完整指南：从点斜式到阿波罗尼斯圆的全景概览¶

本篇博文为您浓缩《直线和圆》全文的核心要点，涵盖直线方程、圆的方程、二者的几何关系以及若干经典定理，帮助您快速抓住解析几何的关键概念，点燃学习兴趣，立即提升解题效率。

直线方程¶

点斜式¶

如果已知直线经过点 \((x_0,y_0)\) 且倾斜角为 \(\theta\)，则其斜率为 \(k=\tan\theta\)，点斜式写作

\[ y - y_0 = k\,(x - x_0) \]

两点式¶

已知两点 \(A(x_1,y_1)\) 与 \(B(x_2,y_2)\)（\(x_1\neq x_2\)），直线方程可写作

\[ y - y_1 = \frac{y_2 - y_1}{x_2 - x_1}\,(x - x_1) \]

或等价形式

\[ \frac{y - y_1}{x - x_1} = \frac{y_2 - y_1}{x_2 - x_1} \]

斜截式¶

当直线与 \(y\) 轴相交于点 \((0,b)\) 时，方程化为

\[ y = kx + b \]

截距式¶

若直线截于 \(x\) 轴、\(y\) 轴的两点分别为 \((a,0)\) 与 \((0,b)\)，则

\[ \frac{x}{a} + \frac{y}{b} = 1 \]

注意：正比例函数 \(y=x\) 无法写成截距式，需要单独说明。

一般式¶

任意直线均可写作

\[ Ax + By + C = 0 \]

其中 \(A,B\) 为法向量分量，斜率为 \(-\dfrac{A}{B}\)（\(B\neq 0\)），横、纵截距分别为 \(-\dfrac{C}{A}\) 与 \(-\dfrac{C}{B}\)。

圆的方程¶

极坐标¶

在极坐标系中，以原点 \(O\) 为极点，极径 \(\rho\) 与极角 \(\varphi\) 满足

\[ x = \rho\cos\varphi ,\qquad y = \rho\sin\varphi \]

标准方程¶

圆心为 \((a,b)\)、半径为 \(r\) 的圆表示为

\[ (x-a)^2 + (y-b)^2 = r^2 \]

一般方程¶

展开后可写成

\[ x^2 + y^2 + Dx + Ey + F = 0 \]

只要 \(D^2 + E^2 > 4F\)，该式即描绘真实圆。圆心与半径分别为

\[ \bigl(-\tfrac{D}{2},\,-\tfrac{E}{2}\bigr),\qquad \tfrac{ \sqrt{D^2 + E^2 - 4F} }{2} \]

向量方程¶

使用向量 \(\mathbf{r}\) 表示点坐标，圆的向量形式为

\[ \bigl\lVert \mathbf{r} - \mathbf{c} \bigr\rVert = r \]

其中 \(\mathbf{c}\) 为圆心向量。

圆心系¶

若圆心随参数 \(\theta\) 旋转，方程

\[ (x-3\cos\theta)^2 + (y-3\sin\theta)^2 = 1 \]

描述半径 \(1\)、圆心在半径 \(3\) 的圆环上运动的所有圆。

直线与圆的交互¶

平行与垂直¶

  • 平行：\(k_1 = k_2\)（或 \(\theta_1 = \theta_2\)）。
  • 垂直：\(k_1k_2 = -1\)（或 \(\theta_1 + \theta_2 = 90^\circ\)）。

交点与距离¶

两条一般式直线

\[ \begin{cases} A_1x + B_1y + C_1 = 0\\ A_2x + B_2y + C_2 = 0 \end{cases} \]

的交点为

\[ x = \dfrac{B_1C_2 - B_2C_1}{A_1B_2 - A_2B_1},\qquad y = \dfrac{A_2C_1 - A_1C_2}{A_1B_2 - A_2B_1} \]

点 \((x_0,y_0)\) 到直线 \(Ax+By+C=0\) 的距离

\[ d = \frac{\bigl|Ax_0 + By_0 + C\bigr|}{\sqrt{A^2 + B^2}} \]

两条平行直线的距离

\[ d = \frac{|C_2 - C_1|}{\sqrt{A^2 + B^2}} \]

位置关系¶

  • 相离：圆心到直线的距离 \(d > r\)。
  • 相切：\(d = r\)，此时唯一交点即切点。
  • 相交：\(d < r\)，交点构成弦，弦长
\[ L = 2\sqrt{r^2 - d^2} \]

切线问题¶

已知圆 \((x-a)^2+(y-b)^2=r^2\)，

  • 圆上点 \(P(x_0,y_0)\) 的切线方程

$$ (x_0-a)(x-a) + (y_0-b)(y-b) = r^2 $$

  • 圆外点 \(Q(x_1,y_1)\) 的两条切线可由上述式子中的 \((x_0,y_0)\) 替换为切点坐标来求解；也可直接使用点到圆心的距离判别。

阿波罗尼斯圆¶

设定点 \(A(x_a,y_a)\)、\(B(x_b,y_b)\)，若点 \(P\) 满足

\[ \frac{PA}{PB}=k\;(k\neq 1) \]

则 \(P\) 的轨迹为阿波罗尼斯圆，方程

\[ \frac{(x-x_a)^2+(y-y_a)^2}{(x-x_b)^2+(y-y_b)^2}=k^2 \]

代数关系¶

参数方程¶

圆的参数方程

\[ \begin{cases} x = a + r\cos\theta\\ y = b + r\sin\theta \end{cases} \]

椭圆、双曲线等曲线均可通过适当的参数化获取。

仿射变换¶

一次线性变换加平移可将椭圆映射为圆：

\[ x' = \frac{b}{a}\,x,\qquad y' = y \]

使得 \(\displaystyle \frac{x'^2}{b^2}+y'^2 = 1\) 变为单位圆方程。

向量关系¶

若 \(\overrightarrow{PA} = \lambda\,\overrightarrow{PB}\)，则 \(P\) 位于 \(AB\) 的延长线上，可通过坐标代入快速求解。

角度问题¶

两条斜率为 \(k_1\)、\(k_2\) 的直线夹角满足

\[ \tan\theta = \frac{k_2 - k_1}{1 + k_1k_2} \]

特别地，\(k_1k_2 = -1\) 时 \(\theta = 90^\circ\)。

轨迹问题¶

动点满足的几何约束往往可转化为代数方程。例如，动点 \(P\) 与固定点 \(A\)、\(B\) 的距离平方和为常数 \(C\)，其轨迹即圆：

\[ (PA)^2 + (PB)^2 = C \]

拓展内容¶

托勒密定理¶

圆内接凸四边形 \(ABCD\) 满足

\[ AB\cdot CD + AD\cdot BC = AC\cdot BD \]

广义形式：左侧大于等于右侧，等号成立当且仅当四边形可内接于同一圆。

相交弦定理¶

两条弦 \(AB\) 与 \(CD\) 在点 \(P\) 相交，则

\[ PA\cdot PB = PC\cdot PD \]

圆幂定理¶

若点 \(P\) 与圆 \(O\) 的交点为 \(A,B\)，则

\[ PA\cdot PB = \bigl|PO\bigr|^2 - r^2 \]

此式用于快速判断点在圆内外及求切线长度。

西姆松定理¶

点 \(P\) 在三角形 \(ABC\) 的外接圆上时，\(P\) 对三边的投影共线，这条共线称为西姆松线。

定点速求¶

对动直线 \(MN\)，若两端点可写成参数形式，可通过齐次坐标求出不随参数变化的定点 \(X\)：

\[ X = L \times L' \quad\bigl(L = M \times N\bigr) \]

该技巧在求包络线、确定不动点时异常高效。
