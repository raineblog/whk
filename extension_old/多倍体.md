## （1）两个单倍体细胞融合后，得到的细胞**一定**有 2 个染色体组吗？

**不一定。**关键看你说的“融合”到底到哪一步，以及后续染色体是否稳定保留。

### 1）只发生“细胞膜融合/胞质融合”时：不一定变成“2个染色体组的一个细胞核”

细胞融合后常见的第一阶段是形成**异核体（heterokaryon，两个细胞核共用一个细胞质）**：细胞已经“一个细胞”了，但**核还没合并**，所以不能简单说“一个核里有 2 个染色体组”。[1](https://en.wikipedia.org/wiki/Cell_fusion)

### （1）两个单倍体细胞融合后，得到的细胞一定有 2 个染色体组吗？

**答：不一定。**
这是一个非常经典的“概念陷阱”。
“单倍体细胞”的定义是：含有本物种**配子**染色体数目的细胞。

- 如果原来的生物是**二倍体**（如人、果蝇），它的单倍体细胞确实只有1个染色体组。两个这样的细胞融合后，确实是 1+1=2 个染色体组。
- **但是**，如果原来的生物是**多倍体**，情况就不同了。例如，普通小麦是六倍体（含有6个染色体组），它的配子（即单倍体细胞）含有3个染色体组。如果将两个小麦的单倍体细胞进行融合，得到的细胞将含有 3+3=**6个染色体组**。
- 此外，在动物细胞融合的实际操作中（例如人-鼠细胞融合），融合后细胞在分裂过程中还经常会发生**染色体随机丢失**的现象。
所以，不能绝对地说一定有2个染色体组。

### （2）题目中说“将两种细胞置于培养基中进行两两细胞融合”，认为有几种得到的细胞（筛选之前）？

**答：有 3 种融合细胞。**
假设这两种细胞分别是 A细胞 和 B细胞。“两两融合”的意思是每次只有两个细胞贴在一起发生融合。那么在培养基中，可能发生的组合只有三种：

- **A-A 融合细胞**（同种融合）
- **B-B 融合细胞**（同种融合）
- **A-B 融合细胞**（杂交细胞，也就是我们制备单抗时真正需要的）
**做题小提示**：如果题目问的是“筛选前，培养基中**发生融合的细胞**有几种？”，答案是 **3种**。如果题目问的是“筛选前，培养基中**所有的细胞**有几种？”，答案则是 **5种**（除了上述3种融合细胞，还要加上没有发生融合的游离A细胞和游离B细胞）。

### （3）单克隆抗体技术中，“浆细胞”与“免疫过的B淋巴细胞”哪个说法更对？

**答：在高中生物考试的标准中，“免疫过的B淋巴细胞”（或新教材中的“能产生特定抗体的B淋巴细胞”）更对。**
你看到的这两道题，其实反映了“生物学绝对事实”与“高中教材术语规范”之间的摩擦：

- **从绝对的生物学事实来看：** 只有高度分化的 **浆细胞（效应B细胞）** 才具有合成和分泌抗体的能力。记忆B细胞或未被激活的B细胞是不能分泌抗体的。因此，实际上真正与骨髓瘤细胞融合并发挥作用的就是浆细胞。
- **从教材规范和考试得分来看：** 人教版高中生物教材（无论是老版还是2019新版），在描述这段制备过程时，原话都是“从小鼠的脾中得到**能产生特定抗体的B淋巴细胞**”（老教材常简称为“已免疫的B淋巴细胞”）。在教材语境中，“B淋巴细胞”是一个统称的概念，涵盖了受抗原刺激后正在增殖分化的那个细胞群体。
**结论：** 考试做题时，遵循**“教材原话至上”**的原则。如果让你判断哪个说法最严谨标准，一定要选**“已免疫的B淋巴细胞”**或**“能产生特定抗体的B淋巴细胞”**。如果有解析判“浆细胞”错，是因为出题人死扣教材字眼，认为浆细胞高度分化不能分裂，而教材为了描述从脾脏提取的整个操作流程使用了“B淋巴细胞”这个略微宏观的词汇。顺着教材的标准答案写是最稳妥的。

### （4）植物叫“杂种细胞”，动物叫“杂交细胞”，是因为得不到动物个体吗？如果植物只得到组织，还能叫杂种细胞吗？

**答：你的直觉非常敏锐，完全正确！叫法差异确实与“发育潜能（能否得到完整的个体）”有关。**

- **为什么有区别？**

- **植物体细胞杂交**：植物细胞具有很容易表达的**全能性**。这项技术的最终目的，通常是为了培育出一个完整的、具有双亲性状的新植物个体——**“杂种植株”**（相当于创造了一个新的“杂种”）。既然它的最终归宿是一个“杂种个体”，那么作为起点的融合细胞，自然就被称为**“杂种细胞”**。
- **动物细胞融合**：动物体细胞的全能性在体外极难实现（必须借助核移植等其他手段）。动物细胞融合后，最多只能在体外分裂形成细胞系（比如杂交瘤细胞），或者最多发育成组织，**无法发育成一个完整的“杂种动物个体”**。它仅仅停留在细胞层面的“杂交”过程，所以被称为**“杂交细胞”**。

- **如果植物的只得到组织（或者没发育到个体），还能叫杂种细胞吗？**

- **依然必须叫“杂种细胞”。** 生物学名词的定义是基于该细胞**内在的遗传组成和潜在的发育能力**，以及该技术的常规应用目标。只要它是通过植物体细胞融合得到的，哪怕你的实验中途失败了，或者只培养到了“愈伤组织”阶段就不培养了，它本身的全能性依然存在，它的专用名词依然是“杂种细胞”。
- **考试提醒：** 答题时千万不要根据实验“进行到了哪一步”去主观替换名词。对于植物，统统写“杂种细胞”；对于动物，统统写“杂交细胞”。不可混用！

## 1）先把三件事分开：2n / n / x（这是所有混乱的根源）

### A. 2n（或“体细胞染色体数”）

- **2n**是“这个物种正常体细胞/合子阶段的染色体数”的记号（传统上这么写，即使它不是二倍体也常写成 2n=…）。
- 关键点：**2n 只是一个“数”，不直接告诉你有几套染色体组。**在多倍体里，2n 可能等于 4x、6x…… [2](https://link.springer.com/article/10.1007/s00425-026-04952-w)

### B. n（配子染色体数 / 减数分裂后的“还原数”）

- **n**指这个物种配子（或配子体阶段）里的染色体数。它本质上是“减数分裂后的数”。 [1](https://mcb.berkeley.edu/courses/mcb140/Syllabus/AmacherLecture/Lecture12.pdf)
- 在**二倍体物种**里通常 n = x（所以高中经常直接把“单倍体=一套”讲死）。
- 在**多倍体物种**里，n 往往 **≠ x**（这就是你现在混乱的核心）。

### C. x（基本染色体数 / 一套“完整不同染色体”的数目）

- **x（basic chromosome number / monoploid number）**指“**一套**完整的不同染色体（一个基因组 set）有多少条”。 [1](https://mcb.berkeley.edu/courses/mcb140/Syllabus/AmacherLecture/Lecture12.pdf)
- 倍性（几倍体：2x、4x…）其实说的是“有几套 x”。

一句最关键的判别：
**‘几倍体’最好用 x 来说（2x、4x…），‘单倍体’最好用 n 来说（n 这个阶段/这类个体）。**
混用就会出现你说的“BB 或 BC 到底是不是单倍体”的矛盾。 [1](https://mcb.berkeley.edu/courses/mcb140/Syllabus/AmacherLecture/Lecture12.pdf)

## 2）把“单倍体、二倍体、四倍体”用最严谨的一句话定义

### （1）“几倍体”（diploid/tetraploid…）——说的是 **x 的倍数**

- 二倍体 = **2x**
- 三倍体 = **3x**
- 四倍体 = **4x**
这叫“倍性/染色体组数”。 [1](https://mcb.berkeley.edu/courses/mcb140/Syllabus/AmacherLecture/Lecture12.pdf)

### （2）“单倍体（haploid）”——严格讲是 **n（配子数）这个核相/个体**

学术上常用的定义之一就是：“**单倍体 = 具有该物种的减数分裂后染色体数 n 的细胞/个体**”。 [3](https://pmc.ncbi.nlm.nih.gov/articles/PMC4246724/)
但注意：文献也指出 “haploid” 这个词历史上在不同类群里有多种用法，尤其在植物和“世代交替/核相”语境下更复杂；因此很多课程会特意强调 n 与 x 的区别来避免误用。 [3](https://pmc.ncbi.nlm.nih.gov/articles/PMC4246724/)

### （3）“一倍体（monoploid）”——说的是 **x（只有一套基本染色体组）**

也就是“只有一个 chromosome set”的意思。 [4](https://www.merriam-webster.com/dictionary/monoploid)

**结论：高中常把“单倍体=一倍体=只有一套染色体组”当成等价，是因为默认讨论对象是‘二倍体物种’。**一旦讨论多倍体物种，这三个概念就不再自动等价。 [1](https://mcb.berkeley.edu/courses/mcb140/Syllabus/AmacherLecture/Lecture12.pdf)

## 3）用你熟悉的 A/B/C 字母法，把“BB、BC、BBBB、BBCC”讲清楚

很多教材用字母表示“来源不同的一整套基因组（genome）”。比如：

- **AAAA**：同源四倍体（autotetraploid）
- **AABB**：异源四倍体（allotetraploid） [2](https://link.springer.com/article/10.1007/s00425-026-04952-w)

### 情况 1：二倍体物种（最符合高中默认）

假设某植物是二倍体：

- 体细胞：**2n = 2x**（两套）
- 基因组式：**BB**（可以理解为两套 B）
- 配子：**n = x**，是 **B**（一套）

这时：

- “单倍体（n）” = “一倍体（x）” = **B**
- 秋水仙素把 **B → BB**，就变成**二倍体（2x）**。

### 情况 2：同源四倍体物种（AAAA 或 BBBB）

假设该物种是同源四倍体：

- 体细胞：**2n = 4x**
- 基因组式：**BBBB**（四套 B） [2](https://link.springer.com/article/10.1007/s00425-026-04952-w)
- 它的配子正常情况下是：**n = 2x**（注意：配子里是两套！），基因组式 **BB**。

这时就出现你觉得“脑袋打结”的点了：

- 这个物种的“**单倍体（haploid）**”（按 n 来说）其实是 **BB（n=2x）**
- 但按“染色体组数（x 的倍数）”它又是 **2x（二倍体）**
也就是说：**“单倍体（n）”不等于“只有一套（x）”**——因为这个物种本身就是 4x。 [1](https://mcb.berkeley.edu/courses/mcb140/Syllabus/AmacherLecture/Lecture12.pdf)

秋水仙素处理这个“单倍体（n=2x）”会发生：

- **BB（n=2x） → BBBB（2n=4x）**
结果是回到该物种的“正常四倍体水平”，育种上这叫“加倍单倍体（doubled haploid, DH）”的思想框架之一（本质：先把基因型变成全纯合，再加倍恢复可育）。这里你会直观看到：**同样叫“加倍”，但最后叫“四倍体”而不是“二倍体”，取决于原物种的 x/倍性背景。**

### 情况 3：异源四倍体物种（BBCC 这种）

假设某物种是异源四倍体：

- 体细胞：**2n = 4x**
- 基因组式：**BBCC**（两套 B + 两套 C） [2](https://link.springer.com/article/10.1007/s00425-026-04952-w)
- 正常配子：**n = 2x**，基因组式 **BC**（各来一套）

于是：

- 这个物种的“单倍体（n）”应当是 **BC**（因为它的配子数就是 BC）。
- **BB 或 CC**在这里一般不作为“该物种正常意义上的单倍体配子类型”（除非出现不正常配子/染色体丢失、远缘杂交不配对等特殊情况，那就另当别论）。

你原话里说“单倍体育种得到的，不管是 BB 还是 BC 还是 B/C 都是单倍体”，如果放到严谨定义下应该改成：

**单倍体 = 拥有该材料的配子染色体数 n 的个体；而 n 可能等于 x，也可能等于 2x、3x……取决于材料本身的倍性。**

## 4）秋水仙素“加倍”到底把它变成二倍体还是四倍体？——规则其实很简单

### 机制（为什么会“加倍”）

秋水仙素会抑制微管/纺锤体形成，使细胞在分裂时**染色体分开受阻（不能正常进入后期/分离）**，随后细胞核重建时可能把同一套复制后的染色体都装回一个核里，从而形成“染色体组加倍”的细胞。经典早期研究就指出它通过抑制纺锤体、阻断后期来导致多倍化，同时也可能出现微核等异常。 [5](https://www.nature.com/articles/1401101a0)

### 结果规律（高中最该记住的“一条”）

如果是“整套都成功加倍”，那它就是：

- **x 倍性翻倍**：1x→2x，2x→4x，3x→6x……
- 或者按染色体数：N 条→2N 条

所以：

- 你拿**一倍体（x）**去加倍 → **二倍体（2x）**
- 你拿**二倍体（2x）**去加倍 → **四倍体（4x）**
- 你拿“某物种的单倍体（n）”去加倍 → 得到的是 **2n**（但 2n 对应几倍体，要看该物种的 x）

这正是你感觉“它可能变二倍体，也可能四倍体”的原因：
你在脑内同时用两套坐标系在说话——**一会儿用 n（单倍体），一会儿用 x（几倍体）**。

额外提醒：实际操作里也常出现**嵌合体/混倍体（mixoploid）**、部分细胞加倍、甚至异常分裂导致的非整倍体等，所以“加倍”在现实中不一定 100%干净整齐；但高中题通常默认“整套成功加倍”。 [5](https://www.nature.com/articles/1401101a0)

## 5）这到底是高中生物的问题，还是学术上确实如此？

两者都有：

### 高中层面的“问题”

高中为了省事，常隐含前提：“讨论对象是二倍体物种”，于是直接把：

- 单倍体 ≈ n ≈ x ≈ 一套染色体组
讲成一个意思。这个在**人类/多数动物/很多经典二倍体例子**里没错（因为 n=x），但一遇到植物多倍体就崩。 [1](https://mcb.berkeley.edu/courses/mcb140/Syllabus/AmacherLecture/Lecture12.pdf)

### 学术层面的“确实如此”

学术上也承认：

- **n 和 x 是不同概念**，在多倍体里必须分开用；甚至课程讲义会直接写“在二倍体中 n=x，但在多倍体中不成立”，并用小麦举例（六倍体 42 条，x=7，n=21）。 [1](https://mcb.berkeley.edu/courses/mcb140/Syllabus/AmacherLecture/Lecture12.pdf)
- “haploid（单倍体）”在不同语境（核相/世代/育种材料）里确实存在历史用法差异，所以才更强调用 **x（monoploid）**去表示“真正的一套”。 [3](https://pmc.ncbi.nlm.nih.gov/articles/PMC4246724/)

## 6）给你一套“做题不混”的固定写法（强烈建议照抄这个套路）

每次看到倍性题，强制自己写三行：

- **写 x**：这个材料的一套染色体是多少（基本数）。
- **写 2n**：体细胞是多少条。
- **算倍性 = (体细胞染色体数) / x**，写成 2x、4x、6x……

然后再说：

- 配子数 **n = ½(2n)**（对偶倍体通常成立；奇倍体会复杂，但高中多不深究）
- “单倍体”按 **n**，“几倍体”按 **x 的倍数**

这样你就不会再被“BB 是不是单倍体”这种表述带跑偏：你会先问一句——**这是在用 n 说核相，还是在用 x 说倍性？**

如果你愿意，我也可以把你教材里常见的几种出题句式（“单倍体经秋水仙素处理”“四倍体的配子”“异源四倍体 BBCC 的单倍体/配子是什么”）各举 2–3 个典型例题，把答案按 n/x 两套坐标系同时写出来，你会很快形成肌肉记忆。

### 第一层：破解教材的“死规定”（高中做题的终极口诀）

高中生物对“几倍体”和“单倍体”的命名，有一个**“一票否决”**的死规定。你只需要记住下面这句口诀：**“先看起点（来源），再看组数”**。

- **如果生物体是由【受精卵】发育来的：**

- 看它细胞内有几个染色体组，就叫**“几倍体”**。
- 有2个组（如AA）叫二倍体；有4个组（如BBBB或BBCC）叫四倍体；有6个组（如普通小麦）叫六倍体。

- **如果生物体是由【配子（精子、卵细胞、花粉）】直接发育来的：**

- **不要管它有几个染色体组，一律强行叫【单倍体】！**
- **举例1：** 二倍体（AA）的配子是 A（1个组）。由它发育成的植株，叫**单倍体**。
- **举例2：** 四倍体（BBBB）的配子是 BB（2个组）。由它发育成的植株，即使它体内有2个染色体组，也**必须叫单倍体**！
- **举例3：** 六倍体小麦（6个组）的花粉有 3个组。由花粉直接培育出的植株，含有3个染色体组，但依然**必须叫单倍体**！

**回到你的疑惑：**
“对单倍体用秋水仙素处理，得到的可能是二倍体，也可能是四倍体，对吗？”
**答：你完全正确！**

- 如果原来的植物是二倍体（AA），它的单倍体（A）用秋水仙素加倍，得到的是**二倍体（AA）**。
- 如果原来的植物是四倍体（BBBB），它的单倍体（BB）用秋水仙素加倍，得到的是**四倍体（BBBB）**。
- **高中答题切记：** “单倍体”里面不一定只有1个染色体组，它可能含有1个、2个、3个甚至更多个染色体组。

### 第二层：为什么会这么混乱？（揭秘学术界的真实情况）

你之所以觉得这不合逻辑——“凭什么有2个组的BB也叫单倍体，这跟二倍体有什么区别？”——是因为**中文教材把英文里的两个词，强行翻译成了一个词。**
在正规的英文遗传学著作中，区分得非常清楚，有两个完全不同的词：

- **Monoploid（一倍体）：**

- 指的是细胞里**绝对只有1个染色体组**（基数 $x$）。
- 无论你怎么来的，只要你只有1个组（比如 A ），你就是 Monoploid。

- **Haploid（单倍体）：**

- 指的是**配子中的染色体数**（通常用 $n$ 表示）。它强调的是**“体细胞的一半”**。
- 如果母本是四倍体（4x），它的配子就是 2x。这个含有2x的配子和由它发育来的个体，在英文里叫 Haploid。

**悲剧来了：**
中文早年引进的时候，把 **Haploid** 翻译成了**“单倍体”**（带个“单”字，听起来就像是只有1个）。而在高中阶段，为了减轻学生负担，几乎不教 **“一倍体 (Monoploid)”** 这个概念。
这就导致了字面意思上的严重冲突：

- 叫“**单**倍体”，里面却可能有**多**个染色体组。
- “二倍体”和“四倍体”是以**数量**命名的，而“单倍体”却是以**发育来源**命名的。它们根本不在一个分类维度上！
