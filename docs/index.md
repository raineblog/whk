# 简介

## 这里是 **RainPPR's WHK Blog**！[![Ask DeepWiki](https://deepwiki.com/badge.svg)](https://deepwiki.com/raineblog/whk)

这个是 WHK 博客，是目前作者比较活跃的百科性质博客。本项目受 [CTF Wiki](https://ctf-wiki.org/)、[OI Wiki](https://oi-wiki.org/) 的启发。在本项目的编写过程中，作者进行了很多创造性（造轮子性）的改造，尤其是对于高中知识体系框架的构建。

在编写过程中参考了诸多资料，主要内容来自于教材、书籍、网课、培训机构（山东青岛地区）等，部分资料来源于一些网站，在此一并致谢！
