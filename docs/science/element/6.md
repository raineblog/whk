# 卤族元素概述

## 氯及其化合物

### 氯单质的性质与应用

氯（Chlorine）是极其活泼的非金属元素，位于元素周期表第三周期第ⅦA族。其单质**氯气（$\text{Cl}_2$）**具有强氧化性，在工业、医疗及日常生活中有着极其广泛的应用。以下是关于氯单质性质及其应用的详细展开讲解：

- **色、态与气味**：在常温常压下，氯气是一种**黄绿色**、具有强烈**刺激性气味**的有毒气体。

- **密度**：氯气的密度比空气大（标准状况下为 $3.214\,\text{g/L}$），约为同体积空气重量的 2.5 倍。在实验室中通常采用向上排空气法收集。
- **溶解性**：氯气可溶于水（25℃时，1 体积水约溶解 2 体积氯气），所得水溶液称为**氯水**。氯气也易溶于四氯化碳等有机溶剂。
- **状态变化**：氯气易液化。在常温下加压或降温至 $-34.6^\circ\text{C}$ 时，可变为黄绿色油状的**液氯**；继续冷却至 $-101^\circ\text{C}$ 则凝固为黄色固体。由于干燥液氯不与铁反应，工业上常用钢瓶盛装液氯储运。

氯原子的最外层有 7 个电子，极易得到 1 个电子达到 8 电子稳定结构，因此氯气表现出极强的活泼性和氧化性。

1. 与金属单质反应：氯气能与绝大多数金属化合，且由于其强氧化性，与变价金属反应时生成**高价态**金属氯化物。
    - **与钠反应**：剧烈燃烧，发出黄色火焰，产生大量**白烟**（$\text{NaCl}$ 微粒）。
    - **与铁反应**：红热的铁丝在氯气中剧烈燃烧，产生**棕褐色烟**（$\text{FeCl}_3$ 微粒）。无论铁是否过量，产物均为 $+3$ 价铁盐。
    - **与铜反应**：灼热的铜丝在氯气中燃烧，产生**棕黄色烟**，产物溶于水后呈绿色或蓝色溶液。

2. 与非金属单质反应：
    - **与氢气反应**：氢气在氯气中安静燃烧，发出**苍白色火焰**，瓶口出现**白雾**（生成的 $\text{HCl}$ 气体吸收空气中的水蒸气形成盐酸小液滴）。若 $\text{H}_2$ 与 $\text{Cl}_2$ 混合后受到强光照射，则会发生**爆炸**。
    - **与磷反应**：磷在氯气中燃烧，少量氯气生成 $\text{PCl}_3$（液体，呈白雾状）；过量氯气生成 $\text{PCl}_5$（固体，呈白烟状）。

3. 与水反应（歧化反应）：氯气溶于水时，部分与水发生可逆反应：**$\text{Cl}_2 + \text{H}_2\text{O} \rightleftharpoons \text{HCl} + \text{HClO}$**。
    - 生成的**次氯酸（$\text{HClO}$）**具有极强的氧化性，是氯水表现漂白和消毒作用的主要成分。
    - $\text{HClO}$ 极其不稳定，见光易分解产生氧气：$2\text{HClO} \xrightarrow{\text{光照}} 2\text{HCl} + \text{O}_2\uparrow$。

4. 与碱反应：氯气易与碱溶液反应，常用于制备漂白性物质或处理尾气。
    - **制漂白液**：常温下通入 $\text{NaOH}$ 溶液，有效成分为 $\text{NaClO}$。
    - **制漂白粉/漂粉精**：将氯气通入冷的石灰乳中，有效成分为 $\text{Ca(ClO)}_2$。
    - **注意**：在加热（如 $70^\circ\text{C}$）或浓碱条件下，氯气会歧化生成高价态的**氯酸盐**（$\text{ClO}_3^-$）。

5. 与还原性物质反应：氯气能氧化 $\text{Fe}^{2+}$、$\text{Br}^-$、$\text{I}^-$ 等离子，并将 $\text{SO}_2$ 氧化为 $\text{H}_2\text{SO}_4$。

氯单质的应用：

- **杀菌消毒**：常用于自来水的杀菌消毒（浓度 0.2-1.0 ppm），次氯酸的强氧化性能杀死多种病原微生物。

- **漂白**：用于棉、麻、纸张的漂白（主要利用 $\text{HClO}$ 或其盐类）。
- **有机合成**：是合成塑料（如聚氯乙烯 $\text{PVC}$）、合成橡胶、人造纤维、农药（如六六六、滴滴涕）、染料、药品（如诺氟沙星）的重要原料。
- **精炼与冶炼**：用于制取高纯度半导体硅（$\text{Si}$）、光纤材料（$\text{SiCl}_4$）以及提取金属钛（$\text{Ti}$）。
- **制备酸类**：与氢气直接化合生产 $\text{HCl}$，进而制取盐酸。

安全与环保建议：

- **中毒防范**：闻氯气需采用“扇闻法”。一旦发生泄漏，应向**高处**并**逆风**方向疏散。

- **急救处理**：可用浸有弱碱性溶液（如肥皂水、纯碱溶液）的毛巾捂住口鼻。
- **混用禁忌**：**严禁将“84 消毒液”与洁厕灵（含盐酸）混用**，否则会产生有毒氯气导致中毒：$\text{ClO}^- + \text{Cl}^- + 2\text{H}^+ = \text{Cl}_2\uparrow + \text{H}_2\text{O}$。

### 氯化氢的性质与应用

氯化氢（$\text{HCl}$）是氯元素的一种重要氢化物，在常温下为气体，其水溶液即为化学中极常用的**盐酸**（$\text{Hydrochloric Acid}$）。以下是对氯化氢物理性质、化学性质及其应用领域的详细展开：

1. **外观与气味**：氯化氢是一种**无色**、具有**强烈刺激性气味**的气体。
2. **溶解性**：氯化氢**极易溶于水**。在 $0^\circ\text{C}$ 时，1体积的水约能溶解 500 体积的氯化氢气体。由于其极高的溶解度，氯化氢常被用于演示**“喷泉实验”**，原理是气体极易溶于水导致容器内压强骤降，从而将下方液体吸入形成喷泉。
3. **状态与密度**：其气态密度比空气大。氯化氢气体易液化，常压下冷却至 $-85^\circ\text{C}$ 即变为液态，降温至 $-114.17^\circ\text{C}$ 则固化为晶体。
4. **在空气中的现象**：氯化氢在空气中会形成**白雾**。这是因为挥发出的 $\text{HCl}$ 气体极易吸收空气中的水蒸气，形成盐酸小液滴。
5. **热效应**：氯化氢气体溶解于水的过程是**显著放热**的。

化学性质：

1. **酸性与电离**：
    - **干燥气体的特殊性**：干燥的氯化氢气体化学性质稳定，**不具有酸性**，不能使干燥的蓝色石蕊试纸变红。
    - **强酸性（盐酸）**：氯化氢溶于水后会完全电离成水合氢离子（$\text{H}_3\text{O}^+$）和氯离子（$\text{Cl}^-$），表现出强酸的通性。它是强电解质，在氢卤酸（$\text{HF, HCl, HBr, HI}$）中，其酸性强于氢氟酸，但弱于氢溴酸和氢碘酸。
2. **与氨气的反应**：氯化氢气体与氨气（$\text{NH}_3$）相遇会立即反应，产生**浓厚的白烟**，该白烟是微小的**氯化铵（$\text{NH}_4\text{Cl}$）**固体颗粒。这是检验 $\text{HCl}$ 或 $\text{NH}_3$ 气体的常用方法。
3. **与金属的反应**：
    - **干燥状态**：干燥的 $\text{HCl}$ 很不活泼，不与锌、镁、铁等活泼金属及大多数金属氧化物反应，但碱金属可在其中燃烧。
    - **溶液状态**：盐酸能与活泼金属（如 $\text{Zn, Al, Fe}$）发生置换反应产生氢气（$\text{H}_2$），并与金属氧化物（如 $\text{CuO, Fe}_2\text{O}_3$）反应生成盐和水。
4. **还原性**：氯化氢具有一定的还原性，可以被强氧化剂（如 $\text{MnO}_2$、$\text{KMnO}_4$）氧化生成**氯气**（$\text{Cl}_2$），这是实验室制取氯气的基本原理。
5. **有机反应**：在有机化学中，氯化氢常用于与烯烃（如乙烯）或炔烃（如乙炔）发生**加成反应**，制备氯代烃（如氯乙烷、氯乙烯）。

氯化氢及盐酸的应用：

1. **工业生产**：
    - 它是合成**染料、香料、药物、塑料**（如聚氯乙烯 $\text{PVC}$）和合成橡胶的重要原料。
    - 用于制备各种金属氯化物，如氯化钡、氯化锌等。
    - 在食品工业中用于制造**葡萄糖**和胶类物质。
2. **金属表面处理**：盐酸广泛用于金属的**“酸洗”**（$\text{Pickle Liquor}$），即除去铁或钢材表面的氧化物膜（锈），以便进行镀锌、镀锡或涂漆。
3. **能源与资源开发**：在石油工业中，盐酸常用于**油井酸化**，以溶解岩石缝隙中的物质，增加油气产量。
4. **生物学作用**：人体胃液中含有约 0.5% 的盐酸，通常称为**“胃酸”**，对消化食物和杀灭进入胃部的细菌具有重要作用。
5. **实验室用途**：作为重要的化学试剂，常用于调节溶液的 $\text{pH}$、中和碱类、制备氢气或作为分析化学中的滴定剂。

制备方法简述：

- **实验室制法**：通过**固体氯化钠（$\text{NaCl}$）与浓硫酸（$\text{H}_2\text{SO}_4$）共热**制得，利用高沸点酸制取低沸点酸的原理。
- **工业制法**：通过**氢气（$\text{H}_2$）与氯气（$\text{Cl}_2$）直接化合**（氢氯合成法），生成的 $\text{HCl}$ 气体随后用水吸收得到浓盐酸。工业盐酸由于含有少量 $\text{FeCl}_3$ 杂质，常呈现**淡黄色**。

### 氯的氧化物与含氧酸

氯元素在含氧化合物中表现出多种氧化态（如 +1, +3, +4, +5, +6, +7），形成了种类丰富的氧化物、含氧酸及其盐类。这些物质在消毒、漂白和工业生产中具有重要地位。

氯的氧化物通常极不稳定，易发生爆炸性分解。

1. **一氧化二氯 ($\text{Cl}_2\text{O}$)**
    - **性质**：棕黄色具有刺激性气味的气体，是次氯酸（$\text{HClO}$）的酸酐。它是一种强氧化剂，极不稳定，在光照或加热下易分解为 $\text{Cl}_2$ 和 $\text{O}_2$。
    - **应用**：主要用于制备高纯度的次氯酸浓溶液。
2. **二氧化氯 ($\text{ClO}_2$)**
    - **性质**：黄绿色或橙黄色有刺激性气味的气体，极易溶于水。它是奇电子化合物，具有顺磁性，中心氯原子采取 $sp^2$ 杂化。$\text{ClO}_2$ 是一种强力氧化剂，见光或受热易分解爆炸，通常需要用空气或 $\text{CO}_2$ 稀释。
    - **应用**：被世界卫生组织（WHO）列为 A1 级高效安全灭菌消毒剂。由于其氧化性强且不会产生致癌的有机氯化物，广泛用于自来水消毒、造纸业漂白及食品保鲜。
3. **七氧化二氯 ($\text{Cl}_2\text{O}_7$)**
    - **性质**：无色油状液体，是高氯酸（$\text{HClO}_4$）的酸酐。它是氯的氧化物中最稳定的一种，但受热（超过 393 K）或受撞击仍会发生剧烈爆炸。

随着氯元素化合价的升高，其含氧酸的**酸性**和**稳定性**逐渐增强，而**氧化性**通常逐渐减弱。

1. +1 氧化态：次氯酸 ($\text{HClO}$) 与次氯酸盐
    - **性质**：次氯酸是极弱的酸（酸性弱于 $\text{H}_2\text{CO}_3$），仅存在于水溶液中。它极不稳定，见光易分解产生 $\text{HCl}$ 和 $\text{O}_2$。
    - **氧化性与漂白性**：$\text{HClO}$ 具有强氧化性，能氧化色素使有色布条褪色，且这种漂白作用是不可逆的。
    - **常见盐类**：
        - **次氯酸钠 ($\text{NaClO}$)**：“84 消毒液”的有效成分，由 $\text{Cl}_2$ 通入 $\text{NaOH}$ 溶液制得。
        - **次氯酸钙 [$\text{Ca(ClO)}_2$]**：漂白粉或漂粉精的有效成分，由 $\text{Cl}_2$ 通入石灰乳制得。
    - **应用与安全**：常用于饮用水消毒和织物漂白。**严禁与酸性洗涤剂（如洁厕灵，含盐酸）混用**，否则会发生归中反应释放剧毒氯气：$\text{ClO}^- + \text{Cl}^- + 2\text{H}^+ = \text{Cl}_2\uparrow + \text{H}_2\text{O}$。

2. +3 氧化态：亚氯酸 ($\text{HClO}_2$) 与亚氯酸盐
    - **性质**：亚氯酸是中强酸，极不稳定，易发生歧化反应。
    - **应用**：**亚氯酸钠 ($\text{NaClO}_2$)** 是一种高级漂白剂，常用于不损伤织物强度的漂白，也可作为制取 $\text{ClO}_2$ 的原料。

3. +5 氧化态：氯酸 ($\text{HClO}_3$) 与氯酸盐
    - **性质**：氯酸是强酸，在酸性介质中是强氧化剂。其浓溶液（超过 40%）极不稳定，易发生爆炸性歧化。
    - **常见盐类**：**氯酸钾 ($\text{KClO}_3$)** 为白色晶体，与易燃物（如硫、磷、碳）混合受撞击易爆炸。
    - **应用**：主要用于制造火药、烟花、火柴及实验室制取氧气。

4. +7 氧化态：高氯酸 ($\text{HClO}_4$) 与高氯酸盐
    - **性质**：高氯酸是无机含氧酸中**酸性最强**的酸。它在氯的含氧酸中热稳定性最好，但在纯质状态下仍不稳定。
    - **氧化性**：在稀溶液中其氧化性较弱，但在受热或浓缩状态下具有强氧化性。
    - **常见盐类**：**高氯酸钾 ($\text{KClO}_4$)** 难溶于水，常用于制备高氯酸及推进剂。

| 性质       | 次氯酸 ($\text{HClO}$) $\rightarrow$ 亚氯酸 ($\text{HClO}_2$) $\rightarrow$ 氯酸 ($\text{HClO}_3$) $\rightarrow$ 高氯酸 ($\text{HClO}_4$) | 来源 |
| :--------- | :---------------------------------------------------------------------------------------------------------------------------------------- | :--- |
| **酸性**   | 逐渐增强（$\text{HClO}$ 为弱酸，$\text{HClO}_4$ 为最强无机酸）                                                                            |      |
| **稳定性** | 逐渐增强（$\text{HClO}$ 见光分解，$\text{HClO}_4$ 相对最稳定）                                                                            |      |
| **氧化性** | 逐渐减弱（通常 $\text{HClO}$ 氧化性最强，$\text{ClO}_4^-$ 在稀溶液中几乎无氧化性）                                                        |      |

**补充**：除高氯酸盐外，其余氯的含氧酸盐在酸性条件下多能与 $\text{Cl}^-$ 发生归中反应生成 $\text{Cl}_2$。

### 氯及其化合物之间的转化关系与规律

氯（Chlorine）是极其活泼的非金属元素，位于元素周期表第三周期第ⅦA族。在中学化学中，氯元素及其化合物的性质常通过**“价—类”二维图**进行系统总结，其核心是以**化合价**为纵轴、**物质类别**为横轴，通过**氧化还原反应**构建起物质间的转化网络。

| 物质类别      | -1价                                        | 0价           | +1价                             | +3价                     | +4价           | +5价                            | +7价                     |
| :------------ | :------------------------------------------ | :------------ | :------------------------------- | :----------------------- | :------------- | :------------------------------ | :----------------------- |
| **氢化物/酸** | $\text{HCl}$ (盐酸)                         | /             | $\text{HClO}$ (次氯酸)           | $\text{HClO}_2$ (亚氯酸) | /              | $\text{HClO}_3$ (氯酸)          | $\text{HClO}_4$ (高氯酸) |
| **单质**      | /                                           | $\text{Cl}_2$ | /                                | /                        | /              | /                               | /                        |
| **氧化物**    | /                                           | /             | $\text{Cl}_2\text{O}$            | /                        | $\text{ClO}_2$ | /                               | $\text{Cl}_2\text{O}_7$  |
| **盐**        | $\text{NaCl}, \text{MgCl}_2, \text{FeCl}_3$ | /             | $\text{NaClO}, \text{Ca(ClO)}_2$ | $\text{NaClO}_2$         | /              | $\text{KClO}_3, \text{NaClO}_3$ | $\text{KClO}_4$          |

氯气（$\text{Cl}_2$）处于中间价态，既可被氧化也可被还原，但由于其极强的电负性，主要表现为**强氧化性**。

- **还原为 -1价**：
    - **与金属反应**：$\text{Cl}_2$ 与绝大多数金属反应生成高价态氯化物，如 $2\text{Fe} + 3\text{Cl}_2 \xrightarrow{\triangle} 2\text{FeCl}_3$。
    - **与氢气反应**：$\text{H}_2 + \text{Cl}_2 \xrightarrow{点燃} 2\text{HCl}$，产生苍白色火焰并伴有白雾。
    - **置换反应**：$\text{Cl}_2$ 能置换出还原性更强的 $\text{Br}^-$ 和 $\text{I}^-$（氧化性：$\text{Cl}_2 > \text{Br}_2 > \text{I}_2$）。
- **歧化反应（价态一升一降）**：
    - **与水反应**：$\text{Cl}_2 + \text{H}_2\text{O} \rightleftharpoons \text{HCl} + \text{HClO}$。
    - **与碱反应**：
        - **常温**：$\text{Cl}_2 + 2\text{NaOH} = \text{NaCl} + \text{NaClO} + \text{H}_2\text{O}$（用于制备漂白液）。
        - **加热**：$3\text{Cl}_2 + 6\text{NaOH} \xrightarrow{\triangle} 5\text{NaCl} + \text{NaClO}_3 + 3\text{H}_2\text{O}$。

**氧化为 0价**：实验室常用强氧化剂氧化浓盐酸制取氯气，如 $\text{MnO}_2 + 4\text{HCl(浓)} \xrightarrow{\triangle} \text{MnCl}_2 + \text{Cl}_2\uparrow + 2\text{H}_2\text{O}$ 或使用 $\text{KMnO}_4$（无需加热）。

**非氧化还原转化**：$\text{HCl}$ 与金属氧化物、碱或盐（如 $\text{CaCO}_3$）反应生成金属氯化物盐。

正价含氧物质间的转化：归中与分解：

- **归中反应（价态靠拢）**：
    - 在酸性条件下，含氧酸根与 $\text{Cl}^-$ 反应生成 $\text{Cl}_2$，如 $\text{ClO}^- + \text{Cl}^- + 2\text{H}^+ = \text{Cl}_2\uparrow + \text{H}_2\text{O}$。
    - $\text{ClO}_3^-$ 与 $\text{Cl}^-$ 的归中：$\text{KClO}_3 + 6\text{HCl} = \text{KCl} + 3\text{Cl}_2\uparrow + 3\text{H}_2\text{O}$。
- **分解反应**：次氯酸极不稳定，见光易分解：$2\text{HClO} \xrightarrow{光照} 2\text{HCl} + \text{O}_2\uparrow$。

氯及其化合物转化的核心规律：

1. **氧化还原规律**：
    - **价态规律**：处于最高价态（+7）的 $\text{Cl}$ 只有氧化性，最低价态（-1）的 $\text{Cl}^-$ 只有还原性，中间价态（0~+5）既有氧化性又有还原性。
    - **邻位转化规律**：在氧化还原反应中，氯元素的化合价变化通常遵循“只靠拢，不交叉”的原则。
    - **歧化与归中**：氯元素丰富的价态使得歧化（如 $\text{Cl}_2$ 在碱中）和归中（如制“84”消毒液失效产生的 $\text{Cl}_2$）成为其最显著的特征反应。

2. **含氧酸的递变规律**：
    - **酸性递增**：$\text{HClO} < \text{HClO}_2 < \text{HClO}_3 < \text{HClO}_4$。高氯酸是无机含氧酸中酸性最强的酸。
    - **稳定性递增**：$\text{HClO} < \text{HClO}_2 < \text{HClO}_3 < \text{HClO}_4$。
    - **氧化性递减**：通常情况下，低价态含氧酸（如 $\text{HClO}$）的氧化性反而比高价态（如 $\text{HClO}_4$）更强。

3. **水溶液中的平衡规律**：
    - **氯水平衡**：新制氯水中存在 $\text{Cl}_2 + \text{H}_2\text{O} \rightleftharpoons \text{H}^+ + \text{Cl}^- + \text{HClO}$ 平衡。加入碱、光照或升温均会使平衡右移，增加 $\text{Cl}^-$ 浓度；而加入 $\text{NaCl}$ 固体（增大 $\text{Cl}^-$ 浓度）会使平衡左移。

实际应用中的转化案例：

- **自来水消毒**：利用 $\text{Cl}_2$ 歧化产生的 $\text{HClO}$ 的强氧化性杀菌。
- **漂白粉的失效**：$\text{Ca(ClO)}_2$ 与空气中 $\text{CO}_2$ 和水反应生成 $\text{HClO}$，后者光解导致失效：$\text{Ca(ClO)}_2 + \text{CO}_2 + \text{H}_2\text{O} = \text{CaCO}_3\downarrow + 2\text{HClO}$。
- **二氧化氯消毒**：$\text{ClO}_2$（+4价）作为新型绿色消毒剂，因其极强的氧化能力且不产生有机氯化物而备受推崇。

### 氯气的实验室与工业制取

氯气（$\text{Cl}_2$）作为重要的化工原料，其制取方法在化学实验与工业生产中具有重要地位。以下是对氯气实验室制法与工业制取的详细展开讲解：

在实验室中，氯气主要通过**强氧化剂氧化浓盐酸**的方法制取。

- **经典方法（舍勒法）**：利用二氧化锰（$\text{MnO}_2$）与浓盐酸共热。
    - **化学方程式**：$\text{MnO}_2 + 4\text{HCl(浓)} \xlongequal{\triangle} \text{MnCl}_2 + \text{Cl}_2\uparrow + 2\text{H}_2\text{O}$。
    - **注意**：此反应必须使用**浓盐酸**且需要**加热**，稀盐酸不与二氧化锰反应。
- **其他强氧化剂法**：可以使用高锰酸钾（$\text{KMnO}_4$）、重铬酸钾（$\text{K}_2\text{Cr}_2\text{O}_7$）或氯酸钾（$\text{KClO}_3$）代替二氧化锰。
    - **高锰酸钾法**：$2\text{KMnO}_4 + 16\text{HCl(浓)} = 2\text{KCl} + 2\text{MnCl}_2 + 5\text{Cl}_2\uparrow + 8\text{H}_2\text{O}$。此反应氧化性极强，**在常温下即可迅速反应**，无需加热。

实验室制取氯气的装置通常遵循：**发生装置 $\rightarrow$ 净化装置 $\rightarrow$ 收集装置 $\rightarrow$ 尾气吸收装置**。

- **发生装置**：采用“固体+液体加热型”装置（若用 $\text{KMnO}_4$ 则无需酒精灯）。通常使用圆底烧瓶和分液漏斗，分液漏斗可控制浓盐酸的滴加速度。
- **净化处理**：
    1. **除杂（除 $\text{HCl}$）**：通过盛有**饱和食盐水**的洗气瓶。原因是 $\text{HCl}$ 极易溶于水，而饱和食盐水中的 $\text{Cl}^-$ 可抑制 $\text{Cl}_2$ 的溶解。
    2. **干燥（除水蒸气）**：通过盛有**浓硫酸**的洗气瓶。净化顺序必须是**先除杂后干燥**。
- **收集方法**：
    - **向上排空气法**：氯气密度比空气大（标准状况下为 $3.214\,\text{g/L}$）。
    - **排饱和食盐水法**：可减少氯气损耗且更纯净。
- **尾气吸收**：氯气有毒，不能直接排入空气。必须使用**强碱溶液（如 $\text{NaOH}$ 溶液）**吸收。
    - 反应原理：$\text{Cl}_2 + 2\text{NaOH} = \text{NaCl} + \text{NaClO} + \text{H}_2\text{O}$。
    - **注意**：不能使用澄清石灰水，因为其 $\text{Ca(OH)}_2$ 浓度过低，吸收不完全。

验满方法：

- **观察法**：集气瓶内充满黄绿色气体。
- **试纸法**：将**湿润的淀粉-碘化钾试纸**靠近瓶口，若试纸**变蓝**则已满（$\text{Cl}_2 + 2\text{KI} = 2\text{KCl} + \text{I}_2$）。
- **石蕊试纸法**：将湿润的蓝色石蕊试纸靠近瓶口，若试纸**先变红后褪色**则已满。

现代工业生产氯气主要依靠电解法，特别是**氯碱工业**。

1. 氯碱工业（电解饱和食盐水）：这是目前生产氯气最主要的方法，同时联产烧碱（$\text{NaOH}$）和氢气（$\text{H}_2$）。
    - **总反应方程式**：$2\text{NaCl} + 2\text{H}_2\text{O} \xlongequal{\text{电解}} 2\text{NaOH} + \text{H}_2\uparrow + \text{Cl}_2\uparrow$。
    - **电极反应**：
        - **阳极（氧化反应）**：$2\text{Cl}^- - 2\text{e}^- = \text{Cl}_2\uparrow$。
        - **阴极（还原反应）**：$2\text{H}_2\text{O} + 2\text{e}^- = \text{H}_2\uparrow + 2\text{OH}^-$。
    - **核心技术：离子交换膜法**。
        - **作用**：使用阳离子交换膜，只允许 $\text{Na}^+$ 通过，**阻止 $\text{Cl}_2$ 与 $\text{NaOH}$ 反应**生成 $\text{NaClO}$，并防止 $\text{H}_2$ 与 $\text{Cl}_2$ 混合发生爆炸。

2. 电冶金副产（电解熔融氯化钠）：在工业上通过电解熔融氯化钠生产金属钠时，氯气作为副产品产出。
    - **反应方程式**：$2\text{NaCl(熔融)} \xlongequal{\text{电解}} 2\text{Na} + \text{Cl}_2\uparrow$。
    - **工艺细节**：由于 $\text{NaCl}$ 熔点高（$801^\circ\text{C}$），常加入 $\text{CaCl}_2$ 作为助熔剂，使熔点降至约 $600^\circ\text{C}$ 以降低能耗。

    电解熔融氯化镁：在工业提炼金属镁的过程中也会产生氯气。**反应方程式**：$\text{MgCl}_2\text{(熔融)} \xlongequal{\text{电解}} \text{Mg} + \text{Cl}_2\uparrow$。

实验安全与防护建议：

- **防毒**：氯气有毒，闻气味时应采用“扇闻法”，即用手轻轻在瓶口扇动，使极少量气体飘进鼻孔。
- **应急处理**：若发生氯气泄漏，应逆风并向高处转移，可用浸有肥皂水或纯碱溶液的毛巾捂住口鼻。
- **混用禁忌**：家庭中严禁将“84 消毒液”与洁厕灵（含盐酸）混用，否则会产生大量有毒氯气：$\text{ClO}^- + \text{Cl}^- + 2\text{H}^+ = \text{Cl}_2\uparrow + \text{H}_2\text{O}$。

### 实验：与氯相关的经典实验

与氯相关的化学实验不仅是中学化学的核心，也体现了人类认识物质、制备物质及利用物质的科学逻辑。以下根据提供的来源，详细展开讲解与氯相关的经典实验。

氯气的实验室制取实验是研究氯及其化合物性质的基础实验。

- **实验操作**：
    1. **发生装置**：在圆底烧瓶中加入二氧化锰（$\text{MnO}_2$）粉末，通过分液漏斗慢慢注入浓盐酸。
    2. **反应条件**：必须使用浓盐酸且需要加热。
    3. **净化处理**：产生的气体依次通过盛有**饱和食盐水**（除去 $\text{HCl}$，利用 $\text{Cl}^-$ 抑制氯气溶解）和**浓硫酸**（除去水蒸气）的洗气瓶。
    4. **收集与验满**：利用向上排空气法收集（氯气密度比空气大）。将湿润的**淀粉-KI试纸**靠近瓶口，观察变色情况。
    5. **尾气处理**：多余的氯气通入盛有 **$\text{NaOH}$ 溶液**的烧杯中。
- **实验现象**：烧瓶中产生黄绿色、有刺激性气味的气体。淀粉-KI试纸立即**变蓝**（反应原理：$\text{Cl}_2 + 2\text{KI} = 2\text{KCl} + \text{I}_2$）。
- **实验意义**：验证了强氧化剂氧化浓盐酸产生单质氯的原理，并掌握了气体制备中“发生-净化-收集-尾气处理”的完整流程。

氯气与金属及非金属的反应实验。通过燃烧实验验证氯气的强氧化性。

- **实验操作**：
    1. **钠与氯气**：将切好的金属钠在石棉网上微热至熔成球状，立即倒扣盛满氯气的集气瓶。
    2. **铁与氯气**：将擦亮的细铁丝绕成螺旋状，系上火柴梗点燃，待即将燃尽时迅速伸入盛有氯气的集气瓶。
    3. **氢气与氯气**：在空气中点燃纯净的 $\text{H}_2$，缓慢伸入盛满 $\text{Cl}_2$ 的集气瓶中。
- **实验现象**：
    1. **钠**：剧烈燃烧，产生黄色火焰和大量**白烟**（$\text{NaCl}$ 微粒）。
    2. **铁**：剧烈燃烧，产生大量**棕褐色烟**（$\text{FeCl}_3$ 微粒），加水后溶液呈黄色。
    3. **氢气**：$\text{H}_2$ 在 $\text{Cl}_2$ 中安静燃烧，发出**苍白色火焰**，瓶口出现**白雾**。
- **实验意义**：证明了氯气具有极强的氧化性，能将变价金属（如 Fe）氧化至高价态。同时说明**燃烧不一定要有氧气参加**，任何剧烈的发光发热的氧化还原反应均可称为燃烧。

氯气（$\text{Cl}_2$）是一种化学性质非常活泼的非金属单质，具有**强氧化性**。在一定条件下，它能与大多数金属和非金属单质发生剧烈的化合反应，这些反应通常伴随着发光、发热的现象，属于不一定要有氧气参加的燃烧反应。

氯气与变价金属反应时，通常会将金属氧化至**最高价态**。

1. 钠（$\text{Na}$）在氯气中燃烧：
    - **操作**：取一块绿豆粒大小的金属钠，吸干表面煤油并切去表层，放在石棉网（或陶土网）上微热。待钠熔成球状时，将盛有氯气的集气瓶迅速倒扣在钠球上方。
    - **现象**：钠在氯气中剧烈燃烧，发出**明亮的黄色火焰**，并产生大量的**白烟**（$\text{NaCl}$ 微粒）。
    - **方程式**：$2\text{Na} + \text{Cl}_2 \xlongequal{\triangle \text{或点燃}} 2\text{NaCl}$。

2. 铁（$\text{Fe}$）在氯气中燃烧：
    - **操作**：将擦亮的细铁丝绕成螺旋状，末端系一段火柴梗。点燃火柴，待其即将燃尽时，迅速将铁丝伸入盛有氯气的集气瓶中。
    - **现象**：铁丝在氯气中剧烈燃烧，**火星四射**，产生大量的**棕褐色烟**（$\text{FeCl}_3$ 微粒）。反应结束后向瓶中加入少量水，振荡，可得到**黄色溶液**。
    - **规律**：无论铁是否过量，产物均为 $+3$ 价的 $\text{FeCl}_3$，而非 $\text{FeCl}_2$。
    - **方程式**：$2\text{Fe} + 3\text{Cl}_2 \xlongequal{\text{点燃}} 2\text{FeCl}_3$。

3. 铜（$\text{Cu}$）在氯气中燃烧：
    - **操作**：用坩埚钳夹住一束铜丝，加热至红热后，立即放入盛有氯气的集气瓶中。
    - **现象**：红热的铜丝在氯气中剧烈燃烧，产生大量的**棕黄色烟**（$\text{CuCl}_2$ 微粒）。反应后向瓶中加少量水，溶液通常呈**绿色或蓝色**。
    - 剧烈燃烧，产生**棕黄色**烟，加水后溶液逐渐由黄色变为**黄绿色、蓝绿色、蓝色**。
    - **方程式**：$\text{Cu} + \text{Cl}_2 \xlongequal{\text{点燃}} \text{CuCl}_2$。

4. 镁（$\text{Mg}$）与锑（$\text{Sb}$）的反应：
    - **镁**：镁条在氯气中点燃可剧烈燃烧，生成白色固体 $\text{MgCl}_2$。
    - **锑**：将锑粉撒入盛有氯气的集气瓶中，锑粉会立即燃烧，产生白色的 $\text{SbCl}_3$ 和 $\text{SbCl}_5$ 混合物烟雾。

非金属单质在氯气中的燃烧：

1. 氢气（$\text{H}_2$）在氯气中燃烧：
    - **操作**：在空气中点燃纯净的氢气（点燃前必须验纯），将导管缓慢伸入盛满氯气的集气瓶中。
    - **现象**：氢气在氯气中**安静地燃烧**，发出**苍白色火焰**。在集气瓶口有**白雾**产生。
    - **注意**：若将氢气与氯气按一定比例混合后光照，则会发生**爆炸**。
    - **原理**：生成的 $\text{HCl}$ 气体极易吸收空气中的水蒸气形成盐酸小液滴（酸雾）。
    - **方程式**：$\text{H}_2 + \text{Cl}_2 \xlongequal{\text{点燃或光照}} 2\text{HCl}$。

2. 磷（$\text{P}$）在氯气中燃烧：
    - **操作**：将少量红磷放在燃烧匙里，在酒精灯上加热至开始燃烧后，迅速伸入盛有氯气的集气瓶中。
    - **现象**：磷在氯气中继续燃烧，产生大量的**白色烟雾**。
    - **产物差异**：
        - **氯气不足**：生成液态的 $\text{PCl}_3$（表现为**白雾**）。
        - **氯气过量**：生成固态的 $\text{PCl}_5$（表现为**白烟**）。
    - **方程式**：$2\text{P} + 3\text{Cl}_2 \xlongequal{\text{点燃}} 2\text{PCl}_3$；$2\text{P} + 5\text{Cl}_2 \xlongequal{\text{点燃}} 2\text{PCl}_5$。

现象总结与实验提示：

1. **“烟”与“雾”的区别**：
    - **烟**：反应生成的固体微粒分散在空气中（如 $\text{NaCl}$、$\text{FeCl}_3$、$\text{CuCl}_2$ 的烟）。
    - **雾**：反应生成的液体小液滴分散在空气中（如 $\text{HCl}$ 遇水形成的盐酸雾）。
2. **燃烧的本质**：通过这些实验可以认识到，燃烧不一定要有氧气参加，其本质是剧烈的、发光发热的氧化还原反应。
3. **安全防护**：氯气有剧毒，以上实验必须在**通风橱**中进行。闻氯气气味时应采用“扇闻法”。

氯气与水反应及漂白性探究实验，是区分“氯气”与“次氯酸”性质的关键实验。

- **实验操作**：将干燥的红色纸条（或布条）和湿润的红色纸条分别放入两瓶盛有干燥氯气的集气瓶中，盖上玻璃片观察。
- **实验现象**：**干燥的红纸条不褪色，湿润的红纸条褪色**。
- **实验意义**：证明了 $\text{Cl}_2$ 本身不具有漂白性，其漂白作用是由 $\text{Cl}_2$ 与水反应生成的**次氯酸（$\text{HClO}$）**产生的。反应方程式为：$\text{Cl}_2 + \text{H}_2\text{O} \rightleftharpoons \text{HCl} + \text{HClO}$。

氯离子（$\text{Cl}^-$）的检验实验：

- **实验操作**：取少量待测液，先加入过量**稀硝酸**酸化，再滴加 **$\text{AgNO}_3$ 溶液**。
- **实验现象**：产生不溶于稀硝酸的**白色沉淀**。
- **实验意义**：确立了检验 $\text{Cl}^-$ 的标准流程。加入稀硝酸是为了排除 $\text{CO}_3^{2-}$、$\text{PO}_4^{3-}$ 等杂质离子的干扰。

氯化氢（$\text{HCl}$）喷泉实验：

- **实验操作**：在干燥的圆底烧瓶中充满 $\text{HCl}$，塞上带有挤压过水的小滴管和玻璃管的橡胶塞，倒置于盛有紫色石蕊溶液（或滴有酚酞的碱液）的水槽中，挤压滴管。
- **实验现象**：烧瓶内形成美丽的**红色喷泉**。
- **实验意义**：形象地展示了 $\text{HCl}$ 极易溶于水（$0^\circ\text{C}$ 时 1:500）的物理性质。

电解饱和食盐水模拟实验（氯碱工业基础）：

- **实验操作**：在 U 形管中注入饱和食盐水，滴入几滴酚酞，插入石墨棒和铁棒电极并通直流电。
- **实验现象**：
    1. **阳极（石墨棒）**：产生黄绿色气体，能使湿润的淀粉-KI试纸变蓝。
    2. **阴极（铁棒）**：产生无色气体，在针尖处点燃有爆鸣声；**阴极附近溶液变红**。
- **实验意义**：模拟了工业制氯气的核心过程。证明了电解饱和食盐水生成 $\text{Cl}_2$、$\text{H}_2$ 和 $\text{NaOH}$ 的原理。

卤素单质间的置换实验：

- **实验操作**：分别向盛有 $\text{NaBr}$ 溶液和 $\text{KI}$ 溶液的试管中加入新制氯水，振荡后加入 $\text{CCl}_4$ 萃取。
- **实验现象**：
    1. 通入 $\text{Cl}_2$ 到 $\text{NaBr}$ 溶液中，加入 $\text{CCl}_4$ 后下层呈**橙红色**。
    2. 通入 $\text{Cl}_2$ 到 $\text{KI}$ 溶液中，加入 $\text{CCl}_4$ 后下层呈**紫红色**。
- **实验意义**：验证了卤素单质氧化性的递变规律：$\text{Cl}_2 > \text{Br}_2 > \text{I}_2$。

特殊安全实验：消毒剂混用危害验证：

- **实验描述（原理分析）**：在日常生活中，**禁止将“84消毒液”（含 $\text{NaClO}$）与洁厕灵（含 $\text{HCl}$）混用**。
- **反应现象（推测）**：混合后会剧烈反应，迅速产生大量黄绿色的剧毒氯气。
- **实验原理**：$\text{ClO}^- + \text{Cl}^- + 2\text{H}^+ = \text{Cl}_2\uparrow + \text{H}_2\text{O}$（归中反应）。
- **意义**：强调了化学知识在居家安全中的应用，防止生命财产损失。

### 应用：与氯相关的实验流程

与氯相关的实验是中学化学的核心内容，涵盖了气体的制备、性质验证、离子检验及定量分析等多个维度。以下是根据提供的来源对与氯相关的实验流程及实验类问题的详细总结。

氯气（$\text{Cl}_2$）的制备流程：

- **实验室制法**：利用强氧化剂氧化浓盐酸。
    - **经典原理**：$\text{MnO}_2 + 4\text{HCl(浓)} \xrightarrow{\triangle} \text{MnCl}_2 + \text{Cl}_2\uparrow + 2\text{H}_2\text{O}$。
    - **改进原理（无需加热）**：可用 $\text{KMnO}_4$、$\text{K}_2\text{Cr}_2\text{O}_7$ 或 $\text{KClO}_3$ 代替 $\text{MnO}_2$。例如使用高锰酸钾：$2\text{KMnO}_4 + 16\text{HCl(浓)} = 2\text{KCl} + 2\text{MnCl}_2 + 5\text{Cl}_2\uparrow + 8\text{H}_2\text{O}$。
- **实验装置序列**：
    1. **发生装置**：固液加热型（或固液不加热型）。
    2. **净化装置**：依次通过盛有**饱和食盐水**（除去 $\text{HCl}$，利用 $Cl^-$ 抑制氯气溶解）和**浓硫酸**（除去水蒸气）的洗气瓶。
    3. **收集装置**：**向上排空气法**（由于 $\text{Cl}_2$ 密度比空气大）或**排饱和食盐水法**。
    4. **尾气处理**：用 **$\text{NaOH}$ 溶液**吸收。**注意**：不能用澄清石灰水，因为 $\text{Ca(OH)}_2$ 溶解度小，吸收不完全。
- **氯碱工业**：电解饱和食盐水，产物为 $\text{NaOH}$、$\text{H}_2$ 和 $\text{Cl}_2$。
- **电解熔融氯化钠**：可同时获得金属钠和氯气。氯及其相关微粒的检验与验证：

氯气（$\text{Cl}_2$）的检验：

- **观察法**：黄绿色气体、有刺激性气味。
- **试纸法**：将**湿润的淀粉-碘化钾试纸**靠近瓶口，试纸变蓝（$\text{Cl}_2 + 2\text{KI} = 2\text{KCl} + \text{I}_2$）。
- **石蕊法**：将湿润的蓝色石蕊试纸靠近瓶口，现象为**先变红后褪色**。

氯离子（$\text{Cl}^-$）的检验：

- **操作规范**：取样，先滴加**稀硝酸**酸化（排除 $\text{CO}_3^{2-}$、$\text{OH}^-$、$\text{PO}_4^{3-}$ 等离子的干扰），再滴加 **$\text{AgNO}_3$ 溶液**。
- **现象**：产生不溶于稀硝酸的**白色沉淀**（$\text{AgCl}$）。
- **易错点**：不能用稀硫酸酸化，因为生成的 $\text{Ag}_2\text{SO}_4$ 是微溶物，会造成干扰。

氯水成分的验证：

- **成分**：新制氯水含“三分四离”。
- **漂白性验证**：干燥氯气不能使干燥红布条褪色，说明 $\text{Cl}_2$ 无漂白性；氯水能使布条褪色，证明 **$\text{HClO}$ 具有漂白性**。
- **不稳定性验证**：氯水光照后 pH 减小，氯离子浓度增大，并产生气泡（$\text{O}_2$）。

实验安全与意外处理：

- **泄漏自救**：$\text{Cl}_2$ 密度大于空气，应**逆风向高处**转移；用浸有肥皂水或纯碱溶液（弱碱性）的毛巾捂住口鼻。
- **混合禁忌**：**严禁将“84 消毒液”与洁厕灵混用**。原理：$\text{ClO}^- + \text{Cl}^- + 2\text{H}^+ = \text{Cl}_2\uparrow + \text{H}_2\text{O}$，会产生大量有毒氯气。
- **加热注意事项**：实验室制取时加热温度不宜过高，以减少浓盐酸的挥发。

装置创新与改进：

- **防倒吸**：对于极易溶于水的气体（如 $\text{HCl}$），需使用倒置漏斗或干燥管等防倒吸装置。
- **除杂效率**：洗气时应遵循“**长进短出**”原则。
- **特殊产物制备**：制备易水解、易升华的物质（如 $\text{FeCl}_3$、$\text{AlCl}_3$）时，需保证氯气干燥，并在装置末端加装干燥管，防止空气中的水蒸气进入。

定量测定类问题：

- **有效氯测定**：通常利用碘量法。关系式常为：$\text{HClO} \sim \text{I}_2 \sim 2\text{S}_2\text{O}_3^{2-}$，通过标准硫代硫酸钠（$\text{Na}_2\text{S}_2\text{O}_3$）滴定来计算。
- **纯度分析**：例如测定 $\text{Cl}_2\text{O}$ 纯度时，需考虑 $\text{Cl}_2$ 的残留情况，利用不同温度下的沸点差异通过抽气或通入氮气法除去多余氯气。

性质探究中的变量控制：

- **置换反应**：通过氯水分别与 $\text{NaBr}$、$\text{KI}$ 反应并配合 $\text{CCl}_4$ 萃取，验证氧化性强弱：$\text{Cl}_2 > \text{Br}_2 > \text{I}_2$。
- **水解平衡**：探究 $AlCl_3$ 溶液中通入 $\text{HCl}$ 析出晶体，是因为 $Cl^-$ 浓度增大使溶解平衡逆向移动。

### 应用：与氯相关的工艺流程

与氯相关的工艺流程是现代化学工业的重要支柱，涵盖了从基础原料提取到高分子材料合成的完整产业链。以下是基于提供的来源对相关工艺流程的详细总结：

海水是氯元素的主要来源，其主要以 $\text{NaCl}$、$\text{MgCl}_2$ 等氯化物的形式存在。

1. **海水晒盐**：利用盐田法，通过日光和风力使水分蒸发，使食盐从饱和溶液中结晶析出得到**粗盐**。
2. **粗盐精制**：粗盐中含有 $\text{Ca}^{2+}$、$\text{Mg}^{2+}$、$\text{SO}_4^{2-}$ 等杂质，在进入后续电解工序前必须除去，以防止堵塞电解槽隔膜。
    - **工艺顺序**：通常依次加入过量的 $\text{BaCl}_2$（除 $\text{SO}_4^{2-}$）、$\text{NaOH}$（除 $\text{Mg}^{2+}$）、$\text{Na}_2\text{CO}_3$（除 $\text{Ca}^{2+}$ 及过量的 $\text{Ba}^{2+}$），过滤后加入适量**盐酸**调节 pH 至中性。

氯碱工业是生产氯气、烧碱和氢气的基础工艺，目前最先进的是**离子交换膜法**。

1. **反应原理**：电解饱和食盐水。阳极（石墨或钛网）发生氧化反应生成 $\text{Cl}_2$；阴极（铁或碳钢网）发生还原反应生成 $\text{H}_2$ 和 $\text{NaOH}$。
    - **方程式**：$2\text{NaCl} + 2\text{H}_2\text{O} \xlongequal{\text{电解}} 2\text{NaOH} + \text{H}_2\uparrow + \text{Cl}_2\uparrow$。
2. **关键技术**：**阳离子交换膜**只允许 $\text{Na}^+$ 通过，阻止 $\text{Cl}^-$ 进入阴极区及 $\text{OH}^-$ 进入阳极区，从而避免 $\text{Cl}_2$ 与 $\text{NaOH}$ 反应生成 $\text{NaClO}$，并防止 $\text{H}_2$ 与 $\text{Cl}_2$ 混合爆炸。

盐酸的工业制取（氢氯合成法）：

- **工艺流程**：将氯碱工业产生的 $\text{H}_2$ 和 $\text{Cl}_2$ 在**石墨合成炉**中点燃直接化合生成 $\text{HCl}$ 气体，随后通过石墨薄膜吸收器进行**多级逆流吸收**得到浓盐酸。
- **特点**：合成盐酸纯度高，生产成本低。

含氯消毒剂与漂白剂的制备：

- **漂白液（84消毒液）**：在常温下将 $\text{Cl}_2$ 通入 $\text{NaOH}$ 溶液中，有效成分为 $\text{NaClO}$。
- **漂白粉/漂粉精**：将 $\text{Cl}_2$ 通入冷的**石灰乳** $[\text{Ca(OH)}_2]$ 中。若使 $\text{Ca(ClO)}_2$ 成为主要成分则得到漂粉精。
    - **反应**：$2\text{Cl}_2 + 2\text{Ca(OH)}_2 = \text{Ca(ClO)}_2 + \text{CaCl}_2 + 2\text{H}_2\text{O}$。

联合制碱法（侯氏制碱法）中的氯循环：

- **工艺核心**：将氨碱法与合成氨工业联合。在滤出 $\text{NaHCO}_3$ 后的母液中通入 $\text{NH}_3$ 并加入食盐粉末，利用**冷析**和**盐析**原理使副产品**氯化铵** ($\text{NH}_4\text{Cl}$) 结晶析出。
- **意义**：实现了食盐中氯元素的综合利用，避免了索尔维法产生大量 $\text{CaCl}_2$ 废渣的问题。

海水提溴与提碘：

- **空气吹出法提溴**：利用 $\text{Cl}_2$ 的强氧化性，将苦卤中的 $\text{Br}^-$ 氧化为 $\text{Br}_2$。
    - **方程式**：$\text{Cl}_2 + 2\text{Br}^- = 2\text{Cl}^- + \text{Br}_2$。
- **海带提碘**：在浸取液中加入适量 $\text{Cl}_2$（或 $\text{H}_2\text{O}_2$）将 $\text{I}^-$ 氧化为 $\text{I}_2$，随后进行萃取。**注意**：$\text{Cl}_2$ 过量可能将 $\text{I}_2$ 进一步氧化为 $\text{IO}_3^-$。

电冶金（活泼金属冶炼）：

- **金属钠**：电解**熔融 $\text{NaCl}$**，加入 $\text{CaCl}_2$ 作为助熔剂以降低熔点。
- **金属镁**：由海水提镁得到 $\text{Mg(OH)}_2$，经盐酸酸化、浓缩、在 $\text{HCl}$ 气流中脱水得到无水 $\text{MgCl}_2$，最后**电解熔融 $\text{MgCl}_2$** 制得。

氯相关的有机化工产业链：

- **聚氯乙烯 (PVC) 生产**：
    1. **乙烯法**：乙烯与 $\text{Cl}_2$ 加成生成 1,2-二氯乙烷，再经裂解脱去 $\text{HCl}$ 得到氯乙烯单体，最后加聚成 PVC。
    2. **乙炔法**：乙炔与 $\text{HCl}$ 在催化剂作用下直接加成生成氯乙烯。
- **氧氯化工艺**：利用乙烯、$\text{HCl}$ 和 $\text{O}_2$ 反应制备 1,2-二氯乙烷，可利用副产物 $\text{HCl}$，提高氯元素的利用率。

工艺流程中的环保与安全总结：

1. **尾气处理**：所有涉及 $\text{Cl}_2$ 或 $\text{HCl}$ 的流程尾气必须经过吸收。$\text{Cl}_2$ 常用 **$\text{NaOH}$ 溶液**吸收；极易溶的 $\text{HCl}$ 需使用**防倒吸装置**（如倒置漏斗）用水吸收。
2. **防腐与防潮**：制备易水解产物（如 $\text{FeCl}_3, \text{AlCl}_3, \text{TiCl}_4$）时，原料氯气必须通过浓硫酸**干燥**，且装置末端需连接干燥管防止外界水蒸气进入。
3. **循环经济**：现代冶金工艺（如电解 $\text{CoCl}_2$）产生的副产物 $\text{Cl}_2$ 常返回前端浸出工段，形成**闭路循环**以降低成本并减少污染。

### 应用：与氯相关的生产生活

氯及其化合物在生产（非化工原料领域）与日常生活中发挥着举足轻重的作用，主要利用其**强氧化性**进行杀菌、消毒与漂白。以下是以漂白粉、漂白液为核心的氯及其化合物应用的详细展开：

**漂白液**的主要有效成分是**次氯酸钠（$NaClO$）**，它是通过将氯气通入氢氧化钠溶液中制得的。

1. **家居与环境消毒**：在日常生活中，漂白液常用于家庭、宾馆及医院的地面、课桌椅、马桶及水槽的擦拭与喷洒消毒。它能杀灭肠道致病菌、化脓性球菌和致病性酵母菌，并能灭活多种病毒。
2. **织物漂白与洗涤**：它常用于棉、麻、纸张的漂白，但需要注意它具有强氧化性，易使有色衣物脱色，且**禁用于丝、毛、织物**的洗涤。
3. **水处理应用**：在部分公共场所，如游泳池的日常水质维持中，漂白液也被用作低成本的消毒剂。
4. **反应原理**：其起效原理是次氯酸钠与空气中的二氧化碳和水反应生成具有强氧化性的**次氯酸（$HClO$）**：$NaClO + H_2O + CO_2 = NaHCO_3 + HClO$。

**漂白粉**是氯气与石灰乳反应生成的混合物，有效成分为**次氯酸钙［$Ca(ClO)_2$］**；若反应充分使次氯酸钙含量大幅提高，则称为**漂粉精**。

1. **大规模环境消毒**：漂白粉广泛用于污水、排泄物及疫区的环境消毒。在野外作业或灾后防疫中，漂白粉因其固体形态比液态漂白液更易于运输和储存而得到广泛应用。
2. **饮用水净化**：在缺乏精密自来水处理设施的地区，漂白粉常被用于饮用水的杀菌消毒。
3. **农业与国防**：在农业上，漂白粉可用于防治农作物病害；在国防领域，它也是中和某些毒气毒害作用的有效药剂。
4. **失效原理与保存**：由于漂白粉中的次氯酸钙会与空气中的二氧化碳和水蒸气反应生成不稳定的次氯酸，且次氯酸见光易分解，因此漂白粉必须**避光、密封并置于阴凉处保存**。

漂白粉的成分：

- 主要成分：$\ce{CaCl2,Ca(ClO)2}$。

- 有效成分：$\ce{Ca(ClO)2}$。

制取方法：

- 将氯气通入**石灰乳**中。

- $\ce{2Ca(OH)2 + 2Cl2 -> CaCl2 + Ca(ClO)2 + 2H2O}$。

漂白原理：

- 生效原理：
    - $\ce{Ca(ClO)2 + H2O + CO2 -> 2HClO + CaCO3 v}$。

- 失效原理：
    - $\ce{Ca(ClO)2 + H2O + CO2 -> 2HClO + CaCO3 v}$。

    - $\ce{2HClO ->[光] 2HCl + O2 ^}$。

漂白液的分：

- 主要成分：$\ce{NaCl,NaClO}$。

- 有效成分：$\ce{NaClO}$。

制取方法：

- 将氯气通入氢氧化钠溶液中。

- $\ce{2NaOH + Cl2 -> NaCl + NaClO + H2O}$。

漂白原理：

- 生效原理：
    - $\ce{NaClO + H2O + CO2 -> HClO + NaHCO3}$。

- 失效原理：
    - $\ce{NaClO + H2O + CO2 -> HClO + NaHCO3}$。

    - $\ce{2HClO ->[光] 2HCl + O2 ^}$。

除了加工好的消毒剂，氯的氧化物和单质在生产中也有直接应用。

1. **自来水杀菌**：氯气（$Cl_2$）是传统大城市自来水厂最常用的消毒剂。它溶于水生成的次氯酸能有效杀死伤寒菌、赤痢菌等微生物。
2. **绿色消毒剂——二氧化氯（$ClO_2$）**：二氧化氯是国际公认的高效安全灭菌剂。相比氯气，它不会与水中的微量有机物反应生成致癌的有机氯化物（如三卤甲烷），因此正逐渐取代氯气用于现代自来水消毒。
3. **农业熏杀**：氯气可直接用于熏杀仓库里的害虫，或将氯气喷射入田鼠洞穴以杀灭有害动物。

其他氯化合物的特殊生活应用：

1. **医疗急救（氯乙烷）**：**氯乙烷**常用作运动员的**局部麻醉剂**。因其沸点极低，喷射在皮肤上迅速汽化吸热，使局部神经末梢因冷却而处于麻醉状态，可紧急处理肌肉疼痛。
2. **干燥与防冻（氯化钙）**：**无水氯化钙**在日常生活中常用作**食品干燥剂**（如装在小纸袋中放入饼干盒内）。同时，因其溶解放热且能降低水的冰点，常用于道路除冰和制备建筑用的水泥防冻液。
3. **建筑材料（PVC）**：**聚氯乙烯（PVC）**是生活中最常见的塑料材质之一，用于制造排水管、电线绝缘外壳、地板砖及人造革等。

在日常生活中，**绝对禁止将漂白液或漂白粉与酸性洗涤剂（如洁厕灵，含盐酸）混合使用**。两者混合后会发生剧烈的氧化还原反应，迅速释放出**剧毒的氯气**，严重时会导致人员中毒死亡。其离子方程式为：$ClO^- + Cl^- + 2H^+ = Cl_2\uparrow + H_2O$。

氯气是一种**黄绿色、有强烈刺激性气味且剧毒**的气体。吸入少量会刺激鼻腔和喉头黏膜，引起胸痛和咳嗽；吸入大量则会导致中毒甚至窒息死亡。

- **逃生方向**：由于氯气的**密度比空气大**，泄露后会向低洼处沉积，因此逃生时应**逆风向高坡处（高处）转移**。
- **个人防护**：由于氯气易与碱性物质反应，在没有专业防毒面具的情况下，应迅速用**浸有肥皂水、纯碱（$Na_2CO_3$）溶液或稀小苏打溶液的毛巾捂住口鼻**，以吸收吸入的氯气。
- **意外处理**：若不慎吸入气体导致身体不适，应立即撤离至室外通风处仰卧并深呼吸，必要时及时就医。

氯气的水溶性及防倒吸问题：

- **水溶性**：氯气在水中的溶解性通常被描述为“可溶”。在常温常压下（25 ℃），**1体积的水约能溶解2体积的氯气**，所得水溶液称为“氯水”。
- **是否需要防倒吸**：在将氯气通入蒸馏水制备氯水或进行实验时，**通常不需要安装防倒吸装置**。
    - **原理分析**：防倒吸装置主要针对“极易溶于水”的气体，如氨气（溶解比1:700）或氯化氢（溶解比1:500）。由于氯气的溶解度相对较小（仅1:2），产生的内外压强差不足以引发严重的倒吸现象。
- **不能通过通入纯水来处理氯气尾气**。
- **效率低下**：氯气与水的反应是**可逆反应**（$Cl_2 + H_2O \rightleftharpoons HCl + HClO$），反应极其不充分，无法将有毒的氯气完全吸收。
- **正确处理方法**：实验室通常使用**强碱溶液（如 $NaOH$ 溶液）**进行尾气吸收。
    - **反应原理**：$Cl_2 + 2NaOH = NaCl + NaClO + H_2O$。
    - **注意**：不宜使用澄清石灰水处理尾气，因为 $Ca(OH)_2$ 是微溶物，溶液浓度太低，吸收不完全。

虽然氯气因成本低、能残留在水中维持持续杀菌效果而被广泛使用，但其缺点也十分显著：

1. **产生致癌副产物**：氯气会与自来水中的**微量有机物**发生反应，生成**有机氯化物**（如三卤甲烷、氯仿等）。这些物质可能在人体内累积，具有**致癌性或慢性毒性**。
2. **气味与口感**：处理后的自来水常带有明显的“余氯”味，影响饮用口感。
3. **替代方案**：为了提高安全性，目前许多现代化水厂已开始使用**二氧化氯（$ClO_2$）或臭氧（$O_3$）**作为消毒剂。特别是二氧化氯，其杀菌能力更强、持效更久，且不会产生上述致癌的有机氯化物，被视为**绿色消毒剂**。

## 溴和碘的概述

### 卤素的物理性质

卤族元素（Halogens）位于元素周期表第ⅦA 族（第 17 列），包括**氟（F）、氯（Cl）、溴（Br）、碘（I）、砹（At）**以及人工合成的**石田（Ts）**。在中学及普通化学研究中，卤族元素的物理性质表现出极强的规律性与递变性。

卤族元素原子的最外电子层均有 **7 个电子**（价电子构型为 $ns^2np^5$），这决定了它们具有相似的化学性质，而物理性质的递变则主要源于核外电子层数的增加。

1. **原子半径**：从 F 到 I，随着核电荷数的增加，电子层数依次增多，原子半径**逐渐增大**。
2. **离子半径**：卤素原子易得到 1 个电子形成 $-1$ 价简单阴离子（$X^-$）。同主族自上而下，阴离子半径也随电子层数的增加而**逐渐增大**，且阴离子的半径总比其对应的原子半径大。

卤素单质（均以双原子分子 $X_2$ 形式存在）的颜色和物理状态随原子序数增大呈现明显的规律性变化：

- **颜色**：单质的颜色**逐渐加深**。
    - **氟（$F_2$）**：淡黄绿色气体。
    - **氯（$Cl_2$）**：黄绿色气体。
    - **溴（$Br_2$）**：深红棕色（深棕红色）液体，其蒸气为红棕色。
    - **碘（$I_2$）**：紫黑色固体，具有金属光泽，其蒸气为紫色。
    - **砹（$At$）**：推测为有色固体，颜色可能比碘更深（黑色）。
- **状态**：在常温常压下，呈现从**气态→液态→固态**的转变。
    - $F_2$、$Cl_2$为气体；$Br_2$是常温下**唯一呈液态的非金属单质**；$I_2$为固体。
- **气味与毒性**：卤素单质均具有强烈的刺激性或窒息性气味，且**均有毒**。

卤素单质均为**分子晶体**，分子间通过范德华力（分子间作用力）结合。

1. **熔沸点**：随着原子序数增大，相对分子质量增大，分子的极化率增强，分子间的范德华力逐渐增强，因此单质的熔点和沸点**逐渐升高**。
    - 具体的熔点数据示例：$F_2$ (53K) < $Cl_2$ (172K) < $Br_2$ (266K) < $I_2$ (387K)。
2. **密度**：单质的密度随原子序数的增大而**逐渐增大**。例如，液溴的密度比水大（约$3.119\,g/cm^3$），碘的密度更大（约$4.93\,g/cm^3$）。

卤素单质在水和有机溶剂中的溶解表现遵循**“相似相溶”**规律：

1. **水溶性**：
    - 卤素单质均为非极性分子，在极性溶剂（水）中的溶解度通常较小。
    - **氟（$F_2$）**：由于其极强的氧化性，进入水后会剧烈反应生成$HF$和$O_2$，因此在水溶液中不存在$F_2$分子。
    - **氯、溴、碘**：在水中的溶解度逐渐减小。碘在水中微溶，溶解度仅约$1g/3L$水。
    - **特殊**：碘在**碘化钾（KI）溶液**中的溶解度会明显增大，这是因为发生了反应 $I_2 + I^- \rightleftharpoons I_3^-$，形成了易溶的配离子。
2. **有机溶剂溶解性**：
    - 卤素单质易溶于**四氯化碳（$CCl_4$）、苯、汽油、酒精**等非极性或弱极性有机溶剂中。
    - **萃取显色**：利用溶解度差异可进行萃取，单质在有机层中呈现特征颜色。
        - **溴**：在有机溶剂中显**橙红色**或橙色。
        - **碘**：在有机溶剂（如$CCl_4$、苯）中显**紫色**或紫红色。

|                  | 氯元素 $\ce{Cl}$ | 溴元素 $\ce{Br}$ | 碘元素 $\ce{I}$ |
| :--------------: | :--------------: | :--------------: | :-------------: |
|       单质       |    黄绿色气体    |   深红棕色气体   |   紫黑色固体    |
|    卤化银沉淀    |       白色       |      淡黄色      |      黄色       |
|    水溶液颜色    |      黄绿色      |       橙色       |     黄褐色      |
| 四氯化碳溶液颜色 |      黄绿色      |      橙红色      |     紫红色      |

特殊物理现象：

1. **溴的挥发性**：液溴极易挥发，蒸气有毒。实验室保存液溴时常采用**“水封法”**，即在液溴表面覆盖一层水，以减少其挥发。
2. **碘的升华**：固态碘受热时，不经过液态而直接变成紫色蒸气，遇冷又重新凝华成固体。这一性质常用于碘的提纯。
3. **碘的特征反应**：碘单质遇**淀粉**溶液显**蓝色**。这是检验碘单质最灵敏的方法（注意：$I^-$离子无此现象）。
4. **导电性**：一般情况下卤素单质不导电，但碘在极高压下可能表现出一定的金属特性；砹作为放射性元素，其金属性比碘更强。

卤族元素的物理性质随原子序数的递增，呈现出**颜色变深、状态由气变固、熔沸点升高、密度增大、原子半径增加**的清晰递变规律。这些规律性变化是分子间作用力随相对分子质量增大而增强的宏观体现。

### 卤素的化学性质

卤族元素（氟、氯、溴、碘、砹）位于元素周期表第ⅦA 族，其化学性质表现出高度的**相似性**与明显的**递变性**。以下是根据提供的来源对卤族元素一般化学性质的详细展开：

1. **价电子构型**：卤族元素原子的最外电子层均有 **7 个电子**（价层电子排布为 $ns^2np^5$），这决定了它们都是极其活泼的非金属元素。
2. **得电子能力**：卤素原子非常容易得到 1 个电子而达到 8 电子稳定结构，形成 $-1$ 价的简单阴离子（$X^-$），因此卤素单质是**强氧化剂**。
3. **主要化合价**：由于最外层有 7 个电子，它们的最高正价通常为 **$+7$ 价**（氟除外），最低负价为 **$-1$ 价**。

随着原子序数的递增（自上而下从 F 到 I），卤族元素的原子半径逐渐增大，原子核对最外层电子的吸引力逐渐减弱：

1. **非金属性逐渐减弱**：得电子能力依次降低。
2. **单质氧化性逐渐减弱**：氧化性顺序为 **$F_2 > Cl_2 > Br_2 > I_2$**。
3. **阴离子还原性逐渐增强**：还原性顺序为 **$I^- > Br^- > Cl^- > F^-$**。
4. **与氢气反应难度增大**：反应越来越难进行，生成的气态氢化物（$HX$）稳定性依次减弱。

典型的化学反应：

1. 与金属单质反应：卤素单质能与绝大多数金属直接化合生成金属卤化物。
    - **反应条件**：$F_2$、$Cl_2$ 等氧化性强的单质在常温或点燃下即可反应，而 $I_2$ 通常需要加热。
    - **产物价态差异**：氧化性较强的 $F_2$、$Cl_2$、$Br_2$ 与变价金属（如铁）反应生成**高价态**卤化物（如 $FeCl_3$、$FeBr_3$）；而氧化性较弱的 $I_2$ 与铁反应仅生成**低价态**的 $FeI_2$。

2. 与氢气反应：卤素单质与 $H_2$ 反应生成卤化氢（$HX$），反应剧烈程度差异显著：
    - **氟（$F_2$）**：在**冷暗处**即可剧烈化合并发生爆炸。
    - **氯（$Cl_2$）**：在**光照或点燃**条件下发生爆炸或燃烧。
    - **溴（$Br_2$）**：需加热至一定温度才能反应。
    - **碘（$I_2$）**：需持续加热才能缓慢反应，且生成的 $HI$ 不稳定，易发生分解，反应具有**可逆性**。

3. 与水反应：
    - **氟的特殊性**：$F_2$ 具有极强的氧化性，能剧烈置换出水中的氧，反应方程式为：**$2F_2 + 2H_2O = 4HF + O_2$**。
    - **氯、溴、碘的歧化反应**：常温下可与水反应生成卤化氢和次卤酸（$HXO$），反应由强到弱：**$X_2 + H_2O \rightleftharpoons HX + HXO$**。

4. 与碱溶液反应：卤素单质（$F_2$ 除外）与碱溶液发生**歧化反应**：
    - 常温下反应生成卤化物、次卤酸盐和水：**$X_2 + 2OH^- = X^- + XO^- + H_2O$**。
    - 在加热或浓碱条件下，$I_2$（及 $Cl_2$、$Br_2$）会进一步歧化生成最高价态为 $+5$ 的卤酸盐：**$3I_2 + 6OH^- = 5I^- + IO_3^- + 3H_2O$**。

5. 卤素单质间的置换反应：根据氧化性强弱，排在前面的卤素单质能将排在后面的卤素从其盐溶液中置换出来：
    - $Cl_2 + 2Br^- = 2Cl^- + Br_2$
    - $Cl_2 + 2I^- = 2Cl^- + I_2$
    - $Br_2 + 2I^- = 2Br^- + I_2$
    - **注意**：不能用 $F_2$ 从水溶液中置换出其他卤素，因为它会优先与水反应产生 $O_2$。

卤素化合物的特殊性质：

1. **氢卤酸的酸性**：在水溶液中，除了 $HF$ 是**弱酸**外，其余（$HCl$、$HBr$、$HI$）均为**强酸**，且酸性顺序为：**$HF < HCl < HBr < HI$**。
2. **$HF$ 的特殊性**：$HF$ 分子间存在**氢键**，导致其沸点反常偏高；此外，$HF$（氢氟酸）能**腐蚀玻璃**（与 $SiO_2$ 反应），需保存在塑料瓶中。
3. **卤化银的溶解性与感光性**：除 $AgF$ 易溶于水外，$AgCl$（白色）、$AgBr$（淡黄色）、$AgI$（黄色）均难溶于水且具有感光性，常用于摄影感光材料。
4. **特征反应**：**淀粉遇碘单质（$I_2$）变蓝色**，这是检验碘单质的灵敏方法。

### 氟及其化合物

氟（Fluorine）是元素周期表第二周期第ⅦA族（第17族）元素，原子序数为 9。它是卤族元素的第一个成员，也是自然界中**非金属性最强、电负性最大**的元素。由于其极端的活泼性，氟在自然界中仅以化合态存在。

- **外观与状态**：在常温常压下，氟气（$F_2$）是一种**淡黄绿色**的气体。
- **气味与毒性**：具有和氯气相似的刺激性臭味，且具有**剧毒**和强腐蚀性。
- **熔沸点与密度**：氟的相对分子质量最小，分子间范德华力最弱，因此在卤素单质中其**熔沸点最低**（熔点约为 -219.67℃，沸点约为 -188.12℃）。其气态密度约为 $1.696 \, g/L$（0℃）。
- **溶解性**：氟气与水会发生剧烈反应，因此**不能单纯地溶解于水**中。

氟原子的最外层有 7 个电子，极易获得 1 个电子达到稳定结构。其化学性质极度活泼，几乎能与除氦、氖、氩以外的所有单质发生反应。

1. **化合价**：氟在化合物中**只有 -1 价**，没有正化合价。
2. **与氢气反应**：氟气与氢气在**冷暗处**即可发生剧烈化合并爆炸，生成极稳定的氟化氢。
    - $H_2 + F_2 = 2HF$
3. **与水反应**：氟气表现出极强的氧化性，能猛烈地置换出水中的氧。
    - $2F_2 + 2H_2O = 4HF + O_2$
4. **与金属反应**：氟能与几乎所有金属反应生成金属氟化物（如 $NaF$、$FeF_3$、$CaF_2$ 等）。
5. **与非金属及稀有气体反应**：
    - 能与碳、硫、磷、硅等直接化合，生成如 $CF_4$、$SF_6$、$SiF_4$ 等高价化合物。
    - 在特定条件下（如加热、光照），氟能与氪、氙等较重的稀有气体反应生成氟化物（如 $XeF_2$、$XeF_4$、$XeF_6$、$KrF_2$）。
6. **与碱反应**：氟气通入 2% 的稀 $NaOH$ 溶液中会生成二氟化氧（$OF_2$）。
    - $2F_2 + 2NaOH = 2NaF + H_2O + OF_2$

氟的重要化合物：

1. 氟化氢（$HF$）与氢氟酸：
    - **物理特性**：氟化氢是无色有刺激性气味的气体。由于分子间存在强烈的**氢键**，其沸点（20℃）在氢卤酸中反常最高，且易发生缔合形成 $(HF)_n$。
    - **弱酸性**：在水溶液中（氢氟酸），$HF$ 是一种**弱酸**。
    - **腐蚀玻璃**：这是氢氟酸最特殊的性质，它能与玻璃中的二氧化硅（$SiO_2$）反应，生成气态的四氟化硅，因此氢氟酸需保存在塑料瓶或铅制器皿中，常用于**刻蚀玻璃**。
        - $SiO_2 + 4HF = SiF_4\uparrow + 2H_2O$
    - **制备**：实验室和工业上通常用**萤石（$CaF_2$）与浓硫酸**共热制取。
        - $CaF_2 + H_2SO_4(浓) \xrightarrow{\Delta} CaSO_4 + 2HF\uparrow$

2. 氟化物盐类：
    - **氟化钙（$CaF_2$）**：俗称萤石，是自然界最重要的含氟矿物，也是制取 $HF$ 的原料。
    - **冰晶石（$Na_3[AlF_6]$）**：在电解铝工业中用作助熔剂。
    - **氟化钠（$NaF$）与氟化亚锡（$SnF_2$）**：常用于含氟牙膏中，通过生成比羟基磷灰石更耐酸的**氟磷灰石** $[Ca_5(PO_4)_3F]$ 来预防龋齿。

3. 有机氟化合物：
    - **聚四氟乙烯（PTFE）**：号称**“塑料王”**，具有极高的化学稳定性、耐高低温和耐腐蚀性能，常用作不粘锅涂层和航天材料。
    - **氟利昂（Freon）**：如 $CF_2Cl_2$，曾广泛用作制冷剂，但因会释放氯自由基（$Cl\cdot$）破坏臭氧层而逐渐被禁。

氟单质的制取：

- **莫瓦桑法（电解法）**：1886 年由法国化学家莫瓦桑首次实现，通过**电解熔融的 $KHF_2$ 与无水 $HF$** 的混合物制得。
- **化学法**：1986 年由德国化学家 Christie 首次实现，利用强 Lewis 酸（$SbF_5$）从稳定的配离子 $[MnF_6]^{2-}$ 盐中置换出不稳定的 $MnF_4$，后者分解释放 $F_2$。

氟在工业工艺中的特殊应用：

- **核工业**：利用六氟化铀（$UF_6$）的易升华特性，通过气体扩散法富集铀同位素（$^{235}U$）。
- **过渡金属冶炼**：在钽（$Ta$）与铌（$Nb$）、锆（$Zr$）与铪（$Hf$）的分离中，利用氟离子极强的**配位能力**形成稳定的氟络阴离子，通过萃取或结晶差异实现高纯度分离。

安全与防护：

- **骨骼毒性**：氟离子能与人体的钙、钾结合，导致剧烈疼痛和组织破坏（氟骨病），高浓度接触氢氟酸可能致命。
- **废水处理**：工业含氟废水通常加入**石灰乳** $[Ca(OH)_2]$，使氟离子转化为难溶的 $CaF_2$ 沉淀析出。

### 溴及其化合物

溴（Bromine）是位于元素周期表第四周期第ⅦA族（第17族）的非金属元素，符号为 **Br**，原子序数为 35。溴在自然界中主要以溴离子的形式存在于海水中，因此被称为“**海洋元素**”。以下是关于溴单质及其化合物的性质、制备、应用及检验的详细讲解。

- **外观与状态**：在常温常压（NTP）下，溴是唯一的**深红棕色（或黑红色）液体**非金属单质。
- **挥发性与气味**：溴极易挥发，产生**红棕色**的有毒蒸气，具有强烈的刺激性窒息臭味（英文名来源于希腊语“bromos”，意为“恶臭”）。
- **溶解性**：
    - **水溶性**：溴在水中溶解度较小（20℃时约为 3.4g/100g），所得溶液称为**溴水**，呈橙色或橙黄色。
    - **有机溶剂**：溴极易溶于乙醇、乙醚、苯、汽油、四氯化碳（$CCl_4$）等有机溶剂。在 $CCl_4$ 或苯中，溴层通常显**橙红色**。
- **安全性**：液溴具有强烈的腐蚀性，接触皮肤会引起难以治愈的溃疡。

由于溴易挥发且蒸气有毒，通常存放在细口磨口塞玻璃瓶中，并向瓶中加入少量水进行“**水封**”，以减少挥发。**注意**：保存液溴的瓶塞**严禁使用橡胶塞**，因为溴会腐蚀橡胶。

溴原子的最外层有 7 个电子，极易得到 1 个电子，表现出强氧化性，但氧化性弱于氟和氯。

- **与金属反应**：能与大多数金属反应生成高价金属溴化物（如 $2Fe + 3Br_2 \xrightarrow{\Delta} 2FeBr_3$）。
- **与氢气反应**：需加热至一定温度（约 500℃）方可缓慢反应生成溴化氢（$H_2 + Br_2 \xrightarrow{\Delta} 2HBr$），化合时不如氯气剧烈。
- **与水反应**：发生微弱的歧化反应，生成氢溴酸和次溴酸（$Br_2 + H_2O \rightleftharpoons HBr + HBrO$）。
- **与碱反应**：发生歧化反应。常温下（0℃左右）生成溴化物和次溴酸盐（$Br_2 + 2NaOH = NaBr + NaBrO + H_2O$）；室温以上或热浓碱中生成溴酸盐（$3Br_2 + 6OH^- \xrightarrow{\Delta} 5Br^- + BrO_3^- + 3H_2O$）。
- **置换反应**：溴的氧化性介于氯和碘之间，因此**氯能置换出溴，溴能置换出碘**。
    - $Cl_2 + 2Br^- = 2Cl^- + Br_2$
    - $Br_2 + 2I^- = 2Br^- + I_2$

溴的无机化合物：

1. 溴化氢 ($HBr$) 与氢溴酸：
    - **溴化氢**：无色、有刺激性气味的有毒气体，极易溶于水，在潮湿空气中形成**白雾**（酸雾）。
    - **氢溴酸**：$HBr$ 的水溶液，是**还原性强酸**。
    - **制备**：实验室通常用溴化钠与**浓磷酸**共热制取，而不使用浓硫酸，因为浓硫酸氧化性太强，会将 $HBr$ 氧化为 $Br_2$。
    - **反应性**：$HBr$ 溶于水是显著的放热过程。

2. 重要溴化物：
    - **溴化银 ($AgBr$)**：**淡黄色**固体，不溶于水和酸。具有强烈的**感光性**，见光分解为银和溴。广泛用于制造摄影底片、感光纸和变色玻璃。
    - **医疗用盐**：$NaBr$、$KBr$、$NH_4Br$ 在医学上常被用作**镇静剂**（神经系统对溴离子敏感）。

3. 溴的含氧酸：
    - 主要包括次溴酸 ($HBrO$)、亚溴酸 ($HBrO_2$)、溴酸 ($HBrO_3$) 和高溴酸 ($HBrO_4$)。
    - **规律**：随着溴元素价态升高，其含氧酸的酸性和稳定性逐渐增强，但氧化性通常减弱。$HBrO_4$ 氧化性极强，且在室温下氧化速率较慢。

溴在有机化学中的应用：溴是重要的有机合成试剂，通过取代或加成反应引入溴原子，常被视为合成的“桥梁”。

- **加成反应**：溴水或溴的 $CCl_4$ 溶液常用于检验有机物中的**碳碳双键**或**三键**，反应会导致溴色褪去。
    - 如乙烯与溴反应生成 1,2-二溴乙烷：$CH_2=CH_2 + Br_2 \rightarrow CH_2Br-CH_2Br$。
- **取代反应**：
    - **苯的溴代**：在铁粉（实际催化剂为 $FeBr_3$）催化下，苯与**液溴**反应生成溴苯。
    - **苯酚反应**：苯酚与浓溴水反应生成**三溴苯酚白色沉淀**，用于苯酚的定性检验。
- **溴代烃**：如**溴乙烷** ($CH_3CH_2Br$)，是无色液体，密度比水大，难溶于水。其在氢氧化钠水溶液中发生**水解反应**生成乙醇，在氢氧化钠醇溶液中发生**消去反应**生成乙烯。

溴及其离子的检验方法：

1. **沉淀法**：向未知液中先加稀硝酸酸化，再滴加 **$AgNO_3$ 溶液**。若产生**淡黄色沉淀** ($AgBr$)，且沉淀不溶于稀硝酸，则证明含有 $Br^-$。
2. **置换-萃取法**：向未知液中滴加少量新制**氯水**，充分振荡后加入 **$CCl_4$（或苯）**萃取。若有机层呈**橙红色**，则证明含有 $Br^-$。

### 碘及其化合物

碘（Iodine）是元素周期表中第五周期第ⅦA族（第17族）的元素，原子序数为53。作为卤族元素中原子序数较大的成员，碘及其化合物在结构、物理性质及化学反应中表现出独特的规律与应用价值。

1. **外观与状态**：常温下，碘是一种呈**紫黑色**、具有**金属光泽**的固体。
2. **升华性**：碘具有显著的**升华**特性。加热时，固态碘不经过液态直接变为**紫色蒸气**；蒸气遇冷则重新凝华为固体。这一性质常用于碘的提纯。
3. **溶解性**：
    - **水**：**微溶**于水，其水溶液（碘水）呈棕黄色或黄色。
    - **有机溶剂**：**易溶**于乙醇、乙醚、苯、四氯化碳 ($CCl_4$) 和二硫化碳等。在 $CCl_4$ 或苯等非极性溶剂中，碘溶液呈**紫色**；在乙醇中呈棕色。
    - **碘化物溶液**：碘在含有碘离子 ($I^-$) 的溶液中溶解度显著增大。这是因为发生了反应 **$I_2 + I^- \rightleftharpoons I_3^-$**，形成了可溶性的三碘阴离子。
4. **毒性与气味**：碘蒸气具有强烈的刺激性、窒息性臭味，且有毒性和腐蚀性。

碘单质的化学性质：

1. **氧化性**：碘是较活泼的非金属，具有强氧化性，但氧化性弱于 $F_2$、$Cl_2$ 和 $Br_2$。
    - **与金属反应**：能与大多数金属反应，但通常生成**低价态**卤化物。例如，铁与碘反应仅生成**碘化亚铁 ($FeI_2$)**，而氯气与铁反应生成氯化铁 ($FeCl_3$)。
    - **与氢气反应**：需在**高温**下才能缓慢化合生成碘化氢 ($HI$)，且该反应是**可逆**的，生成的 $HI$ 不稳定。
2. **歧化反应**：碘与碱溶液反应会发生歧化。即使在常温或微热条件下，碘主要歧化生成碘化物和**碘酸盐**（而非次碘酸盐，因为 $IO^-$ 极不稳定）：**$3I_2 + 6OH^- = 5I^- + IO_3^- + 3H_2O$**。
3. **置换反应**：排在碘前面的卤素单质（$F_2, Cl_2, Br_2$）能从碘化物溶液中置换出碘单质。
    - 例如：$Cl_2 + 2I^- = I_2 + 2Cl^-$。

碘的重要化合物：

1. **碘化氢 ($HI$) 与氢碘酸**：
    - $HI$ 是无色、有刺激性气味的气体，极易溶于水，其水溶液称为**氢碘酸**。
    - **强还原性**：氢碘酸是强酸，也是卤化氢中还原性最强的。它极易被氧化，甚至露置在空气中就会被氧气氧化而显棕色（析出 $I_2$）。
    - **制备**：实验室通常用碘化钠与**浓磷酸**共热制取，**不能使用浓硫酸**，因为浓硫酸会将其氧化为碘单质。
2. **碘酸钾 ($KIO_3$)**：白色固体，是强氧化剂，在酸性条件下可与 $I^-$ 发生归中反应生成 $I_2$。
3. **碘化银 ($AgI$)**：**黄色**沉淀，不溶于稀硝酸。它具有强烈的感光性，且在光照下分解吸热，常用于**人工降雨**。
4. **卤素互化物**：碘可与其他卤素结合，如 **$ICl$**（一氯化碘）和 **$IBr$**（一溴化碘）。在这些化合物中，电负性较弱的碘显正价（如 +1），具有强氧化性，遇水易发生非氧化还原的水解反应。
5. **特殊化合物 $I_4O_9$**：实质上是一种离子型化合物，化学式为 **$I(IO_3)_3$**（碘酸碘），其中含一个 +3 价的碘阳离子和三个 +5 价的碘酸根阴离子。

特征检验与定量分析：

1. **淀粉试验**：**碘单质遇淀粉变蓝色**，这是检验碘最灵敏的方法。
    - **微观原理**：直链淀粉形成左手螺旋空腔，碘分子与多碘离子在腔内形成线性长链，电子高度离域使能级差缩小，从而吸收红橙光而显蓝色。
2. **碘量法滴定**：化学分析中常用单质碘的氧化性或碘离子的还原性进行定量测定。
    - 核心反应：**$I_2 + 2S_2O_3^{2-} = 2I^- + S_4O_6^{2-}$**。
    - 终点判断：滴入最后半滴硫代硫酸钠溶液后，溶液**蓝色褪去**且半分钟内不恢复。

应用与生理作用：

1. **生理健康**：碘是人体必需的微量元素，是合成**甲状腺激素**的重要成分。缺碘会引起**甲状腺肿大**（俗称“大脖子病”）。
2. **加碘食盐**：我国通过在食盐中添加 **$KIO_3$** 来防治碘缺乏病。
3. **医药用途**：碘酒（碘的酒精溶液）是常用的皮肤消毒剂。
4. **同位素应用**：碘-131 等放射性同位素用于放射性治疗和示踪技术。

### 碘量法（碘量滴定法和滴定碘量法）

碘量法是氧化还原滴定法中的一种重要方法，主要利用碘（$\text{I}_2$）的氧化性和碘离子（$\text{I}^-$）的还原性来进行定量分析。根据滴定方式的不同，碘量法可分为**直接碘量法**和**间接碘量法**。

碘量法的基础反应是碘与硫代硫酸钠（$\text{Na}_2\text{S}_2\text{O}_3$）之间的定量反应：

$$
\text{I}_2 + 2\text{S}_2\text{O}_3^{2-} = 2\text{I}^- + \text{S}_4\text{O}_6^{2-}
$$

在这个反应中，$\text{I}_2$ 作为氧化剂被还原为 $\text{I}^-$，而硫代硫酸根（$\text{S}_2\text{O}_3^{2-}$）被氧化为连四硫酸根（$\text{S}_4\text{O}_6^{2-}$）。

**直接碘量法（碘滴定法）**是用**碘标准溶液**直接滴定具有还原性物质的方法。

- **滴定过程**：将碘标准液装在滴定管中，直接滴入盛有待测还原性物质的锥形瓶中。
- **适用对象**：滴定强还原剂，如 $\text{Sn}^{2+}$、$\text{H}_2\text{S}$、$\text{As(III)}$、$\text{SO}_3^{2-}$、维生素 C等。
- **反应条件**：由于碘在碱性溶液中会发生歧化反应，滴定通常在**弱酸性、中性或弱碱性**（$\text{pH} < 9$）溶液中进行。
- **指示剂与终点**：通常使用淀粉溶液作为指示剂。
    - **现象**：滴定至终点时，溶液由**无色变为蓝色**，且半分钟内不褪色。

**间接碘量法（滴定碘法）**是利用 $\text{I}^-$ 的还原性与氧化性待测物反应产生 $\text{I}_2$，再用硫代硫酸钠标准溶液滴定生成的 $\text{I}_2$。

- **实验步骤**：
    1. 向氧化性待测物中加入**过量的碘化钾（$\text{KI}$）**，在适宜条件下待测物将 $\text{I}^-$ 氧化生成 $\text{I}_2$。
    2. 用 $\text{Na}_2\text{S}_2\text{O}_3$ 标准溶液滴定释放出的 $\text{I}_2$。
- **适用对象**：测定强氧化剂，如 $\text{O}_3$、$\text{ClO}^-$、$\text{IO}_3^-$、$\text{Cr}_2\text{O}_7^{2-}$、$\text{MnO}_4^-$、$\text{Cu}^{2+}$ 等。
- **计算逻辑（关系式法）**：
    通过多步反应建立关系式，例如测定 $\text{KIO}_3$ 纯度时：$\text{IO}_3^- \sim 3\text{I}_2 \sim 6\text{S}_2\text{O}_3^{2-}$。
- **终点判断**：
    - **现象**：滴入最后半滴标准液时，溶液由**蓝色变为无色**，且半分钟内不恢复原色。
- **特殊操作**：淀粉指示剂应在**滴定接近终点**（溶液呈微黄色/浅黄色）时加入。若过早加入，大量碘会与淀粉结合成极稳定的深蓝色络合物，导致碘释放缓慢，引起滴定误差。

关键实验细节：

1. **指示剂（淀粉）**：
    - **微观原理**：直链淀粉盘旋成螺旋构象，碘分子进入其疏水空腔形成线性多碘链。电子在长链上的高度离域显著缩小了能级差，使其强烈吸收可见光中的黄橙色光，从而显现其互补色——**深蓝色**。
    - **灵敏度**：该反应非常敏锐，可定性检验极微量的碘。
2. **溶液配制**：
    - **碘液**：碘单质微溶于水，配制时需溶解在浓 $\text{KI}$ 溶液中，利用 $\text{I}_2 + \text{I}^- \rightleftharpoons \text{I}_3^-$ 反应增大溶解度并降低挥发性。
    - **硫代硫酸钠液**：应使用新煮沸并冷却的蒸馏水配制，以除去水中的 $\text{CO}_2$、$\text{O}_2$ 并杀灭细菌，防止溶液变质。
3. **仪器选择（碘量瓶）**：
    - 滴定应在**碘量瓶**（具塞锥形瓶）中进行，其喇叭口设计可加水密封，防止产生的 $\text{I}_2$ 蒸气挥发逸失，并减少空气中氧气对 $\text{I}^-$ 的氧化干扰。
4. **反应环境**：
    - 通常需要在**暗处**静置数分钟（如 5 min），使氧化还原反应进行完全。
    - 碘量法中滴定体系不应受热（高温会加速 $\text{I}_2$ 挥发并使淀粉指示剂灵敏度下降），且滴定速度要适当控制以防过量。

### 溴和碘的工业制取与提纯

溴和碘是海水中蕴藏丰富的卤素资源，工业上针对两者的制取与提纯主要基于氧化还原反应原理，并利用其物理性质（如挥发性、升华性、溶解度差异）进行富集和纯化。

溴被誉为“海洋元素”，地球上 99% 的溴分布在海洋中。工业上从海水中提取溴最成熟的方法是**“空气吹出法”**。

1. **浓缩与酸化（原料预处理）**：
    - 利用海水晒盐后的母液（苦卤）作为原料，此时溴离子（$Br^-$）已得到初步浓缩。
    - 向苦卤中加入硫酸进行**酸化**，目的是控制溶液的 pH 约为 3.5，以抑制氯气（$Cl_2$）和生成的溴（$Br_2$）与水发生反应。

2. **氧化（置换单质）**：
    - 向酸化的浓缩海水中通入**氯气**作为氧化剂，将溴离子氧化为溴单质。
    - 反应的离子方程式为：$Cl_2 + 2Br^- = 2Cl^- + Br_2$。

3. **吹出与吸收（初级富集）**：
    - 利用溴具有**挥发性**的物理性质，向氧化后的溶液中通入**热空气或水蒸气**，将溶解在水中的溴吹出。
    - 吹出的溴蒸气进入吸收塔，通常使用**二氧化硫（$SO_2$）**水溶液吸收，将其还原为氢溴酸（$HBr$），从而实现溴的富集。
    - 吸收环节的离子方程式为：$Br_2 + SO_2 + 2H_2O = 4H^+ + 2Br^- + SO_4^{2-}$。
    - 部分工艺也会采用纯碱（$Na_2CO_3$）溶液吸收溴蒸气，生成溴化钠和溴酸钠。

4. **再次氧化与提取（产物生成）**：
    - 向富集后的 $HBr$ 溶液（或经酸化的纯碱吸收液）中再次通入**氯气**，将溴离子氧化成溴单质。
    - 化学方程式为：$2HBr + Cl_2 = 2HCl + Br_2$。

5. **精制与提纯**：
    - 通过**蒸馏、冷凝、分液**等操作，即可得到高纯度的液溴。
    - 提纯过程中需注意，溴会腐蚀橡胶，因此实验及生产装置中不能使用橡胶塞或乳胶管。

由于海水中碘的浓度极低，工业上主要通过具有富集碘能力的海洋生物（如海带、海藻）或从智利硝石中提取。

从干海带中提取碘（经典工艺）：

1. **灼烧灰化**：将干海带放入坩埚中**灼烧**至完全成灰，目的是使有机碘转化为无机碘盐。
2. **浸取与过滤**：将海带灰加沸水溶解并搅拌，使碘离子充分进入溶液，随后通过**过滤**除去不溶性残渣。
3. **酸化与氧化（核心步骤）**：
    - 在滤液中加入稀硫酸酸化，然后加入氧化剂将 $I^-$ 氧化为 $I_2$。
    - 常用的氧化剂包括**氯气**（$Cl_2 + 2I^- = I_2 + 2Cl^-$）或**双氧水**（$H_2O_2 + 2I^- + 2H^+ = I_2 + 2H_2O$）。
    - **注意**：工业生产中氯气用量需严格控制，过量氯气会将生成的 $I_2$ 进一步氧化为高价态的碘酸根（$IO_3^-$），导致碘的收率降低。

从智利硝石中提取碘：

- 智利硝石中含有碘酸钠（$NaIO_3$），主要采用**亚硫酸氢钠（$NaHSO_3$）还原法**。
- 总反应方程式为：$2IO_3^- + 5HSO_3^- = I_2 + 5SO_4^{2-} + 3H^+ + H_2O$。

碘的富集与精制提纯：

1. **萃取与分液**：
    - 利用碘在有机溶剂（如**四氯化碳 $CCl_4$**、苯）中的溶解度远大于在水中的性质，将碘从水溶液中转移到有机相中。
    - 在 $CCl_4$ 或苯等非极性溶剂中，碘溶液呈特征性的**紫色**。

2. **反萃取（化学富集法）**：
    - 若需进一步浓缩，可向碘的有机溶液中加入**浓 $NaOH$ 溶液**进行反萃取。
    - 利用碘在碱性条件下的歧化反应：$3I_2 + 6OH^- = 5I^- + IO_3^- + 3H_2O$。
    - 分离出水层后再用稀硫酸酸化，发生归中反应重新析出碘单质，过滤后得到粗碘。

3. **升华提纯（最终精制）**：
    - 碘具有显著的**升华**特性，受热易变为紫色蒸气，遇冷直接凝华为固体。
    - 工业和实验室常利用升华法除去碘中不挥发性的杂质，获得高纯度的晶体碘。

### 卤素的相似性与递变性

卤族元素（简称卤素）位于元素周期表第ⅦA 族（第 17 列），包括氟（F）、氯（Cl）、溴（Br）、碘（I）、砹（At）和石田（Ts）。这一族元素在原子结构、物理性质和化学性质上表现出高度的**相似性**与规律性的**递变性**。

原子结构的相似性与递变性：

- **最外层电子数相同**：卤素原子的最外电子层均有 **7 个电子**（价电子构型为 $ns^2np^5$）。
- **成键倾向**：由于最外层电子接近 8 电子稳定结构，卤素原子极易得到 1 个电子形成 $-1$ 价的简单阴离子（$X^-$），是典型的活泼非金属元素。
- **存在形式**：在自然界中均以化合态存在，单质均由双原子分子（$X_2$）组成。
- **原子半径逐渐增大**：随着核电荷数增加，电子层数依次增多，原子半径逐渐增大。
- **电离能与电负性逐渐减小**：原子半径增大导致核对外层电子的束缚能力减弱，因此第一电离能和电负性均依次减小。

卤素单质的物理性质随着相对分子质量的增大呈现出明显的规律性变化：

- **颜色逐渐加深**：
    - **$F_2$**：淡黄绿色气体。
    - **$Cl_2$**：黄绿色气体。
    - **$Br_2$**：深红棕色（深棕红色）液体。
    - **$I_2$**：紫黑色固体。
- **状态变化**：呈现由**气态 $\rightarrow$ 液态 $\rightarrow$ 固态**的转变。其中 $Br_2$ 是常温下唯一呈液态的非金属单质。
- **熔、沸点逐渐升高**：卤素单质均为分子晶体。随着相对分子质量增大，分子间范德华力增强，熔、沸点依次升高。
- **密度逐渐增大**：单质的密度随原子序数增加而逐渐增大。
- **溶解性**：在水中的溶解度一般较小，且逐渐减小（$F_2$ 除外，它与水剧烈反应）；易溶于苯、$CCl_4$等有机溶剂。

化学性质的相似性与递变性：

1. 化学性质的相似性（共性）：
    - **强氧化性**：卤素单质都是强氧化剂。
    - **与金属反应**：能与绝大多数金属直接化合生成金属卤化物（如 $NaCl, MgBr_2, FeI_2$ 等）。
    - **与氢气反应**：均能与 $H_2$ 反应生成气态氢化物 $HX$。
    - **歧化反应（除 $F_2$ 外）**：常温下与水或强碱溶液反应，发生歧化生成卤化物和次卤酸（盐）。

2. 化学性质的递变性（自 F 到 I）：随着原子半径增大，原子核对电子的引力减弱，得电子能力减弱，非金属性随之减弱。
    - **单质氧化性逐渐减弱**：$F_2 > Cl_2 > Br_2 > I_2$。
    - **阴离子还原性逐渐增强**：$I^- > Br^- > Cl^- > F^-$。
    - **单质间的置换反应**：氧化性强的单质可以将氧化性弱的元素从其盐溶液中置换出来（如 $Cl_2 + 2Br^- = 2Cl^- + Br_2$）。
    - **与氢气反应难度增大**：
        - **$F_2$**：冷暗处即可剧烈爆炸化合。
        - **$Cl_2$**：光照或点燃时发生爆炸或燃烧。
        - **$Br_2$**：加热至一定温度方可反应。
        - **$I_2$**：持续加热下缓慢化合，且生成的 $HI$ 不稳定，具有可逆性。
    - **氢化物的递变**：
        - **热稳定性依次减弱**：$HF > HCl > HBr > HI$。
        - **水溶液酸性依次增强**：除了 **$HF$ 为弱酸**外，其余均为强酸，酸性顺序为：$HF < HCl < HBr < HI$。

在遵循递变规律的同时，卤素成员也具有一些“另类”的特质：

1. **氟（F）的特殊性**：
    - **无正价**：由于电负性最大，在化合物中只显 $-1$ 价，无正价，故无含氧酸。
    - **与水反应特殊**：极易与水反应生成 $HF$ 并放出 $O_2$（$2F_2 + 2H_2O = 4HF + O_2$），因此无法在水溶液中进行单质间的置换反应。
    - **$HF$ 的反常**：由于存在分子间**氢键**，$HF$ 的沸点反常地高；此外，$HF$（氢氟酸）能**腐蚀玻璃**，需保存在塑料瓶或铅制器皿中。
2. **溴（Br）的特殊性**：常温下唯一的液态非金属单质，极易挥发，通常需用**水封**保存，且不能使用橡胶塞（会腐蚀）。
3. **碘（I）的特殊性**：具有**升华**特性；单质遇**淀粉**溶液显蓝色；在有机溶剂中呈紫色。此外，$I_2$ 与变价金属（如 $Fe$）反应只能生成低价态产物（如 $FeI_2$），而 $Cl_2, Br_2$ 能将其氧化为 $+3$ 价。

### 卤素互化物和拟卤素

在卤族元素的化学研究中，**卤素互化物**和**拟卤素**是两类极其重要的拓展概念。它们在结构或性质上与卤素单质高度相似，是理解元素周期律和拟合化学反应规律的关键。以下是详细的讲解：

**卤素互化物**是由两种不同的卤素原子通过共价键结合形成的物质。

- **通式**：一般表示为 $XX'_n$，其中 $n$ 通常为奇数（1, 3, 5, 7），且中心原子 $X$ 的原子半径必须大于被包围的原子 $X'$（即 $r_X > r_{X'}$）。
- **形成规律**：$n$ 的数值取决于两个卤素原子半径的比值以及它们电负性的差值；比值和差值越大，$n$ 的最大值也就越大。
- **常见实例**：$\text{ICl}$（一氯化碘）、$\text{IBr}$（一溴化碘）、$\text{BrF}_3$、$\text{ClF}_3$、$\text{IF}_5$、$\text{IF}_7$ 等。其中 $\text{ICl}$ 是 1814 年由给吕萨克首先发现的。

- **化合价规律**：在卤素互化物中，**非金属性强（电负性大）的元素显负价**（通常为 -1），非金属性弱的元素显正价。例如在 $\text{ICl}$ 中，氯（Cl）显 -1 价，碘（I）显 +1 价。
- **状态与稳定性**：这类化合物多为易挥发的液体或固体，熔沸点通常较低，性质与卤素单质相似。例如 $\text{ICl}$ 为红棕色液体，熔点为 $27.2^\circ\text{C}$。

卤素互化物的化学性质与卤素单质非常相似，具有**强氧化性**。

- **与金属反应**：能与金属化合生成相应的卤化物。
    - 例如：$2\text{IBr} + 2\text{Mg} = \text{MgBr}_2 + \text{MgI}_2$。
- **水解反应**：卤素互化物遇水极易发生水解。由于水中的元素极难被氧化，此类反应通常属于**非氧化还原反应**，生成对应化合价的氢代酸。
    - 反应规律：$XX' + \text{H}_2\text{O} = \text{HX'} + \text{HXO}$。
    - 实例：$\text{IBr} + \text{H}_2\text{O} = \text{HBr} + \text{HIO}$。注意产物中电负性强的元素结合氢生成氢卤酸（如 $\text{HBr}$），电负性弱的元素生成次卤酸（如 $\text{HIO}$）。
- **与碱反应**：与水解类似，生成对应的盐，各元素化合价保持不变。
    - 实例：$\text{IBr} + 2\text{NaOH} = \text{NaBr} + \text{NaIO} + \text{H}_2\text{O}$。

**拟卤素**是指由两种或多种非金属原子团形成的二聚体分子，其在游离态时的化学性质与卤素单质十分相似。

- **拟卤离子**：指对应原子团的阴离子，如 $\text{CN}^-$、$\text{SCN}^-$、$\text{OCN}^-$ 等，它们的性质与卤离子（$\text{X}^-$）相似。
- **常见实例**：氰 $(\text{CN})_2$、硫氰 $(\text{SCN})_2$、氧氰 $(\text{OCN})_2$、硒氰 $(\text{SeCN})_2$、叠氮代二硫化碳 $(\text{SCSN}_3)_2$。

拟卤素与卤素的相似性主要体现在以下几个维度：

- **分子结构**：游离态时为双聚分子，且通常易挥发、熔沸点低。
- **与氢结合**：能与氢气反应生成相应的氢化物（$\text{HX}$），其水溶液多显酸性。例如 $\text{HCN}$（氢氰酸）是弱酸，而 $\text{HSCN}$ 是强酸。
- **与金属反应**：能与多种金属反应生成盐，且其银（$\text{Ag}$）、汞（$\text{Hg(I)}$）、铅（$\text{Pb}$）盐多难溶于水。
- **歧化反应**：在水或碱溶液中会发生类似于卤素的歧化反应。
    - 实例：$(\text{CN})_2 + 2\text{OH}^- = \text{CN}^- + \text{OCN}^- + \text{H}_2\text{O}$。
- **置换反应**：可以像卤素单质一样通过置换反应制备。
    - 实例：$\text{Pb(SCN)}_2 + \text{Br}_2 = \text{PbBr}_2 + (\text{SCN})_2$。
- **配位能力**：拟卤离子可以作为配体与各种金属离子形成稳定的配离子。
    - 实例：$[\text{Zn(SCN)}_4]^{2-}$、$[\text{Ag(CN)}_2]^-$。
- **可被氧化性**：拟卤离子可以被强氧化剂氧化回双聚分子状态。

卤素互化物和拟卤素的研究体现了化学中**“类比”**和**“换元”**的思想。**卤素互化物**重点在于不同卤素原子间的电负性差异及其产生的不对称氧化性；而**拟卤素**则展示了特定原子团作为一个整体如何模拟单一原子的化学行为。在处理此类问题时，可将互化物视作极性加强的卤素单质，将拟卤素直接视作 $\text{X}_2$ 进行反应推导。
