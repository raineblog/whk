# 氮族活泼非金属

## 氮的基本性质

### 氮气及其性质

氮气（$\text{N}_2$）是大气中含量最丰富的气体，约占空气总体积的 78%。作为氮族元素（第 15 族）的首位元素，氮气在自然界循环和工业生产中具有举足轻重的地位。

氮气分子的结构特征是其化学性质高度稳定的根本原因：

- **成键方式**：氮原子（$1s^2 2s^2 2p^3$）有 3 个未成对电子。两个氮原子通过共用三对电子结合，形成 **共价三键**（$\text{N}\equiv\text{N}$）。
- **轨道重叠**：三键由一个 **$\sigma$ 键**（原子轨道“头碰头”重叠）和两个 **$\pi$ 键**（原子轨道“肩并肩”重叠）组成。
- **高键能与稳定性**：氮氮三键的键能极高（约为 **941~946 kJ/mol**），断开此键需要巨大的能量。因此，氮气在常温下表现出极强的 **化学惰性**，难以与其他物质反应。

物理性质：

- **基本特征**：纯净的氮气是 **无色、无味、无嗅** 的气体。
- **密度与溶解性**：密度（1.25 g/L）略小于空气，**难溶于水**。
- **相变点**：在常压下，沸点约为 **$-196^\circ\text{C}$**，熔点约为 **$-210^\circ\text{C}$**。液氮常作为低温实验或医学手术的冷冻剂。

尽管氮气性质稳定，但在高温、高压、放电或催化剂等特定条件下，仍可发生反应：

1. 与非金属反应
    - **还原性（与氧气反应）**：在 **放电或高温** 条件下，氮气能与氧气化合生成一氧化氮。
        - $\text{N}_2 + \text{O}_2 \xrightarrow{\text{放电/高温}} 2\text{NO}$
        - 此反应是“雷雨发庄稼”过程的第一步。
    - **氧化性（与氢气反应）**：即工业合成氨（哈伯法），是人工固氮的核心。
        - $\text{N}_2 + 3\text{H}_2 \xrightleftharpoons[\text{催化剂}]{\text{高温、高压}} 2\text{NH}_3$

2. 与金属反应（表现氧化性）
    - **与镁反应**：点燃的镁条能在氮气中燃烧，生成 **淡黄色**（或黄绿色）的氮化镁固体。
        - $3\text{Mg} + \text{N}_2 \xrightarrow{\text{点燃}} \text{Mg}_3\text{N}_2$
        - $\text{Mg}_3\text{N}_2$ 遇水会发生剧烈水解，产生氨气：$\text{Mg}_3\text{N}_2 + 6\text{H}_2\text{O} = 3\text{Mg}(\text{OH})_2 + 2\text{NH}_3 \uparrow$。
    - **与锂反应**：锂是极少数在 **常温** 下能与氮气缓慢反应的金属。
        - $6\text{Li} + \text{N}_2 = 2\text{Li}_3\text{N}$

实验室制备方法：

1. **最常用法**：加热 **饱和氯化铵和亚硝酸钠** 溶液。
    - $\text{NH}_4\text{Cl} + \text{NaNO}_2 \xrightarrow{\Delta} \text{NaCl} + \text{N}_2 \uparrow + 2\text{H}_2\text{O}$
2. **超纯氮制备**：加热 **叠氮化钠**（$\text{NaN}_3$）分解（此原理亦用于汽车安全气囊）。
    - $2\text{NaN}_3 \xrightarrow{\Delta} 2\text{Na} + 3\text{N}_2 \uparrow$
3. **氨气还原法**：用氨气还原灼热的氧化铜。
    - $2\text{NH}_3 + 3\text{CuO} \xrightarrow{\Delta} 3\text{Cu} + \text{N}_2 + 3\text{H}_2\text{O}$

主要用途：

- **保护气**：利用其化学不活泼性，用于金属焊接保护、食品充氮防腐、灯泡填充气防止钨丝氧化。
- **工业原料**：制造氨气、硝酸、化肥、炸药及合成材料。
- **科研医疗**：液氮作为超导材料的低温环境及医疗冷冻麻醉。

### 氮的氧化物及其性质

氮的氧化物种类繁多，涵盖了从 +1 到 +5 的所有正价态。这些化合物在工业生产、环境保护以及生命科学中都具有极其重要的地位。

氮的主要氧化物包括以下六种，其中氮元素的化合价随氧化程度增加而升高：

1. **一氧化二氮 ($\text{N}_2\text{O}$)**：+1 价
2. **一氧化氮 ($\text{NO}$)**：+2 价
3. **三氧化二氮 ($\text{N}_2\text{O}_3$)**：+3 价（亚硝酸酐）
4. **二氧化氮 ($\text{NO}_2$)**：+4 价
5. **四氧化二氮 ($\text{N}_2\text{O}_4$)**：+4 价（$\text{NO}_2$ 的二聚体）
6. **五氧化二氮 ($\text{N}_2\text{O}_5$)**：+5 价（硝酸酐）

重点氧化物详解：

1. 一氧化氮 ($\text{NO}$)
    - **物理性质**：**无色、无味、有毒**的气体，密度比空气略大，**不溶于水**。
    - **化学性质**：
        - **极易氧化**：在常温下即可与空气中的氧气反应生成红棕色的二氧化氮：$2\text{NO} + \text{O}_2 = 2\text{NO}_2$。
        - **毒性原理**：$\text{NO}$ 与血红蛋白的结合能力比 $\text{CO}$ 还强，容易引起人体缺氧窒息。
        - **生物学作用**：在人体内，极少量的 $\text{NO}$ 是重要的“信使分子”，具有扩张血管、提高免疫力和增强记忆力的功能。
    - **实验室制法**：通过铜与**稀硝酸**反应制取：$3\text{Cu} + 8\text{HNO}_3(稀) = 3\text{Cu}(\text{NO}_3)_2 + 2\text{NO}\uparrow + 4\text{H}_2\text{O}$。收集时必须使用**排水法**。

2. 二氧化氮 ($\text{NO}_2$)
    - **物理性质**：**红棕色、有刺激性气味、有毒**的气体，密度比空气大，易液化，**易溶于水**。
    - **化学性质**：
        - **与水反应**：$\text{NO}_2$ 溶于水发生歧化反应，生成硝酸和一氧化氮：$3\text{NO}_2 + \text{H}_2\text{O} = 2\text{HNO}_3 + \text{NO}$。
        - **与碱反应**：可被烧碱溶液吸收生成硝酸盐和亚硝酸盐：$2\text{NO}_2 + 2\text{NaOH} = \text{NaNO}_3 + \text{NaNO}_2 + \text{H}_2\text{O}$。
        - **可逆平衡**：$\text{NO}_2$ 存在二聚反应平衡，$2\text{NO}_2 \rightleftharpoons \text{N}_2\text{O}_4$（$\text{NO}_2$ 为红棕色，$\text{N}_2\text{O}_4$ 为无色，该反应为**放热反应**）。升高温度或降低压强，颜色会加深。
    - **实验室制法**：通过铜与**浓硝酸**反应制取：$\text{Cu} + 4\text{HNO}_3(浓) = \text{Cu}(\text{NO}_3)_2 + 2\text{NO}_2\uparrow + 2\text{H}_2\text{O}$。由于其易溶于水且与水反应，只能使用**向上排空气法**收集。

3. 其他氧化物
    - **一氧化二氮 ($\text{N}_2\text{O}$)**：俗称“**笑气**”，有甜味，可用作麻醉剂。常温下较稳定，但在高温下可助燃。
    - **三氧化二氮 ($\text{N}_2\text{O}_3$)**：亚硝酸的酸酐，在低温下为**深蓝色液体或固体**，常温下极易分解为 $\text{NO}$ 和 $\text{NO}_2$。
    - **五氧化二氮 ($\text{N}_2\text{O}_5$)**：硝酸的酸酐，**无色固体**，极易潮解，不稳定，受热易分解放出氧气。在固态时其结构为离子晶体 $[\text{NO}_2]^+[\text{NO}_3]^-$。

氮氧化物（通常统称为 $\text{NO}_x$）是主要的大气污染物之一。

- **危害**：
    - **酸雨**：形成硝酸型酸雨。
    - **光化学烟雾**：$\text{NO}_x$ 与碳氢化合物在紫外线照射下发生反应，产生臭氧和 $\text{PAN}$ 等有毒物质。
    - **臭氧层破坏**：$\text{NO}_2$ 可与平流层中的臭氧发生作用，导致臭氧层损耗。
- **尾气处理方法**：**碱液吸收法**，利用 $\text{NaOH}$ 溶液吸收，通常需保证 $n(\text{NO}_2) \geq n(\text{NO})$ 才能被完全吸收。

在实验室中，制取氮氧化物（主要是一氧化氮和二氧化氮）通常利用**不活泼金属（如铜）与硝酸的反应**。根据硝酸浓度的不同，其还原产物也会有所区别。以下是实验室制取主要氮氧化物的详细方案：

一氧化氮（NO）的制备：在实验室里，还原硝酸是制备一氧化氮的最佳方法。通常使用**铜片与稀硝酸**反应，一氧化氮是主要的还原产物。

1. **化学方程式**：$3\text{Cu} + 8\text{HNO}_3(\text{稀}) = 3\text{Cu}(\text{NO}_3)_2 + 2\text{NO}\uparrow + 4\text{H}_2\text{O}$。

2. **实验装置**：
    - **发生装置**：选用**固+液不加热型**发生装置（如锥形瓶或具支试管配合分液漏斗）。
    - **收集方法**：只能使用**排水集气法**。
        - **原因**：NO的密度与空气接近，且在常温下极易与空气中的氧气反应生成红棕色的 $\text{NO}_2$ ($2\text{NO} + \text{O}_2 = 2\text{NO}_2$)。

3. 注意事项与改进：
    - **排尽空气**：由于装置内残留的空气会迅速将生成的NO氧化，实验开始前通常需要通入二氧化碳或氮气等惰性气体排尽装置内的空气，以观察到无色的NO气体。
    - **替代方法**：也可以利用亚硝酸钠（$\text{NaNO}_2$）与氯化亚铁（$\text{FeSO}_4$）溶液在稀硫酸中反应制得。

二氧化氮（$\text{NO}_2$）的制备：利用**铜片与浓硝酸**的反应制取。浓硝酸的氧化性极强，反应非常剧烈。

1. **化学方程式**：$\text{Cu} + 4\text{HNO}_3(\text{浓}) = \text{Cu}(\text{NO}_3)_2 + 2\text{NO}_2\uparrow + 2\text{H}_2\text{O}$。

2. 实验装置：
    - **发生装置**：同NO，选用**固+液不加热型**装置。
    - **收集方法**：只能使用**向上排空气法**。
    - **原因**：$\text{NO}_2$的密度比空气大，且极易溶于水并与水发生歧化反应生成硝酸和一氧化氮 ($3\text{NO}_2 + \text{H}_2O = 2\text{HNO}_3 + \text{NO}$)。

3. 注意事项：
    - **腐蚀性**：$\text{NO}_2$具有强氧化性和腐蚀性，制备时**严禁使用橡胶塞或乳胶管**连接仪器，因为它们会被氮氧化物腐蚀，应使用全玻璃连接的仪器（如曲颈甑）。
    - **替代方法**：加热重金属硝酸盐（如硝酸铜、硝酸银）也可以分解产生$\text{NO}_2$。

气体的净化、干燥与尾气处理：

1. 净化与除杂：若NO中混有$\text{NO}_2$杂质，可将混合气体通过**盛水的洗气瓶**，利用$\text{NO}_2$与水反应将其转化为NO除去。

2. 干燥方法：
    - **干燥剂选择**：NO和$\text{NO}_2$均可使用**五氧化二磷（$\text{P}_2\text{O}_5$）**、**浓硫酸**或**无水氯化钙**进行干燥。
    - **注意**：$\text{NO}_2$不能用碱性干燥剂（如碱石灰）干燥，因为它会与之发生反应。

3. 尾气处理（极重要）：氮氧化物均有毒，会引起大气污染、酸雨和光化学烟雾，必须进行尾气处理。
    - **碱液吸收法**：通常使用**氢氧化钠（NaOH）溶液**吸收。
        - 纯$\text{NO}_2$吸收：$2\text{NO}_2 + 2\text{NaOH} = \text{NaNO}_3 + \text{NaNO}_2 + \text{H}_2\text{O}$。
        - NO与$\text{NO}_2$混合吸收（归中反应）：$\text{NO} + \text{NO}_2 + 2\text{NaOH} = 2\text{NaNO}_2 + \text{H}_2\text{O}$。
    - **关键条件**：混合气体被完全吸收的条件是 $n(\text{NO}_2) \geq n(\text{NO})$。若NO过量，需补充空气将其氧化后再吸收。

其他氮氧化物：

- **一氧化二氮（$\text{N}_2\text{O}$，笑气）**：实验室中可通过**加热硝酸铵（$\text{NH}_4\text{NO}_3$）**制取，但需严格控制温度以防爆炸。
- **三氧化二氮（$\text{N}_2\text{O}_3$）**：它是亚硝酸的酸酐，可通过等物质的量的NO和$\text{NO}_2$混合并冷至 $-21^\circ\text{C}$ 以下得到深蓝色液体。

### 硝酸与硝酸盐

硝酸（$\text{HNO}_3$）是氮元素最高价（+5价）含氧酸，具有酸的通性，更以其突出的**不稳定性**和**强氧化性**著称。以下结合来源，对硝酸及其盐的性质、与金属反应的现象及规律进行详细展开。

1. **物理性质**：纯硝酸是**无色、易挥发、有刺激性气味**的液体。市售浓硝酸质量分数通常约为 68%～70%。质量分数在 95% 以上的硝酸在空气中挥发出蒸气，与水蒸气结合成微小液滴，产生“发烟”现象，称为**发烟硝酸**。
2. **酸的通性**：硝酸是一元强酸，在水中完全电离。它能使紫色石蕊试液变红（浓硝酸会使其先变红后褪色），并能与碱性氧化物、碱及弱酸盐发生复分解反应。

硝酸的化学特性：

1. **不稳定性**：硝酸见光或受热易分解，生成 $\text{NO}_2$、$\text{O}_2$ 和 $\text{H}_2\text{O}$。
    - **化学方程式**：$4\text{HNO}_3 \xlongequal{光照或\Delta} 4\text{NO}_2\uparrow + \text{O}_2\uparrow + 2\text{H}_2\text{O}$。
    - **现象**：分解生成的 $\text{NO}_2$ 溶于硝酸中，使**久置的浓硝酸显黄色**。实验室需用**棕色瓶**密封并置于阴凉处保存。
2. **强氧化性**：硝酸无论浓、稀均具有强氧化性，其氧化性随浓度增大而增强。
    - **非金属反应**：加热条件下，浓硝酸能氧化木炭、硫、磷等非金属单质至最高价态，自身通常被还原为红棕色的 $\text{NO}_2$。
        - 例（与木炭）：$\text{C} + 4\text{HNO}_3(浓) \xlongequal{\Delta} \text{CO}_2\uparrow + 4\text{NO}_2\uparrow + 2\text{H}_2\text{O}$。

硝酸与金属反应时，既表现出**强氧化性**又表现出**酸性**。其核心特征是：**反应通常不产生氢气**。

1. 还原产物的浓度规律：硝酸的还原产物十分复杂，主要取决于酸的浓度。
    - **浓硝酸**：还原产物主要为红棕色的 **$\text{NO}_2$**。
        - **Cu 与浓硝酸**：反应剧烈，产生红棕色气体，溶液变蓝。
    - **稀硝酸**：还原产物主要为无色的 **$\text{NO}$**。
        - **Cu 与稀硝酸**：反应较缓慢，产生无色气泡（在瓶口遇空气变为红棕色），溶液变蓝。
    - **极稀硝酸**：与活泼金属（如 Mg、Zn）反应时，产物可能是 $\text{N}_2\text{O}$、$\text{N}_2$ 甚至 $\text{NH}_4^+$。

2. 特殊的“钝化”现象：常温下，铁（Fe）、铝（Al）和铬（Cr）遇到**浓硝酸**（或浓硫酸）时，表面会生成一层**致密的氧化物保护膜**。
    - **实质**：这是一种化学变化，生成的膜阻止了内部金属与酸的进一步反应。
    - **应用**：常温下可用铁质或铝质容器盛装、运输浓硝酸。但在加热条件下，钝化膜会被破坏，反应将剧烈进行。

3. 与不同金属的反应性：
    - **不活泼金属**：Cu、Ag、Hg 等能溶于硝酸，产生相应的硝酸盐和氮氧化物。
    - **贵金属（Au、Pt）**：不溶于单一的硝酸，但可溶于**王水**（浓硝酸与浓盐酸体积比 $1:3$ 的混合液）。
    - **反应计算模型**：
        - **N 原子守恒**：$n(\text{HNO}_3)_{参加} = n(\text{还原产物}) + n(\text{NO}_3^-)_{盐}$。
        - **得失电子守恒**：金属失去电子总数等于硝酸中氮原子得到电子总数。

硝酸盐的性质与热分解规律：

1. **物理性质**：硝酸盐大多数为无色晶体，**均易溶于水**。
2. **热稳定性与分解规律**：硝酸盐在高温下不稳定，受热分解产物随金属阳离子的活动性不同而异：
    - **活泼金属（如 K、Na、Ca 之前）**：分解生成**亚硝酸盐和氧气**。
        - 例：$2\text{NaNO}_3 \xlongequal{\Delta} 2\text{NaNO}_2 + \text{O}_2\uparrow$。
    - **较活泼金属（如 Mg 到 Cu 之间）**：分解生成**金属氧化物、二氧化氮和氧气**。
        - 例：$2\text{Cu(NO}_3)_2 \xlongequal{\Delta} 2\text{CuO} + 4\text{NO}_2\uparrow + \text{O}_2\uparrow$。
    - **不活泼金属（如 Hg、Ag 之后）**：分解生成**金属单质、二氧化氮和氧气**。
        - 例：$2\text{AgNO}_3 \xlongequal{\Delta} 2\text{Ag} + 2\text{NO}_2\uparrow + \text{O}_2\uparrow$。
3. **氧化性表现**：固态硝酸盐受热是强氧化剂（如黑火药成分之一）。在水溶液中，$\text{NO}_3^-$ 本身几乎无氧化性，但若**加入强酸使溶液显酸性**，则表现出与硝酸类似的强氧化性。

### 亚硝酸与亚硝酸盐

**亚硝酸**（$\text{HNO}_2$）与**亚硝酸盐**（$\text{NO}_2^-$）是氮元素处于中间价态（**+3价**）的重要化合物，在化学合成、食品工业及医学领域均有广泛应用。

亚硝酸（$\text{HNO}_2$）：

1. 物理性质与稳定性
    - **存在形式**：亚硝酸是一种**弱酸**（$K_a \approx 5 \times 10^{-4}$），它**极不稳定**，仅存在于稀的水溶液中。
    - **不稳定性**：亚硝酸在冷水中即可发生**歧化反应**，生成硝酸、一氧化氮和水：$3\text{HNO}_2 = \text{HNO}_3 + 2\text{NO}\uparrow + \text{H}_2\text{O}$。微热时分解更剧烈。
    - **酸酐**：亚硝酸的酸酐是**三氧化二氮**（$\text{N}_2\text{O}_3$），一种在低温下呈深蓝色的液体。

2. 氧化性与还原性：亚硝酸中氮元素处于中间价态，既有氧化性又有还原性，但以**氧化性**为主。
    - **氧化性**：在酸性条件下，它能将碘离子（$\text{I}^-$）氧化为单质碘（用于检验）：$2\text{HNO}_2 + 2\text{I}^- + 2\text{H}^+ = \text{I}_2 + 2\text{NO}\uparrow + 2\text{H}_2\text{O}$。也能将 $\text{Fe}^{2+}$ 氧化为 $\text{Fe}^{3+}$。
    - **还原性**：遇到强氧化剂（如 $\text{KMnO}_4$、$\text{O}_2$）时表现出还原性，被氧化为硝酸。

3. 制备方法：
    - **混合气体法**：将 $\text{NO}$ 和 $\text{NO}_2$ 混合物溶解在接近零度的水中：$\text{NO} + \text{NO}_2 + \text{H}_2\text{O} = 2\text{HNO}_2$。
    - **酸化法**：向亚硝酸盐溶液中加入强酸：$\text{NaNO}_2 + \text{HCl} = \text{NaCl} + \text{HNO}_2$。

亚硝酸盐（以亚硝酸钠 $\text{NaNO}_2$ 为代表）：

1. 物理性质与结构
    - **外观**：亚硝酸钠为**无色或微黄色**晶体，外观与口感均与食盐（$\text{NaCl}$）相似，因此常发生误食中毒事故。
    - **溶解性**：易溶于水，水溶液呈**碱性**（因 $\text{NO}_2^-$ 水解）。
    - **微观结构**：$\text{NO}_2^-$ 离子中心氮原子采取 **$sp^2$ 杂化**，空间构型为 **V形**。

2. 化学性质
    - **热稳定性**：比亚硝酸稳定。碱金属的亚硝酸盐加热至熔融不分解。通常由活泼金属硝酸盐加热分解制得：$2\text{NaNO}_3 \xrightarrow{\Delta} 2\text{NaNO}_2 + \text{O}_2\uparrow$。
    - **氧化还原反应**：
        - **氧化性**：酸性条件下氧化 $\text{KI}$（淀粉变蓝）和 $\text{FeCl}_2$（溶液变黄）。
        - **还原性**：能使酸性 $\text{KMnO}_4$ 溶液或 $\text{K_2Cr_2O_7}$ 溶液褪色/变色：$\text{Cr}_2\text{O}_7^{2-} + 3\text{NO}_2^- + 8\text{H}^+ = 2\text{Cr}^{3+} + 3\text{NO}_3^- + 4\text{H}_2\text{O}$。

3. 鉴别 $\text{NaNO}_2$ 与 $\text{NaCl}$
    - **溶液酸碱性**：$\text{NaNO}_2$ 溶液呈碱性，$\text{NaCl}$ 呈中性。
    - **沉淀法**：加入 $\text{AgNO}_3$ 溶液，均产生沉淀，但 $\text{AgNO}_2$ 沉淀可溶于稀硝酸，$\text{AgCl}$ 不溶。
    - **氧化还原法**：加入酸化的 $\text{KI}$ 溶液，使淀粉变蓝的是 $\text{NaNO}_2$。

有机化学中的转化：

1. 重氮化反应（Diazotization）：这是芳香族合成的核心反应。在 $0 \sim 5^\circ\text{C}$ 下，**芳香族伯胺**与亚硝酸（由 $\text{NaNO}_2 + \text{HCl}$ 现场产生）反应生成**重氮盐**。重氮盐（$\text{Ar-N}_2^+$）是极佳的合成中间体，可转化为酚（水解）、卤代烃（Sandmeyer 反应）或偶氮染料。

2. 亚硝胺的形成：**仲胺**与亚硝酸反应生成 **$N$-亚硝基化合物**（亚硝胺，$R_2\text{N-NO}$），此类物质是世界公认的**强致癌物**。

毒性、用途与环境保护：亚硝酸盐有毒，它能将人体血红蛋白中的 **$\text{Fe}^{2+}$ 氧化为 $\text{Fe}^{3+}$**，使其失去与氧气结合的能力，导致组织缺氧（高铁血红蛋白症）。

临床解毒：

- **维生素 C**：作为强还原剂，可直接还原高铁血红蛋白，缓解中毒。
- **亚甲蓝**（美蓝）：作为电子传递体，在体内酶的作用下加速血红蛋白的还原。

用途：

- **食品添加剂**：作为抗氧化剂、防腐剂（抑制细菌生长）和增色剂（使肉类颜色鲜红），常用于腊肉、火腿等，但须严格控制用量。
- **工业用途**：用于织物染色（偶氮染料合成）、电镀及金属热处理剂。

尾气处理：工业生产硝酸排放的尾气（$\text{NO}_x$）可用碱液吸收产生亚硝酸钠，达到“变废为宝”的目的：$\text{NO} + \text{NO}_2 + 2\text{NaOH} = 2\text{NaNO}_2 + \text{H}_2\text{O}$。

### 金属氮化物及其性质

金属氮化物是由金属元素与氮元素形成的化合物。根据其成键性质和结构特征，金属氮化物主要分为**类盐氮化物**和**间充型氮化物**两大类。

类盐氮化物通常由活泼金属（如碱金属和碱土金属）与氮气在高温下直接反应制得。

1. **成键与结构**：
    - 它们是**离子化合物**，氮元素以 **$N^{3-}$（氮根离子）** 形式存在，表现出 **-3 价**。
    - **典型代表**：氮化锂（$Li_3N$）、氮化镁（$Mg_3N_2$）、氮化钙（$Ca_3N_2$）等。其中 **$Li_3N$** 是极少数能在室温下直接与氮气化合生成的产物。

2. **物理性质**：
    - 多为**固体**。例如，$Mg_3N_2$ 是**淡黄色**或黄绿色的固体。

3. **化学性质：剧烈的水解性**：
    - 类盐氮化物是**强碱**或强亲核试剂，遇水会发生剧烈且不可逆的**“双水解”反应**，生成对应的金属氢氧化物沉淀并放出**氨气（$NH_3$）**。
    - **反应方程式（以 $Mg_3N_2$ 为例）**： $Mg_3N_2 + 6H_2O = 3Mg(OH)_2\downarrow + 2NH_3\uparrow$。
    - **现象**：产生的氨气具有刺激性气味，能使**湿润的红色石蕊试纸变蓝**。这是检验氮气（通过生成氮化物再水解）的间接方法。
    - **与酸反应**：会生成两种盐（金属盐和铵盐）。 $Mg_3N_2 + 8HCl = 3MgCl_2 + 2NH_4Cl$。

间充型氮化物主要由**过渡金属**（如 Ti, V, Zr, Nb, W, Fe 等）形成。

1. **结构特征**：
    - 原子半径较小的氮原子填充在金属晶格的**间隙位置**中，称为金属间隙化合物。
    - 这种结构保留了金属键的性质，因此它们通常具有**金属光泽和良好的导电、导热性**。

2. **卓越的物理性能**：
    - **超高硬度**：其硬度通常极大，例如碳化钛（$TiC$）和某些氮化物的硬度等级可大于 9。
    - **极高熔点**：它们的熔点往往比原来的纯金属更高。例如，$TiN$ 等化合物具有高熔点、耐磨损和耐腐蚀的特性。

3. **化学稳定性**：
    - 与类盐氮化物不同，间充型氮化物通常**不发生水解反应**，化学性质非常稳定。

其他重要的金属氮化物：

1. **氮化铝 (AlN)**：
    - 属于**共价化合物（共价晶体）**。
    - **性质**：具有**高强度、高导热性**、优异的绝缘性和耐热冲击性。
    - **用途**：常用于制造熔铸纯铁、铝及其合金的理想**坩埚材料**，以及新型陶瓷材料。

2. **钢中的氮化物**：
    - 在冶金中，氮常与钢中的 Al, V, Nb, Ti 等元素结合，形成弥散分布的微小氮化物（如 $AlN, VN, TiN$），起到**细化晶粒、强化基体**的作用，从而提高钢的强度和韧性。

| 性质          | 类盐氮化物 (如 $Mg_3N_2$) | 间充型氮化物 (如 $TiN$) |
| :------------ | :------------------------ | :---------------------- |
| **主要金属**  | 活泼金属（I A, II A 族）  | 过渡金属                |
| **化学键**    | 离子键为主                | 金属键 + 部分共价键     |
| **水解性**    | **剧烈水解**，产生 $NH_3$ | **不水解**，性质稳定    |
| **硬度/熔点** | 相对较低                  | **极高**，属于超硬材料  |
| **导电性**    | 固体不导电                | **良好**，具金属特性    |

### 氨和铵盐及其实验室制法

氨和铵盐是氮循环中的重要物质，在工业生产（如化肥、硝酸、纯碱工业）和实验室研究中占有核心地位。以下结合提供的来源，对氨、铵盐及其制法进行详细展开：

- **结构**：氮原子采取 **$sp^3$ 杂化**，分子构型为 **三角锥形**，氮原子上有一对孤对电子，使其具有碱性和配位能力。
- **物理性质**：无色、有强烈 **刺激性气味** 的气体，密度比空气小。
- **溶解性**：**极易溶于水**（常温常压下 1:700），溶解原因包括氨是极性分子（相似相溶）、能与水分子形成 **氢键**，以及部分与水反应。
- **液化**：易液化，液氨汽化时 **吸收大量的热**，常用作 **制冷剂**。

化学性质：

- **弱碱性**：氨溶于水形成氨水，发生可逆反应：$NH_3 + H_2O \rightleftharpoons NH_3 \cdot H_2O \rightleftharpoons NH_4^+ + OH^-$。一水合氨是弱电解质。
- **与酸反应**：生成铵盐。与 **挥发性酸**（如浓盐酸、浓硝酸）靠近时会产生 **大量白烟**（$NH_4Cl$ 或 $NH_4NO_3$ 微晶），可用于检验氨气。
- **还原性**：
    - **催化氧化**：在催化剂（如铂丝）作用下与 $O_2$ 反应生成 **$NO$**，是工业制硝酸的基础：$4NH_3 + 5O_2 \xrightarrow[催化剂]{\Delta} 4NO + 6H_2O$。
    - **与氧化铜反应**：$2NH_3 + 3CuO \xrightarrow{\Delta} 3Cu + N_2 + 3H_2O$，可用于实验室制备纯净氮气。
- **配位性**：作为中等强度的单牙配体，能与 $Ag^+$、$Cu^{2+}$ 等形成稳定的配离子，如 $[Ag(NH_3)_2]^+$ 和 $[Cu(NH_3)_4]^{2+}$。

铵盐：

- 铵盐是由铵根离子（$NH_4^+$）和酸根离子构成的离子化合物。
- 绝大多数铵盐为 **无色晶体**，**均易溶于水**。

化学性质：

- **不稳定性（热解）**：铵盐受热易分解，但产物视酸根而定。
    - $NH_4Cl$ 分解生成 $NH_3$ 和 $HCl$，但在试管口遇冷会重新化合为 $NH_4Cl$ 固体，该过程属于化学变化而非升华。
    - $NH_4HCO_3$ 受热完全分解为 $NH_3$、$H_2O$ 和 $CO_2$。
    - **注意**：具有强氧化性酸根的铵盐（如 $NH_4NO_3$）受热可能发生氧化还原反应，甚至发生 **爆炸**，因此不能用于实验室加热制氨气。
- **碱解**：铵盐与强碱共热释放出氨气，这是实验室制取氨气及检验铵根离子的原理。
- **$NH_4^+$ 的检验**：取样，加入 **浓 $NaOH$ 溶液** 并 **加热**，将 **湿润的红色石蕊试纸** 置于试管口，若试纸 **变蓝**（或用蘸有浓盐酸的玻璃棒靠近，产生白烟），则证明含 $NH_4^+$。

氨气的实验室制法：

1. 经典制法：固固加热法
    - **反应原理**：$2NH_4Cl + Ca(OH)_2 \xrightarrow{\Delta} CaCl_2 + 2NH_3 \uparrow + 2H_2O$。
    - **发生装置**：使用“**固体 + 固体 $\xrightarrow{\Delta}$ 气体**”装置（与实验室制氧气类似）。试管口需 **略向下倾斜**，防止冷凝水回流炸裂试管。
    - **干燥方法**：使用 **碱石灰**（$NaOH$ 与 $CaO$ 混合物）。
        - **禁忌**：不能使用浓硫酸、$P_2O_5$ 等酸性干燥剂；不能使用 **无水 $CaCl_2$**，因为氨会与之结合形成配合物 $CaCl_2 \cdot 8NH_3$。
    - **收集方法**：**向下排空气法**（氨气密度小于空气）。导管应伸入试管底部。
    - **验满**：将湿润的红色石蕊试纸放在管口，变蓝即满；或用蘸有浓盐酸的玻璃棒靠近，产生白烟。
    - **尾气处理**：在试管口塞一团 **润湿的棉花**，可减少空气对流并吸收多余氨气，防止污染。

2. 快速制备法（简易法）
    - **加热浓氨水**：利用一水合氨的不稳定性，$NH_3 \cdot H_2O \xrightarrow{\Delta} NH_3 \uparrow + H_2O$。
    - **浓氨水 + 固体物质**：
        - 向 **$NaOH$ 固体** 或 **生石灰（$CaO$）** 中滴加浓氨水。
        - **原理**：$NaOH$ 或 $CaO$ 遇水放热，且增加 $OH^-$ 浓度，促使氨水的电离及分解平衡向生成 $NH_3$ 的方向移动。

氨的典型实验——喷泉实验：

- **原理**：利用氨气极易溶于水的性质，使烧瓶内外产生 **压强差**，在大气压作用下液体快速进入烧瓶形成喷泉。
- **现象**：若烧杯中水含有酚酞，喷泉呈现 **红色**，证明氨气极易溶于水且溶液显碱性。

### 叠氮、重氮、联氨及其化合物

氮（N）原子因其多样的成键方式和氧化态（从 -3 到 +5），能形成一系列结构复杂的链状或簇状含氮化合物。**叠氮、重氮和联氨（肼）**是这类化学中的核心成员，在能源材料、有机合成和工业生产中扮演着关键角色。

联氨（肼，$\text{N}_2\text{H}_4$）：联氨可以看作是氨（$\text{NH}_3$）分子中的一个氢原子被氨基（$-\text{NH}_2$）取代的衍生物，分子中含有 $\text{N}-\text{N}$ 单键。

- **分子结构与物理性质**：
    - **结构**：氮原子采取 **$sp^3$ 杂化**。分子结构呈不对称的交叉构型，具有极强的极性。
    - **性质**：联氨是一种无色、有刺激性气味的油状液体，在空气中发烟，极毒且具有强腐蚀性。
- **化学性质**：
    - **弱碱性**：联氨是 **二元弱碱**，碱性弱于氨，在水中发生二级电离：$\text{N}_2\text{H}_4 + \text{H}_2\text{O} \rightleftharpoons \text{N}_2\text{H}_5^+ + \text{OH}^-$。
    - **强还原性**：能被卤素、$\text{H}_2\text{O}_2$、$\text{AgNO}_3$ 等氧化生成氮气。例如与 $H_2O_2$ 的反应：$\text{N}_2\text{H}_4 + 2\text{H}_2\text{O}_2 = \text{N}_2 \uparrow + 4\text{H}_2\text{O}$。
- **制备与应用**：
    - **制备**：常用 $\text{NaClO}$ 溶液氧化过量的 $\text{NH}_3$（拉西法）：$2\text{NH}_3 + \text{ClO}^- = \text{N}_2\text{H}_4 + \text{Cl}^- + \text{H}_2\text{O}$。也可利用尿素与 $\text{NaClO}$ 和 $\text{NaOH}$ 反应制取。
    - **用途**：主要用作 **喷气式发动机和火箭的高能燃料**（氧化剂常为 $\text{N}_2\text{O}_4$ 或 $\text{H}_2\text{O}_2$），也用于制药和有机合成试剂。

重氮化合物（Diazonium Compounds）：重氮化合物是一类含有重氮基（$-\text{N}\equiv\text{N}^+$ 或 $-\text{N}_2$）的物质，通常由芳香族伯胺在酸性条件下与亚硝酸钠反应制得，此过程称为 **重氮化反应**。

- **反应条件与机理**：
    - **低温控制**：反应必须在 **$0 \sim 5^\circ\text{C}$** 冰水浴中进行。温度过高会导致重氮盐分解生成酚并放出氮气。
    - **强酸过量**：酸的作用是活化亚硝酸产生活性亲电试剂 $\text{NO}^+$，并抑制生成的重氮盐与未反应的胺发生偶联副反应。
- **结构与稳定性**：
    - **芳香族重氮盐**：相对稳定，因为氮原子与苯环之间存在共轭作用，其结构中的两个氮原子均处于 **$sp$ 杂化** 状态，体系呈直线型。
    - **脂肪族重氮化合物**：如 **重氮甲烷（$\text{CH}_2\text{N}_2$）**，极不稳定且具有爆炸性，但在有机合成中是重要的甲基化试剂和扩环反应试剂。
- **合成“中转站”作用**：
    - **取代反应**：重氮基是“超级离去基团”。通过 **Sandmeyer 反应**（用 $\text{CuCl/CuBr/CuCN}$ 催化）可引入 $\text{Cl/Br/CN}$ 基团；通过 **Schiemann 反应**（用 $\text{HBF}_4$）引入 $\text{F}$ 原子。
    - **还原反应**：加入次磷酸（$\text{H}_3\text{PO}_2$）可除去重氮基，将 $-\text{NH}_2$ 还原为 $-\text{H}$，这在多取代苯的合成中常用于“占位后切除”策略。
    - **偶联反应**：重氮盐与酚或芳胺反应生成 **偶氮化合物**（$-\text{N=N}-$），是制造偶氮染料的基础。

叠氮化合物（Azides）：叠氮化合物含有叠氮根离子（$\text{N}_3^-$）或基团，其核心是 **叠氮酸（$\text{HN}_3$）**。

- **叠氮酸与无机叠氮盐**：
    - **叠氮酸**：无色、易爆炸的液体，具有强烈的细胞质毒性。它是一种弱酸（$K_a \approx 1.8 \times 10^{-5}$），氧化性与硝酸相似。
    - **热分解特性**：叠氮盐（如 $\text{NaN}_3$）受热剧烈分解产生大量氮气和金属单质。
        - **应用**：**汽车安全气囊** 利用叠氮化钠在碰撞时瞬间分解产生氮气来充气：$2\text{NaN}_3 \xrightarrow{\Delta} 2\text{Na} + 3\text{N}_2 \uparrow$。
- **有机叠氮化合物**：
    - **合成用途**：在有机化学中，叠氮根离子（$\text{N}_3^-$）是优秀的亲核试剂，常用于合成伯胺。
    - **转化**：烷基叠氮化物（$\text{RN}_3$）通过氢化铝锂（$\text{LiAlH}_4$）或催化氢化还原，可以高产率地转化为伯胺（$\text{RNH}_2$）。

拓展：超高能含氮物质。

- **全氮阴离子（$\text{N}_5^-$）**：近年来，南京理工大学团队成功合成了室温下稳定存在的全氮阴离子盐。由于其完全由氮原子组成，分解产物仅为氮气，具有极高的能量密度，被称为“**超级炸药**”，威力可达 $\text{TNT}$ 的数倍。

### 氢氰酸、氰酸与异氰酸及其化合物

氢氰酸（HCN）、氰酸（HOCN）与异氰酸（HNCO）是氮族元素中极具代表性的含碳、氮化合物。它们虽然分子式简单，但在分子结构、酸性强度以及工业应用上存在显著差异。

这三者在结构上呈现出独特的化学键特征：

- **氢氰酸 (HCN)**：呈直线形分子结构（$\text{H—C}\equiv\text{N}$），碳原子采取 **$sp$ 杂化**。
- **异氰酸 (HNCO)**：其结构为 $\text{H—N=C=O}$，具有累积双键结构，是三者中含氧化合物形式最稳定的。
- **氰酸 (HOCN)**：结构为 $\text{H—O—C}\equiv\text{N}$。
- **互变异构关系**：氰酸与异氰酸是一对**互变异构体**。在室温下，平衡绝对偏向于**异氰酸（HNCO）**。实际上，常温下可以分离和操作的游离酸基本都是异氰酸，而游离的氰酸（HOCN）极不稳定，仅能在极低温度下检测到。

| 性质           | 氢氰酸 (HCN)                            | 异氰酸 (HNCO)                        | 氰酸 (HOCN)                       |
| :------------- | :-------------------------------------- | :----------------------------------- | :-------------------------------- |
| **外观与气味** | 无色液体/气体，具**苦杏仁味**           | 无色液体，具强烈**刺鼻气味**         | 挥发性液体                        |
| **沸点**       | 25.6℃~25.7℃                             | 23.5℃~23.6℃                          | ~23℃                              |
| **酸性 (pKa)** | **极弱酸** ($pK_a \approx 9.2\sim 9.3$) | **中强酸/弱酸** ($pK_a \approx 3.7$) | 强酸 ($pK_a \approx -0.5$ 计算值) |
| **稳定性**     | 相对稳定，碱性杂质可诱导其聚合          | **极不稳定**，易自发聚合成三聚氰酸   | 极不稳定，水中快速水解            |
| **毒性**       | **剧毒**，锁死细胞色素氧化酶导致窒息    | 高毒，具强烈刺激性                   | 有毒                              |

氢氰酸及其化合物的应用：

1. 有机合成：氢氰酸是有机化学中重要的“一碳试剂”：
    - **生成氰醇（羟基腈）**：醛或酮与 HCN 发生亲核加成反应生成 $\alpha$-羟基腈。此反应通常在弱碱性条件下进行，因为真正的亲核试剂是 **$\text{CN}^-$**。
    - **水解反应**：氰基（$\text{-CN}$）水解是合成 **$\alpha$-羟基酸**（如乳酸）的核心步骤。其机理遵循 **“腈 $\rightarrow$ 酰胺 $\rightarrow$ 羧酸”** 的路径。
    - **Strecker 氨基酸合成**：通过醛、氨和 HCN 反应生成 $\alpha$-氨基腈，随后水解得到 **$\alpha$-氨基酸**。

2. 无机化学与工业应用：
    - **黄金冶炼（氰化法）**：利用 $\text{CN}^-$ 极强的配位能力，在氧气存在下将金溶解为可溶性的 $[Au(CN)_2]^-$。
    - **金属配合物**：形成极其稳定的络合物，如**亚铁氰化钾（黄血盐）** $\text{K}_4[Fe(CN)_6]$ 和**铁氰化钾（赤血盐）** $\text{K}_3[Fe(CN)_6]$。
    - **工业原料**：大量用于生产丙烯腈（合成纤维）、己二腈（尼龙-6,6前体）和甲基丙烯酸甲酯（有机玻璃）。

氰酸/异氰酸及其化合物的应用：

1. 异氰酸酯与聚合物：
    - **聚氨酯 (PU) 工业**：有机异氰酸酯（如 **TDI**、**MDI**）是制造海绵、涂料和泡沫塑料的核心单体。
    - **碳酰胺化反应**：异氰酸与醇反应生成**氨基甲酸酯**，与胺反应生成**脲**（尿素衍生物）。

2. 化学史上的里程碑：
    - **维勒合成 (Wöhler synthesis)**：1828年维勒通过加热无机盐**氰酸铵**（实为异氰酸铵）合成了有机物**尿素**，打破了“生命力论”。

3. 拟卤素性质：
    - 氰酸根（$\text{OCN}^-$）属于**拟卤素离子**，其性质与卤素离子相似，可形成难溶的银盐 $\text{AgOCN}$。

安全与环境保护：

- **无害化处理**：工业上常用**氯氧化法**处理含氰废水，过量氯气可将 $\text{CN}^-$ 氧化为无毒的 $\text{CO}_2$ 和 $\text{N}_2$。
- **泄漏应急**：HCN 挥发性极强且具瞬时致死性（“内窒息”），发生泄漏时需极速撤离。氢氟酸（HF）虽也剧毒，但其致死主要通过干扰钙、镁离子引发心脏毒性，且伴随严重的局部骨骼溶解痛感。

### 氮的卤化物及卤氧化物

氮的卤化物及卤氧化物是氮族元素化学中具有独特成键特征和反应机理的化合物。由于氮原子处于第二周期，其价层轨道的局限性（无 $d$ 轨道）以及电负性的特殊性，使得这些化合物表现出与同族其他元素（如磷）截然不同的性质。

氮的常见卤化物主要包括三氟化氮（$\text{NF}_3$）和三氯化氮（$\text{NCl}_3$）。

1. 三氯化氮 ($\text{NCl}_3$)：三氯化氮是一种非常经典且容易引起直觉误判的分子。
    - **化合价悖论与判定**：
        - 根据鲍林电负性，$\chi(\text{Cl}) = 3.16$，$\chi(\text{N}) = 3.04$，氯的电负性略大于氮。若机械判定，氮应显 +3 价，氯显 -1 价。
        - 但在化学界，最终认定 **$\text{N}$ 为 -3 价，$\text{Cl}$ 为 +1 价**。其核心依据是其 **水解产物** 为氨气（$\text{NH}_3$）和次氯酸（$\text{HOCl}$）：$\text{NCl}_3 + 3\text{H}_2\text{O} \longrightarrow \text{NH}_3 + 3\text{HClO}$。
    - **独特的亲核水解机理**：
        - **常规卤化物（如 $\text{PCl}_3$）**：水分子（亲核试剂）进攻中心原子（磷），生成 $\text{H}_3\text{PO}_3$ 和 $\text{HCl}$。
        - **三氯化氮 ($\text{NCl}_3$)**：水分子进攻 **外围的氯原子**。这是因为氮原子半径极小，且自身拥有一对孤对电子，产生强烈的空间位阻和静电排斥，水分子的氧原子无法靠近氮。相反，氯原子位于第三周期，拥有 **价层空 $3d$ 轨道**（或易被进攻的 $\text{N-Cl}$ $\sigma^*$ 反键轨道），更容易接受外来电子云。
    - **物理性质**：$\text{NCl}_3$ 是一种极其不稳定的油状液体，受热或潮湿环境易分解甚至爆炸。

2. 三氟化氮 ($\text{NF}_3$)：
    - **性质与应用**：$\text{NF}_3$ 是微电子工业中优良的 **等离子刻蚀气体**。
    - **化学行为**：它在潮湿环境中能发生反应生成一氧化氮和硝酸：$3\text{NF}_3 + 5\text{H}_2\text{O} = 2\text{NO} + \text{HNO}_3 + 9\text{HF}$。在该反应中，$\text{NF}_3$ 表现为氧化剂。

氮的卤氧化物主要分为亚硝酰卤（$\text{NOX}$，氮为 +3 价）和硝酰卤（$\text{NO}_2\text{X}$，氮为 +5 价）。

1. 亚硝酰卤 (Nitrosyl Halides, $\text{NOX}$)：
    - **代表物质**：亚硝酰氯（$\text{NOCl}$）。
    - **制备方法**：
        - 实验室通常由一氧化氮与氯气在催化剂作用下反应制得：$2\text{NO} + \text{Cl}_2 \xrightarrow{\text{催化剂}} 2\text{NOCl}$。
        - 王水（浓硝酸与浓盐酸混合液）在起作用时也会产生 $\text{NOCl}$ 和 $\text{Cl}_2$。
    - **物理性质**：$\text{NOCl}$ 的沸点约为 $-6^\circ\text{C}$，在常温下为气体。
    - **应用**：$\text{NOCl}$ 是重要的有机合成试剂，在酸性条件下可作为亚硝酰正离子（$\text{NO}^+$）的载体，参与重氮化等反应。

2. 硝酰卤 (Nitryl Halides, $\text{NO}_2\text{X}$)：
    - **代表物质**：硝酰氟（$\text{FNO}_2$）、硝酰氯（$\text{ClNO}_2$）。
    - **反应特性**：
        - **自由基加成**：硝酰氯（$\text{NO}_2\text{Cl}$）可以加成到烯烃上，生成 $\beta$-卤代硝基化合物，这通常是一个自由基反应过程。
        - **亲电加成**：在特定条件下，$\text{FNO}_2$ 或 $\text{NO}_2\text{Cl}$ 也可以加成到三键上，得到硝基氯代烯烃。

关键离子与结构分析：

- **亚硝酰正离子 ($\text{NO}^+$)**：
    - 它是重氮化反应中的真正亲电试剂。
    - 其结构与 $\text{CO}$、$\text{N}_2$ 互为 **等电子体**，拥有一个 $\sigma$ 键、两个 $\pi$ 键及空的反键 $\pi^*$ 轨道。
- **硝酰正离子 ($\text{NO}_2^+$)**：
    - 存在于固态五氧化二氮中（固态 $\text{N}_2\text{O}_5$ 为离子晶体 $[\text{NO}_2]^+[\text{NO}_3]^-$）。
    - 它是强烈的硝化试剂，广泛应用于芳香烃的硝化反应中。
- **成键限制**：由于氮原子位于第二周期，只有 $2s$ 和 $2p$ 共 4 个价轨道，因此在形成的共价化合物中，氮原子的 **最大成键数（共价键数）只能为 4**。例如在 $\text{NO}_3^-$ 或 $\text{N}_2\text{O}_5$ 中，氮的氧化数虽然是 +5，但周围只有 4 个共价键。

### 其他非金属氮化物

除了前述提及的金属氮化物，氮还可以与硼、硅、磷、硫等非金属或类金属元素形成多种具有独特性质的**共价型氮化物**。这些化合物通常具有极高的硬度、热稳定性和特殊的电学性质，是现代材料科学中的重要研究对象。

1. 氮化硼 (Boron Nitride, BN)：氮化硼是由硼和氮组成的共价网状固体，因其结构与碳的同素异形体极其相似，常被称为“类碳材料”。
    - **同素异形体与结构：**
        - **六方氮化硼 (h-BN)：** 俗称**“白石墨”**。其结构与石墨类似，为层状结构，具有良好的润滑性、介电性和耐热性（可耐 2000℃ 高温），常用作高温润滑剂和脱模剂。
        - **立方氮化硼 (c-BN)：** 具有**钻石（金刚石）结构**。其硬度极高，略小于钻石，但热稳定性优于钻石，是工业上极其重要的磨料和钻石替代品。
    - **物理与化学性质：** BN 昇华点高达 2973℃，化学性质极不活泼，不溶于水，也不与盐酸、氢氟酸反应。
    - **制备方法：** 工业上常用**硼砂与氨气或尿素**在氮气气氛中高温反应制得。实验室中也可由三氧化二硼与氨气反应生成：$B_2O_3 + 2NH_3 \xrightarrow{1000^\circ C} 2BN + 3H_2O$。
    - **用途：** 用于制造特种耐火材料、高温坩埚、绝缘材料以及高级研磨抛光剂。此外，**氮化硼纳米管**（BN nanotube）也是一种具有高迁移率的一维管状材料。

2. 氮化硅 (Silicon Nitride, $Si_3N_4$)：氮化硅是一种高性能的新型无机非金属材料（工程陶瓷），以其卓越的机械强度和稳定性著称。
    - **物理性质：** 灰白色固体，属于**共价晶体**。它具有**超硬、耐磨、耐高温**（熔点 1900℃）且**自润滑**的特性。
    - **化学性质：** 抗腐蚀能力极强，除氢氟酸外，不与其他无机酸反应。在浓强酸或浓强碱中会缓慢水解，但在高温下能抗氧化且抗热震。
    - **制备方法：**
        - 硅粉与氮气在 1300～1400℃ 下直接反应：$3Si + 2N_2 \xrightarrow{\Delta} Si_3N_4$。
        - 硅与氨气在 1500℃ 下反应。
    - **用途：** 广泛用于制造**火箭发动机尾管、燃气轮机叶片**、耐高温轴承、切削刀具及机械密封环等受力构件。

3. 硼氮苯 (Borazine, $B_3N_3H_6$)：硼氮苯与苯（$C_6H_6$）是等电子体，分子结构对称且呈平面六角形，因此俗称**“无机苯”**。
    - **结构特点：** 分子中 B 和 N 原子交替排列，均采取 **$sp^2$ 杂化**，存在类似苯环的大 $\pi$ 键。
    - **性质：** 属于分子晶体，其熔沸点取决于分子间作用力而非化学键。尽管结构相似，但化学活性高于苯，能与 HCl 等发生加成反应。
    - **制备：** 可由乙硼烷与氨气加成反应制得：$3B_2H_6 + 6NH_3 \rightarrow 2B_3N_3H_6 + 12H_2$。

4. 硫的氮化物 (Sulfur Nitrides)：硫与氮可形成多种结构复杂的化合物，其中最具代表性的是四氮化四硫和聚氮化硫。
    - **四氮化四硫 ($S_4N_4$)：** 分子呈非平面的八元环状结构。由于硫原子利用 3d 轨道参与成键，环内存在较强的离域 $\pi$ 键。
    - **聚氮化硫 $(SN)_x$：** 呈金黄色并具有金属光泽。它是单双键交替排列的 $\pi$ 共轭体系，表现出良好的导电性，在 0.26 K 低温下可转变为**超导体**，被视为一种“一维金属”。

5. 磷的氮化物 (Phosphorus Nitrides)：
    - **$P_3N_5$：** 是一种固体氮化物，通常被归类为两性或偏酸性的氮化物。在高温高压下，它与 $Si_3N_4$ 一样，表现出较高的热稳定性。

### 有机物中的氮与胺

在有机化学中，氮元素是构成生命物质（如蛋白质和核酸）以及合成药物、染料和炸药的核心元素。含氮有机化合物种类繁多，其中最基础且重要的类别包括**胺、酰胺、氨基酸、硝基化合物及重氮盐**等。

胺可以看作是**氨（$\text{NH}_3$）分子中的氢原子被烃基取代**后的衍生物。

1. 结构与分类：
    - **按取代数目分类**：
        - **一级胺（伯胺）**：$\text{R-NH}_2$，如甲胺、苯胺。
        - **二级胺（仲胺）**：$\text{R}_2\text{NH}$，如二甲胺、四氢吡咯。
        - **三级胺（叔胺）**：$\text{R}_3\text{N}$，如三甲胺。
        - **四级铵盐（季铵盐）**：$\text{R}_4\text{N}^+\text{X}^-$，氮原子带正电荷。
    - **按烃基类型分类**：分为**脂肪胺**（如乙胺）和**芳香胺**（如苯胺）。

2. 系统命名法：
    - **主链选择**：以与氮原子相连的最长碳链为主链，称为“某胺”。
    - **取代基标注**：对于二级和三级胺，将较短的烃基视作取代基，并在其名称前加上字母 **“N-”**，以表示该基团直接连在氮原子上。例如，$\text{HCONHCH}_3$ 称为 **N-甲基甲酰胺**。

3. 物理性质：
    - **气味**：低级胺通常具有强烈的**鱼腥味**或类似氨的气味，腐胺和尸胺具有难闻的腐败臭味。
    - **氢键与沸点**：一级胺和二级胺分子间能形成**氢键**，因此其沸点高于分子量相近的烷烃，但低于相应的醇。
    - **溶解性**：低级胺（碳数小于5）易溶于水，因为能与水分子形成氢键。

4. 化学性质：
    - **碱性**：由于氮原子上有一对**孤对电子**，胺能接受质子（$\text{H}^+$），水溶液显弱碱性。其碱性强弱顺序通常为：**二级胺 > 一级胺 > 三级胺 > 氨 > 苯胺**（受溶剂化效应和诱导效应共同影响）。
    - **成盐反应**：胺能与酸反应生成可溶于水的铵盐。医药上常利用此性质将难溶的胺类药物制成盐酸盐（如麻黄碱盐酸盐），以增强其水溶性和稳定性。
    - **亲核性**：胺是良好的亲核试剂，可发生**烷基化反应**。但由于生成的产物（如二级胺）往往比原料更亲核，容易导致多重烷基化，最终生成季铵盐。

酰胺是羧酸分子中羟基被氨基或烃氨基取代得到的化合物，结构通式为 $\text{R-CONR'R''}$。

1. 结构特征与杂化：
    - **共轭体系**：酰胺中氮上的孤对电子与羰基的 $\pi$ 轨道发生**离域共轭**（$\text{p-}\pi$ 共轭）。
    - **杂化状态**：为了实现最大程度的轨道重叠，酰胺中的氮原子通常采取 **$\text{sp}^2$ 杂化**，分子呈现平面结构。
    - **C-N键特征**：由于共轭作用，C-N键具有**部分双键性质**，旋转受阻（如DMF的旋转能垒约为 88 kJ/mol）。

2. 性质对比：
    - **碱性极弱**：受羰基强吸电子效应的影响，氮上的电子云密度降低，酰胺几乎呈**中性**，不能与酸成盐。
    - **稳定性与水解**：酰胺在强酸或强碱存在下加热可发生**不可逆水解**，生成羧酸（盐）和铵盐（或氨）。
    - **用途**：液态酰胺（如 **DMF**，N,N-二甲基甲酰胺）是优良的极性非质子溶剂。

氨基酸既含有酸性的**羧基**（$-\text{COOH}$）又含有碱性的**氨基**（$-\text{NH}_2$）。

1. **两性与内盐**：氨基酸在水中会发生分子内质子转移，形成**两性离子（内盐）**。
2. **等电点 (pI)**：当溶液处于某一特定 pH 时，氨基酸净电荷为零，此时溶解度最小，容易析出晶体。
3. **成肽反应**：一分子氨基酸的羧基与另一分子的氨基脱水缩合形成**肽键（酰胺键）**，多个氨基酸连接形成多肽和蛋白质。
4. **显色反应**：$\alpha$-氨基酸遇**茚三酮**溶液并加热会产生紫色物质，可用于氨基酸的定性和定量分析。

硝基化合物与重氮盐：

1. 硝基化合物 (R-NO2)：
    - **电子效应**：硝基是极强的**吸电子基团**，会强烈钝化苯环，且属于间位定位基。
    - **还原转化**：硝基最核心的用途是通过还原反应生成**一级胺**。常用方法包括催化加氢（$\text{H}_2/\text{Pd}$）或金属/酸还原（如 $\text{Fe/HCl}$）。

2. 重氮盐 (Ar-N2+)：
    - **形成**：芳香族一级胺在低温（$0 \sim 5^\circ\text{C}$）下与亚硝酸（$\text{NaNO}_2 + \text{HCl}$）反应生成重氮盐，此过程称为**重氮化反应**。
    - **合成中转站**：重氮基是一个“超级离去基团”，可以通过不同的反应转化为多种官能团：
        - **Sandmeyer反应**：引入 $-\text{Cl}, -\text{Br}, -\text{CN}$。
        - **Schiemann反应**：引入 $-\text{F}$。
        - **水解反应**：转化为酚（$-\text{OH}$）。
        - **脱氨基还原**：用次磷酸（$\text{H}_3\text{PO}_2$）还原为 $-\text{H}$。

| 特征         | 胺 ($\text{R-NH}_2$)                | 酰胺 ($\text{R-CONH}_2$) | 硝基化合物 ($\text{R-NO}_2$)         |
| :----------- | :---------------------------------- | :----------------------- | :----------------------------------- |
| **氮杂化**   | $\text{sp}^3$（趋向 $\text{sp}^2$） | $\text{sp}^2$            | ——                                   |
| **酸碱性**   | 弱碱性                              | 中性/极弱碱性            | 强吸电子性，$\alpha\text{-H}$ 具酸性 |
| **水溶性**   | 低级胺易溶                          | 低级酰胺易溶             | 难溶于水，密度大                     |
| **典型反应** | 与酸成盐、烷基化                    | 酸/碱催化水解            | 还原为胺                             |

## 氮的相关题型

### 工业合成氨

工业合成氨（哈伯-博施法，Haber-Bosch process）是人类历史上最伟大的化学成就之一，它实现了人工固氮，为化肥工业奠定了基础，解决了地球上因粮食不足导致的饥饿问题。

合成氨的反应方程式为：

$$
\mathrm{N_{2}(g) + 3H_{2}(g) \rightleftharpoons 2NH_{3}(g) \quad \Delta H = -92.4\ kJ \cdot mol^{-1}}
$$

该反应具有以下特点：

1. **可逆性**：在工业条件下，反应物无法完全转化为产物。
2. **放热反应 ($\Delta H < 0$)**：降低温度有利于提高平衡转化率。
3. **气体分子数减小的反应**：根据勒夏特列原理，增大压强有利于平衡向生成氨的方向移动。
4. **活化能极高**：氮气分子内含有氮氮三键（键能高达 $946\text{ kJ/mol}$），极难断裂，因此在常温下反应速率近乎为零。

合成氨反应的关键在于使用**催化剂**。

- **催化剂成分**：目前工业上普遍使用以**铁**为主体的多成分催化剂，称为**铁触媒**（主要成分为 $\mathrm{Fe_2O_3}$ 和 $\mathrm{FeO}$，并加入 $\mathrm{Al_2O_3, K_2O, CaO}$ 等助剂以增强活性）。
- **表面化学历程**：德国科学家格哈德·埃特尔（Gerhard Ertl）证实，反应是在催化剂表面进行的非均相催化过程，包括吸附、解离、表面反应、脱附和扩散五个步骤。
- **限速步骤**：**氮分子的吸附解离**（将 $\mathrm{N_2}$ 拆解为原子氮）活化能最高，是控制总反应速率的关键步骤。

工业生产需综合平衡“反应速率”与“平衡限度”的矛盾，并考虑设备成本。

1. **温度 (400~500 ℃)**：从平衡角度看需要低温，但从速率角度看需要高温。选择该温度是因为它是**铁触媒活性最大**的温度区间。
2. **压强 (10~30 MPa)**：理论上压强越大越好，但压强过高对设备材料（防氢脆、高强度钢材）和动力要求极高，目前多采用中压法。
3. **投料比 ($n(\mathrm{N_2}):n(\mathrm{H_2}) \approx 1:2.8$)**：虽然理论计量比为 $1:3$，但为保证氮气在催化剂表面占有足够份额以提高吸附速率，通常适当提高氮气比例。
4. **及时分离产物**：采用迅速冷却法将气态氨液化分离，可打破平衡限制，促使反应持续正向进行。

合成氨工业通常分为造气、净化、合成和分离四个主要阶段。

1. **造气 (Gas Production)**：
    - **氮气来源**：来自空气（液化空气分馏或通过化学方法除去氧气）。
    - **氢气来源**：以水和煤、天然气等为原料。例如天然气重整：$\mathrm{CH_4 + H_2O \xrightarrow{\text{高温}} CO + 3H_2}$，以及水煤气变换：$\mathrm{CO + H_2O \xrightarrow{\text{催化剂}} CO_2 + H_2}$。
2. **净化 (Purification)**：除去原料气中的 $\mathrm{CO, CO_2, H_2S}$ 和水蒸气。这些杂质会使铁触媒“**中毒**”失效或腐蚀设备。
3. **合成 (Synthesis)**：在**合成氨塔**中进行。原料气经过**热交换器**预热（利用反应放出的热量，节约能源）后进入催化剂层反应。
4. **分离与循环 (Separation & Recycling)**：反应后的混合气（含氨约 10%~20%）进入氨分离器，液化分离出液氨。**未反应的氮氢混合气**经循环压缩机再次压入合成塔，实现原料的循环利用，大幅提高原子利用率。

合成氨不仅直接产出液氨和氨水（氮肥），还与许多工业过程紧密相连：

- **联合制碱法（侯氏制碱法）**：中国科学家**侯德榜**将合成氨工业与制碱工业联合，利用合成氨产生的废气 $\mathrm{CO_2}$ 制取纯碱和氯化铵，极大地提高了资源利用率。
- **制硝酸**：氨经催化氧化生成 $\mathrm{NO}$，是工业制硝酸的基础。
- **制尿素**：氨与二氧化碳在高压下反应可制备尿素 $\mathrm{CO(NH_2)_2}$，这是目前含氮量最高的优质固态氮肥。

### 硝酸的工业制备

现代工业生产硝酸普遍采用的是**奥斯特瓦尔德法（Ostwald process）**，其核心原理是利用合成氨工业提供的氨气进行**催化氧化**。这一工艺通常分为氨的催化氧化、一氧化氮的氧化以及二氧化氮的吸收三个主要阶段。

1. 氨的催化氧化（核心阶段）：氨气与过量的空气（提供 $O_2$）混合后，进入装有催化剂的氧化炉。
    - **催化剂**：通常使用**铂铑合金网（$Pt\text{-}Rh$）**。
    - **反应条件**：温度通常控制在 **800℃～900℃**。
    - **化学方程式**： $$4\mathrm{NH}_3(g) + 5\mathrm{O}_2(g) \xrightarrow[800^\circ\text{C}\sim 900^\circ\text{C}]{\text{催化剂}} 4\mathrm{NO}(g) + 6\mathrm{H}_2\mathrm{O}(g) \quad \Delta H = -906\text{ kJ/mol} \quad$$
    - **工艺特点**：该反应是剧烈的放热反应，反应产生的热量除维持催化剂温度外，常通过热交换器预热原料气或进入余热锅炉产生水蒸气以节约能源。

2. 一氧化氮的氧化：从氧化炉出来的混合气体经过冷却，一氧化氮与过量的氧气进一步反应生成二氧化氮。
    - **化学方程式**： $$2\mathrm{NO}(g) + \mathrm{O}_2(g) \rightleftharpoons 2\mathrm{NO}_2(g) \quad \Delta H = -114\text{ kJ/mol} \quad$$
    - **工艺控制**：由于此反应是放热且体积缩小的，降低温度有利于平衡向生成 $\mathrm{NO}_2$ 的方向移动。

3. 二氧化氮的吸收：将产生的二氧化氮通入**吸收塔**，利用水或稀硝酸进行逆流吸收。
    - **化学方程式**： $$3\mathrm{NO}_2(g) + \mathrm{H}_2\mathrm{O}(l) = 2\mathrm{HNO}_3(aq) + \mathrm{NO}(g) \quad$$
    - **循环利用**：生成的 $\mathrm{NO}$ 会被补充的空气再次氧化为 $\mathrm{NO}_2$ 并循环吸收，整个过程的总反应可表示为： $$4\mathrm{NO}_2(g) + \mathrm{O}_2(g) + 2\mathrm{H}_2\mathrm{O}(l) = 4\mathrm{HNO}_3(aq) \quad$$
    - **吸收设备**：吸收塔内部通常填充瓷环以增大接触面积，采用**逆流原理**（酸液自上而下，气体自下而上）以实现充分吸收。

通过上述吸收法直接制得的硝酸通常为**稀硝酸**（质量分数约 50%～68%）。若要生产 **98% 以上的浓硝酸**，需采用以下方法：

1. **萃取精馏法**：在稀硝酸中加入**脱水剂**（如浓硫酸或高浓度的 $\mathrm{Mg(NO_3)_2}$ 溶液），这些物质能强烈夺取体系中的水分，打破硝酸与水的共沸平衡，从而精馏出高纯硝酸蒸气。
2. **加压合成法**：在约 5.0 MPa 的压强下进行二氧化氮的吸收，利用压力促进平衡移动，提高产得硝酸的浓度。

硝酸工业产生的尾气中含有少量未被吸收的 $\mathrm{NO}$ 和 $\mathrm{NO}_2$（统称 $\mathrm{NO}_x$），直接排放会造成酸雨和光化学烟雾。常用处理方法包括：

- **碱液吸收法**：使用 $\mathrm{NaOH}$ 或 $\mathrm{Na_2CO_3}$ 溶液吸收，生成亚硝酸钠和硝酸钠，实现变废为宝。 $$\mathrm{NO} + \mathrm{NO}_2 + 2\mathrm{NaOH} = 2\mathrm{NaNO}_2 + \mathrm{H}_2\mathrm{O} \quad$$
- **催化还原法**：在催化剂作用下，利用 $\mathrm{NH}_3$ 将 $\mathrm{NO}_x$ 还原为无毒的 $\mathrm{N}_2$。 $$6\mathrm{NO} + 4\mathrm{NH}_3 \xrightarrow{\text{催化剂}} 5\mathrm{N}_2 + 6\mathrm{H}_2\mathrm{O} \quad$$

在合成氨工业成熟之前，历史上曾使用**钠硝石与浓硫酸共热**的方法制取硝酸。

- **原理**：利用高沸点酸制低沸点酸。
- **方程式**：$\mathrm{NaNO}_3(s) + \mathrm{H}_2\mathrm{SO}_4(\text{浓}) \xrightarrow{\Delta} \mathrm{NaHSO}_4 + \mathrm{HNO}_3\uparrow$。
- **评价**：该法成本较高且受原料矿藏限制，现已被奥斯特瓦尔德法全面取代。

### 氮及其化合物的应用、氮的固定、氮在自然界中的循环、废气中氮氧化物的处理

氮及其化合物在现代工业、农业、国防及生命科学中发挥着核心作用，同时在自然界维持着动态的平衡。以下结合提供的来源，对氮及其化合物的应用、氮的固定、氮在自然界中的循环以及废气中氮氧化物的处理进行详细展开讲解。

氮及其化合物的应用极其广泛，涵盖了从基础民生到尖端科技的各个领域：

1. **氮气 ($N_2$) 的应用**：
    - **保护气**：利用氮气化学性质极不活泼的特点，在工业上常用于填充灯泡以防止钨丝氧化，或在焊接金属时作为保护气，防止金属在高温下被氧化。此外，它还广泛用于食品包装以防腐保鲜。
    - **冷冻剂**：液氮沸点极低（-196℃），常用于医学冷冻手术（冷冻麻醉）以及使某些超导材料获得超导性能。
    - **工业原料**：是合成氨、制取硝酸、氮肥、炸药、合成纤维及合成树脂的重要基础原料。

2. **氨 ($NH_3$) 及铵盐的应用**：
    - **化肥工业**：氨是化肥工业的支柱，约 90% 的氨用于生产各种氮肥。常见的铵态氮肥包括硫酸铵、硝酸铵、氯化铵及碳酸氢铵。
    - **制冷剂**：由于液氨汽化时会吸收大量的热，使周围温度急剧降低，因此在工业上常被用作制冷剂。
    - **其他工业**：作为制造硝酸、纯碱及有机合成（如合成纤维、染料）的原料。

3. **硝酸 ($HNO_3$) 的应用**：
    - **国防与工业**：硝酸是“三酸二碱”之一，主要用于制造炸药（如三硝基甲苯 TNT、硝化甘油）、染料、塑料及硝酸盐。
    - **分析与蚀刻**：在实验室中作为重要试剂，并在工业上用于金属蚀刻和矿石浮选。

4. **其他化合物的应用**：
    - **尿素 [$CO(NH_2)_2$]**：目前使用量最大、含氮量最高（约 46.7%）的固态有机氮肥，性质稳定且属于中性肥料。
    - **叠氮化钠 ($NaN_3$)**：主要用于**汽车安全气囊**，在撞击瞬间剧烈分解产生大量氮气使气囊弹出。
    - **联氨 ($N_2H_4$)**：及其衍生物常用作火箭及喷气式发动机的高能燃料。

**氮的固定**是指将大气中游离态的氮转化为氮的化合物的过程。

1. **自然固氮**：
    - **生物固氮**：指自然界中某些微生物（如豆科植物根部的**根瘤菌**）在固氮酶作用下，将空气中的氮气转化为氨的过程。这种方式约占自然固氮总量的 90%。
    - **高能固氮（大气固氮）**：雷电天气时，大气中的氮气在放电或高温条件下与氧气反应生成一氧化氮，随后经一系列转化最终形成硝酸随雨水降落到土壤中（即“**雷雨发庄稼**”）。其反应链为：$N_2 + O_2 \xrightarrow{放电} 2NO \rightarrow 2NO_2 \xrightarrow{H_2O} 2HNO_3 + NO$。

2. **人工固氮**：
    - **工业合成氨**：目前最主要的人工固氮途径，由哈伯和博施发明。反应原理为：$N_2(g) + 3H_2(g) \rightleftharpoons 2NH_3(g) \quad \Delta H = -92.4\text{ kJ/mol}$。
    - **仿生固氮**：科学家正在研究利用金属有机化合物作为催化剂，模拟根瘤菌的功能，实现在常温、常压下固定空气中的氮。

氮元素在地球的大气圈、岩石圈和生物圈中不停地循环着，其主要历程如下：

1. **进入生物圈**：大气中的氮气通过固氮作用（生物固氮和雷电固氮）转化为化合态氮（氨或硝酸盐）进入土壤和水体。
2. **同化作用**：植物根系从土壤中吸收 $NH_4^+$ 和 $NO_3^-$，通过生物转化将其合成**蛋白质**和核酸等有机物。动物通过摄食植物获得植物蛋白并转化为动物蛋白。
3. **氨化作用与分解**：动植物的排泄物和残骸在土壤中经微生物分解，有机氮重新转化为 $NH_4^+$、 $NO_3^-$ 和 $NH_3$，再次被植物吸收利用。
4. **脱氮作用 (Denitrification)**：在土壤氧气不足的情况下，**反硝化细菌**将硝酸根还原为氮气返回大气，从而完成一个完整的循环。

氮氧化物是主要的大气污染物，会导致**酸雨**（硝酸型）、**光化学烟雾**以及破坏臭氧层。

1. **碱液吸收法**：
    - 适用于处理工业尾气（如硝酸生产尾气）。
    - 利用 $NaOH$ 或 $Na_2CO_3$ 溶液吸收 $NO$ 和 $NO_2$ 的混合物，通过**归中反应**生成亚硝酸钠：$NO + NO_2 + 2NaOH = 2NaNO_2 + H_2O$。
    - 完全吸收的条件是：$n(NO_2) \geq n(NO)$；若 $NO$ 过量，则需通入空气氧化后再吸收。

2. **催化转化法**：
    - **汽车尾气处理**：在汽车排气管上安装“催化转换器”，利用 $CO$ 的还原性将 $NO_x$ 转化为无毒的 $N_2$ 和 $CO_2$。反应式为：$2CO + 2NO \xrightarrow{催化剂} N_2 + 2CO_2$。
    - **氨还原法**：在催化剂作用下，通入适量 $NH_3$ 将 $NO_x$ 还原为 $N_2$。例如：$6NO + 4NH_3 \xrightarrow{\Delta, 催化剂} 5N_2 + 6H_2O$。

3. **其他治理措施**：
    - 调整能源结构，发展清洁能源（如电动汽车、氢能）。
    - 改进燃烧技术，控制化石燃料燃烧时氮氧化物的产生量。

### 氮及其化合物的相互转化

氮及其化合物的相互转化是元素化学的核心内容，通常可以通过**“价—类”二维图**（即化合价与物质类别）来系统梳理。氮元素拥有从 -3 到 +5 的多种化合价，构成了复杂的转化网络。

氮气分子内含有极牢固的**共价三键**（$N \equiv N$），能量高达 946 kJ/mol，这使得氮气在常温下化学性质极其稳定。**氮的固定**是指将大气中游离态的氮转化为含氮化合物的过程。

1. **自然固氮**：
    - **高能固氮（大气固氮）**：雷电条件下，氮气与氧气直接化合生成无色的一氧化氮：$N_2 + O_2 \xlongequal{\text{放电或高温}} 2NO$。**注意，该反应不能直接生成 $NO_2$**。
    - **生物固氮**：豆科植物根部的根瘤菌通过固氮酶在常温常压下将 $N_2$ 转化为氨或铵盐。
2. **人工固氮（工业合成氨）**：
    - 在高温、高压和催化剂（铁触媒）条件下，氮气与氢气发生可逆反应：$N_2 + 3H_2 \rightleftharpoons 2NH_3$。
3. **其他转化**：
    - 氮气与活泼金属（如镁、锂）反应生成金属氮化物：$N_2 + 3Mg \xlongequal{\text{点燃}} Mg_3N_2$（黄绿色固体）。

氨（$NH_3$）与铵盐（$NH_4^+$）的相互转化：这是氮元素在 -3 价态下的物质类别转化，通常不涉及氧化还原。

1. **$NH_3 \rightarrow NH_4^+$（生成铵盐）**：
    - 氨气遇酸（如盐酸、硝酸、醋酸）生成铵盐，反应现象通常为产生**白烟**：$NH_3 + HCl = NH_4Cl$。
    - 氨气溶于水形成一水合氨，存在电离平衡：$NH_3 + H_2O \rightleftharpoons NH_3 \cdot H_2O \rightleftharpoons NH_4^+ + OH^-$。
2. **$NH_4^+ \rightarrow NH_3$（释放氨气）**：
    - **铵盐热分解**：不同铵盐分解产物不同。氯化铵受热分解生成 $NH_3$ 和 $HCl$，但在试管口遇冷又重新化合：$NH_4Cl \xrightarrow{\Delta} NH_3 \uparrow + HCl \uparrow$。**不能利用 $NH_4Cl$ 直接加热制备纯净氨气**。
    - **实验室制法**：铵盐与强碱（如消石灰）共热：$2NH_4Cl + Ca(OH)_2 \xrightarrow{\Delta} CaCl_2 + 2NH_3 \uparrow + 2H_2O$。
    - **快速制氨**：向浓氨水中加入 $NaOH$ 固体或生石灰（$CaO$），利用平衡移动和放热促使氨气逸出。

工业制硝酸的“连续氧化”链：工业生产硝酸的过程体现了氮元素从 -3 价向 +5 价阶梯式升高的转化。

1. **氨的催化氧化**：氨气在铂铑合金网催化下与氧气反应生成一氧化氮：$4NH_3 + 5O_2 \xlongequal[\Delta]{\text{催化剂}} 4NO + 6H_2O$。这是工业制硝酸的基础。
2. **一氧化氮氧化**：$NO$ 在空气中迅速与氧气化合生成红棕色 $NO_2$：$2NO + O_2 = 2NO_2$。
3. **二氧化氮吸收**：$NO_2$ 溶于水发生**歧化反应**：$3NO_2 + H_2O = 2HNO_3 + NO$。
4. **循环与净化**：生成的 $NO$ 经循环氧气进一步转化为硝酸。

硝酸（$HNO_3$）的还原转化：硝酸具有强氧化性，与金属或非金属反应时，根据浓度不同，还原产物各异。

1. **与不活泼金属（如铜、银）反应**：
    - **浓硝酸**还原产物主要是 **$NO_2$**：$Cu + 4HNO_3(\text{浓}) = Cu(NO_3)_2 + 2NO_2 \uparrow + 2H_2O$。
    - **稀硝酸**还原产物主要是 **$NO$**：$3Cu + 8HNO_3(\text{稀}) = 3Cu(NO_3)_2 + 2NO \uparrow + 4H_2O$。
2. **与活泼金属（如镁、锌、铁）反应**：
    - 硝酸浓度越低，还原产物价态越低，可能生成 $N_2O$、$N_2$ 甚至 **$NH_4^+$**。
3. **与非金属反应**：
    - 热的浓硝酸能将木炭、硫、磷氧化至最高价态，自身还原为 $NO_2$：$C + 4HNO_3(\text{浓}) \xrightarrow{\Delta} CO_2 \uparrow + 4NO_2 \uparrow + 2H_2O$。
4. **钝化现象**：常温下，铁、铝遇浓硝酸表面形成致密氧化膜而**钝化**，阻止进一步反应。

氮氧化物在处理尾气和环境化学中表现出多样的转化。

1. **归中反应（尾气处理）**：
    - **氨还原法**：利用氨气的还原性将 $NO_x$ 转化为无害的 $N_2$：$6NO + 4NH_3 \xrightarrow{\text{催化剂}} 5N_2 + 6H_2O$ 或 $6NO_2 + 8NH_3 \xrightarrow{\text{催化剂}} 7N_2 + 12H_2O$。
    - **汽车尾气催化转化**：$2NO + 2CO \xlongequal{\text{催化剂}} N_2 + 2CO_2$。
2. **歧化反应（碱液吸收）**：
    - $NO_2$ 遇碱溶液歧化生成硝酸盐和亚硝酸盐：$2NO_2 + 2NaOH = NaNO_3 + NaNO_2 + H_2O$。
    - 若 $NO$ 与 $NO_2$ 按 1:1 比例被吸收，则发生归中式歧化生成纯亚硝酸盐：$NO_2 + NO + 2NaOH = 2NaNO_2 + H_2O$。

亚硝酸（$HNO_2$）及其盐的转化：

1. **亚硝酸的不稳定性**：亚硝酸仅存在于稀溶液中，受热易歧化：$3HNO_2 = HNO_3 + 2NO \uparrow + H_2O$。
2. **氧化性与还原性**：
    - **氧化性**（主）：在酸性条件下将 $I^-$ 氧化为 $I_2$：$2NO_2^- + 2I^- + 4H^+ = 2NO \uparrow + I_2 + 2H_2O$。
    - **还原性**：遇强氧化剂（如酸性高锰酸钾）转化为硝酸根：$5NO_2^- + 2MnO_4^- + 6H^+ = 5NO_3^- + 2Mn^{2+} + 3H_2O$。

有机氮化合物的转化：在有机合成中，硝基与氨基及其衍生物的相互转化非常关键。

1. **硝基 $\rightarrow$ 氨基（还原）**：芳香族硝基化合物（如硝基苯）在铁/盐酸或催化加氢条件下还原为苯胺。
2. **氨基 $\rightarrow$ 重氮盐（重氮化）**：芳香族伯胺在 $0 \sim 5^\circ C$ 及强酸条件下，与亚硝酸钠反应生成不稳定的重氮盐（$Ar-N_2^+$）。
3. **重氮盐的衍生化（合成中转站）**：
    - **Sandmeyer 反应**：引入 $-Cl$、$-Br$ 或 $-CN$。
    - **Schiemann 反应**：引入 $-F$。
    - **水解**：转化为酚（$-OH$）。
    - **脱氨基还原**：用次磷酸还原为 $-H$。
4. **尿素（$CO(NH_2)_2$）的转化**：
    - 工业合成：$NH_3 + CO_2 \xrightarrow{\text{高压}} CO(NH_2)_2 + H_2O$。
    - 水解：在微生物或酸碱作用下，尿素水解产生氨气和二氧化碳。

### 以金属与硝酸反应产生的气体、转移的电子数及产生的盐、产生的盐与其他物质后续反应为核心的计算

以金属与硝酸（$\text{HNO}_3$）反应为核心的计算是化学中的重点与难点，其核心在于理解硝酸的**双重性质**（酸性与强氧化性）以及遵循的**三大守恒规律**（得失电子守恒、原子守恒、电荷守恒）。

硝酸具有极强的氧化性，其还原产物随硝酸浓度和金属活泼性的不同而表现出复杂性。

1. **还原产物的规律**：
    - **浓硝酸**（$12 \sim 16 \text{ mol/L}$）：还原产物主要是红棕色的 **$\text{NO}_2$**。
    - **稀硝酸**（$6 \sim 8 \text{ mol/L}$）：还原产物主要是无色的 **$\text{NO}$**。
    - **极稀硝酸**：还原产物可能进一步降至低价态，如 **$\text{N}_2\text{O}$**、**$\text{N}_2$** 甚至 **$\text{NH}_4^+$**。金属越活泼，硝酸越稀，还原产物中氮元素的价态通常越低。
    - **注意**：硝酸与金属反应一般**不产生 $\text{H}_2$**。

2. **得失电子守恒计算**：计算的核心公式为：**金属失去电子总数 = 氮原子得到电子总数**。
    - $\text{N}(+5) \rightarrow \text{NO}_2(+4)$：得到 $1e^-$。
    - $\text{N}(+5) \rightarrow \text{NO}(+2)$：得到 $3e^-$。
    - $\text{N}(+5) \rightarrow \text{NH}_4^+(-3)$：得到 $8e^-$。
    - **混合气体计算**：若产生 $\text{NO}$ 和 $\text{NO}_2$ 混合气体，可设 $\text{NO}$ 为 $x \text{ mol}$，$\text{NO}_2$ 为 $y \text{ mol}$，建立方程组：① $x + y = n(\text{总气体})$ ② $3x + y = n(e^-)_{\text{转移}}$。

在计算中，硝酸的作用通常分为两部分：**起氧化剂作用**（转化为还原产物气体）和**起酸性作用**（转化为硝酸盐中的 $\text{NO}_3^-$）。

1. **原子守恒模型**： **$n(\text{HNO}_3)_{\text{消耗}} = n(\text{氧化性}) + n(\text{酸性})$**。
    - $n(\text{氧化性}) = n(\text{氮原子在气体产物中})$。
    - $n(\text{酸性}) = n(\text{盐中 } \text{NO}_3^-) = n(\text{金属离子}) \times \text{金属化合价}$。

2. **铁与硝酸反应的特殊性（量变引起质变）**：铁与硝酸反应生成的盐取决于两者的物质的量之比：
    - **硝酸过量**（或铁不足）：生成 $\text{Fe}^{3+}$。离子方程式为 $\text{Fe} + 4\text{H}^+ + \text{NO}_3^- = \text{Fe}^{3+} + \text{NO}\uparrow + 2\text{H}_2\text{O}$。
    - **铁过量**（或硝酸不足）：生成 $\text{Fe}^{2+}$。反应分为两阶段：先生成 $\text{Fe}^{3+}$，过量的铁再将 $\text{Fe}^{3+}$ 还原为 $\text{Fe}^{2+}$。总反应为 $3\text{Fe} + 8\text{H}^+ + 2\text{NO}_3^- = 3\text{Fe}^{2+} + 2\text{NO}\uparrow + 4\text{H}_2\text{O}$。
    - **计算技巧**：可以通过比较 $n(\text{Fe})$ 与 $n(\text{HNO}_3)$ 的比例来快速判断产物。若比例介于两者之间，则产物为 $\text{Fe}^{2+}$ 与 $\text{Fe}^{3+}$ 的混合盐。

生成的硝酸盐溶液在特定条件下会发生进一步反应，这常作为综合计算题的后续环节。

1. **隐性硝酸的再反应**：
    - 当反应停止（如铜过量、硝酸耗尽）后，向溶液中加入**稀硫酸**，由于引入了 $\text{H}^+$，溶液中的 $\text{NO}_3^-$ 在酸性环境下恢复强氧化性，可继续氧化剩余的金属。
    - 计算时应使用离子方程式：$3\text{Cu} + 8\text{H}^+ + 2\text{NO}_3^- = 3\text{Cu}^{2+} + 2\text{NO}\uparrow + 4\text{H}_2\text{O}$。

2. **沉淀反应与滴定计算**：
    - **加碱沉淀**：硝酸盐与 $\text{NaOH}$ 或氨水反应产生沉淀（如 $\text{Fe(OH)}_3$、$\text{Cu(OH)}_2$）。计算时遵循元素守恒，如 $n(\text{Fe}) = n(\text{Fe(OH)}_3)$。
    - **氧化还原滴定**：生成的 $\text{Fe}^{2+}$ 盐常用 $\text{K}_2\text{Cr}_2\text{O}_7$ 或 $\text{KMnO}_4$ 滴定。关系式：$6\text{Fe}^{2+} \sim \text{Cr}_2\text{O}_7^{2-}$ 或 $5\text{Fe}^{2+} \sim \text{MnO}_4^-$。

3. **电荷守恒在后续溶液中的应用**：反应后的混合溶液中，如果不考虑 $\text{OH}^-$（酸性环境），阴、阳离子电荷平衡： **$c(\text{NO}_3^-) = c(\text{H}^+) + \sum n \cdot c(\text{M}^{n+})$**。这在计算未反应完的 $\text{H}^+$ 或剩余硝酸根浓度时极为高效。

典型计算模型总结：

- **金属 + 混合酸（如 $\text{H}_2\text{SO}_4 + \text{KNO}_3$）**：必须使用**离子方程式**进行计算，判断哪种离子优先耗尽（通常是 $\text{H}^+$）。
- **连续反应计算**：对于“金属 $\xrightarrow{\text{硝酸}}$ 盐 $\xrightarrow{\text{还原}}$ 金属”的多步过程，可跳过中间步骤，直接利用**起始物与最终产物的原子守恒和得失电子守恒**快速求解。例如，泡铜（$\text{Cu}_2\text{O, Cu}$）溶于硝酸生成 $\text{Cu(NO}_3)_2$，再被还原为 $\text{Cu}$，整个过程中铜原子数恒定。

### 以氮氧化物与二氧化氮与水的反应为核心的物质的量和气体计算

以**二氧化氮（$\text{NO}_2$）与水**的反应为核心的计算，是中学化学中研究氮氧化物（$\text{NO}_x$）性质、大气污染治理以及工业制硝酸的重要内容。这类计算通常结合**阿伏加德罗定律**、**氧化还原电子守恒**和**原子守恒**，具有较强的逻辑性。以下是详细的展开讲解：

二氧化氮与水发生歧化反应，氮元素的化合价从 +4 价一部分升高到 +5 价（生成 $\text{HNO}_3$），一部分降低到 +2 价（生成 $\text{NO}$）。

1. **反应方程式**： $$3\text{NO}_2 + \text{H}_2\text{O} = 2\text{HNO}_3 + \text{NO}$$
2. **物质的量与体积关系**：
    - **转化比**：每消耗 $3\text{ mol}$ 的 $\text{NO}_2$，生成 $1\text{ mol}$ 的 $\text{NO}$ 气体。
    - **体积变化**：在同温同压下，若反应前 $\text{NO}_2$ 为 $V\text{ L}$，反应后剩余气体的体积（即 $\text{NO}$）为 $\frac{1}{3}V\text{ L}$。
    - **液面上升**：在试管倒立于水槽的实验中，液体会进入试管并上升至试管高度的 **2/3** 处。
3. **电子转移计算**：
    - 该反应中，$\text{NO}_2$ 既是氧化剂又是还原剂。
    - $3\text{ mol}$ 的 $\text{NO}_2$ 反应时，有 $2\text{ mol}$ 氮原子失去电子（+4 $\rightarrow$ +5），$1\text{ mol}$ 氮原子得到电子（+4 $\rightarrow$ +2），总共转移 **$2\text{ mol}$ 电子**。

当混合气体中存在氧气时，生成的 $\text{NO}$ 会被进一步氧化为 $\text{NO}_2$ 并继续与水反应，最终可能将氮氧化物完全转化为硝酸。

1. $\text{NO}_2$ 与 $\text{O}_2$ 混合溶于水：
    - **总反应式**：$4\text{NO}_2 + \text{O}_2 + 2\text{H}_2\text{O} = 4\text{HNO}_3$。
    - **体积比判断**：
        - 若 $V(\text{NO}_2) : V(\text{O}_2) = 4 : 1$：两者**恰好完全反应**，若为喷泉实验则液体充满整个容器。
        - 若 $V(\text{NO}_2) : V(\text{O}_2) > 4 : 1$：$\text{NO}_2$ 过量，剩余气体为 **$\text{NO}$**（注意：过量的 $\text{NO}_2$ 会继续与水反应生成 $\text{NO}$）。
        - 若 $V(\text{NO}_2) : V(\text{O}_2) < 4 : 1$：$\text{O}_2$ 过量，剩余气体为 **$\text{O}_2$**。

2. $\text{NO}$ 与 $\text{O}_2$ 混合溶于水：
    - **总反应式**：$4\text{NO} + 3\text{O}_2 + 2\text{H}_2\text{O} = 4\text{HNO}_3$。
    - **体积比判断**：
        - 若 $V(\text{NO}) : V(\text{O}_2) = 4 : 3$：两者**恰好完全反应**。
        - 若 $V(\text{NO}) : V(\text{O}_2) > 4 : 3$：$\text{NO}$ 过量，剩余气体为 **$\text{NO}$**。
        - 若 $V(\text{NO}) : V(\text{O}_2) < 4 : 3$：$\text{O}_2$ 过量，剩余气体为 **$\text{O}_2$**。

3. $\text{NO}$、$\text{NO}_2$ 与 $\text{O}_2$ 三者混合：先根据 $3\text{NO}_2 + \text{H}_2\text{O} = 2\text{HNO}_3 + \text{NO}$ 将所有 $\text{NO}_2$ 虚拟转化为 $\text{NO}$，再加上原有的 $\text{NO}$ 得到总的 $\text{NO}$ 体积，最后利用 $4 : 3$ 的比例与 $\text{O}_2$ 进行比较计算。

在微观粒子数计算中，$\text{NO}_2$ 存在一个极重要的隐含平衡：**$2\text{NO}_2 \rightleftharpoons \text{N}_2\text{O}_4$**。

1. **分子数问题**：由于该平衡的存在，标准状况下 $22.4\text{ L}$ 的 $\text{NO}_2$ 所含的分子总数**小于 $N_A$**，因为部分 $\text{NO}_2$ 聚合成了 $\text{N}_2\text{O}_4$。
2. **原子数问题（守恒思想）**：虽然分子数不确定，但由于 $\text{NO}_2$ 和 $\text{N}_2\text{O}_4$ 的最简式相同（均为 $\text{NO}_2$），因此 **$46\text{ g}$ 的混合气体中所含有的原子总数恒定为 $3N_A$**（$1\text{ mol}$ 氮原子和 $2\text{ mol}$ 氧原子）。
3. **密度与平均相对分子质量**：混合气体的平均相对分子质量介于 46（$\text{NO}_2$）与 92（$\text{N}_2\text{O}_4$）之间。通过测定混合气体的密度，可以计算出其离解度 $\alpha$ 和各组分的分压。

在复杂的氮氧化物反应计算中（如金属与硝酸反应产生混合气体后再通入水），应优先使用守恒法：

1. **电子守恒（得失电子守恒）**：
    - 金属失电子总数 = 氮氧化物中氮原子得电子总数。
    - 当混合气体再通入 $\text{O}_2$ 并被水完全吸收时，实质是**金属失去的电子全部交给了 $\text{O}_2$**，即：$n(\text{金属}) \times \text{化合价} = n(\text{O}_2) \times 4$。
2. **氮原子守恒**：
    - 在工业制硝酸或循环吸收实验中，氮原子的总量在气体、硝酸和硝酸盐中保持平衡。
    - 例如：$n(\text{参加反应的 } \text{HNO}_3) = n(\text{气相中的 } \text{N}) + n(\text{盐中的 } \text{NO}_3^-)$。
3. **状态判定**：
    - 注意标准状况下 $\text{NO}_2$ 并非理想气体（其沸点约 $21^\circ\text{C}$，常温下易液化），在严谨的计算中需考虑其存在形式。

若标准状况下容积为 $V\text{ L}$ 的烧瓶充满 $\text{NO}_2$，做喷泉实验：

- **溶质的物质的量**：根据 $3\text{NO}_2 \sim 2\text{HNO}_3$，可知 $n(\text{HNO}_3) = \frac{2}{3} n(\text{NO}_2) = \frac{2}{3} \times \frac{V}{22.4} \text{ mol}$。
- **溶液的体积**：由于剩余 $1/3$ 的 $\text{NO}$ 气体，进入烧瓶的液体体积为 $\frac{2}{3} V\text{ L}$。
- **物质的量浓度**：$c = \frac{n}{V_{溶液}} = \frac{\frac{2}{3} \times \frac{V}{22.4} \text{ mol}}{\frac{2}{3} V\text{ L}} = \mathbf{\frac{1}{22.4} \text{ mol/L}}$。
- **结论**：充满 $\text{NH}_3$、$\text{HCl}$ 或 $\text{NO}_2$ 的烧瓶完成喷泉实验后（不考虑溶质扩散），其溶液的物质的量浓度在标准状况下**均为 $\frac{1}{22.4} \text{ mol/L}$**。

## 磷及其化合物

### 磷单质及其同素异形体

磷（Phosphorus, P）位列元素周期表第三周期、第 VA 族（氮族元素），原子序数为 15，其基态原子的价电子排布式为 **$3s^23p^3$**。磷是一种活泼的非金属元素，在自然界中主要以磷酸盐的形式（如磷灰石）存在，而不以游离态存在。磷具有多种同素异形体，其结构与性质的差异极其显著。

磷的常见同素异形体包括**白磷、红磷和黑磷**。

1. 白磷（White Phosphorus，又称黄磷）：
    - **分子结构**：白磷由 **$P_4$ 分子**构成，呈**正四面体**构型。四个磷原子位于正四面体的顶点，每个磷原子与另外三个磷原子以共价单键结合。
    - **成键特征**：$P_4$ 分子中的 $P-P-P$ 键角为 **$60^\circ$**。由于纯 $p$ 轨道的自然夹角应为 $90^\circ$，这使得 $P-P$ 键承受了巨大的张力（弯曲键），键能较低（约 $200\text{ kJ/mol}$），因此白磷在常温下表现出**极高的化学活性**。
    - **物理性质**：白磷是白色或淡黄色蜡状固体，有特殊的臭味，**剧毒**（误食 0.11g 即可致死）。它不溶于水，微溶于乙醇，**易溶于二硫化碳（$CS_2$）**。

2. 红磷（Red Phosphorus，又称赤磷）：
    - **结构特征**：红磷的结构至今尚无定论，一般认为它是 $P_4$ 四面体断裂一个 $P-P$ 键后，相互连接形成的**链状或网状巨大分子（聚合物）**。
    - **性质对比**：红磷呈红褐色粉末状，**无毒**，不溶于水及 $CS_2$。其化学性质比白磷稳定得多，不发磷光。

3. 黑磷（Black Phosphorus）：
    - **结构特征**：黑磷是磷**最稳定的同素异形体**，具有类似石墨的**片层结构**。层内每个磷原子与其他三个磷原子形成共价键，构成二维蜂巢型结构。
    - **物理特性**：黑磷具有金属光泽，能够导电，因此又称“金属磷”。

4. 相互转化关系：磷的各同素异形体在一定条件下可以相互转化：
    - **白磷转红磷**：白磷在隔绝空气的情况下加热至 **$260^\circ\text{C}$** 或光照，可转变为红磷。
    - **红磷转白磷**：红磷隔绝空气加热到 **$416^\circ\text{C}$** 升华，蒸气迅速冷却后可冷凝为白磷。
    - **黑磷的制备**：通常由白磷在高压（如 $1.2 \times 10^{11}\text{ Pa}$）下加热制得。

磷的化学性质活泼，特别是白磷。磷主要表现出**强还原性**，其最高正价为 +5，最低负价为 -3。

1. 与非金属的反应：
    - **与氧气反应**：磷极易与氧化合。
        - **自燃**：白磷的着火点很低（约 **$40^\circ\text{C}$**），在空气中缓慢氧化产生的热量若不能散发，极易引发自燃。
        - **燃烧**：磷在充足的氧气或空气中燃烧，产生浓厚的**白烟**（$P_4O_{10}$ 固体微粒），化学方程式为：$4P + 5O_2 \xlongequal{\text{点燃}} 2P_2O_5$（或 $P_4O_{10}$）。在氧气不足时，会生成 $P_2O_3$（$P_4O_6$）。
    - **与卤素反应**：磷在氯气中可以自发燃烧。
        - 氯气不足：生成三氯化磷（$PCl_3$），现象为**白雾**。
        - 氯气过量：生成五氯化磷（$PCl_5$），现象为**白烟**。
    - **与氢气反应**：磷与氢气很难直接化合，需通过间接方法制得**磷化氢（$PH_3$）**。

2. 与金属反应：磷几乎能与所有金属反应，生成相应的**金属磷化物**。例如与镁反应：

    $$
    P_4 + 6Mg \xrightarrow{\Delta} 2Mg_3P_2
    $$

    类盐磷化物（如 $Ca_3P_2$、$AlP$）遇水或酸会发生剧烈水解，产生剧毒且易自燃的 $PH_3$ 气体。

3. 与强碱的歧化反应：白磷在热的浓碱溶液中会发生**自身氧化还原（歧化）反应**，生成磷化氢气体和次磷酸盐：

    $$
    P_4 + 3NaOH + 3H_2O \xlongequal{\Delta} PH_3\uparrow + 3NaH_2PO_2
    $$

4. 还原性与强氧化性酸反应：磷能被硝酸等强氧化性酸氧化。

    例如，磷与稀硝酸反应生成磷酸和一氧化氮：

    $$
    3P + 5HNO_3 + 2H_2O = 3H_3PO_4 + 5NO\uparrow
    $$

5. 与金属盐溶液的反应：磷具有强还原性，能从某些金属盐（如 $Ag^+$、$Cu^{2+}$）溶液中置换出金属。

    **解毒反应**：**硫酸铜溶液**是白磷中毒的解毒剂。磷与硫酸铜在不同条件下反应如下：
    - $11P_4 + 60CuSO_4 + 96H_2O = 20Cu_3P + 24H_3PO_4 + 60H_2SO_4$。
    - $P_4 + 10CuSO_4 + 16H_2O = 10Cu + 4H_3PO_4 + 10H_2SO_4$。

### 磷的氧化物及磷的含氧酸及其盐

磷（P）由于其价层具有空的 $3d$ 轨道，能够形成多种氧化态（主要为 -3、+1、+3、+5）的化合物，其中以磷的氧化物及含氧酸最为重要。

磷最常见的两种氧化物是**三氧化二磷**（$\text{P}_4\text{O}_6$）和**五氧化二磷**（$\text{P}_4\text{O}_{10}$）。虽然它们的实验式常写为 $\text{P}_2\text{O}_3$ 和 $\text{P}_2\text{O}_5$，但研究表明其真实的分子结构是基于 $\text{P}_4$ 正四面体骨架的笼状结构。

1. **三氧化二磷 ($\text{P}_4\text{O}_6$)**：
    - **结构与制备**：磷在不充足的空气中燃烧或缓慢氧化可制得。分子由 4 个 $\text{P}$ 原子和 6 个 $\text{O}$ 原子组成，呈三角锥构型。
    - **物理性质**：白色晶体，具有大蒜气味，有剧毒且易潮解。
    - **化学性质**：它是亚磷酸的酸酐，与**冷水**缓慢反应生成亚磷酸 ($\text{H}_3\text{PO}_3$)。但在热水中反应剧烈，会发生歧化反应生成红磷、磷化氢和磷酸。

2. **五氧化二磷 ($\text{P}_4\text{O}_{10}$)**：
    - **结构与制备**：磷在充足的氧气中燃烧产生浓厚的**白烟**，即为五氧化二磷固体微粒。结构中每个磷原子再与一个氧原子以双键结合。
    - **极强的吸水性**：$\text{P}_4\text{O}_{10}$ 是实验室中**最强的酸性干燥剂**，甚至能使浓硝酸脱水成 $\text{N}_2\text{O}_5$，使浓硫酸脱水成 $\text{SO}_3$。
    - **与水反应**：吸水过程放出大量的热。与**冷水**反应生成极毒的**偏磷酸** ($\text{HPO}_3$)；与**热水**反应或煮沸则生成无毒的**正磷酸** ($\text{H}_3\text{PO}_4$)。

磷的含氧酸种类丰富，其酸性强弱和“元数”取决于分子结构中**可电离的羟基（$-\text{OH}$）数目**，而直接连在磷原子上的氢原子（$\text{P}-\text{H}$ 键）不表现酸性。

| 名称       | 分子式                  | 氧化态 | 元数       | 酸性/特性                                         |
| :--------- | :---------------------- | :----- | :--------- | :------------------------------------------------ |
| **正磷酸** | $\text{H}_3\text{PO}_4$ | +5     | 三元酸     | 中强酸，非氧化性，高沸点难挥发                    |
| **亚磷酸** | $\text{H}_3\text{PO}_3$ | +3     | **二元酸** | 中强酸，含一个 $\text{P}-\text{H}$ 键，具强还原性 |
| **次磷酸** | $\text{H}_3\text{PO}_2$ | +1     | **一元酸** | 中强酸，含两个 $\text{P}-\text{H}$ 键，极强还原性 |

1. **正磷酸 ($\text{H}_3\text{PO}_4$) 详解**：
    - **物理性质**：纯质为无色透明晶体，易潮解。
    - **电离特性**：它是三元弱电解质，在水中分三步电离，电离常数 $K_{a1} \gg K_{a2} \gg K_{a3}$。
    - **多磷酸的形成**：磷酸分子间脱水缩合可形成**焦磷酸** ($\text{H}_4\text{P}_2\text{O}_7$)、**三聚磷酸** ($\text{H}_5\text{P}_3\text{O}_{10}$) 等链状或环状缩合酸。

2. **还原性酸**：
    - **亚磷酸与次磷酸**：由于磷原子氧化态较低，它们及其盐能将 $\text{Ag}^+$、$\text{Cu}^{2+}$ 等金属离子还原为单质。例如亚磷酸能还原银镜反应中的银。

磷酸盐及其性质：

1. **分类与生成**：由于磷酸是三元酸，与碱反应视物质的量之比可生成三类盐：**磷酸二氢盐** ($\text{H}_2\text{PO}_4^-$)、**磷酸氢盐** ($\text{HPO}_4^{2-}$) 和**磷酸正盐** ($\text{PO}_4^{3-}$)。

2. **溶解性规律**：
    - **易溶盐**：所有的磷酸二氢盐都易溶于水；钾、钠、铵的磷酸盐也都易溶。
    - **难溶盐**：除上述碱金属和铵盐外，其余金属的磷酸氢盐和正盐大多难溶于水（如白色 $\text{Ca}_3(\text{PO}_4)_2$）。

3. **水溶液的酸碱性**：
    - **磷酸二氢钠 ($\text{NaH}_2\text{PO}_4$)**：由于 $\text{H}_2\text{PO}_4^-$ 的电离程度大于水解程度，水溶液显**酸性**。
    - **磷酸氢二钠 ($\text{Na}_2\text{HPO}_4$)**：由于 $\text{HPO}_4^{2-}$ 的水解程度大于电离程度，水溶液显**碱性**。
    - **磷酸钠 ($\text{Na}_3\text{PO}_4$)**：强碱弱酸盐，在水中强烈水解，显**强碱性**。

4. **检验与鉴别**：
    - **硝酸银法**：磷酸根与 $\text{Ag}^+$ 反应生成**黄色**的 $\text{Ag}_3\text{PO}_4$ 沉淀，该沉淀溶于稀硝酸。
    - **钼酸铵法**：在浓硝酸中与钼酸铵反应生成**黄色**结晶状的钼磷酸铵沉淀。

工业制备与应用：

1. **磷单质的制备**：工业上在电炉中于 1500℃ 利用焦炭还原磷酸钙和二氧化硅混合物。
2. **磷酸的工业生产**：
    - **湿法**：用硫酸分解磷矿石（如氟磷灰石），副产石膏，主要用于制造化肥。
    - **热法**：将磷燃烧生成的 $\text{P}_4\text{O}_{10}$ 与水结合，纯度高，用于食品、医药和电子工业。
3. **磷肥的应用**：常用的可溶性磷肥是**过磷酸钙**（$\text{Ca}(\text{H}_2\text{PO}_4)_2$ 与 $\text{CaSO}_4$ 的混合物）和**重过磷酸钙**。注意过磷酸钙不能与消石灰混用，否则会生成难溶的正盐降低肥效。

### 金属磷化物及其性质

金属磷化物是由金属元素与磷元素形成的化合物，在化学性质、结构特征以及工业应用上呈现出多样性。根据磷原子的结合方式和成键性质，金属磷化物主要可分为**类盐磷化物**和**间充型（或金属型）磷化物**。

以下结合来源对金属磷化物的性质进行详细展开：

类盐磷化物：这类化合物主要由活泼金属（如碱金属、碱土金属及铝、锌等）与磷化合而成。

1. **成键特征与结构**：
    - **阴离子形式**：磷以 **$P^{3-}$（磷根离子）** 形式存在，表现出 **-3 价**。
    - **极化作用**：$P^{3-}$ 离子具有较大的半径且易变形，由于其电荷高，附加极化作用强，导致许多磷化物具有明显的**共价向离子型过渡**的特征。
    - **典型代表**：磷化锂（$Li_3P$）、磷化钠（$Na_3P$）、磷化镁（$Mg_3P_2$）、磷化钙（$Ca_3P_2$）、磷化铝（$AlP$）等。

2. **核心化学性质：剧烈的水解性**：
    - **水解反应**：类盐磷化物极不稳定，遇水或酸会发生剧烈的“双水解”反应，生成对应的金属氢氧化物（或盐）并放出剧毒的**磷化氢（$PH_3$）**。
    - **反应方程式**：
        - $AlP + 3H_2O = Al(OH)_3\downarrow + PH_3\uparrow$。
        - $Ca_3P_2 + 6H_2O = 3Ca(OH)_2\downarrow + 2PH_3\uparrow$。
    - **现象特征**：产生的 $PH_3$ 是一种无色、具有**强烈大蒜味（或鱼腥味）**的剧毒气体。由于制取过程中常伴有微量的**联膦（$P_2H_4$）**，导致其在空气中极易发生**自燃**。

3. **实际应用**：
    - **农药与熏蒸剂**：磷化铝（$AlP$）是农业上极其常用的**粮仓熏蒸杀虫剂**，利用其吸收空气水分后缓慢释放 $PH_3$ 来杀灭害虫。
    - **杀鼠剂**：磷化锌（$Zn_3P_2$）和磷化钙常因其强毒性被用作杀鼠药。

间充型与过渡金属磷化物：这类磷化物由过渡金属（如 Fe, Mn, Cu, Ti 等）形成，性质更趋向于金属或合金。

1. **结构与物理性质**：
    - 这类化合物通常具有**金属光泽、良好的导电性和高硬度**。
    - 它们通常不发生类似类盐磷化物那样的剧烈水解反应，化学性质相对稳定。

2. **典型实例：磷化亚铜 ($Cu_3P$)**：
    - 当单质磷与**热的硫酸铜溶液**反应时，会生成磷化亚铜沉淀。
    - **反应方程式**：$11P + 15CuSO_4 + 24H_2O \xrightarrow{\Delta} 5Cu_3P + 6H_3PO_4 + 15H_2SO_4$。
    - **对比**：在冷溶液中，磷通常直接将 $Cu^{2+}$ 还原为金属铜单质。

3. **在冶金与半导体中的存在**：
    - **钢中的磷**：磷在钢中具有强烈的**固溶强化**作用，能提高钢的强度和硬度，但会显著降低其塑性和韧性，尤其是导致**“冷脆”现象**（低温韧性剧降）。在炼钢过程中，常通过加入 $CaO$ 造渣来除去有害的磷。
    - **特种合金**：磷青铜和磷铁是工业上重要的特种合金。
    - **半导体材料**：磷化铟（$InP$）和磷化镓（$GaP$）是高性能的半导体材料，广泛用于光纤通信和电子芯片。

| 性质         | 类盐磷化物 (如 $AlP, Ca_3P_2$)  | 过渡金属磷化物/半导体 (如 $Cu_3P, InP$) |
| :----------- | :------------------------------ | :-------------------------------------- |
| **主要金属** | 活泼金属（I A, II A, III A 族） | 过渡金属、主族类金属                    |
| **磷的价态** | -3 价（$P^{3-}$）               | 复杂的共价或金属键结合                  |
| **水解性**   | **极易水解**，产生剧毒 $PH_3$   | **不水解**或极难水解，性质稳定          |
| **外观特性** | 多为蜡状固体或粉末              | 具有金属光泽，硬度大                    |
| **导电性**   | 固体通常不导电                  | **良好导电性**或半导体特性              |

**安全提示**：金属磷化物（尤其是类盐类）在储存和使用时必须严禁烟火并保持干燥，因其水解产生的 $PH_3$ 既有剧毒又极易自燃。

### 其他非金属磷化物及其性质

非金属磷化物是指磷元素与其他非金属元素（如氢、卤素、氧、硫、氮、硼等）通过共价键结合形成的化合物。由于磷原子具有空的 **$3d$ 轨道**且电负性适中，其非金属化合物表现出多样的成键方式（如 $sp^3$、$sp^3d$ 杂化）及独特的化学活性。以下是主要的非金属磷化物及其性质的详细展开：

磷的氢化物：膦（$\text{PH}_3$）与联膦（$\text{P}_2\text{H}_4$）：

1. **磷化氢（$\text{PH}_3$，又称膦）**
    - **分子结构**：磷原子采取 **$sp^3$ 杂化**，拥有一对孤对电子，分子构型为**三角锥形**，与氨（$\text{NH}_3$）类似，但键角更小（约 $93.6^\circ$）。
    - **物理性质**：无色、**剧毒**气体，具有极难闻的**鱼腥味或大蒜味**，微溶于水。
    - **化学性质**：
        - **弱碱性**：碱性远弱于氨。仅能与强质子酸（如 $\text{HI}$、$\text{HClO}_4$）反应生成**钅粦盐**（如 $\text{PH}_4\text{I}$），且这类盐极不稳定，遇水即分解。
        - **强还原性**：极易被氧化。在空气中燃烧生成磷酸：$\text{PH}_3 + 2\text{O}_2 = \text{H}_3\text{PO}_4$。
        - **置换反应**：能将某些金属（如 $\text{Ag}$、$\text{Cu}$、$\text{Au}$）从其盐溶液中还原为单质。例如与硝酸银反应：$6\text{AgNO}_3 + \text{PH}_3 + 3\text{H}_2\text{O} = 6\text{Ag}\downarrow + 6\text{HNO}_3 + \text{H}_3\text{PO}_3$。
    - **自燃特性**：实验室制取的 $\text{PH}_3$ 常混有微量的**联膦（$\text{P}_2\text{H}_4$）**，导致其在空气中能自燃，这是民间所谓“鬼火”的主要化学成因。

2. **联膦（$\text{P}_2\text{H}_4$）**
    - 结构与联氨（$\text{N}_2\text{H}_4$）相似，是一种无色液体，由于含有 $\text{P-P}$ 键，其化学性质比 $\text{PH}_3$ 更活泼，极易自燃。

磷的卤化物（$\text{PX}_3, \text{PX}_5$）：磷能与卤素形成三卤化磷和五卤化磷。由于磷有空的 $3d$ 轨道，其卤化物的性质与氮的卤化物显著不同。

1. **三卤化磷（以 $\text{PCl}_3$ 为例）**
    - **结构**：$sp^3$ 杂化，三角锥形，具有极性。
    - **性质**：无色液体，在潮湿空气中强烈发雾（因生成 $\text{HCl}$）。它极易水解生成**亚磷酸**和卤化氢：$\text{PCl}_3 + 3\text{H}_2\text{O} = \text{H}_3\text{PO}_3 + 3\text{HCl}$。
    - **配合能力**：$\text{PF}_3$ 是一种重要的配体，能与过渡金属形成稳定的配合物（如 $\text{Pd}(\text{PF}_3)_4$），这是因为磷原子能接受金属反馈的 $d$ 电子进入其 $3d$ 空轨道。

2. **五卤化磷（以 $\text{PCl}_5$ 为例）**
    - **结构**：气态时为 **$sp^3d$ 杂化**，呈**三角双锥**构型。
    - **离子性**：在固态时，五氯化磷实际上是**离子晶体**，由 $[\text{PCl}_4]^+$（正四面体）和 $[\text{PCl}_6]^-$（正八面体）组成。
    - **性质**：强烈的氯化剂和脱水剂。遇水剧烈水解生成**磷酸**：$\text{PCl}_5 + 4\text{H}_2\text{O} = \text{H}_3\text{PO}_4 + 5\text{HCl}$。

磷的氧化物（$\text{P}_4\text{O}_6, \text{P}_4\text{O}_{10}$）：虽然实验式常写为 $\text{P}_2\text{O}_3$ 和 $\text{P}_2\text{O}_5$，但它们真实的分子结构是基于磷原子的笼状结构。

1. **三氧化二磷（$\text{P}_4\text{O}_6$）**：亚磷酸的酸酐。白色剧毒固体，具大蒜气味，易潮解。与冷水反应生成亚磷酸，遇热水则发生歧化反应生成单质磷、磷化氢和磷酸。
2. **五氧化二磷（$\text{P}_4\text{O}_{10}$）**：磷酸的酸酐。白色粉末，具有极强的**吸水性**，是实验室中除氨气外最有效的**酸性干燥剂**。它与水反应放出巨大的热，先生成偏磷酸（$\text{HPO}_3$），最终转化为正磷酸（$\text{H}_3\text{PO}_4$）。

磷的硫化物（$\text{P}_4\text{S}_n$）：磷与硫化合可生成一系列硫化物，如 $\text{P}_4\text{S}_3$、$\text{P}_4\text{S}_7$ 和 $\text{P}_4\text{S}_{10}$ 等。

- **结构**：与磷的氧化物类似，具有笼状结构。
- **性质**：硫化磷水解的产物非常复杂。例如 $\text{P}_4\text{S}_3$ 与水反应可产生 $\text{PH}_3$、$\text{H}_2$、$\text{H}_2\text{S}$ 以及多种磷的含氧酸。

磷的氮化物（$\text{P-N}$ 化合物）：

- **膦氮烯（Phosphazenes）**：含有 $\text{P=N}$ 双键的环状或链状化合物（如 $\text{NPCl}_2$），具有显著的共振结构。
- **聚膦腈**：由六氯环三磷腈开环聚合而成，是一种重要的**无机-有机杂化高分子**，可用于制造高性能橡胶、阻燃材料和生物医用材料。

其他化合物：

1. **磷化硼（$\text{BP}$）**：一种硬度极高的共价化合物，具有类似金刚石的结构，磷原子在其中的配位数为 4。
2. **半导体应用**：在半导体工业中，将单质磷掺入超纯硅中，利用磷原子多出的价电子形成 **N 型半导体**。此外，**磷化铟（$\text{InP}$）**也是一种高性能的光纤通信半导体材料。

**总结**：非金属磷化物的共性在于磷原子多样的杂化方式和较强的**亲氧性**（形成稳固的 $\text{P=O}$ 键是许多反应的驱动力），且大多具有较高的反应活性（如易水解、易氧化）。

### 磷及其常见化合物工业制备与实验室制取

磷（P）及其化合物在工业和实验室中具有多样的制备路径。根据提供的来源，磷单质、磷的氧化物、磷酸及其盐、磷化氢以及卤化磷的制法详细展开如下：

磷单质（P）的制备与转化：

1. 工业制备（电炉法）：工业上主要通过在**电炉**中于 **1500 ℃** 高温还原磷酸钙（取自磷矿石）来制取单质磷。
    - **反应原理**：将磷酸钙、石英砂（$SiO_2$）和焦炭（C）混合加热，石英砂作为助熔剂与钙结合生成硅酸钙，焦炭将磷还原为气态单质。
    - **化学方程式**：$2Ca_3(PO_4)_2 + 10C + 6SiO_2 \xlongequal{1500^\circ C} 6CaSiO_3 + 10CO\uparrow + P_4\uparrow$。
    - **收集**：生成的磷蒸气通过蒸馏从炉中逸出，并在水中冷凝成固体（白磷）。

2. 同素异形体的相互转化（实验室及工业）：
    - **白磷转红磷**：将白磷隔绝空气加热到 **260~300 ℃** 或进行光照，可转变为红磷。
    - **红磷转白磷**：红磷隔绝空气加热到 **416 ℃** 时升华，其蒸气在冷凝后重新结晶为白磷。

磷的氧化物主要通过磷单质的**直接燃烧**制得，产物取决于氧气的供应量。

1. **五氧化二磷（$P_4O_{10}$ 或 $P_2O_5$）**
    - **方法**：在充足的氧气或空气中点燃磷单质，产生浓厚的**白烟**，即为五氧化二磷固体微粒。
    - **方程式**：$4P + 5O_2 \xlongequal{点燃} 2P_2O_5$（或 $P_4O_{10}$）。

2. **三氧化二磷（$P_4O_6$ 或 $P_2O_3$）**
    - **方法**：在**氧气不充足**或限氧条件下燃烧过量的白磷，或使其在常温下缓慢氧化制得。
    - **方程式**：$4P(\text{过量}) + 3O_2 \xlongequal{点燃} 2P_2O_3$。

磷酸（$H_3PO_4$）的制备：

1. 工业制备：湿法（酸分解法）是工业生产化肥级磷酸的主流方法。
    - **原理**：利用硫酸等强酸分解磷矿石（如氟磷灰石或磷钙石）。
    - **方程式**：$Ca_3(PO_4)_2 + 3H_2SO_4 = 2H_3PO_4 + 3CaSO_4\downarrow$。
    - **特点**：成本低，但产品含有较多杂质，副产物为磷石膏（$CaSO_4 \cdot 2H_2O$）。

2. 工业制备：热法用于生产高纯度的食品级、医药级或电子级磷酸。
    - **流程**：先通过电炉还原得黄磷，点燃黄磷生成五氧化二磷，最后将其水合。
    - **步骤方程式**：
        1. $P_4 + 5O_2 \xrightarrow{点燃} P_4O_{10}$。
        2. $P_4O_{10} + 6H_2O \xrightarrow{\Delta} 4H_3PO_4$。

3. 实验室制备：
    - **方法**：可用白磷与**稀硝酸**在加热条件下反应制得，硝酸作为强氧化剂将磷氧化至最高价态。
    - **方程式**：$3P + 5HNO_3 + 2H_2O = 3H_3PO_4 + 5NO\uparrow$。

磷化氢不能由磷与氢气直接化合制得（反应为吸热且不自发），需通过间接方法制取。

1. **歧化法（常用）**
    - **原理**：将白磷与**热的浓碱溶液**（如 $NaOH$ 或 $Ba(OH)_2$）共热，发生歧化反应。
    - **方程式**：$P_4 + 3NaOH + 3H_2O \xrightarrow{\Delta} PH_3\uparrow + 3NaH_2PO_2$（次磷酸钠）。

2. **磷化物水解法**
    - **原理**：利用**类盐磷化物**（如磷化铝、磷化钙）与水或酸剧烈反应释放出 $PH_3$。
    - **方程式**：$AlP + 3H_2O = Al(OH)_3\downarrow + PH_3\uparrow$。

3. **磷盐分解法**
    - **方法**：加热**钅粦盐**（如碘化钅粦 $PH_4I$）或使其与碱反应。
    - **方程式**：$PH_4I + NaOH = NaI + PH_3\uparrow + H_2O$。

卤化磷的制备：

1. 三氯化磷（$PCl_3$）与五氯化磷（$PCl_5$）
    - **原理**：通过磷单质与氯气的**直接化合**实现，氯气的量决定产物。
    - **$PCl_3$**：在**不足量**的氯气中燃烧。方程式：$2P + 3Cl_2 \xlongequal{点燃} 2PCl_3$。
    - **$PCl_5$**：在**过量**的氯气中燃烧，或使 $PCl_3$ 继续与氯气反应。方程式：$PCl_3 + Cl_2 \xrightarrow{\Delta} PCl_5$。

2. 三氯氧磷（$POCl_3$）
    - **实验室法**：可以利用 $PCl_3$ 与 $SO_2$ 及 $Cl_2$ 为原料反应制取。
    - **另一种方式**：通过 $PCl_5$ 与**限量水**发生部分水解制得。方程式：$PCl_5 + H_2O = POCl_3 + 2HCl$。

磷肥（过磷酸钙）的生产：**过磷酸钙（普钙）**是应用最广泛的磷肥。

- **生产方法**：工业上用适量的**硫酸**跟磷矿粉（主要成分 $Ca_3(PO_4)_2$）反应。
- **原理**：将难溶的正磷酸盐转化为易溶的磷酸二氢盐。
- **方程式**：$Ca_3(PO_4)_2 + 2H_2SO_4 = Ca(H_2PO_4)_2 + 2CaSO_4$。
- **产物**：所得的磷酸二氢钙与硫酸钙的混合物即为过磷酸钙。

### 磷及其化合物的应用与生物学意义

磷（P）是元素周期表第VA族（氮族）的一种活泼非金属元素，在自然界中不以游离态存在，主要以磷酸盐矿物（如磷灰石）的形式分布。磷及其化合物不仅在**现代工业与农业**中具有举足轻重的地位，更是构成**生命物质的基础**，在生物能量转化与遗传信息传递中扮演核心角色。

磷的用途涵盖了从基础化工、农业生产到高科技材料的各个领域：

1. 农业：化肥与农药的核心
    - **磷肥生产**：这是磷最主要的用途。植物生长需要氮、磷、钾三大营养元素。磷能促进作物发育、成熟，使果实饱满。工业上利用硫酸分解磷矿石制得**过磷酸钙**（普钙），或通过更高效的制法生产重过磷酸钙和磷酸铵等肥料。
    - **农药制备**：许多高效农药属于有机磷化合物，如乐果、敌百虫、敌敌畏等，它们在自然界中易分解，残留时间短。此外，**磷化铝（AlP）**常用于粮仓熏蒸，通过与空气中水分反应释放剧毒的磷化氢（$PH_3$）气体来杀灭害虫。

2. 工业：从火柴到高新材料
    - **火柴工业**：红磷（赤磷）是制造安全火柴的主要原料，涂在火柴盒侧面的红磷与火柴头上的氧化剂（如氯酸钾）摩擦生热，引发燃烧。
    - **高纯磷酸与磷酸盐**：白磷（黄磷）常用于制造高纯度的**正磷酸**（$H_3PO_4$），用于食品添加剂、制药及金属表面处理（磷化）以防腐蚀。**三聚磷酸钠**曾广泛用作洗涤剂助剂以软化硬水，但因环境污染问题正被替代。
    - **特种合金与半导体**：磷用于制造磷青铜和磷铁等合金。在电子工业中，向纯硅中掺入微量磷可制得 **N 型半导体材料**。
    - **军事用途**：利用白磷在空气中自燃产生浓厚白烟的性质，用于制造**烟幕弹、燃烧弹**及曳光弹。

磷被称为“生命元素”，它参与了生命体内几乎所有的关键生理过程：

1. 遗传信息的物质基础（**核酸 DNA 和 RNA**）：磷酸基团是构成核苷酸的核心组分。核苷酸单体通过**磷酸二酯键**连接成生物大分子——核酸。磷酸、戊糖和碱基共同构成了携带遗传密码的支架。

2. 生物体内的“能量货币”（**腺苷三磷酸 ATP**）：磷是生物能量代谢的核心。ATP分子含有高能磷酸键，通过水解脱去磷酸基团释放大量能量，为肌肉收缩、神经传导及各种生化反应提供直接动力。

3. 骨骼与牙齿的坚硬支撑（**无机盐组分**）：脊椎动物的骨骼和牙齿主要由**羟基磷灰石** [$Ca_{10}(PO_4)_6(OH)_2$] 构成。在牙齿保健中，含氟牙膏能将羟基磷灰石转化为更耐酸蚀、结构更匹配的**氟磷灰石**，从而预防龋齿。

4. 细胞膜的结构框架（**磷脂**）：磷脂是构成生物膜（如细胞膜、核膜）的主要成分。磷脂分子具有亲水端的磷酸基团和疏水端的烃链，在水中自动排布成双分子层，维持细胞的完整性和物质交换功能。

5. 生化反应的调节开关（**磷酸化与去磷酸化**）：在化学生物学中，蛋白质的磷酸化状态（由蛋白激酶和磷酸酶调节）充当了生物“开关”，控制着信号转导、酶的活性以及细胞的生长与凋亡。

尽管磷必不可少，但其不合理利用也带来了挑战：

- **水体富营养化**：过量的氮、磷物质（来自化肥流失或含磷洗涤剂）进入水体后，会导致藻类疯长，消耗水中氧气，引发“水华”或“赤潮”，破坏生态平衡。
- **毒性差异**：磷的同素异形体毒性差异巨大。**白磷具剧毒**（误食 0.1g 可致死），而红磷基本无毒。此外，有机磷农药中毒会抑制神经系统功能，需紧急催吐并送医。

### 磷与氮的结构与性质的相似性与差异性

磷（P）与氮（N）同属于元素周期表第 VA 族（第 15 族），统称为**氮族元素**。它们在原子结构上具有相似性，但由于磷位于第三周期且拥有**空的 3d 轨道**，其成键特征、分子结构及化学性质展现出显著的多变性和差异。

1. **价电子构型相同**：氮和磷原子的最外层电子排布均为 **$ns^2 np^3$**，即都有 5 个价电子。
2. **主要氧化态一致**：两者均能形成 **-3、+3、+5** 等氧化态的化合物。
3. **氢化物结构相似**：氨（$NH_3$）与膦（$PH_3$）均采取 **$sp^3$ 杂化**，分子构型皆为**三角锥形**。
4. **成键规律**：在形成简单离子时，都有获得 3 个电子达到 8 电子稳定结构的趋势。

尽管处于同族，氮与磷在单质形态和化学键能上存在巨大差异：

- **单质存在形式**：
    - **氮气（$N_2$）**：氮原子半径小，由于其 $p$ 轨道重叠效率极高，倾向于形成极其牢固的**氮氮三键（$N \equiv N$）**。在能量上，1 个三键的键能（946 kJ/mol）远大于 3 个单键的键能之和，因此氮以双原子分子形式存在，性质极稳定。
    - **单质磷**：磷原子半径较大，其 $3p$ 轨道“肩并肩”重叠效率低，难以形成稳定的经典 $\pi$ 键。相反，磷倾向于形成单键，如**白磷（$P_4$）**呈正四面体结构，包含 6 个 $P-P$ 单键。
- **物理性质**：氮气是无色气体，而磷是固体（有白磷、红磷、黑磷等多种同素异形体）。
- **氢键与溶解性**：由于氮的电负性（3.04）远大于磷（2.19），氮能形成**分子间氢键**，导致 $NH_3$ 极易溶于水且沸点反常高于 $PH_3$。

磷原子具有空的 **3d 轨道**，这是其化学性质比氮更加多变和复杂的根本原因：

1. 价层扩充与超价分子（Octet Expansion）：氮原子位于第二周期，受限于 $2s$ 和 $2p$ 轨道（最多容纳 8 电子），其**最大共价键数为 4**，无法形成 $NCl_5$。

    磷由于 3d 轨道的参与，可以发生 **$sp^3d$ 或 $sp^3d^2$ 杂化**，形成 5 或 6 个共价键，生成**超价分子**。

    **典型例子**：五氯化磷（$PCl_5$）呈三角双锥构型，六氟磷酸根（$PF_6^-$）呈正八面体构型。

2. $p\pi - d\pi$ 键的形成：磷可以利用其空 3d 轨道与第二周期元素（如 O、N、F）充满电子的 $p$ 轨道发生重叠，形成所谓的 **$d\pi - p\pi$ 键（反馈 $\pi$ 键）**。
    - **强化化学键**：这种相互作用使得磷氧键（$P=O$）表现出极强的稳定性，键能高达 575 kJ/mol。在磷酸（$H_3PO_4$）和氧化膦（$R_3P=O$）中，磷原子实际上通过 1 个 $\sigma$ 键和 2 个 $d\pi - p\pi$ 配键与氧结合。
    - **改变分子构型**：例如 $(SiH_3)_3N$ 呈现平面三角形，而其氮类类似物 $(CH_3)_3N$ 为三角锥形。这是因为氮上的孤对电子离域进入了硅（类似磷）的 3d 空轨道。

3. 增强的配位能力：磷原子作为配体时（如 $PF_3$），除了能通过孤对电子提供 $\sigma$ 键外，还能接受中心金属原子反馈回来的 $d$ 电子进入其 3d 空轨道，形成**反馈 $\pi$ 键**。
    - 这使得 $PH_3$ 的配位能力强于 $NH_3$，$PF_3$ 甚至能像 $CO$ 一样与过渡金属形成稳定的配合物。

4. 水解机理的多样化：氮的卤化物（如 $NCl_3$）水解时，由于氮没有空轨道，水分子（亲核试剂）无法进攻氮，而是进攻氯原子。

    磷的卤化物（如 $PCl_3$）水解时，水分子可以利用磷的 **3d 空轨道** 进行亲核进攻，这使得磷的卤化物水解速度极快且机理与氮截然不同。

磷与氮虽同族，但**磷的 3d 轨道**使其从“八隅体规则”的束缚中解脱出来。这不仅赋予了磷形成**高配位数化合物**的能力，还通过 **$d\pi - p\pi$ 相互作用** 显著增强了它与氧等元素的结合力，并在**生化反应（如磷脂和核酸中的磷酰基转移）**及**工业催化**中展现出氮所不具备的多变性。
