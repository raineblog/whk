# 过渡元素概述

## 铁单质

### 铁的物理性质

铁（Fe）是人类使用量最大、用途最广的金属元素，也是地壳中含量第四高的元素（在金属中居第二位，仅次于铝）。

- **周期表位置**：铁原子序数为 26，位于元素周期表**第四周期、第 VIII 族**，属于过渡元素。
- **电子排布**：铁原子的外围电子层排布为 **$3d^6 4s^2$**。
- **化合价**：在化学反应中，铁原子容易失去最外层的 2 个 $4s$ 电子形成 **$Fe^{2+}$**，也容易进一步失去 1 个 $3d$ 电子（形成 $3d^5$ 半充满的稳定结构）得到 **$Fe^{3+}$**，因此铁通常显 **+2 价和 +3 价**。

铁单质的性质：

- **外观与磁性**：纯净的铁是光亮的**银白色**金属，具有**铁磁性**，能被磁铁吸引并在磁场作用下产生磁性。
- **基本常数**：密度约为 **$7.86\text{ g/cm}^3$**，熔点为 **1538℃**，沸点为 2861℃。
- **特性**：具有良好的**导电性、导热性和延展性**，但其导电性不如银、铜和铝。

自然界中的铁主要以化合态形式存在于矿石中：

- **赤铁矿 (Hematite)**：主要成分为 **$Fe_2O_3$**，呈红褐色。
- **磁铁矿 (Magnetite)**：主要成分为 **$Fe_3O_4$**，具有磁性，呈黑色。
- **菱铁矿 (Siderite)**：主要成分为 **$FeCO_3$**。
- **黄铁矿 (Pyrite)**：主要成分为 **$FeS_2$**，又称硫铁矿，常用于工业制硫酸。

纯铁质地较软，工业和生活中大量使用的是铁的合金。生铁和钢的主要区别在于**含碳量**的不同：

| 类别 | 含碳量 | 机械性能 | 机械加工 | 主要用途 |
| :--- | :--- | :--- | :--- | :--- |
| **生铁** | 2% ~ 4.3% | 硬而脆，韧性差 | 可铸不可锻 | 制造机床底座、暖气片、井盖等 |
| **钢** | 0.03% ~ 2% | 硬而韧，延展性好 | 可铸可锻 | 建筑、交通工具、机械零件、工具等 |

常见的钢材分类：

- **碳素钢**：仅含铁、碳及少量杂质。
    - **低碳钢**（含碳量 < 0.3%）：韧性好，用于制造铁丝、铁钉等。
    - **中碳钢**（含碳量 0.3%~0.6%）：用于制造铁轨、钢梁等结构材料。
    - **高碳钢**（含碳量 > 0.6%）：硬度大，用于制造刀具、钻头、手术器械等。
- **合金钢（特种钢）**：在碳素钢中加入铬、镍、锰、钨、钒等元素以获得特殊性能。
    - **不锈钢**：含有 **铬 (Cr)**（通常需 $\geq 10.5\% \sim 13\%$）和 **镍 (Ni)**。其表面能生成致密氧化膜，抗腐蚀性极强，用于医疗器械、餐具等。
    - **锰钢**：含有锰元素，韧性好、硬度大。用于制造钢轨、挖掘机铲斗、坦克装甲等。
    - **钨钢**：硬度大、耐高温，用于制造切削刀具。
    - **硅钢**：导磁性强，用于制造变压器铁芯。

### 化学性质和工业炼铁

根据提供的来源，铁（Iron, Fe）作为应用最广泛的过渡金属，其化学性质极其丰富，工业炼铁则是人类文明进步的重要标志。铁是较活泼的金属，在化学反应中主要表现为**还原性**。其产物的价态取决于氧化剂的强弱：

- **与 $O_2$**：在纯氧中剧烈燃烧，火星四射，生成黑色的 **$Fe_3O_4$**。在潮湿空气中则会发生复杂的电化学腐蚀形成铁锈（主成分 $Fe_2O_3 \cdot xH_2O$）。
- **与 $Cl_2$**：与强氧化剂反应，生成棕褐色的烟（**$FeCl_3$**），无论氯气是否过量。
- **与 $S$**：与弱氧化剂反应，加热生成黑色的 **$FeS$**。
- **与 $I_2$**：反应生成 **$FeI_2$**。

与水反应：

- 常温下铁与水不反应。
- **高温**条件下，铁能与水蒸气反应生成 **$Fe_3O_4$ 和 $H_2$**：$3Fe + 4H_2O(g) \xrightarrow{高温} Fe_3O_4 + 4H_2$。
- 这个实验通常将导管口插入肥皂液中，现象是，用火柴靠近肥皂泡，发出爆鸣声。

与酸反应：

- **与非氧化性酸（如稀盐酸、稀硫酸）**：发生置换反应，生成 **$Fe^{2+}$** 和 $H_2$。

- **与氧化性酸（如硝酸、浓硫酸）**：
    - **钝化**：常温下，铁遇浓硫酸或浓硝酸会在表面生成致密的氧化膜，阻止进一步反应（可用铁罐车运输）。
    - **加热反应**：加热时钝化被破坏。与**过量稀硝酸**反应生成 **$Fe^{3+}$**；若**铁过量**，则最终产物为 **$Fe^{2+}$**。

与盐溶液反应（置换反应）：

- 铁能将金属活动性顺序中排在其后的金属（如铜、银）从其盐溶液中置换出来。例如：$Fe + Cu^{2+} = Fe^{2+} + Cu$（湿法炼铜原理）。

- 铁还能与 $Fe^{3+}$ 反应：$Fe + 2Fe^{3+} = 3Fe^{2+}$。

工业炼铁（高炉炼铁）：工业炼铁是一个利用还原剂在高温下将铁从矿石中还原出来的复杂过程。

- **设备**：**高炉**。

- **原料**：
    - **铁矿石**：提供铁元素。常用赤铁矿（$Fe_2O_3$）、磁铁矿（$Fe_3O_4$）、菱铁矿（$FeCO_3$）等。
    - **焦炭**：既是燃料提供热量，又是生产还原剂 $CO$ 的原料。
    - **石灰石 ($CaCO_3$)**：作为造渣剂，除去矿石中的脉石（主要成分 $SiO_2$）。
    - **空气**：提供氧气使焦炭燃烧。

基本原理与反应过程：炼铁的核心反应是利用 **$CO$ 在高温下还原铁的氧化物**。

1. **还原剂的产生**：
    - $C + O_2 \xrightarrow{高温} CO_2$（放热反应，维持炉温）。
    - $CO_2 + C \xrightarrow{高温} 2CO$（吸热反应，生成还原剂）。
2. **矿石的还原（分步进行）**：
    - $3Fe_2O_3 + CO \rightarrow 2Fe_3O_4 + CO_2$。
    - $Fe_3O_4 + CO \rightarrow 3FeO + CO_2$。
    - $FeO + CO \rightarrow Fe + CO_2$。
    - 总反应方程式：**$Fe_2O_3 + 3CO \xrightarrow{高温} 2Fe + 3CO_2$**。
3. **造渣反应（除去杂质）**：
    - $CaCO_3 \xrightarrow{高温} CaO + CO_2 \uparrow$。
    - $CaO + SiO_2 \xrightarrow{高温} CaSiO_3$（炉渣，浮在铁水表面防止铁被氧化）。

高炉炼出的铁是**生铁**，含碳量为 **2% ~ 4.3%**，并含有硫、磷等杂质，硬而脆，可铸不可锻。为了提高性能，生铁需进一步炼制成**钢**。在高温下通入氧气，将生铁中过量的碳、硫、磷等杂质氧化成气体或炉渣除去。含碳量降至 **0.03% ~ 2%**，硬而韧，具有良好的延展性和弹性，可锻造。

### 铁的氧化物

铁的氧化物主要有三种：**氧化亚铁（FeO）**、**氧化铁（$Fe_2O_3$）**和**四氧化三铁（$Fe_3O_4$）**。它们都是**不溶于水**的固体，且均不与水反应。

| 物质名称 | **氧化亚铁 ($FeO$)** | **氧化铁 ($Fe_2O_3$)** | **四氧化三铁 ($Fe_3O_4$)** |
| :--- | :--- | :--- | :--- |
| **俗称** | - | **铁红** | **磁性氧化铁** |
| **矿物名称** | 方铁矿 | **赤铁矿** | **磁铁矿** |
| **颜色状态** | **黑色**粉末 | **红棕色**粉末 | **黑色**有磁性的晶体 |
| **铁的价态** | +2 | +3 | +2、+3（可以看作 $FeO \cdot Fe_2O_3$） |
| **稳定性** | 不稳定，受热易被氧化 | **稳定** | **稳定** |
| **主要用途** | 瓷器颜料（使釉呈绿色） | **红色颜料**、炼铁 | **磁性材料**、墨粉、炼铁 |

与酸的反应（碱性氧化物特性）：$FeO$ 和 $Fe_2O_3$ 属于**碱性氧化物**，能与酸反应生成对应的盐和水。

- **氧化亚铁**：与非氧化性酸（如稀盐酸）反应生成亚铁盐。
    $FeO + 2H^+ = Fe^{2+} + H_2O$
- **氧化铁**：与酸反应生成三价铁盐。
    $Fe_2O_3 + 6H^+ = 2Fe^{3+} + 3H_2O$
- **四氧化三铁**：由于其含有两种价态的铁，与酸反应时会生成 $Fe^{2+}$ 和 $Fe^{3+}$ 的混合溶液。
    $Fe_3O_4 + 8H^+ = Fe^{2+} + 2Fe^{3+} + 4H_2O$

与特殊酸的氧化还原反应：由于铁离子的氧化性或还原性，这些氧化物在遇到某些特定酸时会发生氧化还原反应。

- **遇强氧化性酸（如 $HNO_3$）**：$FeO$ 和 $Fe_3O_4$ 中的 +2 价铁会被硝酸氧化为 $Fe^{3+}$。
- **遇还原性酸（如氢碘酸 $HI$）**：$Fe_2O_3$ 和 $Fe_3O_4$ 中的 $Fe^{3+}$ 会将 $I^-$ 氧化为 $I_2$，自身被还原为 $Fe^{2+}$。
    $Fe_2O_3 + 6HI = 2FeI_2 + I_2 + 3H_2O$

还原性与热稳定性：

- **FeO 的不稳定性**：$FeO$ 在空气中受热会迅速被氧化成 $Fe_3O_4$。
- **高温下的还原反应**：在高温条件下，三种氧化物都能被还原剂（如 **$CO$、$H_2$、$C$、$Al$**）还原为金属铁。
    - **高炉炼铁原理**：$Fe_2O_3 + 3CO \xrightarrow{高温} 2Fe + 3CO_2$。
    - **铝热反应**：用于焊接钢轨，放出大量热使生成的铁呈熔融态。
        $2Al + Fe_2O_3 \xrightarrow{高温} 2Fe + Al_2O_3$

铁的氧化物在自然界与生产中的联系：

- **铁锈的形成**：铁在潮湿空气中发生复杂的电化学腐蚀（吸氧腐蚀），最终生成的铁锈主要成分是**氧化铁的水合物**（通常简写为 **$Fe_2O_3 \cdot xH_2O$**）。
- **特殊应用**：
    - **$Fe_2O_3$**：因其红棕色且无毒，被广泛用于建筑涂料、油漆和橡胶的红色颜料。
    - **$Fe_3O_4$**：利用其磁性，常用于制造激光打印机的墨粉及其他磁性材料。

### 铁的氢氧化物

铁的氢氧化物主要包括**氢氧化亚铁 [$Fe(OH)_2$]** 和 **氢氧化铁 [$Fe(OH)_3$]**。它们都是铁的重要化合物，在物理性质、化学稳定性以及制备方法上存在显著差异。

氢氧化亚铁 $Fe(OH)_2$：

1. 物理性质

    - **外观**：纯净的氢氧化亚铁是**白色**固体。
    - **溶解性**：**难溶于水**。

2. 化学性质（强还原性）

    - **不稳定性**：$Fe(OH)_2$ 极不稳定，在空气中极易被氧化。
    - **特征变色反应**：当 $Fe(OH)_2$ 暴露在空气中时，会迅速由**白色**变为**灰绿色**，最后变为**红褐色**。这是因为 $Fe(OH)_2$ 被空气中的氧气氧化成了 $Fe(OH)_3$。
        - 反应方程式：**$4Fe(OH)_2 + O_2 + 2H_2O = 4Fe(OH)_3$**。
    - **与酸反应**：作为不溶性碱，能与非氧化性酸反应生成亚铁盐。
        - 离子方程式：$Fe(OH)_2 + 2H^+ = Fe^{2+} + 2H_2O$。
    - **与氧化性酸反应**：如遇稀硝酸，其中的 $+2$ 价铁会被氧化为 $+3$ 价。
        - 离子方程式：$3Fe(OH)_2 + 10H^+ + NO_3^- = 3Fe^{3+} + NO \uparrow + 8H_2O$。

3. 实验制备与条件控制：由于 $Fe(OH)_2$ 极易被氧化，实验制备时必须严格**隔绝氧气**。

    - **基本原理**：$Fe^{2+} + 2OH^- = Fe(OH)_2 \downarrow$。
    - **操作要点**：
        - 使用**煮沸后冷却**的蒸馏水配制溶液，以除去溶解氧。
        - 将吸有 $NaOH$ 溶液的胶头滴管**伸入 $FeSO_4$ 溶液液面以下**再挤出碱液。
        - **液封法**：在溶液表面覆盖一层苯或煤油，以阻断空气。
        - **还原性气氛保护法**：利用铁粉与稀硫酸反应产生的 $H_2$ 排出装置内的空气，并在 $H_2$ 压力下将亚铁盐压入碱液中。

氢氧化铁 $Fe(OH)_3$：

1. 物理性质

    - **外观**：**红褐色**固体。
    - **溶解性**：**难溶于水**。

2. 化学性质

    - **热稳定性**：受热易分解生成氧化铁和水。
        - 反应方程式：**$2Fe(OH)_3 \xrightarrow{\triangle} Fe_2O_3 + 3H_2O$**。
    - **与酸反应**：能与酸反应生成三价铁盐。
        - 离子方程式：$Fe(OH)_3 + 3H^+ = Fe^{3+} + 3H_2O$。
    - **氧化性（遇强还原剂）**：例如与氢碘酸（$HI$）反应时，$Fe^{3+}$ 会被 $I^-$ 还原。
        - 离子方程式：$2Fe(OH)_3 + 6H^+ + 2I^- = 2Fe^{2+} + I_2 + 6H_2O$。

3. 制备与转化

    - **普通制备**：向铁盐溶液中滴加碱液即可生成红褐色沉淀：$Fe^{3+} + 3OH^- = Fe(OH)_3 \downarrow$。
    - **胶体制备**：向沸水中逐滴加入饱和 $FeCl_3$ 溶液，继续煮沸至液体呈**红褐色**，停止加热即可得到 $Fe(OH)_3$ 胶体。注意：不可过度加热或用玻璃棒搅拌，否则胶体易聚沉。

| 特性 | **氢氧化亚铁 [$Fe(OH)_2$]** | **氢氧化铁 [$Fe(OH)_3$]** |
| :--- | :--- | :--- |
| **铁的化合价** | $+2$ | $+3$ |
| **颜色状态** | **白色**絮状沉淀 | **红褐色**沉淀 |
| **水溶性** | 难溶 | 难溶 |
| **空气中稳定性** | 极不稳定，易被氧化 | 较稳定，加热易分解 |
| **制备试剂** | $Fe^{2+}$ 盐 + 强碱（需无氧环境） | $Fe^{3+}$ 盐 + 碱 |
| **二者转化** | **$4Fe(OH)_2 + O_2 + 2H_2O = 4Fe(OH)_3$** | - |
| **应用** | 工业制取铁黑（磁性氧化铁）的中间体 | 净水剂（利用其胶体的吸附性） |

**在工艺流程中的应用**：常利用 $Fe(OH)_3$ 极小的溶度积（$K_{sp}$），通过**调节 pH 值**的方法将 $Fe^{3+}$ 转化为 $Fe(OH)_3$ 沉淀除去。通常在 **pH 为 3.2~4.7** 左右，$Fe^{3+}$ 即可基本完全沉淀。若体系中含有 $Fe^{2+}$，一般需先加入 $H_2O_2$ 将其氧化为 $Fe^{3+}$，再调 pH 除去。

### 铁盐和亚铁盐

铁盐（含 $\ce{Fe^{3+}}$）的性质深化：

- **颜色细节**：
    - **溶液颜色**：含 $\ce{Fe^{3+}}$ 的稀溶液通常呈**黄色或棕黄色**；但在强酸性且高浓度时，$\ce{Fe^{3+}}$ 实际上以淡紫色的水合离子 $[\ce{Fe(H2O)6}]^{3+}$ 形式存在。
    - **固体颜色**：$\ce{FeCl3 \cdot 6H2O}$ 为**黄色**晶体；**无水 $\ce{FeCl3}$** 则呈**黑褐色**或**墨绿色**片状晶体，且具有极强的吸湿性。$\ce{Fe2(SO4)3}$ 固体为**淡黄色**。
- **溶解性**：铁盐极易溶于水，也易溶于乙醇、乙醚等有机溶剂。

$\ce{Fe^{3+}}$ 处于铁元素的最高价态，具有较强的氧化性。

- **与还原性离子的反应**：
    - **与 $\ce{I-}$**：$2\ce{Fe^{3+}} + 2\ce{I-} \rightleftharpoons 2\ce{Fe^{2+}} + \ce{I2}$。这是一个**可逆反应**，在平衡体系中加入 $\ce{CCl4}$ 萃取 $\ce{I2}$ 可使平衡正向移动。
    - **与 $\ce{S^{2-}}$ / $\ce{H2S}$**：$\ce{Fe^{3+}}$ 会将 $\ce{S^{2-}}$ 氧化为硫单质（淡黄色沉淀）：$2\ce{Fe^{3+}} + \ce{H2S} = 2\ce{Fe^{2+}} + \ce{S \downarrow} + 2\ce{H+}$。
    - **与 $\ce{SO3^{2-}}$**：在不同条件下，$\ce{Fe^{3+}}$ 可能与 $\ce{SO3^{2-}}$ 发生水解竞争或氧化还原反应，生成 $\ce{Fe^{2+}}$ 和 $\ce{SO4^{2-}}$。
- **与金属的反应**：
    - **蚀刻铜箔**：工业上利用 $\ce{FeCl3}$ 腐蚀覆铜板制造印刷电路板。

水解性质与胶体应用：

- **极易水解**：$\ce{Fe^{3+}}$ 水解能力极强，$\ce{pH}$ 达到 **3.2 左右**时即基本沉淀完全（生成 $\ce{Fe(OH)3}$）。
- **净水原理**：$\ce{Fe^{3+}}$ 水解生成的 **$\ce{Fe(OH)3}$ 胶体** 具有吸附性，能吸附水中的悬浮杂质。
- **溶液配制**：配制 $\ce{FeCl3}$ 溶液时，需将其先溶于**浓盐酸**中，再加水稀释，以抑制其水解。

亚铁盐（含 $\ce{Fe^{2+}}$）的性质深化：

- **颜色**：含有 $\ce{Fe^{2+}}$ 的溶液呈**浅绿色**。
- **绿矾 ($\ce{FeSO4 \cdot 7H2O}$)**：蓝绿色单斜晶体，在干燥空气中易风化，受热易分解。
- **莫尔盐 [$\ce{(NH4)2Fe(SO4)2 \cdot 6H2O}$]**：浅绿色晶体，比普通亚铁盐更稳定，在分析化学中常用作还原剂的标准物质。

$\ce{Fe^{2+}}$ 是铁元素的中间价态，**还原性**是其最核心的性质。

- **被强氧化剂氧化**：遇 $\ce{Cl2}$、$\ce{Br2}$、$\ce{H2O2}$、稀硝酸、酸性 $\ce{KMnO4}$ 均会被氧化为 $\ce{Fe^{3+}}$。
    - **与硝酸反应**：$\ce{Fe^{2+}}$ 能将 $\ce{NO3-}$（酸性环境）还原为 $\ce{NO}$ 气体。
- **空气中氧化**：亚铁盐溶液露置于空气中会逐渐变质（变黄）：$4\ce{Fe^{2+}} + \ce{O2} + 4\ce{H+} = 4\ce{Fe^{3+}} + 2\ce{H2O}$。

亚铁盐的保存：**“防氧化”**加入少量**铁粉**。**“防水解”**加入少量对应的**酸**（如稀硫酸）。

| 检验离子 | 试剂名称 | 化学式 / 俗称 | 反应现象 | 离子方程式或原理 |
| :--- | :--- | :--- | :--- | :--- |
| **$\ce{Fe^{3+}}$** | **硫氰酸钾** | $\ce{KSCN}$ | **血红色**溶液 | $\ce{Fe^{3+} + 3SCN- = Fe(SCN)3}$ |
| | **苯酚** | $\ce{C6H5OH}$ | **紫色**溶液 | $\ce{Fe^{3+}}$ 与苯酚的络合反应 |
| | **亚铁氰化钾** | **黄血盐** | **深蓝色**沉淀 | 生成普鲁士蓝 $\ce{Fe4[Fe(CN)6]3}$ |
| **$\ce{Fe^{2+}}$** | **铁氰化钾** | **赤血盐** | **深蓝色**沉淀 | 生成滕氏蓝（与普鲁士蓝组分相同） |
| | **邻二氮菲** | 邻菲啰啉 | **橙红色**溶液 | 生成稳定的螯合物 |
| | **间接法** | $\ce{KSCN}$+氯水 | 先无色后变**红** | $\ce{Fe^{2+}}$ 被氧化为 $\ce{Fe^{3+}}$ 后显色 |

在有色金属（如 $\ce{Cu, Zn, Mg, Ni}$）的提纯流程中，铁通常作为杂质出现。资料显示其标准套路如下：

1. **“先氧化”**：加入 **$\ce{H2O2}$**（首选，不引入杂质）或 $\ce{NaClO}$、$\ce{MnO2}$ 等，将溶液中的 $\ce{Fe^{2+}}$ 全部转化为 $\ce{Fe^{3+}}$。
    - **原因**：$\ce{Fe^{2+}}$ 沉淀所需 $\ce{pH}$ 较高（$\approx 8.3$），此时目标离子（如 $\ce{Cu^{2+}}$）往往也已沉淀；而 $\ce{Fe^{3+}}$ 在 $\ce{pH} \approx 3.2$ 时即可完全沉淀，能实现有效分离。
2. **“后调 pH”**：加入目标金属的氧化物、氢氧化物或碳酸盐（如制备 $\ce{CuSO4}$ 时加 $\ce{CuO}$ 或 $\ce{Cu2(OH)2CO3}$），消耗 $\ce{H+}$，促进 $\ce{Fe^{3+}}$ 水解生成 **$\ce{Fe(OH)3}$ 沉淀** 除去。

生物与生活联系补充：

- **贫血与吸收**：口服补铁剂多为亚铁盐（如绿矾、琥珀酸亚铁）。**维生素 C** 具有还原性，常与补铁剂同服，以防止 $\ce{Fe^{2+}}$ 被氧化或将食物中的 $\ce{Fe^{3+}}$ 还原为易吸收的 $\ce{Fe^{2+}}$。
- **暖贴原理**：暖贴内部含有铁粉、活性炭、食盐等，利用铁在潮湿空气中发生**吸氧腐蚀**放出热量的原理取暖。

### 检验和除杂

铁及其离子的检验是元素化学中的重点内容，主要通过特定的显色反应或沉淀反应来辨别 $Fe^{2+}$ 和 $Fe^{3+}$。以下是根据提供的来源对**硫氰酸钾、亚铁氰化钾、铁氰化钾及临二氮菲**这四种常用检验试剂的详细展开讲解：

- 硫氰化钾 $\ce{KSCN}$ 法：

    - 检验亚铁离子：加入 $\ce{KSCN}$ 无明显现象，加入适量氧化剂（$\ce{Cl2}$）变血红色。

    - 检验铁离子：加入 $\ce{KSCN}$ 变血红色，加入适量还原剂（$\ce{Fe}$）变无色。

    - 注意：$\ce{Fe^3+ + 3SCN- -> Fe(SCN)3}$ 为配合反应，故即使 $\ce{Fe(SCN)3}$ 不是沉淀也不能大量共存。

- 铁氰化钾（$\ce{K3[Fe(CN)6]}$）、亚铁氰化钾（$\ce{K4[Fe(CN)6]}$）法：

    - 检验亚铁离子：加入铁氰化钾，产生蓝色沉淀（铁氰化亚铁）。

    - 检验铁离子：加入亚铁氰化钾，产生蓝色沉淀（亚铁氰化铁）。

- 邻二氮菲在 pH 为 $2\sim9$ 时，会与亚铁离子（$\ce{Fe^2+}$）形成稳定的橙红色邻二氮菲亚铁离子。

硫氰酸钾 (KSCN) —— 检验 $Fe^{3+}$ 的灵敏试剂：硫氰酸钾（或硫氰酸铵）是检验 $Fe^{3+}$ 最常用且最灵敏的试剂。

- **反应原理**：$Fe^{3+}$ 与 $SCN^-$ 能够以不同比例通过配位键结合，生成一系列特征**血红色**的配合物 $[Fe(SCN)_n]^{3-n}$（$n=1 \sim 6$）。
- **反应现象**：向含 $Fe^{3+}$ 的溶液中滴加 KSCN 溶液，溶液立即变为**血红色**。
- **离子方程式**：$Fe^{3+} + 3SCN^- = Fe(SCN)_3$。
- **注意事项**：
    - **$Fe^{2+}$ 的干扰**：$Fe^{2+}$ 遇 KSCN 溶液不显红色。利用此性质可以检验 $Fe^{2+}$ 是否被氧化：先加 KSCN 若不变红，再加氯水或双氧水变红，则证明原溶液含有 $Fe^{2+}$。
    - **酸碱性影响**：该反应须在**酸性条件**下进行。若 pH > 3，$Fe^{3+}$ 易水解生成 $Fe(OH)_3$，且 $OH^-$ 会与 $SCN^-$ 竞争结合 $Fe^{3+}$，导致无法观察到红色。

亚铁氰化钾 ($K_4[Fe(CN)_6]$，黄血盐) —— 检验 $Fe^{3+}$：亚铁氰化钾，俗称**黄血盐**，呈黄色晶体，易溶于水。

- **反应原理**：$Fe^{3+}$ 与亚铁氰根离子 $[Fe(CN)_6]^{4-}$ 反应，生成深蓝色的不溶性配合物。
- **反应现象**：产生**深蓝色沉淀**，这种沉淀被称为**普鲁士蓝**（Prussian blue）。
- **离子方程式**：$4Fe^{3+} + 3[Fe(CN)_6]^{4-} = Fe_4[Fe(CN)_6]_3 \downarrow$。
- **应用**：此反应特征明显，常用于鉴定 $Fe^{3+}$。

铁氰化钾 ($K_3[Fe(CN)_6]$，赤血盐) —— 检验 $Fe^{2+}$：铁氰化钾，俗称**赤血盐**，红色晶体，在水中离解为 $K^+$ 和 $[Fe(CN)_6]^{3-}$。

- **反应原理**：$Fe^{2+}$ 与铁氰根离子 $[Fe(CN)_6]^{3-}$ 反应，生成深蓝色的沉淀。
- **反应现象**：产生**深蓝色沉淀**，传统上称为**滕氏蓝**（Turnbull's blue）。
- **离子方程式**：$3Fe^{2+} + 2[Fe(CN)_6]^{3-} = Fe_3[Fe(CN)_6]_2 \downarrow$。
- **注意事项**：
    - **物质本质**：现代光谱分析发现，滕氏蓝与普鲁士蓝实际上是同一种物质（晶型略有差异），但在中学化学中仍常作为检验 $Fe^{2+}$ 的特征手段。
    - **干扰排除**：在检验 $FeCl_3$ 中是否混有 $FeCl_2$ 时，使用铁氰化钾溶液比酸性高锰酸钾更可靠，因为 $Cl^-$ 也会使高锰酸钾褪色造成干扰。

临二氮菲 (1,10-phenanthroline) —— $Fe^{2+}$ 的特效显色剂：临二氮菲（又称邻菲啰啉）是测定水溶液中微量 $Fe^{2+}$ 的灵敏试剂。

- **反应原理**：临二氮菲分子中的氮原子与 $Fe^{2+}$ 通过配位键形成非常稳定的螯合物 $[Fe(phen)_3]^{2+}$。
- **反应现象**：生成**橙红色**溶液。
- **应用**：
    - **定量分析**：由于配合物颜色深且稳定，常利用**分光光度法**或比色法来测量菠菜或其他食品中铁元素的含量。
    - **指示剂**：由于其可逆性好且终点变化敏锐，也可用作氧化还原滴定的指示剂。
- **环境要求**：
    - **pH 范围**：检验的适宜 pH 范围为 **2 ~ 9**。若酸性太强（pH 过低），$H^+$ 会与临二氮菲结合；若碱性太强，$OH^-$ 会与 $Fe^{2+}$ 反应，均会干扰显色。
    - **价态要求**：该试剂仅对 $Fe^{2+}$ 显色。若需测定总铁含量，通常需加入还原剂（如盐酸羟胺）将 $Fe^{3+}$ 还原为 $Fe^{2+}$ 后再进行显色。

| 试剂名称 | 常用俗称 | 主要检验离子 | 反应现象 | 主要用途 |
| :--- | :--- | :--- | :--- | :--- |
| **硫氰酸钾** | - | $Fe^{3+}$ | **血红色**溶液 | 定性检验 $Fe^{3+}$ |
| **亚铁氰化钾** | 黄血盐 | $Fe^{3+}$ | **深蓝色**沉淀 (普鲁士蓝) | 鉴定 $Fe^{3+}$ |
| **铁氰化钾** | 赤血盐 | $Fe^{2+}$ | **深蓝色**沉淀 (滕氏蓝) | 鉴定 $Fe^{2+}$ |
| **临二氮菲** | 邻菲啰啉 | $Fe^{2+}$ | **橙红色**溶液 | 微量 $Fe^{2+}$ 定量测定 |

当体系中同时存在 **$Fe^{2+}$** 和 **$Fe^{3+}$** 时，最核心且常用的除杂策略是：**“先氧化，后调 pH”**。这种方法利用了 $Fe^{2+}$ 和 $Fe^{3+}$ 沉淀时对 **pH 环境要求差异巨大**的物理化学特性。以下是详细的讲解及其在工艺流程题中的应用。

为什么要“先氧化，后调 pH”？在工业生产（如制备硫酸铜、硫酸镁或锌盐）中，铁通常作为杂质存在，且往往是 $Fe^{2+}$ 和 $Fe^{3+}$ 的混合状态。

1. **沉淀 pH 的差异**：
    - **$Fe^{3+}$**：在 **pH $\approx$ 1.1~2.7** 时开始沉淀，在 **pH $\approx$ 3.2~4.7** 时即可完全沉淀为 $Fe(OH)_3$。
    - **$Fe^{2+}$**：在 **pH $\approx$ 5.8~7.6** 时才开始沉淀，完全沉淀则需要 **pH $\approx$ 8.3~9.6**。
2. **分离的难点**：
    许多目标产物金属离子（如 $Cu^{2+}, Zn^{2+}, Mg^{2+}, Ni^{2+}$）的沉淀 pH 与 $Fe^{2+}$ 非常接近。例如，$Cu^{2+}$ 在 pH $\approx$ 4.2 左右就开始沉淀了。如果直接调 pH 除去 $Fe^{2+}$，会导致目标金属离子也一起沉淀，造成产物损失。
3. **解决思路**：
    通过氧化剂将 $Fe^{2+}$ 全部转化为 $Fe^{3+}$，此时只需将 pH 控制在 **3.2~4.7** 之间，即可使铁元素以 $Fe(OH)_3$ 的形式彻底除去，而目标离子（如 $Cu^{2+}$）仍留在溶液中。

步骤 1：氧化（$Fe^{2+} \rightarrow Fe^{3+}$）加入氧化剂，使溶液中残留的亚铁离子完全转化为铁离子。

- **常用氧化剂**：**$H_2O_2$（双氧水）** 是工艺流程题中的首选“绿色氧化剂”，因为它不引入杂质，且还原产物是水。
- **反应方程式**：$2Fe^{2+} + H_2O_2 + 2H^+ = 2Fe^{3+} + 2H_2O$。
- **其他选择**：$Cl_2$、空气/氧气、$NaClO$、$MnO_2$、$KMnO_4$ 等。

步骤 2：调节 pH（沉淀铁离子）加入碱性物质中和溶液中的酸（消耗 $H^+$），促进 $Fe^{3+}$ 的水解平衡正向移动，生成 $Fe(OH)_3$ 沉淀。

- **原理**：$Fe^{3+} + 3H_2O \rightleftharpoons Fe(OH)_3 \downarrow + 3H^+$。
- **试剂选择原则**：**“不引入新杂质”**。通常选择目标产物金属对应的氧化物、氢氧化物或碳酸盐。
    - 若主产物为 $CuSO_4$：加入 **$CuO, Cu(OH)_2, CuCO_3$** 或 **$Cu_2(OH)_2CO_3$**。
    - 若主产物为 $MgSO_4$：加入 **$MgO, Mg(OH)_2$** 或 **$MgCO_3$**。
- **控制范围**：pH 需控制在 **$Fe(OH)_3$ 完全沉淀** 与 **目标离子开始沉淀** 之间的范围（例如 pH 3.2~4.2）。

步骤 3：过滤：通过过滤操作将生成的 $Fe(OH)_3$ 沉淀除去，得到纯净的含有目标离子的滤液。

在高考或模考的工艺流程大题中，这一环节经常以以下形式出现：

1. **考察试剂的作用**：
    - **问**：流程中加入 $H_2O_2$ 的目的是什么？
    - **答**：将 $Fe^{2+}$ 氧化为 $Fe^{3+}$，便于后续通过调节 pH 的方法使其转化为 $Fe(OH)_3$ 沉淀而除去。

2. **考察 pH 的范围控制**：
    - **题干信息**：常给出“几种金属离子沉淀的 pH 范围表”。
    - **问**：调节 pH 的范围应是多少？
    - **分析**：查看表中 $Fe^{3+}$ 完全沉淀的 pH（下限）和目标离子（如 $Cu^{2+}$）开始沉淀的 pH（上限）。例如 $Fe^{3+}$ 完全沉淀为 3.2，$Cu^{2+}$ 开始沉淀为 4.2，则 pH 范围应选 **3.2~4.2**。

3. **考察调节 pH 试剂的选择**：
    - **问**：制备硫酸镁过程中，调节 pH 应选用 $MgO$ 还是 $NaOH$？
    - **答**：选用 **$MgO$**。理由是 $MgO$ 能中和酸且不引入 Na+ 等杂质离子。

4. **考察实验现象与检验**：
    - **问**：如何检验除杂后的溶液中是否还含有铁元素？
    - **答**：取少量滤液于试管中，滴加 **KSCN 溶液**，若溶液不变红，则说明 $Fe^{3+}$ 已除尽。若要检验 $Fe^{2+}$ 是否氧化完全，可滴加 **$K_3[Fe(CN)_6]$（赤血盐）溶液**，若无蓝色沉淀生成，则证明 $Fe^{2+}$ 已除尽。

| 步骤 | 操作 | 化学反应/原理 | 目的/意义 |
| :--- | :--- | :--- | :--- |
| **氧化** | 加 $H_2O_2$ 等 | $2Fe^{2+} + H_2O_2 + 2H^+ = 2Fe^{3+} + 2H_2O$ | 将 $Fe^{2+}$ 转化为更易沉淀的 $Fe^{3+}$ |
| **调 pH** | 加金属氧化物等 | $Fe^{3+} + 3H_2O \rightleftharpoons Fe(OH)_3 + 3H^+$ | 消耗 $H^+$，使 $Fe^{3+}$ 彻底转化为沉淀 |
| **过滤** | 物理分离 | - | 除去铁杂质沉淀，获得纯净滤液 |

### 补铁剂的化学原理

补铁的核心难点在于铁离子极易发生**水解**和**氧化**，这与人体的生理环境存在天然冲突。

1. **铁的生理形式与分布**：
    - 人体内含铁约 **4~5 g**，其中约 **72%** 存在于红细胞的**血红蛋白**中，负责运输氧气。
    - 血红蛋白中的铁以 **$\ce{Fe^{2+}}$** 形式与原卟啉环形成**配合物（血红素）**。若 $\ce{Fe^{2+}}$ 被氧化为 $\ce{Fe^{3+}}$，则会丧失载氧能力。

2. **$\ce{Fe^{3+}}$ 的“沉淀陷阱”**：
    - 资料显示，$\ce{Fe^{3+}}$ 极易水解，在 **pH $\approx$ 3.2** 时即基本完全沉淀（生成 $\ce{Fe(OH)3}$）。
    - 胃酸（HCl）环境有利于铁盐溶解，但当食糜进入**十二指肠**（pH 约为 6）时，游离的 $\ce{Fe^{3+}}$ 会迅速转化为不溶性的氢氧化物，导致无法被肠道黏膜吸收。
    - 相比之下，$\ce{Fe^{2+}}$ 在酸性到中性环境下稳定性较好，能保持溶解态。

3. **转运蛋白的“特异性”**：
    - 肠黏膜细胞上的转运蛋白（如 DMT1）主要识别并转运**二价金属阳离子**，因此人体对 $\ce{Fe^{2+}}$ 的吸收率远高于 $\ce{Fe^{3+}}$。

补铁剂的三代演变深析：

1. 第一代：无机亚铁盐（简单粗暴但“副作用”大）

    - **典型代表**：**硫酸亚铁（$\ce{FeSO4 \cdot 7H2O}$，俗称绿矾）**。
    - **化学局限**：
        - **稳定性差**：固体易风化，溶液易被空气中的 $O_2$ 氧化变质。
        - **胃肠道刺激**：释放的游离 $\ce{Fe^{2+}}$ 浓度过高，除了你提到的芬顿反应产生羟基自由基外，还会与胃粘膜直接接触产生收敛作用，引起恶心。
        - **食物拮抗**：极易与食物中的单宁酸、植酸等发生反应生成不溶性沉淀。

2. 第二代：有机弱酸亚铁盐（平衡溶解度与刺激性）

    - **典型代表**：**琥珀酸亚铁（速力菲）、富马酸亚铁、葡萄糖酸亚铁**。
    - **改进原理**：
        - 有机酸根（如琥珀酸根）与 $\ce{Fe^{2+}}$ 具有一定程度的配位作用，这种“半束缚”状态延缓了铁离子的释放速度，降低了局部瞬时浓度，从而减轻了对肠胃的灼伤。

3. 第三代：螯合物与多糖铁（分子态整体吸收）

    - **典型代表**：**甘氨酸亚铁、右旋糖酐铁、蛋白琥珀酸铁、乙二胺四乙酸铁钠（EDTA-Fe-Na）**。
    - **深度特性**：
        - **高度稳定性**：以甘氨酸亚铁为例，其形成的**五元环螯合结构**具有极高的形成常数，能有效屏蔽外部阴离子（如草酸根）的干扰。
        - **吸收路径改变**：这类补铁剂可以不经过 DMT1 的离子通道，而是通过氨基酸或多糖的吸收通道以**分子态整体吸收**，避免了铁离子在肠道内的化学竞争。
        - **生物安全性**：铁原卟啉（血红素铁）是天然的螯合物原型，它在小肠内直接被吸收，不受 pH 或其他膳食因素影响。

补铁的“协同”与“禁忌”：

1. 维生素 C：最强辅助剂

    - **还原保护**：维生素 C 具有强还原性，能将食物中不易吸收的 $\ce{Fe^{3+}}$ 还原为易吸收的 $\ce{Fe^{2+}}$。
    - **防止失活**：在血红蛋白层面，维生素 C 也能防止血红素中的 $\ce{Fe^{2+}}$ 被氧化。
    - **酸性增溶**：维生素 C 作为抗坏血酸，能维持肠道的弱酸性环境，防止铁离子水解。

2. 补铁的“饮食禁区”

    - **茶、咖啡、牛奶**：补血剂说明书通常注明服用后 2 小时内不得饮用这些物质。
        - **茶多酚/鞣酸**：会与铁离子结合生成极难溶的**鞣酸铁**沉淀。
        - **钙离子**：牛奶中的高浓度钙会与铁离子竞争转运通道，降低铁的吸收率。

3. 补铁剂的质量检测（家庭或实验室）

    - **检测氧化程度**：
        - 取少量补铁剂溶液，滴加 **KSCN 溶液**，若出现**血红色**，说明部分 $\ce{Fe^{2+}}$ 已被氧化为 $\ce{Fe^{3+}}$。
    - **检测铁元素存在**：
        - 利用**邻二氮菲**（邻菲啰啉）与 $\ce{Fe^{2+}}$ 结合生成**橙红色**配合物 $[Fe(phen)_3]^{2+}$ 来进行定量测定，其适宜 pH 范围为 2 ~ 9。

| 维度 | 关键点 |
| :--- | :--- |
| **吸收核心** | **$\ce{Fe^{2+}}$ 可溶且被 DMT1 识别** |
| **水解限制** | **$\ce{Fe^{3+}}$ 在 pH > 3 时基本沉淀完全** |
| **辅助方案** | **维生素 C 还原 $\ce{Fe^{3+}} \rightarrow \ce{Fe^{2+}}$** |
| **干扰因素** | **茶（鞣酸）、咖啡、牛奶（钙）抑制吸收** |
| **检测手段** | **KSCN 检 $\ce{Fe^{3+}}$（变红），邻二氮菲检 $\ce{Fe^{2+}}$（变橙红）** |

### 高铁酸和高铁酸盐

高铁酸（$H_2FeO_4$）及其盐（高铁酸盐，如 $K_2FeO_4$、$Na_2FeO_4$）是铁元素处于 **+6 价** 态形成的化合物。由于其极高的氧化性和在水处理中的双重功能，高铁酸盐在现代化学工业和环境工程中具有重要地位。

在铁的“价—类”二维图中，高铁酸根（$FeO_4^{2-}$）处于铁元素的**最高常见价态（+6 价）**。

- **原子结构**：铁原子失去 $4s$ 轨道的 2 个电子和 $3d$ 轨道的 4 个电子，达到一种不稳定的高价态。
- **存在形式**：高铁酸（$H_2FeO_4$）极不稳定，很难以游离态存在，通常以**高铁酸盐**（如高铁酸钾 $K_2FeO_4$）的形式存在。
- **外观**：高铁酸钾（$K_2FeO_4$）为**暗紫色**固体晶体。
- **溶解性**：极易溶于水，溶解后形成**浅紫红色**至**紫色**的溶液（类似于高锰酸钾溶液的颜色）。

高铁酸盐表现出**极强的氧化性**，其氧化能力甚至超过了高锰酸钾和氯气。

1. **不稳定性与产物**：高铁酸根在水溶液中不稳定，会发生氧化还原反应并产生氧气。
    - **反应原理**：$FeO_4^{2-}$ 被还原为 **$Fe^{3+}$**，同时水中的氧元素被氧化生成 $O_2$。
2. **酸碱性的影响**：
    - 在**酸性或中性**条件下，高铁酸盐极不稳定，反应剧烈。
    - 在**强碱性**环境下，其稳定性相对较高，利于保存。

高铁酸盐被誉为“绿色环保高效”的消毒净化剂，主要因为它在水处理中具有**双重功能**：

1. **杀菌消毒（强氧化性）**：高铁酸根具有超强的氧化能力，能够迅速破坏细菌、病毒的细胞结构及酶系统，从而起到杀毒、除臭和脱色作用。

2. **絮凝净水（吸附性）**：高铁酸根被还原后的产物是 **$Fe^{3+}$**。$Fe^{3+}$ 随后会发生水解反应：

    $$
    Fe^{3+} + 3H_2O \rightleftharpoons Fe(OH)_3(\text{胶体}) + 3H^+
    $$

    生成的 **$Fe(OH)_3$ 胶体** 具有极大的比表面积，能吸附水中的悬浮颗粒物并沉降，达到净化水质的效果。

3. **绿色评价**：与氯气消毒相比，高铁酸盐不会与水中的有机物产生致癌的有机氯化物；与铝盐净化相比，它不产生对人体有害的铝残留。

高铁酸钾也可作为**高能电池材料**。由于 +6 价铁到 +3 价铁的转化涉及 3 个电子的转移，且铁的原子量较小，高铁酸盐电池具有极高的理论能量密度和放电电位。

高铁酸盐的制备通常利用强氧化剂在碱性条件下氧化三价铁化合物：

1. **化学氧化法（湿法）**：在强碱溶液中，用次氯酸盐（如 $NaClO$）氧化 $Fe(OH)_3$：

    $$
    2Fe(OH)_3 + 3ClO^- + 4OH^- = 2FeO_4^{2-} + 3Cl^- + 5H_2O
    $$

2. **电解法（阳极合成）**：以**金属铁为阳极**，铂为阴极，在浓 NaOH 溶液中进行电解。**阳极反应** $Fe - 6e^- + 8OH^- = FeO_4^{2-} + 4H_2O$。

| 维度 | 高铁酸盐 ($FeO_4^{2-}$) | 铁盐 ($Fe^{3+}$) | 亚铁盐 ($Fe^{2+}$) |
| :--- | :--- | :--- | :--- |
| **化合价** | **+6** | +3 | +2 |
| **氧化还原性** | **强氧化性** | 氧化性 | 还原性（为主） |
| **溶液颜色** | **紫红色** | 黄棕色 | 浅绿色 |
| **主要功能** | 消毒 + 净水 | 净水 | 补血剂 |

## 其他过渡金属

### 汞

汞，化学符号为 **Hg**，原子序数为 **80**，位于元素周期表第六周期、第 12 族（IIB 族）。

1. **独特的物理性质**：
    - **物态**：汞是常温（25 °C）下**唯一呈液态的金属**。其原子间的作用力较弱，使得它在室温下易于流动。
    - **外观**：呈银白色，具有金属光泽。
    - **熔沸点**：熔点约为 **-38.8 °C**，是熔点最低的金属；沸点约为 **356.6 °C**。
    - **密度**：密度极大，约为 **13.55~13.59 g/cm³**，属于重金属。
    - **挥发性**：在常温下即可微量蒸发，其饱和蒸气压约为 0.0012 mmHg。**汞蒸气具有剧毒**，吸入后会对健康造成严重损害。

2. **化学性质**：
    - **稳定性**：汞的化学性质相对安定，其标准还原电位高于氢，因此**不能置换出酸中的氢**。
    - **与酸反应**：虽不与稀盐酸或稀硫酸反应，但可溶于强氧化性酸，如**硝酸**和**热浓硫酸**。
    - **氧化态**：汞常见的氧化态为 **+1** 和 **+2**。
        - **+1 价（亚汞）**：通常以多汞阳离子形式存在，最常见的是具有金属间共价键的 **$Hg_2^{2+}$**。
        - **+2 价（汞）**：是汞最稳定的氧化态，如氧化汞（HgO）。

汞的天然存在与冶炼：

1. **天然存在**：汞在自然界中主要以硫化矿的形式存在，最重要的矿石是**辰砂**（又称硃砂、丹砂，主要成分为 **$α-HgS$**），呈红色。
2. **冶炼方法**：
    - **热分解法**：由于汞的化合物较不稳定，加热即可分解。例如加热红色氧化汞粉末：$2HgO \xrightarrow{\Delta} 2Hg + O_2 \uparrow$。
    - **焙烧辰砂**：在空气中焙烧硫化汞矿石，可直接还原出金属汞：$HgS + O_2 \xrightarrow{\text{焙烧}} Hg + SO_2$。

汞能与许多金属形成合金，这类合金统称为**汞齐**（亦称汞合金、软银）。

1. **物理性状**：汞齐的物理状态取决于汞的比例，可以是**液态、膏状或固态**。
2. **形成能力**：大多数金属都能与汞形成汞齐，但**铁（Fe）、铂（Pt）、钨（W）和钽（Ta）**等金属是例外，它们不与汞形成汞齐。因此，实验室中常用铁罐来盛装和运输汞。
3. **常见的汞齐及其应用**：
    - **牙科汞齐（Dental Amalgam）**：由汞（约 50%）与银、铜、锡等金属组成的合金，长期用于牙齿填充，具有寿命长、安全性高的特点。
    - **钠汞齐（Sodium Amalgam）**：将金属钠溶于汞中（过程放热），常用作有机合成中的**温和还原剂**，且比纯钠更易于安全操作。
    - **铝汞齐（Aluminum Amalgam）**：汞能破坏铝表面的致密氧化膜保护层，防止其钝化，使铝能持续与空气或水反应，保持高活性。
    - **金汞齐（Gold Amalgam）**：金箔接触液态汞会自发溶解，此性质在古代曾用于提金术或镀金工艺。
    - **铊汞齐**：其凝固点低至 **-58 °C**，可用于制造耐低温的低温温度计。
    - **锌汞齐**：过去用于干电池的锌片表面处理，以防止锌的自氧化腐蚀。

汞及其化合物（尤其是**有机汞**）对生物体具有极强的毒性。

1. **水俣病（Minamata Disease）**：是由于人类食用被**甲基汞**等有机汞污染的鱼贝类而引起的中枢神经系统疾病。汞具有脂溶性，易在生物链中富集。
2. **重金属中毒急救**：若误食汞盐等重金属盐，应立即服用大量**鲜牛奶、豆浆或蛋清**，利用蛋白质变性原理结合重金属离子，减少人体吸收。
3. **泄漏处理**：若汞撒落在地面，应立即撒上**硫粉**（Sulfur powder）。汞与硫在常温下即可反应生成极难溶且毒性极低的黑色**硫化汞（HgS）**，从而防止汞蒸气挥发：$Hg + S = HgS$。
4. **环境防治**：废旧电池（如纽扣电池）含有汞、镉等重金属，严禁随意丢弃，必须分类回收处理，以防污染土壤和水源。

在高级有机合成中，汞盐具有重要用途：

- **羟汞化 - 脱汞反应（Oxymercuration-Demercuration）**：使用乙酸汞 $[Hg(OAc)_2]$ 与烯烃反应，随后用硼氢化钠 $(NaBH_4)$ 还原，可以将烯烃转化为符合马氏规则的醇，且不发生重排。
- **氨汞化反应（Aminomercuration）**：类似于羟汞化，用于在底物中引入氨基。

## 钛（Ti）

- **钛白粉（$\ce{TiO2}$）工业**：
  $\ce{FeTiO3}$（钛铁矿）与硫酸反应，或通过氯化法制备。氯化法：$\ce{TiO2 + 2C + 2Cl2 ->[高温] TiCl4 + 2CO}$，随后 $\ce{TiCl4}$ 氧化。

- **海绵钛制备（克罗尔法）**：
  $\ce{TiCl4 + 2Mg ->[高温，Ar] Ti + 2MgCl2}$。

## 钒（V）

- **催化应用**：$\ce{V2O5}$ 是接触法制硫酸中 $\ce{SO2}$ 氧化的核心催化剂。

- **变价与颜色**：
  $\ce{VO2^+}$（黄，+5）$\rightarrow$ $\ce{VO^2+}$（蓝，+4）$\rightarrow$ $\ce{V^3+}$（绿，+3）$\rightarrow$ $\ce{V^2+}$（紫，+2）。

## 铬（Cr）

- **平衡与颜色**：
  $\ce{2CrO4^2-}$（黄）$+ \ce{2H+ <=> Cr2O7^2-}$（橙）$+ \ce{H2O}$。

- **强氧化性**：$\ce{Cr2O7^2- + 14H+ + 6e- -> 2Cr^3+ + 7H2O}$。

- **工业除杂**：$\ce{Cr^6+}$ 具有强毒性，工业废水处理需先将其还原为 $\ce{Cr^3+}$，再调 pH 生成 $\ce{Cr(OH)3}$ 沉淀。

## 钴（Co）与镍（Ni）

- **钴**：变色硅胶中含有 $\ce{CoCl2}$，无水时显蓝色，吸水变为粉红色 $\ce{CoCl2*6H2O}$。

- **镍**：工业上常用羰基镍法（Mond 过程）提纯镍：$\ce{Ni + 4CO <=> Ni(CO)4}$。镍也是常用的加氢催化剂（如雷尼镍）。

## 铜（Cu）与锌（Zn）

- **铜的冶炼**：

- 火法炼铜：$\ce{Cu2S + O2 ->[高温] 2Cu + SO2}$。

    - 电解精炼：粗铜作阳极，纯铜作阴极。

- **锌的工业**：

- 湿法炼锌：$\ce{ZnS}$ 焙烧生成 $\ce{ZnO}$，酸浸后电解。

    - **两性性质**：$\ce{Zn(OH)2}$ 与 $\ce{Al(OH)3}$ 类似，既能溶于酸也能溶于强碱：$\ce{Zn(OH)2 + 2OH- -> [Zn(OH)4]^2-}$。

## 稀有金属

稀土元素（“冶金工业的维生素”）：元素周期表中原子序数从 57-71（从镧至镥，称为镧系元素）的 15 种元素以及钪和钇，共 17 种元素。

稀土金属广泛应用在冶金、石油化工、材料工业（电子材料、荧光材料、发光材料、永磁材料、超导材料、染色材料、纳米材料、引火合金和催化剂等）、医药及农业等领域。

我国的稀土储量、稀土产量、稀土用量和稀土出口量均居世界第一位。我国化学家徐光宪院士与其研究团队在稀土元素的分离及应用中作出了重要贡献。

## 其他过渡元素

### 高锰酸钾的还原产物

高锰酸钾（$\ce{KMnO4}$）氧化有机物时，其中的锰元素初始化合价为 $+7$ 价，还原产物**取决于反应体系的酸碱度**。

**酸性条件（最常用）**

在酸性溶液中（通常用稀硫酸酸化），高锰酸钾的氧化性最强。

- **还原产物**：$\ce{Mn^2+}$（无色，极浅的粉红色）。

- **化合价变化**：$+7 \rightarrow +2$。

- **半反应式**：

    $$
    \ce{MnO4- + 8H+ + 5e- -> Mn^2+ + 4H2O}
    $$

- **应用**：氧化烯烃生成羧酸或酮、氧化醇生成醛或酸、滴定分析等。

    示例反应：

    <center markdown>
    $\ce{2MnO4- + 5H2C2O4 + 6H+ -> 2Mn^2+ + 10CO2 ^ + 8H2O}$
    </center>

**中性或微碱性条件**

不加酸酸化，或者在弱碱性环境中进行反应。

- **还原产物**：$\ce{MnO2}$（棕褐色沉淀，不溶于水）。

- **化合价变化**：$+7 \rightarrow +4$。

- **半反应式**：

    $$
    \ce{MnO4- + 2H2O + 3e- -> MnO2 v + 4OH-}
    $$

- **应用**：**贝耶尔试验**（Baeyer test）检验碳碳双键。冷稀的高锰酸钾水溶液将烯烃氧化为邻二醇，自身褪色并生成棕褐色的 $\ce{MnO2}$ 沉淀。

    示例反应：

    <center markdown>
    $\ce{3R-CH=CH2 + 2MnO4- + 4H2O -> 3R-CH(OH)-CH2OH + 2MnO2 v + 2OH-}$
    </center>

**强碱性条件**

在氢氧化钠或氢氧化钾浓度较高的强碱性溶液中。

- **还原产物**：$\ce{MnO4^2-}$（锰酸根，墨绿色溶液）。

- **化合价变化**：$+7 \rightarrow +6$。

- **半反应式**：

    $$
    \ce{MnO4- + e- -> MnO4^2-}
    $$

- 如果体系中存在过量的还原剂或随着反应时间的推移，绿色的锰酸根可能会进一步被还原为棕褐色的 $\ce{MnO2}$ 沉淀。

**总结**

| 反应条件 | 还原产物 | 化合价变化 | 典型现象 |
| :-: | :-: | :-: | :-: |
| 酸性 | $\ce{Mn^2+}$ | $+7 \rightarrow +2$ | 溶液褪色（紫红 $\to $ 无色） |
| 中性 / 弱碱性 | $\ce{MnO2}$（沉淀） | $+7 \rightarrow +4$ | 生成棕褐色沉淀 |
| 强碱性 | $\ce{MnO4^2-}$ | $+7 \rightarrow +6$ | 溶液由紫变绿 |
