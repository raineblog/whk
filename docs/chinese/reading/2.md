# 红楼梦

## 各章内容概括

| 章序号 | 章标题                                   | 内容概况                                                                                                                                                                                                 |
|--------|------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 1      | 甄士隐梦幻识通灵 贾雨村风尘怀闺秀       | 甄士隐在梦中见到一僧一道，听闻通灵宝玉的来历，醒来后遇见贾雨村。贾雨村因家贫无依，寄居庙中，与甄士隐相识并受到资助。甄士隐的女儿英莲被拐，家中失火，他看破红尘随僧道而去，贾雨村则赴京赶考。             |
| 2      | 贾夫人仙逝扬州城 冷子兴演说荣国府       | 贾雨村得官后因贪酷被革职，偶遇冷子兴。冷子兴向他讲述贾府家族兴衰及主要人物关系，提到贾宝玉天生异禀、衔玉而生的奇事，并透露贾府内部矛盾重重，预示其衰败命运，为后续故事埋下伏笔。                     |
| 3      | 托内兄如海荐西宾 接外孙贾母怜孤女       | 林黛玉母亲病逝，父亲林如海将她送往贾府寄养。贾母疼爱外孙女黛玉，安排她入住潇湘馆。黛玉初入贾府，敏感多疑，与宝玉初次见面便心生亲近。同时，贾雨村受林如海推荐成为黛玉的老师，进入贾府圈子。             |
| 4      | 薄命女偏逢薄命郎 葫芦僧乱判葫芦案       | 贾雨村审理一起人命官司，原告冯渊被恶霸薛蟠打死。贾雨村本欲秉公执法，但因门子献“护官符”，最终徇私枉法，草草结案。此案揭示了封建社会权贵横行、司法腐败的现象，也为薛蟠一家搬入贾府铺垫背景。               |
| 5      | 贾宝玉神游太虚境 警幻仙曲演《红楼梦》   | 宝玉梦游太虚幻境，警幻仙子引导他参观金陵十二钗的命运册子，并聆听《红楼梦》仙曲。这些预言暗示了众女子的悲剧结局，尤其是黛玉和宝钗的命运。警幻还让宝玉体验男女之情，希望点化他，却未能改变他的痴性。         |
| 6      | 贾宝玉初试云雨情 刘姥姥一进荣国府       | 宝玉从梦中醒来后与袭人发生亲密关系，开始对情感有更深刻的认识。与此同时，刘姥姥为求救济首次来到荣国府，通过周瑞家的帮助见到了王熙凤。凤姐表面客气，实则敷衍，展现了贾府上层对底层人的冷漠态度。             |
| 7      | 送宫花贾琏戏熙凤 赴家宴宝玉会秦钟       | 周瑞家的奉命分送宫花给各位小姐，途中窥见贾琏与凤姐调笑。宝玉受邀参加宁国府宴会，结识秦可卿之弟秦钟，两人一见如故。宝玉羡慕秦钟读书的机会，回家后央求父亲让他也去学堂念书，为下一回情节发展做铺垫。           |
| 8      | 贾宝玉奇缘识金锁 薛宝钗巧合认通灵       | 宝玉与宝钗正式相见，发现彼此佩戴的金锁与通灵宝玉竟似天作之合。众人议论纷纷，认为这是命中注定的姻缘。然而，宝玉心中仍倾慕黛玉，对金玉良缘不以为然。这一回进一步刻画了宝钗端庄贤淑的形象及其复杂性格。           |
| 9      | 训劣子李贵承申饬 嗔顽童茗烟闹书房       | 宝玉在学堂调皮捣蛋，惹怒塾师贾代儒，被严厉训斥。茗烟仗着宝玉宠爱，在学堂大闹，引发混乱。宝玉虽受责罚，却依然我行我素，表现出他对传统教育的抗拒以及叛逆个性。此回还描写了学堂中的种种荒唐景象。             |
| 10     | 金寡妇贪利权受辱 张太医论病细穷源       | 秦可卿患病，贾珍请来名医张友士诊治。张太医详细分析病因，提出治疗方案，但秦氏病情未见好转。另一边，金寡妇因儿子金荣在学堂争执之事求助于贾府，反遭羞辱，反映出底层人在贵族家庭面前的卑微地位。                 |
| 11     | 庆寿辰宁府排家宴 见熙凤贾瑞起淫心       | 宁国府为贾敬庆寿，贾母率众人前往祝贺。宴会上热闹非凡，贾瑞却对凤姐起了歹意，频频献媚。凤姐表面上不动声色，内心已萌生报复之意。此回展示了贾府奢华的生活场景，同时也埋下了贾瑞悲剧命运的种子。                   |
| 12     | 王熙凤毒设相思局 贾天祥正照风月鉴       | 凤姐设计引诱贾瑞上钩，假意答应与他幽会，实则百般折磨。贾瑞沉迷美色，屡次落入圈套，最终染病不起。道士赠予风月宝鉴，告诫他只可背面照，不可正面观，但他违背嘱咐，终致丧命。此回讽刺了纵欲者的悲惨下场。           |
| 13     | 秦可卿死封龙禁尉 王熙凤协理宁国府       | 秦可卿突然病逝，贾珍悲痛万分，不惜重金操办丧事，甚至捐官以提升规格。凤姐受托协助管理宁国府事务，展现卓越才能，整顿秩序井然。此回既表现了贾府的奢靡风气，又凸显了凤姐精明强干的一面。                       |
| 14     | 林如海捐馆扬州城 贾宝玉路谒北静王       | 林如海病逝，黛玉返回扬州奔丧。期间，贾宝玉随贾政拜见北静王，得到对方赏识。北静王赠送礼物，与宝玉结下友谊。此回不仅推动了林黛玉身世的发展，还为后来北静王与贾府的互动奠定基础。                           |

| 16     | 贾元春才选凤藻宫 秦鲸卿夭逝黄泉路       | 元春被选入宫，封为凤藻宫尚书，为贾府带来荣耀。然而，秦钟因病早逝，令人唏嘘不已。此回形成鲜明对比：一边是家族的显赫风光，另一边则是青春生命的陨落，寓意着繁华背后的悲哀。                               |

| 19     | 情切切良宵花解语 意绵绵静日玉生香       | 宝玉与黛玉共度温馨时光，彼此倾诉衷肠。黛玉劝慰宝玉不要过于痴迷，以免耽误学业；宝玉则表达对黛玉的深情。此回细腻刻画了二人的纯真爱情，同时也展现了他们性格中的互补之处。                                 |
