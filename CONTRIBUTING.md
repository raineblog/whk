# 贡献指南 (Contributing Guide)

感谢你对 **whk** 项目的关注！这个项目致力于整理高质量、结构清晰的高中理科笔记。虽然项目主要由个人维护，但我们非常欢迎来自社区的纠错、补充和建议，以共同维护一个专业的知识库。

## 🚀 如何参与

### 1. 提交 Issue

如果你发现了以下问题，请优先提交 [Issue](https://github.com/raineblog/whk/issues/new/choose)：

-   **学术错误**：公式错误、概念偏差或逻辑漏洞。
-   **文档错误**：错别字、错误的标点符号或排版问题。
-   **功能建议**：对文档结构、呈现方式或技术设施的改进建议。

### 2. 提交 Pull Request

如果你希望直接修改代码或内容：

1.  **Fork** 本仓库并克隆到本地。
2.  **创建分支**：建议从 `main` 分支切出一个具有描述性的功能分支（例：`fix/typo-physics`）。
3.  **遵循规范**：修改过程中请务必遵守下文中的 [格式规范](#-格式规范)。
4.  **提交 PR**：确保你的 commits 信息清晰明了。

## 🎨 格式规范

为了保证笔记的专业性和统一性，所有贡献必须严格遵循以下规则：

### 通用排版

-   **中英文间距**：在中文和数字、英文单词、数学公式之间必须添加空格。

    -   ✅ `2025 年`、`公式 $a + b = c$`、`使用 Python 构建`
    -   ❌ `2025年`、`公式$a+b=c$`、`使用Python构建`

-   **空行管理**：段落、列表、行间公式、HTML 块之间应留有足够的空行，以保证渲染后的呼吸感。

### 数学公式与符号

-   **公式环境**：尽量使用行间公式 `$$ ... $$`，并确保 `$$` 与公式内容分行书写。
-   **物理单位**：物理量及单位建议使用 `\pu` 命令，例如 $\pu{1kg}$、$\pu{9.8m/s^2}$。
-   **化学式**：化学方程式及物质建议使用 `\ce` 命令，例如 $\ce{H2 + O2 ->[点燃] H2O}$。
-   **公式内中文**：数学公式内部应避免直接出现中文。若必需，请使用 `<center>` 标签包裹。

    <center>
    输入 $=$ 输出（有效）$+$ 损耗（无效）
    </center>

### 标题层级

-   整个文档仅允许一个一级标题（`#`）。
-   目录最多展示至三级标题（`###`）。

## 报告学术/内容安全性问题

虽然本项目不涉及传统意义上的软件安全漏洞，但我们非常重视内容的 **科学性** 和 **准确性**。

如果你发现：
- 严重的学术错误可能误导他人。
- 引用内容涉及侵权。
- 文档中存在不当内容。

请通过以下方式联系：
- 开启一个 [Issue](https://github.com/raineblog/whk/issues/new)。
- 或者发送邮件至项目维护者（如有）。

感谢你为维护学术严谨性做出的支持！

## 🛠️ 开发环境

本项目基于 [MkDocs Material](https://squidfunk.github.io/mkdocs-material/) 构建。推荐使用 Docker 以获得一致的开发环境：

```bash
# 本地预览 (实时更新)
make serve

# 手动构建静态站点
make build
```

## 📝 提交规范

我们鼓励透明且有意义的 commit 记录，建议遵循以下前缀：

-   `docs:` 修改文档内容（增加笔记、修复错误）。
-   `style:` 改进排版格式，不涉及学术内容修改。
-   `feat:` 增加新的基础设施功能。
-   `fix:` 修复项目本身的技术性 Bug。

感谢你的每一份贡献！✨
