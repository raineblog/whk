# 氢氰酸、氰酸与异氰酸的比较

## 一、分子结构差异

| 化合物 | 分子式 | 结构式 | 分子量 |
|--------|--------|--------|--------|
| 氢氰酸 | HCN | H—C≡N | 27.03 |
| 氰酸 | HOCN | H—O—C≡N | 43.03 |
| 异氰酸 | HNCO | H—N=C=O | 43.03 |

> **关键区别**：氰酸和异氰酸是**同分异构体**，而氢氰酸结构完全不同。

---

## 二、物理性质对比

| 性质 | HCN | HOCN（氰酸） | HNCO（异氰酸） |
|------|-----|-------------|---------------|
| 外观 | 无色液体/气体 | 无色液体 | 无色液体 |
| 气味 | 苦杏仁味 | 刺激性 | 刺激性 |
| 沸点 | 25.7°C | ~23°C | ~23°C |
| 熔点 | -13.4°C | -86°C | -86°C |
| 溶解性 | 与水任意比例混溶 | 可溶于水 | 可溶于水 |
| 稳定性 | 相对稳定 | **极不稳定**，易三聚 | 相对稳定 |

---

## 三、化学性质差异

### 1. 酸性强度
```
HCN (pKa ≈ 9.2)  <  HOCN (pKa ≈ 3.7) ≈ HNCO (pKa ≈ 3.7)
   弱酸                中等强度酸
```

### 2. 稳定性与反应性

**HCN（氢氰酸）**
- 容易发生聚合反应（储存时需加稳定剂）
- 与醛、酮发生加成反应（氰醇合成）
- 易与金属形成稳定配合物

**HOCN（氰酸）**
- 极不稳定，室温下迅速三聚：
$$3 \text{HOCN} \rightarrow \text{C}_3\text{H}_3\text{N}_3\text{O}_3 \text{（氰尿酸）}$$
- 通常以盐的形式（如KOCN）稳定存在

**HNCO（异氰酸）**
- 相对稳定，但仍会缓慢聚合
- 亲电性强，易与含活泼氢的化合物反应

### 氢氰酸（HCN）

|------|----------|
| **工业合成** | 丙烯腈（ABS塑料、合成纤维）|

|------|----------|
| **有机合成** | 合成脲类化合物、医药中间体 |

### 异氰酸及其酯类

| 领域 | 具体应用 |
|------|----------|

## 五、安全注意事项

| 化合物 | 毒性等级 | 主要危害 |
|--------|----------|----------|
| HCN | **剧毒** | 抑制细胞色素c氧化酶，致死量约50mg |
| HOCN/盐 | 中等毒性 | 刺激性，遇酸释放HCN |
| HNCO | 中等毒性 | 刺激呼吸道和皮肤 |

> ⚠️ **警示**：这些化合物都需要在专业通风条件下操作，并配备相应的解毒剂（如亚硝酸钠-硫代硫酸钠疗法用于HCN中毒）。

**氢氰酸 (HCN)、氰酸 (HCNO) 和 异氰酸 (HNCO) 的比较**  
（下面以“酸‑盐‑衍生物”体系为主线，先说明结构、再列出性质、应用，最后给出安全提示。）

---

## 1. 结构与命名

| 化合物 | 分子式 | 结构式（最常见的形式） | 互变异构 |
|--------|--------|-----------------------|----------|
| **氢氰酸** (Hydrogen cyanide) | HCN | H–C≡N（线性分子，sp‑杂化） | – |
| **氰酸** (Cyanic acid) | HCNO | HO–C≡N（羟基‑氰，HO‑C≡N） | ⇌ HNCO（互变异构） |
| **异氰酸** (Isocyanic acid) | HNCO | H–N=C=O（氮上带氢，N=C=O） | ⇌ HCNO（互变异构） |

- **互变异构**：在气相或非极性介质中，HCNO 与 HNCO 能互相转化，能量差约为 10 kJ mol⁻¹，绝大多数实验表明 HNCO 更为稳定，因而常以 HNCO 为“单体”使用。  
- **离子形式**：  
  - HCN ⇌ H⁺ + CN⁻（氰离子）  
  - HCNO/HNCO ⇌ H⁺ + OCN⁻（氰酸根）  

---

## 2. 物理性质（室温≈20 °C）

| 性质 | HCN | HCNO | HNCO |
|------|-----|------|------|
| **聚集状态** | 无色液体（bp 25.6 °C）→气体 | 无色气体（bp ≈ 23 °C）→液体 | 无色液体（bp ≈ 71 °C） |
| **熔点** | –13.3 °C | ≈ –13 °C | –86 °C |
| **沸点** | 25.6 °C | 23 °C (可快速挥发) | 71 °C |
| **密度 (20 °C, 液)** | 0.687 g cm⁻³ | ≈ 0.93 g cm⁻³ (估算) | 1.14 g cm⁻³ |
| **气味** | 典型苦杏仁味 | 刺激性、类似氢氰酸但更刺鼻 | 刺激性、刺鼻 |
| **水中溶解度** | 与水任意比例混溶（约 10 % w/w 在 20 °C） | 可溶，但在水中快速水解 → NH₃ + CO₂ | 同 HCNO，易水解 → NH₃ + CO₂ |
| **在有机溶剂** | 可溶于醇、醚、酮 | 易溶于醇、醚，但因易聚合不易长期保存 | 可溶于醇、醚，同样易聚合 |
| **稳定性** | 在中性/碱性条件下可聚合（生成聚氰） | 在气相或低温下可二聚/三聚 → 氰尿酸 | 同 HCNO，常温下缓慢三聚 → 氰尿酸（C₃N₃(OH)₃） |

> **备注**：由于 HCNO 与 HNCO 极易相互转化且易聚合，文献中常把两者统称为 “氰酸/异氰酸”。实验操作时往往使用 HNCO（液体）作为前体。

---

## 3. 化学性质

| 性质 | HCN | HCNO / HNCO |
|------|-----|------------|
| **酸强度 (pKa)** | 9.2（弱酸） | 2.5 – 3.5（相对 HCN 强） |
| **主要离子** | CN⁻（强亲核） | OCN⁻（氰酸根） |
| **水溶液行为** | 大部分保持分子态，仅少量离解 | 快速水解 → NH₃ + CO₂（生成碳酸铵） |
| **与金属离子** | 形成金属氰配合物（Fe(CN)₆⁴⁻、Au(CN)₂⁻、AgCN 等） | 生成金属氰酸盐 (M(OCN)₂) 与配合物 |
| **与亲电试剂** | 亲核加成：氰醇、腈、酰胺、醛/酮 → 羟基腈 | 与醇 → 碳酰胺（氨基甲酸酯）<br>与胺 → 脲<br>与卤代烃 → 氰酸酯 |
| **聚合倾向** | 在酸/碱或加热下可聚合成聚氰（聚丙烯腈前体） | 在气相/液相中二聚/三聚 → 氰尿酸；热、光可加速 |
| **其他重要反应** | - 与 Ag⁺ 生成 AgCN 沉淀（检测氰离子）<br>- 与 Fe³⁺ → 普鲁士蓝 (Fe₄[Fe(CN)₆]₃)<br>- 与 Cu²⁺ → CuCN | - 与金属盐形成金属氰酸酯（M‑O‑C≡N）<br>- 与甲醛反应生成甲酰胺<br>- 与氨反应生成尿素（NH₂CONH₂）<br>- 在有机合成中作 “异氰酸酯” 前体 |

## 4. 物理化学性质的简要对比表

| 项目 | HCN | HCNO (≈ HNCO) |
|------|-----|---------------|
| **化学式** | HCN | HCNO (HNCO) |
| **结构** | H–C≡N（线性） | H–N=C=O（或 HO–C≡N） |
| **熔点** | –13.3 °C | –86 °C（HNCO） / –13 °C（HCNO） |
| **沸点** | 25.6 °C | 71 °C（HNCO） / 23 °C（HCNO） |
| **密度** | 0.687 g cm⁻³ | 1.14 g cm⁻³（HNCO） |
| **pKa** | 9.2 | 2.5 – 3.5 |
| **主要离子** | CN⁻（强亲核） | OCN⁻（相对弱亲核） |
| **水溶性/水解** | 完全混溶，基本不水解 | 易水解 → NH₃ + CO₂ |
| **稳定性** | 相对稳定，需碱/热才能聚合 | 易聚合成氰尿酸，需低温、惰性气体保存 |
| **主要工业用途** | 丙烯腈、己二腈、氰化钠/钾、金浸出 | 异氰酸酯（TDI/MDI）、聚氨酯、碳酰胺农药、碳酰胺树脂 |
| **典型实验用途** | 金属氰化物合成、氰离子分析、普鲁士蓝 | 金属氰酸盐制备、尿素/酰胺合成、聚异氰酸酯交联、碳酰胺化反应 |

## 5. 在无机化学实验中的典型应用

### 5.1 氢氰酸 (HCN)

| 实验 | 关键反应 | 目的/意义 |
|------|----------|-----------|
| **金属氰配合物的合成** | Fe²⁺ + 6 CN⁻ → [Fe(CN)₆]⁴⁻（亚铁氰）<br>Fe³⁺ + [Fe(CN)₆]⁴⁻ → Fe₄[Fe(CN)₆]₃（普鲁士蓝） | 研究配位化学、磁性、颜色变化 |
| **氰离子滴定** | Ag⁺ + CN⁻ → AgCN↓（白色）<br>2 Ag⁺ + [Fe(CN)₆]⁴⁻ → Ag₃[Fe(CN)₆]↓（红） | 定量分析氰化物、金属离子 |
| **金属沉积** | Cu²⁺ + 4 CN⁻ → [Cu(CN)₄]²⁻（无色）<br>Zn²⁺ + 2 CN⁻ → Zn(CN)₂↓ | 用于电镀或沉淀分离 |
| **氰醇反应** | R–CHO + HCN → R–CH(OH)CN | 合成 α‑羟基腈（有机合成中间体） |
| **金/银氰化浸出** | 4 Au + 8 NaCN + O₂ + 2 H₂O → 4 Na[Au(CN)₂] + 4 NaOH | 矿石中贵金属的提取（工业） |

> **实验技巧**：氢氰酸极易挥发且剧毒，实验必须在通风橱内进行，常用 1 % NaOH 溶液捕集产生的 HCN 气体，或直接使用市售的 0.1 M NaCN 溶液在酸性条件下现场生成 HCN。

### 5.2 氰酸 / 异氰酸 (HCNO / HNCO)

| 实验 | 关键反应 | 目的/意义 |
|------|----------|-----------|
| **金属氰酸盐的合成** | M²⁺ + 2 OCN⁻ → M(OCN)₂ (如 Zn(OCN)₂) | 制备低温磁性材料、催化剂前体 |
| **碳酰胺（氨基甲酸酯）合成** | R–OH + HNCO → R–O–C(O)NH₂（氨基甲酸酯） | 用于聚氨酯预聚体、药物中间体 |
| **尿素/酰胺的制备** | 2 NH₃ + HNCO → NH₂CONH₂（尿素） | 实验室合成尿素、氮肥模型 |
| **酰胺化反应** | R–NH₂ + HNCO → R–NH–C(O)–NH₂（对称尿素） | 合成双取代尿素（药物、农药） |
| **氰尿酸（三聚）** | 3 HNCO → C₃N₃(OH)₃（氰尿酸） | 研究聚合机理、制备阻燃剂、氯稳定剂 |
| **聚异氰酸酯交联** | 多羟基聚合物 + HNCO → 交联聚氨酯网络 | 制备高性能弹性体、泡沫塑料（实验规模） |

> **注意**：氰酸/异氰酸在水中会快速水解，产生氨气和二氧化碳，必须在无水的有机溶剂（如 THF、CH₂Cl₂）或干燥惰性气体环境下操作。

## 6. 工业流程中的主要应用

### 6.1 氢氰酸 (HCN)

| 行业 | 关键工艺 / 产品 | 备注 |
|------|----------------|------|
| **合成纤维** | **丙烯腈**（SOHIO 过程）<br>CH₃CH=CH₂ + 2 NH₃ + 1.5 O₂ → CH₂=CHCN + 3 H₂O | 丙烯腈是尼龙、聚丙烯腈纤维的原料 |
| **塑料/树脂** | **己二腈 (Adiponitrile)**（Dupont Hydro‑cyanation）<br>CH₂=CH–CH=CH₂ + 2 HCN → NC–(CH₂)₄–CN | 己二腈 → 己二胺 → 尼龙‑66 |
| **石化** | **甲基丙烯酸甲酯 (MMA)**（氰醇路线）<br>CH₃COCH₃ + HCN → (CH₃)₂C(OH)CN → MMA | MMA 用于光学树脂、涂料 |
| **冶金** | **氰化钠/氰化钾**（NaCN / KCN）<br>用于金、银的**氰化浸出**：<br>4 Au + 8 NaCN + O₂ + 2 H₂O → 4 Na[Au(CN)₂] + 4 NaOH | 黄金提取的主要工业工艺 |
| **化学试剂** | **氢氰酸本身**作为原料、催化剂、硝化剂、燃气脱硫剂 | 需要专门的储运系统（高压、低温） |

### 6.2 氰酸 / 异氰酸 (HCNO / HNCO)

| 行业 | 关键工艺 / 产品 | 备注 |
|------|----------------|------|
| **聚氨酯** | **二异氰酸酯 (TDI, MDI)** 用来生产聚氨酯泡沫、弹性体、涂料<br>HNCO（通过尿素的裂解或氰尿酸的热解得到）与多元醇反应形成氨基甲酸酯链 | 异氰酸酯是聚氨酯工业的核心单体 |
| **农药 & 医药** | **氨基甲酸酯类**（如 carbaryl、aldicarb）<br>HNCO + 醇/胺 → 氨基甲酸酯 → 再经酰化得到农药 | 高效、低毒的广谱杀虫剂 |
| **树脂/复合材料** | **氰酸酯树脂**（二氰酸酯 + 双酚 A）<br>通过 HCNO 与双酚 A 反应得到双氰酸酯，随后热聚得到耐高温树脂 | 用于航空航天、电子基板 |
| **氯稳定剂** | **氰尿酸**（HCNO 的三聚体）<br>用作游泳池氯的稳定剂，防止光解 | 商品名为 “Cyanuric acid” |
| **肥料** | **氰氨 (Cyanamide)**（NH₂–C≡N）可通过 HCNO 氨化得到，进一步加工为缓释氮肥 | 氰氨肥料可提供氮源并具除草活性 |
| **金属表面处理** | **氰化钠**或**氰化钾**用于金属的**渗碳、渗氮**（氰化处理）<br>氰酸盐在高温下可分解产生活性氮/碳，提升钢的硬度 | 在钢铁热处理行业有重要应用 |
| **合成中间体** | **氨基腈、酰胺、尿素**等。<br>HNCO 与胺反应直接得到对称尿素或不对称尿素，是药物合成的常用路线 | 用于制药、染料、功能高分子的合成 |

## 7. 安全与环境保护

| 物质 | 主要危害 | 防护要点 | 废弃物处理 |
|------|----------|----------|------------|
| **HCN** | - 极强毒性（致死剂量 1–2 mg kg⁻¹）<br>- 皮肤、呼吸道吸收快<br>- 与空气形成可燃混合物 (爆炸下限 5.6 %) | - 必须在负压通风柜或手套箱内操作<br>- 佩戴防毒面具 (过滤 HCN) 与防护手套、防护服<br>- 使用氰离子检测仪监测空气中浓度 (< 5 ppm) | - 用 5 % NaOH 溶液捕集形成 NaCN，后用 10 % H₂O₂ 氧化成 NaOCN（低毒）<br>- 高浓度废液送专业危废处理中心 |
| **HCNO / HNCO** | - 刺激性、腐蚀性；吸入可导致呼吸道炎症、肺水肿<br>- 对皮肤、眼睛有强刺激<br>- 可在低温下自行聚合放热 | - 操作时同样需在通风柜中进行<br>- 防护眼镜、耐化学手套、实验服<br>- 避免与酸或碱混合，以免剧烈放热 | - 废液先用稀碱（NaOH）中和，生成氰酸盐后再用 5 % H₂O₂ 氧化成碳酸盐 + 氨水<br>- 对于聚合产物（氰尿酸）可直接回收或送填埋 |
| **通用** | - 所有氰系化合物在环境中都有持久性，可能转化为更毒的氰化物 | - 实验室产生的所有含氰废液必须分类收集、标识<br>- 在处理前进行现场快速氰离子测定（AgNO₃ 法） | - 废气通过活性炭吸附或碱性洗涤塔<br>- 废渣（氰尿酸）在高温下焚烧或作为工业原料回收 |

## 8. 小结（关键要点）

| 项目 | HCN | HCNO / HNCO |
|------|-----|------------|
| **结构** | H–C≡N（线性，强 C≡N 三键） | HO–C≡N ↔ H–N=C=O（可互变，后者的 N=C=O 双键体系更常见） |
| **酸度** | pKa ≈ 9.2（弱酸） | pKa ≈ 2.5–3.5（相对强酸） |
| **离子** | CN⁻（强亲核） | OCN⁻（相对弱的亲核） |
| **水溶液行为** | 易溶，基本不水解 | 快速水解 → NH₃ + CO₂ |
| **主要工业应用** | 丙烯腈、己二腈、MMA、氰化钠/钾、金银浸出 | 异氰酸酯（TDI/MDI）→聚氨酯、碳酰胺农药、氰尿酸、碳酰胺树脂 |
| **实验室常见用途** | 金属氰化物/配位化学、氰离子滴定、沉积反应 | 金属氰酸盐、尿素/酰胺合成、聚异氰酸酯交联、三聚制氰尿酸 |
| **安全性** | 极高毒性、挥发性、易燃 | 毒性、刺激性、易聚合放热 |
