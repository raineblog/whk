---

要“写出所有的置换反应”在数学上是不可能的，因为元素周期表中s区、p区、d区元素的组合可以产生数以万计的化学方程式。但是，化学是一门有规律的科学。我们可以通过**分类总结**的方式，将所有满足你条件的置换反应（即符合 $A + BC \rightarrow AC + B$ 形式的反应）全部囊括在以下几个大类中。

根据你的要求，我们**排除了同位素、f区元素（镧系和锕系）以及钋(Po)之后的放射性元素**，并且**特别加入了稀有气体的反应**。

以下是所有置换反应的全集分类与经典代表方程式：

---

### 一、 涉及稀有气体的置换反应（特约要求）
虽然稀有气体极难发生反应，但较重的稀有气体（如氪 $Kr$、氙 $Xe$）能形成氟化物和氧化物。在这些反应中，**稀有气体通常作为“被置换”的元素出现**，因为它们的化合物具有极强的氧化性和氟化能力。

**1. 非金属置换稀有气体**
*   **氢气置换氙/氪：**
    $$H_2 + XeF_2 \xrightarrow{\Delta} 2HF + Xe$$
    $$H_2 + KrF_2 \rightarrow 2HF + Kr$$
*   **其他非金属置换氙：** 硫、磷等与二氟化氙反应。
    $$S + 3XeF_2 \rightarrow SF_6 + 3Xe$$

**2. 金属置换稀有气体**
二氟化氙是非常干净的氟化剂，很多d区或p区金属能将其中的氙置换出来：
*   $$Ti + 2XeF_2 \rightarrow TiF_4 + 2Xe$$
*   $$2W + 6XeF_2 \rightarrow 2WF_6 + 6Xe$$

*(注：由于稀有气体单质极难抢夺其他元素的电子或阴离子，因此不存在纯稀有气体去置换其他元素的反应。)*

---

### 二、 金属 置换 金属
这类反应的本质是：**活泼性强的金属将活泼性弱的金属从其盐溶液或熔融物中还原出来。**（依据金属活动性顺序表）

**1. 溶液中的置换（湿法冶金 / 盐溶液反应）**
*   **规则：** 活动性排在前面的金属（排除极活泼的K、Ca、Na，它们会直接和水反应）置换后面的金属。
*   **铁置换铜（水法炼铜）：** $$Fe + CuSO_4 \rightarrow FeSO_4 + Cu$$
*   **铜置换银：** $$Cu + 2AgNO_3 \rightarrow Cu(NO_3)_2 + 2Ag$$
*   **锌置换铅：** $$Zn + Pb(NO_3)_2 \rightarrow Zn(NO_3)_2 + Pb$$

**2. 高温熔融状态下的置换（火法冶金 / 铝热反应）**
*   **铝热反应（铝置换d区过渡金属）：**
    $$2Al + Fe_2O_3 \xrightarrow{高温} Al_2O_3 + 2Fe$$
    $$8Al + 3Mn_3O_4 \xrightarrow{高温} 4Al_2O_3 + 9Mn$$
*   **克罗尔法（Kroll Process，活泼金属置换钛/锆等）：**
    $$2Mg + TiCl_4 \xrightarrow{高温} 2MgCl_2 + Ti$$
    $$4Na + TiCl_4 \xrightarrow{高温} 4NaCl + Ti$$

---

### 三、 金属 置换 非金属
这类反应通常是活泼金属将氢或碳置换出来。

**1. 金属置换氢气（从水中）**
*   **极活泼金属（s区，碱金属/碱土金属）遇冷水：**
    $$2Na + 2H_2O \rightarrow 2NaOH + H_2 \uparrow$$
    $$Ca + 2H_2O \rightarrow Ca(OH)_2 + H_2 \uparrow$$
*   **较活泼金属（d区前段或p区）遇高温水蒸气：**
    $$3Fe + 4H_2O(g) \xrightarrow{高温} Fe_3O_4 + 4H_2$$
    $$Mg + H_2O(g) \xrightarrow{高温} MgO + H_2$$

**2. 金属置换氢气（从酸中）**
*   **规则：** 排在氢(H)之前的金属置换非氧化性酸（如HCl、稀H2SO4）中的氢。
*   $$Zn + 2HCl \rightarrow ZnCl_2 + H_2 \uparrow$$
*   $$2Al + 3H_2SO_4 \rightarrow Al_2(SO_4)_3 + 3H_2 \uparrow$$

**3. 金属置换碳（从二氧化碳中）**
*   镁条在干冰/二氧化碳中燃烧（经典的“灭火器失效”反应）：
    $$2Mg + CO_2 \xrightarrow{点燃} 2MgO + C$$

---

### 四、 非金属 置换 非金属
非金属单质依据其电负性或氧化性/还原性的强弱，互相置换。

**1. 卤素单质间的置换**
*   **规则：** 氧化性 $F_2 > Cl_2 > Br_2 > I_2$。
*   **氯气置换溴/碘：**
    $$Cl_2 + 2NaBr \rightarrow 2NaCl + Br_2$$
    $$Cl_2 + 2KI \rightarrow 2KCl + I_2$$
*   **溴置换碘：**
    $$Br_2 + 2KI \rightarrow 2KBr + I_2$$

**2. 卤素置换氧气/硫**
*   **氟气置换水中的氧气（特例）：**
    $$2F_2 + 2H_2O \rightarrow 4HF + O_2$$
*   **卤素置换硫化氢中的硫：**
    $$Cl_2 + H_2S \rightarrow 2HCl + S \downarrow$$

**3. 氧气置换硫/碘**
*   **氧气置换硫（不完全燃烧）：**
    $$O_2 + 2H_2S \rightarrow 2H_2O + 2S \downarrow$$
*   **氧气置换碘（在酸性环境中）：**
    $$O_2 + 4HI \rightarrow 2H_2O + 2I_2$$

**4. 碳置换其他非金属（或类金属）**
*   **水煤气反应（碳置换水中的氢）：**
    $$C + H_2O(g) \xrightarrow{高温} CO + H_2$$
*   **碳置换硅（工业制粗硅）：**
    $$2C + SiO_2 \xrightarrow{高温} 2CO + Si$$

---

### 五、 非金属 置换 金属
这类反应广泛应用于工业冶金，通常是利用廉价的非金属还原剂（碳、氢气）在高温下夺取金属氧化物中的氧。

**1. 碳置换金属（碳还原法）**
*   **炼铜/炼锌/炼铁等（d区及p区金属）：**
    $$C + 2CuO \xrightarrow{高温} 2Cu + CO_2 \uparrow$$
    $$C + ZnO \xrightarrow{高温} Zn + CO \uparrow$$

**2. 氢气置换金属（氢还原法）**
氢气常用于还原纯度要求较高的金属，或者还原性不如碳强的体系。
*   **氢气还原氧化铜/氧化钨：**
    $$H_2 + CuO \xrightarrow{\Delta} Cu + H_2O$$
    $$3H_2 + WO_3 \xrightarrow{高温} W + 3H_2O$$

---

### 总结
只要符合以上**5个大类、12个小类**的反应规则，且反应物为“单质 + 化合物”，产物也是“单质 + 化合物”，那么（在不考虑放射性元素和f区元素的前提下）你就可以通过在周期表中挑选相应的元素，无限制地写出成百上千个合理的置换反应方程式。
