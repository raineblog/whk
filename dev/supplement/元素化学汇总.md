---

## 一、前三周期简述

### 第一周期

- **H(氢)**：最轻元素，工业制氨、石油加工的重要原料
- **He(氦)**：惰性气体，低温冷却剂

### 第二周期

- **Li(锂)**：锂电池材料
- **C(碳)**：有机化学基础
- **N(氮)**：制氨工业、硝酸工业核心
- **O(氧)**：氧化剂、助燃剂
- **F(氟)**：氟化工、制冷剂原料

### 第三周期

- **Na(钠)**：氯碱工业基础
- **Mg(镁)**：海水提镁工业
- **Al(铝)**：电解铝工业，两性金属
- **Si(硅)**：半导体、玻璃工业
- **P(磷)**：磷肥工业
- **S(硫)**：硫酸工业
- **Cl(氯)**：氯碱工业、漂白剂

## 二、第四周期（重点）

### 主族元素

#### **K(钾)、Ca(钙)**

- K：比Na更活泼，KOH强碱，钾肥重要
- Ca：石灰石(CaCO₃)→生石灰(CaO)→熟石灰[Ca(OH)₂]工业链

### 第一过渡系列（重要）

#### **Ti(钛)**

- 钛白粉(TiO₂)工业：FeTiO₃(钛铁矿)→TiCl₄→TiO₂
- 海绵钛制备：TiCl₄ + 2Mg → Ti + 2MgCl₂

#### **V(钒)**

- V₂O₅催化剂：接触法制硫酸
- 钒的变价：VO₂⁺(黄)→VO²⁺(蓝)→V³⁺(绿)→V²⁺(紫)

#### **Cr(铬)**

- 重铬酸钾氧化性：Cr₂O₇²⁻ + 14H⁺ + 6e⁻ → 2Cr³⁺ + 7H₂O
- 铬酸根-重铬酸根平衡：2CrO₄²⁻ + 2H⁺ ⇌ Cr₂O₇²⁻ + H₂O
- 铬的颜色：Cr³⁺(绿)、CrO₄²⁻(黄)、Cr₂O₇²⁻(橙)

#### **Mn(锰)**

- KMnO₄强氧化剂：酸性、中性、碱性条件下产物不同
- 软锰矿(MnO₂)制备KMnO₄：MnO₂ → K₂MnO₄ → KMnO₄
- 锰的价态：Mn²⁺(浅粉)、MnO₄²⁻(绿)、MnO₄⁻(紫)

#### **Fe(铁)**

- 炼铁工业：Fe₂O₃ + 3CO → 2Fe + 3CO₂
- Fe²⁺/Fe³⁺转化：Fe²⁺易被氧化，Fe³⁺可被还原
- 铁的配合物：Fe(SCN)³⁺(血红)、[Fe(CN)₆]⁴⁻(黄血盐)

#### **Co(钴)、Ni(镍)**

- Co：变色硅胶(CoCl₂)，维生素B₁₂含钴
- Ni：Ni(CO)₄羰基镍提纯，镍氢电池

#### **Cu(铜)**

- 火法炼铜：Cu₂S + O₂ → 2Cu + SO₂
- 电解精炼铜：粗铜作阳极
- Cu⁺/Cu²⁺：CuCl难溶白色，Cu²⁺常见蓝色

#### **Zn(锌)**

- 湿法炼锌：ZnS焙烧→ZnO酸浸→电解
- 两性：Zn(OH)₂ + 2OH⁻ → [Zn(OH)₄]²⁻

### 主族元素（续）

#### **Ga(镓)、Ge(锗)**

- 半导体材料，伴生矿产品

#### **As(砷)**

- As₂O₃(砒霜)有毒
- As₂S₃(雄黄)、As₄S₄(雌黄)

#### **Se(硒)**

- 半导体材料，硒鼓

#### **Br(溴)**

- 海水提溴：Cl₂ + 2Br⁻ → Br₂ + 2Cl⁻
- 有机溴代反应

## 三、第五周期（重点）

### 过渡金属

#### **Mo(钼)**

- 钼酸铵[(NH₄)₂MoO₄]分析试剂
- 钼蓝检测磷

#### **Ag(银)**

- 银镜反应：Ag(NH₃)₂⁺ + e⁻ → Ag↓
- AgCl光分解，感光材料
- 溶解性：AgCl < AgBr < AgI

#### **Cd(镉)**

- 镉污染问题
- CdS黄色颜料

### 主族元素

#### **Sn(锡)**

- SnCl₂还原剂
- 锡的两性：Sn(OH)₂

#### **Sb(锑)**

- 锑白(Sb₂O₃)阻燃剂
- SbCl₃水解：SbCl₃ + H₂O → SbOCl↓ + 2HCl

#### **I(碘)**

- 海带提碘工业流程
- I₂ + I⁻ → I₃⁻络离子
- 碘的歧化：3I₂ + 6OH⁻ → IO₃⁻ + 5I⁻ + 3H₂O

## 四、第六周期（重点）

### 镧系元素

- **稀土元素**：分离困难，萃取分离技术
- Ce(铈)：CeO₂抛光粉，Ce⁴⁺/Ce³⁺氧化还原
- Nd(钕)：钕铁硼永磁材料

### 过渡金属

#### **W(钨)**

- 钨酸钠Na₂WO₄
- 仲钨酸铵(APT)工业中间体

#### **Pt(铂)**

- 催化剂：汽车尾气净化
- 王水溶解：生成H₂PtCl₆

#### **Au(金)**

- 氰化法提金：4Au + 8CN⁻ + O₂ + 2H₂O → 4[Au(CN)₂]⁻ + 4OH⁻
- 王水溶金：Au + HNO₃ + 4HCl → HAuCl₄ + NO + 2H₂O

#### **Hg(汞)**

- Hg₂²⁺二价汞离子
- 汞齐(汞合金)

### 主族元素

#### **Pb(铅)**

- PbO₂强氧化剂
- 铅蓄电池：Pb + PbO₂ + 2H₂SO₄ ⇌ 2PbSO₄ + 2H₂O

#### **Bi(铋)**

- 低熔点合金
- BiOCl珠光颜料

## 五、锕系元素

### **U(铀)**

- 核燃料：U-235裂变
- 铀的提取：沥青铀矿→黄饼(U₃O₈)→UF₆
- UO₂²⁺铀酰离子

### 其他锕系

- Th(钍)：钍基熔盐堆
- Pu(钚)：核武器材料

## 工业流程题常见考点总结

1. **氧化还原调节**：Cr、Mn、Fe变价
2. **pH调节沉淀**：Fe³⁺、Al³⁺分离
3. **萃取分离**：稀土元素、贵金属
4. **电解工艺**：Al、Cu、Zn精炼
5. **络合反应**：Ag、Au、Cu配合物
6. **焙烧工艺**：硫化物→氧化物
7. **酸碱两性**：Al、Zn、Sn、Pb的氢氧化物好的，这是一个很有价值的请求。元素周期表是化学的基石，但课本上的知识往往比较零散。我将按照你的要求，以前6周期为主线，结合高中化学（特别是高考工业流程题）的常见考点，为你系统性地梳理一遍。

**核心思路：** 高考工业流程题，本质上是考查**元素在真实化学反应中的行为**。重点关注：1）**变价**（氧化还原）；2）**溶解性**（沉淀与分离）；3）**酸碱性**（与酸碱反应）；4）**配合性**（形成配合物）；5）**水解性**（尤其对金属离子）。

### 第一周期（简单提纲）

- **1 H 氢**：同位素（H, D, T）。还原性（与金属生成氢化物）、酸性（H⁺）、碱性（H⁻，如NaH）。**工业流程**：常作为酸提供H⁺，或作为还原剂（如电解水制H₂）。有机化学中的氢化反应。
- **2 He 氦**：惰性，几乎不参与任何反应。只在稀有气体背景或性质比较题中出现。

### 第二周期（简单提纲）

- **3 Li 锂**：最轻金属。**高考热点**：与Mg的**对角线规则**（性质相似，如碳酸锂微溶，磷酸锂难溶，硝酸锂分解产物与Mg类似）。电池材料。
- **4 Be 铍**：**对角线规则**：与Al相似（两性，Be(OH)₂溶于强碱）。剧毒。流程中常作为杂质去除。
- **5 B 硼**：缺电子原子。硼酸(H₃BO₃)为一元弱酸（不是三元）。硼砂（Na₂B₄O₇·10H₂O）是重要的缓冲溶液。**工业流程**：硼镁矿制硼酸。
- **6 C 碳**：CO（还原剂）、CO₂（酸性氧化物）、碳酸盐（沉淀与溶解平衡，如CaCO₃溶垢问题）。
- **7 N 氮**：氨（配体、碱）、硝酸（强氧化剂）。**工业流程**：硝酸盐的氧化性（如在酸性条件下氧化Fe²⁺）。
- **8 O 氧**：氧化物（碱性、酸性、两性、过氧化物、超氧化物）。最普遍氧化剂。
- **9 F 氟**：最强氧化性。HF（弱酸，但能腐蚀SiO₂，用于刻蚀玻璃）。**高考偶有出现**：萤石(CaF₂)与浓硫酸反应制HF。
- **10 Ne 氖**：惰性，不反应。

### 第三周期（简单提纲）

- **11 Na 钠**：强还原剂。Na₂O₂（强氧化剂，与H₂O/CO₂反应生成O₂）。NaOH（强碱）。**工业流程**：钠盐的溶解度（NaCl、Na₂SO₄、Na₂CO₃等）。
- **12 Mg 镁**：Mg(OH)₂（中强碱，难溶）。**工业流程**：从海水中提取Mg（加石灰乳沉淀Mg(OH)₂，加盐酸溶解，电解熔融MgCl₂）。
- **13 Al 铝**：**绝对重点**。**两性**：Al₂O₃和Al(OH)₃溶于强酸和强碱。明矾(KAl(SO₄)₂·12H₂O)净水（Al³⁺水解生成Al(OH)₃胶体）。**工业流程**：铝土矿制Al₂O₃（拜耳法：加NaOH溶解，过滤，通CO₂沉淀Al(OH)₃）。**易形成配合物**：[Al(OH)₄]⁻（四羟基合铝酸盐）。
- **14 Si 硅**：SiO₂（酸性氧化物，但只溶于HF和强热浓碱）。硅酸盐（复杂，常以难溶形式存在）。**工业流程**：除杂时，SiO₂常作为不反应的滤渣被除去。
- **15 P 磷**：P₂O₅（酸性干燥剂，强脱水性）。磷酸（中强酸，可形成多种磷酸盐，如Ca₃(PO₄)₂难溶）。**工业流程**：磷肥工业。
- **16 S 硫**：SO₂（还原性为主，也有氧化性，漂白性）。浓H₂SO₄（强氧化性、脱水性）。硫化物（许多金属硫化物难溶，如CuS、PbS，是沉淀分离的依据）。
- **17 Cl 氯**：Cl₂（强氧化剂，可用于氧化Br⁻、I⁻）。次氯酸盐（强氧化剂，漂白）。**工业流程**：氯碱工业。
- **18 Ar 氩**：惰性，不反应。

### 第四周期（重点开始）

**特点**：**第一过渡系**（Sc到Zn），变价丰富，配合物众多，是工业流程的核心。

- **19 K, 20 Ca**：碱金属/碱土金属。Ca²⁺与F⁻、CO₃²⁻、C₂O₄²⁻（草酸根）形成难溶物。**K**：焰色反应（紫色，需透过蓝色钴玻璃）。
- **21 Sc - 23 V**：**高考偶见**。
    - **Sc**：性质与Al相似（两性）。
    - **Ti**：TiO₂（钛白粉，白色颜料）。TiCl₄（遇水强烈水解，用于制造烟幕弹）。**工业流程**：钛铁矿(FeTiO₃)制取TiO₂，常涉及浓硫酸酸溶和高温水解。
    - **V**：**变价丰富** (+2,+3,+4,+5)。V₂O₅（工业制硫酸的催化剂）。**考点**：VO₂⁺（钒酰离子，黄色）、VO²⁺（蓝色）、V³⁺（绿色）、V²⁺（紫色）。偏钒酸铵(NH₄VO₃)难溶，可用于沉淀钒。
- **24 Cr 铬**：**高频考点**。
    - **变价**：Cr³⁺（紫色/绿色，两性）和Cr₂O₇²⁻/CrO₄²⁻（橙色/黄色，强氧化性）。
    - **重要转化**：2CrO₄²⁻ + 2H⁺ ⇌ Cr₂O₇²⁻ + H₂O；Cr₂O₇²⁻ + H₂O ⇌ 2CrO₄²⁻ + 2H⁺（加酸/碱平衡移动）。
    - **工业流程**：**铬铁矿(FeCr₂O₄)制K₂Cr₂O₇**。核心步骤：高温碱熔（氧化焙烧）将Cr(III)氧化为Cr(VI)（可溶性铬酸盐），然后加酸酸化转化为重铬酸盐。**毒性**：Cr(VI)是致癌物，流程中常涉及还原处理（如加FeSO₄，将Cr(VI)还原为Cr(III)沉淀为Cr(OH)₃）。
- **25 Mn 锰**：**高频考点**。
    - **变价**：最常见是Mn(II)（肉粉色，浅色溶液近无色）和Mn(VII)（MnO₄⁻，紫红色，强氧化剂）。
    - **重要物质**：KMnO₄（酸性条件下氧化性极强，自身被还原为Mn²⁺）。MnO₂（强氧化剂，如与浓HCl制Cl₂，与H₂O₂分解）。
    - **工业流程**：**软锰矿(MnO₂)制KMnO₄或MnSO₄**。常涉及：还原焙烧（将MnO₂还原为MnO，然后用酸溶解得Mn²⁺）或 碱熔氧化（将MnO₂氧化为MnO₄²⁻，再歧化）。
- **26 Fe 铁**：**绝对核心，必考**。
    - **变价**：Fe(II)（浅绿色，还原性，易被空气氧化）和Fe(III)（黄色，氧化性，易水解）。
    - **检验**：Fe³⁺ + SCN⁻ → [Fe(SCN)]²⁺（血红色）；Fe²⁺ + [Fe(CN)₆]³⁻ → 深蓝色沉淀（滕氏蓝）；Fe³⁺ + [Fe(CN)₆]⁴⁻ → 深蓝色沉淀（普鲁士蓝）。
    - **重要化合物**：Fe₃O₄（磁性氧化铁，可写作FeO·Fe₂O₃，含Fe(II)和Fe(III)）。Fe(OH)₂（白色，迅速变灰绿，最后成红褐Fe(OH)₃）。FeS（难溶于稀酸）。
    - **工业流程**：**几乎无处不在**。1）**除杂**：调pH使Fe³⁺水解为Fe(OH)₃沉淀除去。2）**氧化还原**：用Fe²⁺还原某些高价离子（如Cr₂O₇²⁻、Cu²⁺）；用H₂O₂/O₂将Fe²⁺氧化为Fe³⁺以便于沉淀。3）**黄铁矿(FeS₂)制硫酸**。4）**高炉炼铁**（CO还原Fe₂O₃）。
- **27 Co, 28 Ni**：**高考偶见**。性质相似。**考点**：Co²⁺ (粉红色)，Ni²⁺ (绿色)。能形成众多配合物。**工业流程**：常与Fe、Cu共生，在分离流程中作为过渡元素出现，可能涉及萃取分离。
- **29 Cu 铜**：**高频考点**。
    - **变价**：Cu(I)（不稳定，易歧化，Cu₂O红色，CuI白色难溶）和Cu(II)（蓝色）。
    - **重要物质**：CuSO₄·5H₂O（胆矾）。Cu(OH)₂（蓝色，能溶于浓碱形成[Cu(OH)₄]²⁻深蓝色溶液）。**配合物**：[Cu(NH₃)₄]²⁺（深蓝色，用于检验Cu²⁺）。
    - **工业流程**：**辉铜矿(Cu₂S)或黄铜矿(CuFeS₂)制铜**。常涉及：1）**焙烧**：将硫化物转化为氧化物或硫酸盐。2）**浸出**：用稀硫酸（常加氧化剂，如O₂或Fe³⁺）溶解铜。3）**置换**：用Fe置换Cu²⁺得到海绵铜。4）**电解精炼**。
- **30 Zn 锌**：**两性**：Zn(OH)₂溶于强碱生成[Zn(OH)₄]²⁻（锌酸盐）。**工业流程**：闪锌矿(ZnS)制锌（焙烧→酸浸→净化→电解）。常与Cd伴生。用于**置换**贵金属（如Au、Ag）。
- **31 Ga, 32 Ge**：**半导体材料**。**考点**：Ga与Al性质相似（两性），Ge与Si相似。在粉煤灰、铅锌矿中可能作为稀有元素回收。
- **33 As, 34 Se**：**有毒，两性**。As₂O₃（砒霜，剧毒）。H₃AsO₄（砷酸，弱氧化性）。**工业流程**：有色金属冶炼烟气中回收。**考点**：砷化氢(AsH₃)的生成（马氏试砷法）。
- **35 Br, 36 Kr**：Br：常温唯一液态非金属，氧化性介于Cl₂和I₂之间。**工业流程**：海水提溴（氯气氧化→空气吹出→二氧化硫吸收→再氧化）。Kr：惰性。

### 第五周期

**特点**：性质与第四周期同族元素相似，但更稳定（惰性电子对效应开始显现）。

- **37 Rb, 38 Sr**：性质分别同K、Ca。Rb、Cs常用于**光电管**。SrCO₃是红色烟花。工业流程中少见。
- **39 Y**：与镧系性质极相似（“重稀土”），常与镧系元素共生。
- **40 Zr - 43 Tc**：Zr、Nb、Mo、Tc。**Mo**：**高频考点（选考）**。辉钼矿(MoS₂)制钼。**考点**：Mo(VI)形成同多酸和杂多酸（如磷钼酸铵黄色沉淀，用于检验PO₄³⁻）。**Tc**：人造元素，放射性强。
- **44 Ru - 47 Ag**：**铂系金属（Ru,Rh,Pd）和Ag**。
    - **Ag**：**高频考点**。1）**配合物**：[Ag(NH₃)₂]⁺（银氨溶液，用于银镜反应和醛基检验）、[Ag(CN)₂]⁻（剧毒，用于电镀）。2）**沉淀**：AgCl（白，不溶于酸，溶于氨水）、AgBr（淡黄，感光）、AgI（黄，不溶于氨水）、Ag₂S（黑，银器变黑原因）。**工业流程**：从废定影液或阳极泥中回收银（常用置换法，如加Zn或Cu）。
- **48 Cd**：常与Zn伴生。CdS（黄色，难溶）。Cd(OH)₂不溶于过量碱（与Zn不同，用于分离）。剧毒（痛痛病）。
- **49 In, 50 Sn, 51 Sb**：**Sn和Sb**是**高频考点**。
    - **Sn**：**两性**。Sn(II)（强还原性，易被空气氧化为Sn(IV)）。SnCl₂（常用还原剂，易水解，配制时常加盐酸和锡粒）。**考点**：Sn²⁺ + Hg²⁺ → Hg₂Cl₂（白）→ Hg（黑），用于检验Sn²⁺或Hg²⁺。
    - **Sb**：**两性**。Sb(III)易水解生成SbOCl（白色，碱式氯化锑）。**工业流程**：锑精矿(Sb₂S₃)制锑（焙烧→还原熔炼）。
- **52 Te, 53 I**：**I是重点**。I₂（紫黑固体，易升华，淀粉变蓝）。**工业流程**：海带提碘（灰化→水浸→氧化→萃取或离子交换）。I⁻（强还原性，可被H₂O₂、Cl₂等氧化为I₂）。IO₃⁻（碘酸根，氧化性，在酸性条件下可氧化I⁻）。
- **54 Xe**：惰性，但有化合物（如XeF₂，强氟化剂），高中不常见。

### 第六周期

**特点**：**镧系收缩**（导致Zr与Hf、Nb与Ta等性质极相似，难分离）。**惰性电子对效应**：Tl(I)、Pb(II)、Bi(III)更稳定。

- **55 Cs, 56 Ba**：Cs与K相似。**Ba**：**BaSO₄**（钡餐，难溶于酸和水）；**BaCO₃**（剧毒，因溶于胃酸生成Ba²⁺）；检验SO₄²⁻时先加稀盐酸排除Ag⁺、CO₃²⁻、SO₃²⁻、PO₄³⁻等的干扰。
- **57-71 镧系（Ln）**：**“稀土元素”**。**高考高频区域**（特别是选考物质结构或工业流程）。性质极其相似（+3价为主），**分离困难**，常利用**微小溶解度差异**（如分步结晶）或**配合物稳定性差异**（如离子交换、萃取）。**个别元素**：
    - **Ce（铈）**：有+4价，Ce⁴⁺是强氧化剂。可利用此性质将Ce与其它+3价镧系分离。
    - **Eu（铕）、Yb（镱）**：有+2价。
    - **工业流程**：从**独居石、氟碳铈矿**提取稀土。核心步骤：**酸溶**（常加NaOH分解矿物）→ **萃取分离**（如P507萃取剂，利用不同稀土离子与萃取剂结合能力的微小差异）。是工业流程大题绝佳背景。
- **72 Hf - 78 Pt**：**Hf、Ta、W、Re、Os、Ir、Pt**。**W和Pt**是重点。
    - **W（钨）**：**高温金属**。**工业流程**：黑钨矿(Fe,Mn)WO₄制钨。核心：**碱熔**生成Na₂WO₄，然后加酸生成H₂WO₄（黄钨酸，难溶），再热分解为WO₃，最后用H₂还原得W粉。
    - **Pt（铂）**：惰性，抗腐蚀。**王水**（浓HNO₃+浓HCl，体积比1:3）能溶解Pt和Au，形成氯铂酸(H₂PtCl₆)或氯金酸(HAuCl₄)。
- **79 Au, 80 Hg**：**Au和Hg**。
    - **Au**：**王水溶解**（见上）。**氰化法提金**：4Au + 8NaCN + O₂ + 2H₂O → 4Na[Au(CN)₂] + 4NaOH，然后用Zn置换。
    - **Hg**：唯一常温液态金属。**HgS**（辰砂，难溶）。**Hg₂²⁺**（亚汞离子，存在Hg₂²⁺形式，与HCl生成Hg₂Cl₂白）。**毒性**。
- **81 Tl, 82 Pb, 83 Bi**：**Tl、Pb、Bi**。
    - **Tl**：+1价稳定（与碱金属相似）。
    - **Pb**：**绝对重点**。**Pb(II)**：PbCl₂（微溶，热水中溶解度增大）、PbSO₄（难溶，但溶于浓NH₄Ac）、PbS（黑色，极难溶）、PbCrO₄（黄色，铬黄颜料）。**工业流程**：**方铅矿(PbS)制铅**（焙烧→还原熔炼）。**铅蓄电池反应**：Pb + PbO₂ + 2H₂SO₄ ⇌ 2PbSO₄ + 2H₂O。**Pb(IV)**：PbO₂（强氧化剂）。
    - **Bi**：+3价稳定。BiOCl（白色，用于珠光粉）。**考点**：Bi³⁺易水解生成BiO⁺（氧基铋离子），加Cl⁻可得BiOCl沉淀。
- **84 Po, 85 At, 86 Rn**：放射性，极毒或稀有，高中基本不涉及。

### 锕系元素

**特点**：全部放射性。**Th和U**在核化学中重要，高中流程题极少涉及（属于超纲安全范围）。**考点**：仅可能在背景信息中提及（如“含铀废水处理”），只需知道它们主要是**+4和+6价**（如U⁴⁺和UO₂²⁺铀酰离子），性质与镧系中相应元素有些相似。

### 总结与解题策略

面对工业流程题，你看到任何一种元素，立刻在脑中过一遍这个**思维清单**：

1. **它最常见的氧化态是什么？能否变价？** → 氧化还原反应。（Fe²⁺/Fe³⁺, Cr³⁺/Cr₂O₇²⁻, Mn²⁺/MnO₄⁻）
2. **它的氢氧化物或盐是否难溶？在酸性还是碱性条件下难溶？** → 沉淀分离。（调pH除Fe³⁺、Al³⁺；加S²⁻除Cu²⁺、Hg²⁺）
3. **它是否具有两性？** → 溶于过量酸碱。（Al³⁺, Zn²⁺, Pb²⁺, Cr³⁺）
4. **它是否容易水解？** → 注意溶液配制（SnCl₂, BiCl₃, TiCl₄），或利用水解进行分离。
5. **它能否形成稳定的配合物？** → 用于溶解、分离或检验。（Ag⁺的氨配合，Cu²⁺的氨配合，Fe³⁺的SCN⁻配合）

这份总结覆盖了高中化学的绝大部分考点和深度。建议你结合具体的高考真题（特别是全国卷、北京卷、上海卷的工业流程大题）来对照学习，效果会最好。祝你学习顺利！这是一份专门为**高中化学（特别是高考工业流程/实验大题）**量身定制的元素周期表讲解。

**核心思路**：不背性质，背**“坑”**和**“用途”**。高考考的就是你能不能在陌生流程里认出老熟人，以及知道哪些元素有**特殊价态**、**特殊颜色**、**两性**、**氧化性/还原性反常**。

| **加H₂O₂**             | 绿色氧化剂，把Fe²⁺→Fe³⁺（方便沉淀），不引入杂质。  |
| **加Na₂CO₃ / NH₄HCO₃** | 除杂，沉淀金属离子，生成沉淀式碳酸盐或碱式碳酸盐。 |
| **加NH₃·H₂O / 调pH**   | 沉淀Fe³⁺/Al³⁺，但Cu²⁺/Zn²⁺不沉淀（分离）。         |
| **加HF**               | 溶解SiO₂（除硅）。                                 |
| **加NaOH至过量**       | 溶解Al(OH)₃（分离Al和Fe）。                        |
| **加CS₂ / CCl₄**       | 萃取Br₂或I₂。                                      |
| **加SCN⁻**             | 检验Fe³⁺（血红色）。                               |
| **加K₃[Fe(CN)₆]**      | 检验Fe²⁺（蓝色沉淀）。                             |
| **加BaCl₂**            | 检验SO₄²⁻（不溶于酸）。                            |
| **加AgNO₃/HNO₃**       | 检验Cl⁻（白色沉淀）。                              |
| **通Cl₂**              | 氧化Br⁻/I⁻/Fe²⁺/SO₂。                              |
| **通SO₂**              | 还原Cr₂O₇²⁻/MnO₄⁻/Fe³⁺。                           |
| **加Na₂S / H₂S**       | 沉淀Cu²⁺/Hg²⁺/Pb²⁺（极难溶）。                     |
| **加NaClO / NaClO₃**   | 氧化Mn²⁺→MnO₂（除锰）。                            |
| **加NaBiO₃(s)**        | **酸性**条件下，Mn²⁺→MnO₄⁻（检验Mn²⁺的大杀器）。   |
| **加H₂C₂O₄ (草酸)**    | 还原剂，除KMnO₄，或沉淀Ca²⁺/稀土。                 |
| **加铁粉**             | 还原Fe³⁺→Fe²⁺，防止氧化。                          |
| **“灼烧”、“煅烧”**     | 分解碳酸盐/氢氧化物/除去结晶水/氧化。              |
| **“水解”**             | 制胶体(Fe(OH)₃) / 制备TiO₂·xH₂O。                  |

这是一份针对高考化学（尤其是针对大题中的**工业流程题**和**综合元素化学题**）的深度元素周期表解析。

在高考的工业流程题中，核心逻辑往往是：**“目标金属的提取与提纯” + “杂质金属的分离”**。这就要求不仅要掌握元素的基础性质，还要精通它们的**氧化还原性、水解特性、络合特性以及两性特征**。

下面为您分周期、分重点进行梳理：

---

### 一、 第一至第三周期（基础提纲，快速掠过）

前三周期是高中化学的基础，在工业流程中主要作为**试剂、辅助元素、配体或基础矿物骨架**出现。

- **H, O**：流程中的灵魂。调节pH（$H^+/OH^-$），提供氧化剂（$H_2O_2$、空气/氧气），提供还原剂（$H_2$）。
- **Li, Na, K**：
    - **Li**：常作为产品出现（如从锂辉石、盐湖卤水中提取碳酸锂，涉及沉淀转化）。
    - **Na/K**：流程中的“绿叶”，提供成盐离子。常考点包括：焙烧时的“钠化”（如纯碱焙烧）、调pH用的$NaOH/K_2CO_3$、以及利用溶解度差异结晶分离。
- **Mg, Ca**：
    - **Mg**：海水提镁流程。杂质形式存在时，通常在流程最后阶段用碱沉淀。
    - **Ca**：除杂常客（如加氟化物沉淀$Ca^{2+}$，加石灰乳除磷、除硫酸根）。
- **Al, Si**：
    - **Al**：**两性**是绝对核心。铝土矿提铝（拜耳法），利用$AlO_2^-$与杂质（如$Fe_2O_3$）分离。
    - **Si**：半导体高纯硅制备（$SiHCl_3$精馏）。在流程首步的“酸浸”中，含硅矿物常转化为**滤渣（$SiO_2$或硅酸）**。
- **N, P, S, Cl**：
    - **N**：硝酸（强氧化性浸出），氨水（**弱碱调pH、络合剂**如$[Cu(NH_3)_4]^{2+}$）。
    - **P**：磷酸精制，次磷酸盐（$H_2PO_2^-$）作化学镀的强还原剂。
    - **S**：硫铁矿焙烧（产生$SO_2$涉及环保与尾气处理），硫酸浸提。连二亚硫酸钠（$Na_2S_2O_4$）等特殊还原剂。
    - **Cl**：氯碱工业，氯气氧化，氯化冶金（加$NH_4Cl$焙烧或通$Cl_2$），次氯酸盐氧化除杂。

---

### 二、 第四周期（工业流程的“绝对主力”，重中之重）

高考70%以上的工业流程题以第四周期过渡金属（特别是**Ti到Zn**）的矿物（如黄铜矿、软锰矿、钛铁矿、闪锌矿、废旧电池等）为载体。

- **22 Ti（钛）**：
    - **核心考点**：钛铁矿（$FeTiO_3$）的提取。$Ti^{4+}$**极易水解**，常考“为什么加水/加热能促进$TiO_2 \cdot nH_2O$沉淀”。
    - **反应**：克罗尔法（$TiO_2 + 2C + 2Cl_2 \xrightarrow{\Delta} TiCl_4 + 2CO$），$TiCl_4$与Mg或Na的热还原（需在Ar气保护下进行）。
- **23 V（钒）**：
    - **核心考点**：多变价态（+2, +3, +4, +5），钒电池，以及从钒渣提钒。
    - **流程**：钠化焙烧转化为可溶的$VO_3^-$（偏钒酸根），加酸沉淀出多钒酸铵，煅烧得$V_2O_5$。
- **24 Cr（铬）**：
    - **核心考点**：**酸碱平衡与氧化还原的结合**。$CrO_4^{2-}$（黄色，碱性稳定）与$Cr_2O_7^{2-}$（橙色，酸性稳定）的转化。
    - **环保考向**：剧毒的六价铬（$Cr^{6+}$）需用还原剂（如$Fe^{2+}$、$SO_3^{2-}$）还原为无毒的$Cr^{3+}$，再调pH沉淀为$Cr(OH)_3$。
- **25 Mn（锰）**：
    - **核心考点**：软锰矿（$MnO_2$）与菱锰矿（$MnCO_3$）。$MnO_2$既是产品，又常作为**氧化剂**（如氧化浸出中的$Fe^{2+}$为$Fe^{3+}$）。
    - **除杂**：$Mn^{2+}$在酸性下较稳定。通常通过调pH，让$Fe^{3+}$、$Al^{3+}$先沉淀，保留$Mn^{2+}$在溶液中。
- **26 Fe（铁）**：
    - **核心考点**：**“永恒的杂质”**。只要是矿石，必有铁。
    - **除杂金标准**：$Fe^{2+}$开始沉淀的pH很高（约7.5），而$Fe^{3+}$很低（约2.7）。所以流程中遇到$Fe^{2+}$，第一步永远是**“加氧化剂（如$H_2O_2$、$MnO_2$、$O_2$）将其氧化为$Fe^{3+}$”**，然后微调pH至3.2左右，使铁以$Fe(OH)_3$形式形成滤渣，而不损失其他目标金属（如$Cu^{2+}$、$Mn^{2+}$、$Zn^{2+}$）。
    - **黄铁矿/赤铁矿**：炼铁及相关的高炉尾气分析。
- **27 Co（钴）、28 Ni（镍）**：
    - **核心考点**：三元锂电池（NMC）回收。
    - **分离难点**：Co和Ni性质极相似。常考**有机溶剂萃取**（如P204, P507）分离。
    - **沉淀特性**：将$Co^{2+}$氧化为$Co^{3+}$，可形成$Co(OH)_3$沉淀从而与Ni分离（利用$Co^{3+}$极易水解沉淀的特性）。
- **29 Cu（铜）**：
    - **核心考点**：黄铜矿（$CuFeS_2$）的焙烧、生物浸出、电解精炼。
    - **特殊化合物**：亚铜离子（$Cu^+$），如生成白色的$CuCl$沉淀、红色的$Cu_2O$。常考$Cu^{2+}$与$I^-$的反应（$2Cu^{2+} + 4I^- = 2CuI\downarrow + I_2$），这是碘量法滴定铜的核心。
- **30 Zn（锌）**：
    - **核心考点**：闪锌矿（$ZnS$）。Zn的**两性**（类似Al，能溶于强碱生成$[Zn(OH)_4]^{2-}$）。
    - **置换除杂**：在提锌流程中，常加入**锌粉**置换除去电位更正的杂质金属（如Cu, Cd）。
- **31 Ga（镓）、32 Ge（锗）、33 As（砷）**：
    - **Ga/Ge**：半导体材料，常作为粉煤灰或铝土矿的副产品提取，常考萃取。
    - **As**：通常作为有害毒物（毒砂$FeAsS$）出现，需在流程中固化（如转化为难溶的砷酸铁$FeAsO_4$弃渣）。
    - **Br, Kr**：Br主要考海水提溴（氯气氧化、空气吹出、二氧化硫吸收，再氯化），Kr不考。

---

### 三、 第五周期（贵金属、催化剂与特殊氧化还原）

这一周期的元素多用于高端制造、催化剂回收或具有特殊的分析化学用途。

- **38 Sr（锶）**：类似钙、钡，考点多为难溶盐（$SrSO_4$、$SrCO_3$）的溶度积$K_{sp}$计算及沉淀转化（加碳酸钠把硫酸锶转化为碳酸锶）。
- **40 Zr（锆）**：高温陶瓷。考点较少，偶尔以$ZrOCl_2$（氯氧化锆）形式出现，考察水解。
- **42 Mo（钼）**：
    - **核心考点**：辉钼矿（$MoS_2$）。焙烧得$MoO_3$，溶于碱生成钼酸盐（$MoO_4^{2-}$）。
    - **特性**：多酸化学（同多酸），酸性条件下易聚合，常考滤液pH变化对产物的影响。
- **44 Ru（钌）、45 Rh（铑）、46 Pd（钯）**：
    - **核心考点**：铂族贵金属，常见于**废旧汽车尾气催化剂回收**流程。
    - **化学特性**：极难溶于普通酸，常需用**王水**或“盐酸+氧化剂（如氯酸钠、$H_2O_2$）”进行浸出。通常考复杂的配离子（如$[PdCl_4]^{2-}$）以及氨水络合除杂。
- **47 Ag（银）**：
    - **核心考点**：感光材料回收、电解铜的阳极泥提银。
    - **络合特性**：银极易络合。溶于氨水（银氨溶液）、溶于硫代硫酸钠（定影液原理）、溶于氰化物（提取金银）。还原提银常用水合肼（$N_2H_4$）或甲醛。
- **50 Sn（锡）**：
    - **核心考点**：+2和+4价的转化。$Sn^{2+}$是**强还原剂**（易被空气氧化），$Sn^{4+}$极易水解生成$SnO_2 \cdot xH_2O$（锡酸）。
- **51 Sb（锑）、52 Te（碲）**：
    - 作为铜/铅精炼阳极泥中的副产物。碲常在提纯流程中以$TeO_2$两性氧化物形式考查（溶于碱生成碲酸盐，酸化沉淀析出）。
- **53 I（碘）**：
    - **核心考点**：海带提碘，或从智利硝石（含$IO_3^-$）提碘。
    - **反应**：$IO_3^- + 5I^- + 6H^+ = 3I_2 + 3H_2O$（归中反应），萃取（四氯化碳/苯），单质碘的升华。

---

### 四、 第六周期（重金属与稀土元素）

- **56 Ba（钡）**：
    - 重晶石（$BaSO_4$）制备立德粉（锌钡白）或碳酸钡。
    - **核心反应**：硫酸钡极难溶，无法直接加酸反应。必须**加碳（C）高温还原**（$BaSO_4 + 4C \xrightarrow{\Delta} BaS + 4CO$），将其转化为可溶性的硫化钡（$BaS$），再进行后续转化。

- **57-71 镧系元素（Rare Earth 稀土元素）**：
    - **统称核心考点**：La（镧）、Ce（铈）、Nd（钕）最常考。它们性质极相似，被统称为$RE$（稀土）。
    - **共性**：主要呈+3价，草酸盐（$RE_2(C_2O_4)_3$）和碳酸盐难溶。
    - **Ce（铈）的特异性**：Ce有稳定的+4价，且$Ce^{4+}$氧化性极强（可用于滴定还原性物质）。流程中常利用“将溶液氧化后，$Ce^{3+}$变成$Ce^{4+}$，由于$Ce^{4+}$水解pH很低，从而与其他+3价稀土元素实现沉淀分离”。
    - **萃取分离**：高考极爱考用有机酸（如P204，用$HR$表示）萃取稀土离子：$RE^{3+} + 3HR \rightleftharpoons RE(R)_3 + 3H^+$。常考“分液漏斗”操作及“反萃取（加酸使平衡左移）”。
- **74 W（钨）**：
    - 黑钨矿/白钨矿提取。和钼类似，焙烧或碱浸生成钨酸钠（$Na_2WO_4$），加酸沉淀出钨酸（$H_2WO_4$），煅烧得$WO_3$，氢气还原得W粉。
- **78 Pt（铂）、79 Au（金）**：
    - **提取**：金矿的氰化法（$4Au + 8NaCN + O_2 + 2H_2O = 4Na[Au(CN)_2] + 4NaOH$）是经典，也可能考环保替代品（如硫脲提金）。王水溶金铂，锌粉置换。
- **82 Pb（铅）**：
    - **废铅酸蓄电池回收**：铅膏（含$PbSO_4$、$PbO_2$、$PbO$）。
    - **脱硫转化**：$PbSO_4$加$Na_2CO_3$转化为$PbCO_3$（沉淀转化，计算$K_{sp}$），再用酸溶。
    - **还原**：$PbO_2$加$H_2O_2$或其他还原剂还原为$Pb^{2+}$浸出。
- **83 Bi（铋）**：
    - **核心特性**：极易水解。将$BiCl_3$溶于水会立刻生成白色的氯氧化铋（$BiOCl$）沉淀。配制铋盐溶液必须加相应的强酸抑制水解。

---

### 五、 锕系元素（Ac-Lr）

- 在高中和高考的“化学反应”层面，锕系元素**极少考查实际的工业提取流程**。

- **考点分布**：
    - **核素与同位素**：铀（$^{235}U$、$^{238}U$）的质子数、中子数、质量数计算。
    - **半衰期与核反应**：作为阿伏加德罗常数、衰变方程（$\alpha$衰变、$\beta$衰变）的背景出现（物理化学交叉）。
    - **偶尔的流程图背景**：如果出现Th（钍）或U（铀）的提取，**无需恐慌**，把它们当做普通的过渡金属（如具备高价态、可与碳酸根络合等）去看待，题目必然会给出未知反应的信息（如$UO_2^{2+}$双氧铀离子的配位特性），只需套用“酸碱中和、氧化还原、沉淀溶解平衡”的三大基本规律即可。

### 总结：如何用周期表思维破解“高考工业流程大题”

当你看到一个流程图，出现上述元素时，大脑中应立刻建立以下映射：

1. **看到 Fe / Mn / O / S** $\rightarrow$ 想 **氧化还原反应**（调价态，特别是Fe的除杂）。
2. **看到 Al / Zn / Cr** $\rightarrow$ 想 **两性**（强碱中可溶，控制pH不能过高）。
3. **看到 Ti / Sn / Bi** $\rightarrow$ 想 **极易水解**（常需要浓酸抑制，或加水加热促使其沉淀析出）。
4. **看到 Co / Ni / Rare Earths(稀土)** $\rightarrow$ 想 **有机溶剂萃取分离**（因为沉淀法难以分开性质太近的元素）。
5. **看到 Ag / Cu / Au / Pt** $\rightarrow$ 想 **配位络合**（氨水、氰化物、硫代硫酸根浸出）。
6. **看到 Ba / Sr / Pb 的硫酸盐** $\rightarrow$ 想 **加碳高温还原** 或 **加碳酸钠沉淀转化**。这份元素周期表讲解将严格按照你的要求：**前三周期提纲式带过，第4、5、6周期及锕系元素作为核心重点**，紧密结合**高中化学课内知识**与**高考工业流程题**中的常考考点（如：焙烧、浸出、除杂、氧化还原调价、沉淀转化等），生僻元素一语带过。

### 第一部分：前三周期（提纲挈领，基础骨架）

**第1周期**

- **H (氢)**：最轻的气体。高考核心：有机物骨架；无机中作为还原剂（如$H_2$还原$CuO$）；水溶液中的氢键（影响沸点、溶解度）。
- **He (氦)**：稀有气体，核外电子排布满足“2电子稳定结构”，用于保护气、气球充气。

**第2周期 (Li ~ Ne)**

- **Li (锂)**：最轻的金属，碱金属。高考热点：锂电池（负极失去电子，$Li^+$嵌入正极）。
- **B (硼)**：缺电子原子，常考硼酸 ($H_3BO_3$) 是一元弱酸（结合水电离出的$OH^-$）。
- **C (碳)**：有机化学的核心。无机重点：碳酸盐/碳酸氢盐的热稳定性（$M_2CO_3 > MHCO_3$）、$CO$的还原性（冶金还原剂）、$CO_2$的温室效应。
- **N (氮)**：三键极其稳定。重点：合成氨、硝酸工业（$NO_x$的吸收与循环）、铵盐受热分解。
- **O (氧)**：氧化剂的代表。臭氧 ($O_3$) 强氧化性，过氧化氢 ($H_2O_2$) 在流程中常作“绿色氧化剂”（产物为水）。
- **F (氟)**：氧化性最强。氢氟酸 (HF) 刻蚀玻璃（与$SiO_2$反应）。

**第3周期 (Na ~ Ar)**

- **Na (钠)**：碱金属代表。焰色试验（黄色，需透过蓝色钴玻璃看钾）。$Na_2O_2$（过氧化钠）常考呼吸面具供氧、强氧化性漂白。
- **Mg (镁)**：轻金属。常考：与$N_2$反应生成$Mg_3N_2$；海水提镁（石灰乳沉淀→盐酸溶解→电解）。
- **Al (铝)**：**两性金属**。高频考点：铝热反应（焊接铁轨）；$Al_2O_3$、$Al(OH)_3$既溶于酸又溶于强碱；工业流程中“调节pH沉淀$Al^{3+}$”。
- **Si (硅)**：地壳中含量第二。常考：半导体材料、光导纤维（$SiO_2$）、硅酸盐（水泥/玻璃工业）、氢氟酸刻蚀。
- **P (磷)**：白磷（$P_4$，正四面体，剧毒，保存在水中）；红磷（安全火柴）；磷酸（中强酸，工业流程中常用来沉淀某些金属离子）。
- **S (硫)**：硫酸工业（接触法：沸腾炉→接触室→吸收塔）。$SO_2$（漂白性、还原性、大气污染物）。浓硫酸（强氧化性、脱水性、吸水性）。
- **Cl (氯)**：氯碱工业（电解食盐水）；“84消毒液”与洁厕灵混合产生氯气；氯气泄漏用碱液吸收。

---

### 第二部分：第4周期（高考工业流程的“绝对主力”）

本周期是高考元素化学和工业流程题的“修罗场”，几乎每个过渡金属都是考点。

- **K (钾)**：草木灰（$K_2CO_3$，水溶液呈碱性）；钾肥。焰色试验紫色（透过钴玻璃）。
- **Ca (钙)**：石灰石/生石灰/熟石灰的三角转化是流程题中“调节pH、提供碱性环境、沉淀除杂”的万能工具。硬水软化。
- **Sc (钪)**：轻稀土元素，常作为合金添加剂，高中基本不考，略过。
- **Ti (钛) —— 【流程题常客】**
    - **地位**：被誉为“21世纪金属”、“太空金属”，强度大、密度小、耐腐蚀。
    - **工业制取**：常从钛铁矿 ($FeTiO_3$) 提取。经典流程：用浓硫酸/氯气焙烧将其转化为 $TiOSO_4$（硫酸氧钛），水解得到 $TiO_2·xH_2O$（偏钛酸），最后高温煅烧得 $TiO_2$（钛白粉，白色颜料）。如果要制金属钛，则用 $TiCl_4$ 与 **Mg** 在氩气保护下发生还原反应（这与炼硅类似）。
    - **考点**：$Ti^{4+}$ 容易水解，因此溶液必须保持强酸性。
- **V (钒) —— 【催化剂与电池新贵】**
    - **核心性质**：接触法制硫酸的催化剂 ($V_2O_5$)。
    - **流程考点**：钒有多种化合价 (+2, +3, +4, +5)。工业上从石煤/钒矿提钒，常采用“钠化焙烧”将低价钒转化为可溶性的钒酸钠，水浸后，再加入铵盐沉淀出偏钒酸铵 ($NH_4VO_3$)，煅烧得 $V_2O_5$。
    - **前沿**：全钒液流电池（正极 $VO_2^+$ / $VO^{2+}$，负极 $V^{3+}$ / $V^{2+}$），常在电池题中出现。
- **Cr (铬) —— 【电镀与除杂明星】**
    - **核心性质**：两性金属。$Cr^{3+}$ 绿色，$Cr_2O_7^{2-}$ 橙色，$CrO_4^{2-}$ 黄色。
    - **考点1（相互转化）**：$Cr_2O_7^{2-}$ (酸性) $\rightleftharpoons$ $CrO_4^{2-}$ (碱性)，受pH控制。
    - **考点2（电镀/防腐）**：铬是坚硬金属，常用于电镀（镀硬铬）。
    - **考点3（环保除杂）**：废水中除 $Cr(VI)$ 剧毒，流程中常加入还原剂（如$FeSO_4$、$Na_2SO_3$）将其还原为 $Cr^{3+}$，再调节pH至弱碱性，生成 $Cr(OH)_3$ 沉淀除去。
- **Mn (锰) —— [氧化剂之王]**
    - **核心性质**：+2价（浅粉色，酸性溶液中极稳定）、+4价（黑色 $MnO_2$，氧化性）、+6价（绿色 $MnO_4^{2-}$，极不稳定）、+7价（紫色 $MnO_4^-$，强氧化剂）。
    - **工业流程**：软锰矿 ($MnO_2$) 制备高锰酸钾。通常先与 $KOH$ 和 $O_2$ 熔融（或加氧化剂）生成 $K_2MnO_4$，再通入 $CO_2$ 或电解使其歧化为 $KMnO_4$ 和 $MnO_2$。
    - **考点**：炼钢时锰用作脱氧剂（脱去铁水中的$O$）。
- **Fe (铁) —— 【工业流程的基石】**
    - **绝对核心**：高中化学分量最重的元素之一。
    - **性质**：$Fe^{2+}$（浅绿色，还原性，易被氧化）与 $Fe^{3+}$（黄色/棕色，氧化性，易水解）。
    - **工业流程角色**：绝大多数金属矿（如钛铁矿、铜矿、锌矿）中都含铁，铁往往作为**“杂质”**被除去。经典操作：用氧化剂（如$H_2O_2$、$NaClO$、$MnO_2$）将 $Fe^{2+}$ 氧化为 $Fe^{3+}$，然后调节pH至 3~4 左右，使 $Fe^{3+}$ 水解产生 $Fe(OH)_3$ 沉淀过滤除去（“除铁”是高考流程最频繁的操作之一）。
- **Co (钴) —— 【电池材料新宠】**
    - **地位**：锂离子电池正极材料（钴酸锂 $LiCoO_2$）的核心元素。
    - **性质**：+2价（粉红色，较稳定），+3价（强氧化性，在水溶液中极易被还原）。
    - **流程考点**：常从含钴废料（如废旧电池）中回收。通常用酸浸出 $Co^{2+}$，然后加入氧化剂（如次氯酸钠）将 $Co^{2+}$ 氧化为 $Co^{3+}$ 并沉淀为 $Co(OH)_3$，再与碳酸锂高温煅烧制得正极材料。
- **Ni (镍) —— 【催化剂与合金】**
    - **性质**：+2价为主（绿色）。常作为催化剂（如有机化学中的加氢催化剂：兰尼镍）。
    - **流程考点**：电镀工业、不锈钢成分。常考从废催化剂中回收镍，除去铁、钴杂质时，利用它们形成不同配合物或氧化还原电位的差异进行萃取或沉淀分离。
- **Cu (铜) —— 【湿法冶金代表】**
    - **性质**：+1价（$Cu_2O$ 红色，$Cu^+$在溶液中歧化）、+2价（蓝色 $Cu^{2+}$，$CuO$ 黑色）。
    - **工业流程**：黄铜矿 ($CuFeS_2$) 炼铜。火法炼铜（冰铜吹炼）；湿法炼铜（酸浸→萃取→电积）。有时流程中会涉及“细菌浸出”（生物冶金）。
    - **考点**：铜氨络合物 ($[Cu(NH_3)_4]^{2+}$，深蓝色）常用于检验 $Cu^{2+}$ 或在除杂时掩蔽铜离子；$CuSO_4$ 溶液配制时加稀硫酸防止水解。
- **Zn (锌) —— 【两性金属】**
    - **性质**：$Zn(OH)_2$ 既溶于酸又溶于强碱（与铝相似）。$Zn^{2+}$ 容易发生水解。
    - **工业流程**：闪锌矿 ($ZnS$) 提锌。常先焙烧成 $ZnO$，再用稀硫酸浸出。除铁后，加锌粉置换出较不活泼的金属（如$Cu、Cd$），最后电解得到纯锌。
    - **考点**：锌锰干电池（负极反应）、镀锌防腐（白铁皮）。
- **Ga (镓)、Ge (锗)、As (砷)、Se (硒)**
    - **Ga**：熔点极低（手心能融化），高温温度计，半导体材料（GaAs，砷化镓）。
    - **Ge**：半导体材料，晶体结构类似金刚石。
    - **As**：砒霜 ($As_2O_3$) 的成分。工业流程中砷常作为剧毒杂质，需氧化为砷酸盐沉淀除掉。
    - **Se**：光电材料。$SeO_2$ 易升华，常考其提纯方法。
- **Br (溴)**：常温下唯一的液态非金属。流程题常考“海水提溴”：氯气氧化溴离子→空气吹出→二氧化硫吸收→再氧化提取。
- **Kr (氪)**：稀有气体，光源用，高中不考。

### 第三部分：第5周期（催化剂、贵金属与类金属）

本周期在高考中的出场率略低于第4周期，但有几个绝对的“明星元素”。

- **Rb (铷)、Sr (锶)、Y (钇)、Zr (锆)**：高中极罕见。$Sr$ 焰色为洋红色（烟火）；$Y$ 常用于超导材料 ($YBaCuO$) 的背景信息。
- **Nb (铌)、Mo (钼)**：
    - **Mo**：合金添加剂（耐高温、耐腐蚀）。其化合物 $MoS_2$ 常作为固体润滑剂出现在流程题背景中。
- **Tc (锝)、Ru (钌)、Rh (铑)、Pd (钯)**：
    - **Tc**：放射性元素，最轻的人工元素，略过。
    - **Pd/Rh**：常作有机合成催化剂（如汽车尾气催化转化器），高中流程题偶尔会以“钯渣回收”出题，主要考查其易被王水溶解，或形成配合物的性质。
- **Ag (银) —— 【摄影与冶炼】**
    - **性质**：导电性最好。$Ag^+$ 遇光照易分解（感光性）。
    - **流程考点**：废旧胶片/电解废液中回收银。常用沉淀法（加$Na_2S$生成$Ag_2S$黑沉淀）或置换法（用铁/锌置换）。$Ag^+$ 检验用稀盐酸/硝酸酸化的硝酸银溶液。氨水溶解 $Ag_2O$ 生成 $[Ag(NH_3)_2]^+$（银镜反应）。
- **Cd (镉)**：镍镉电池成分。剧毒重金属（“痛痛病”元凶），流程中通常需要转化为难溶物固化处理。
- **In (铟)、Sn (锡)、Sb (锑)、Te (碲)**
    - **Sn**：马口铁（镀锡铁）。$SnCl_2$ 是强还原剂（常用于检验$Hg^{2+}$）。焊锡（Pb-Sn合金）。
    - **Sb**：两性偏弱。常考锑酸（用于阻燃剂）。冶炼中常通过“火法”提炼，生成 $Sb_2O_3$（锑白）。
- **I (碘) —— 【生命元素与升华提纯】**
    - **性质**：升华特性（流程中常考“加热升华→冷凝”提纯碘）。$I_2$ 遇淀粉变蓝（特征检验）。
    - **流程考点**：海带提碘（灼烧灰化→水浸→氧化（氯气/双氧水）→萃取（四氯化碳/苯））。
- **Xe (氙)**：稀有气体中唯一能较容易形成化合物的元素（如 $XeF_2$, $XeF_4$），高考偶尔在物质结构题中考查其空间构型（$XeF_2$ 直线形，$XeF_4$ 平面正方形）。

### 第四部分：第6周期及镧系（重金属、贵金属与核工业）

- **Cs (铯)、Ba (钡)**：
    - **Cs**：光电效应的极佳材料（逸出功最小）。
    - **Ba**：焰色反应黄绿色。$BaSO_4$ 极难溶且不溶于酸，X光造影剂（“钡餐”）。工业流程中常利用 $BaCl_2$ 除去溶液中的 $SO_4^{2-}$。
- **Hf (铪)、Ta (钽)、W (钨) —— 【耐高温霸主】**
    - **W**：熔点最高的金属。常考其提取：黑钨矿碱熔后转化为钨酸钠，加酸沉淀为钨酸，煅烧得 $WO_3$，最后用氢气还原得到纯钨（注意不用碳还原，怕生成碳化钨影响性能）。碳化钨 (WC) 极硬，用于切削刀具。
- **Re (铼)、Os (锇)、Ir (铱)**：极其生僻。$OsO_4$ 剧毒，有机化学氧化剂；$Ir$ 常用于高温坩埚或火花塞，高考基本不涉及其化学反应。
- **Pt (铂) —— 【催化之王】**
    - **性质**：极其稳定，不溶于单一酸（除王水）。催化活性极高。
    - **考点**：接触法制硫酸的催化剂（早期用Pt，现用$V_2O_5$）；燃料电池电极材料；氨氧化制硝酸的催化剂（Pt-Rh合金网）。
- **Au (金) —— 【王水溶解与氰化提金】**
    - **性质**：极不活泼，但溶于王水（配位作用降低了还原电位）。
    - **工业流程考点（重点）**：现代提金工业采用**氰化法**——用 $NaCN$ 溶液在氧气存在下溶解金：$4Au + 8NaCN + O_2 + 2H_2O \rightarrow 4Na[Au(CN)_2] + 4NaOH$。然后用锌粉置换出金。此过程涉及配位化学和置换反应，常在流程题中出现（同时会强调含氰废水的毒性处理，用漂白粉氧化分解）。
- **Hg (汞) —— 【液态毒王】**
    - **性质**：常温下唯一呈液态的金属，易挥发，剧毒。
    - **考点**：汞齐（汞的合金，如钠汞齐常作温和还原剂）；“水俣病”元凶；温度计。$HgS$（朱砂）是自然界最稳定的汞矿物。
- **Tl (铊)、Pb (铅)、Bi (铋)**
    - **Tl**：剧毒（曾被用作杀鼠剂），高中不考。
    - **Pb —— 【蓄电池老将】**：铅酸蓄电池（正极 $PbO_2$，负极 $Pb$，电解质 $H_2SO_4$）是高考电化学“常青树”。$PbO_2$ 具有强氧化性。工业上铅常作为防辐射材料。
    - **Bi**：熔点低，易形成低熔点合金（如保险丝）。$BiOCl$ 常作为珠光颜料出现在流程题背景中。+3价稳定，+5价具有强氧化性（类似锑）。
- **Po (钋)、At (砹)、Rn (氡)**：放射性元素/极稀有，高中化学直接忽略。
- **镧系元素 (57~71) —— 【稀土家族】**
    - **总体特征**：被称为“工业维生素”。化学性质极其相似（由于半径相近、化合价多为+3），导致**分离极其困难**。工业上常采用“萃取法”利用微小的分配系数差异进行分离。
    - **Ce (铈)**：镧系中常考的例外，除了+3价，还有+4价。$Ce^{4+}$ 是强氧化剂，在流程中常被用作氧化剂（如氧化除杂其他稀土元素）。
    - **应用**：抛光粉（$CeO_2$）、永磁材料（$Nd_2Fe_{14}B$，钕铁硼）、荧光粉。

### 第五部分：锕系元素 (89~103)

锕系元素全部具有放射性，在高中常规的元素化合物题中基本不出现，但需了解几个概念性考点：

- **Th (钍)、U (铀) —— 【核燃料双雄】**
    - **U (铀)**：最重要的核燃料。高考常考其提纯：从沥青铀矿提取，通过转化得到 $UF_6$（六氟化铀）。$UF_6$ 易升华，利用其气体进行**气体扩散法**或**离心法**分离同位素 $^{235}U$（可裂变）与 $^{238}U$（不可裂变）。这里常结合分子运动速率（相对分子质量越轻，扩散越快）出物理/化学综合题。
    - **Th (钍)**：被视作未来核能的“第三步燃料”（钍基熔盐堆），近年常在信息题中作为前沿背景出现。
- **Np (镎) 到 Lr (铹)**：超铀元素，多为人工合成，极不稳定。高中化学不涉及具体反应。唯一需要注意的是，流程题如果给出它们，通常考查**“结构推断”**或**“氧化还原调价”**（由于它们具有多种化合价，常考查通过调节pH或加入氧化/还原剂使其沉淀分离）。

### 💡 高考工业流程题“陌生元素”解题心法（附赠）

当你遇到第4、5、6周期的陌生元素（如 $Mo, V, Ti, W$ 等）时，不要慌，记住以下规律：

1. **“价态决定命运”**：流程题一定会给信息，比如“某元素+5价以含氧酸根形式存在”。记住，**高价具有氧化性**（需加还原剂才能沉淀），**低价具有还原性**（需加氧化剂防跑漏），**中间价态容易水解沉淀**。
2. **“两性定律”**：如果是主族金属或两性区的金属（如Be, Al, Ga, Zn, Pb, Sn, Sb），只要题目没见过，大胆猜测其最高价氧化物或氢氧化物**既能溶于强酸，又能溶于强碱**。流程中如果出现“加过量NaOH过滤，某物质不溶”，它一定不是两性物质（如 $Mg(OH)_2$, $Fe(OH)_3$）。
3. **“焙烧/熔融目的”**：难溶的过渡金属矿（如钛铁矿、黑钨矿、铬铁矿），第一步永远是加碱（$Na_2CO_3$、$NaOH$）或加氧化剂（$O_2$、$NaNO_3$）在高温下焙烧。目的只有一个：**将难溶的低价金属转化为可溶的高价含氧酸盐**（如 $MoS_2 \rightarrow Na_2MoO_4$），以便水浸分离。
4. **“除杂铁、铝”**：不管什么流程，只要看到浸出液，下一步往往是“调节pH”。90%的情况是为了让 $Fe^{3+}$ 或 $Al^{3+}$ 水解沉淀掉。如果发现加了氧化剂再调pH，那说明原矿里铁是+2价，必须先氧化再除。这是一份面向高考化学（特别是工业流程大题）的元素周期表精读。前3周期为基础，略作提纲；第4-6周期+镕系为**工业流程高频考点**，将结合**氧化还原、水解、两性、沉淀转化、配合物**等核心模型深度讲解。

|       | **Mg**   | **海水提镁**：石灰乳沉淀Mg(OH)₂→盐酸溶解→蒸发结晶→熔融电解（阴极得Mg，阳极得Cl₂）                    |
|       | **Al**   | **铝土矿提纯**：NaOH溶解Al₂O₃（Fe₂O₃不溶）→过滤→通CO₂酸化→灼烧→电解冰晶石熔体；Al³⁺与AlO₂⁻双水解彻底 |
|       | **Si**   | **粗硅提纯**：SiO��+2C→Si(粗)+2CO↑；Si+3HCl→SiHCl₃+H₂；SiHCl₃+H₂→Si(纯)+3HCl；硅酸凝胶干燥剂         |
|       | **P**    | 磷酸制取（Ca₃(PO₄)₂+3H₂SO₄→3CaSO₄+2H₃PO₄），磷肥（过磷酸钙Ca(H₂PO₄)₂+CaSO₄）                         |
|       | **S**    | **接触法制硫酸**：4FeS₂+11O₂→2Fe₂O₃+8SO₂；V₂O₅催化；98.3%浓硫酸吸收SO₃；硫酸渣(Fe₂O₃)炼铁            |
|       | **Cl**   | **氯碱工业**：2NaCl+2H₂O→2NaOH+H₂↑+Cl₂↑（离子交换膜法）；**海水提溴**中Cl₂作氧化剂                   |
|       | **Ar**   | 保护气（不锈钢焊接，比He便宜）                                                                       |

## 第四周期：过渡金属的入门与工业基石（重点）

### K、Ca：碱金属与碱土金属的延伸

- **K**：与Na性质极似，但**焰色反应为紫色**（透过蓝色钴玻璃），化肥（KCl、K₂SO₄）。
- **Ca**：**水泥工业**核心（CaCO₃高温分解→CaO，与SiO₂反应生成硅酸盐）；**钙三角**：CaCO₃⇌CaO⇌Ca(OH)₂；Ca²⁺与C₂O₄²⁻（草酸根）生成CaC₂O₄白色沉淀（检验Ca²⁺）。

### Ti（钛）：航空材料与钛白粉

- **工业流程核心**：TiO₂（钛白粉，最佳白色颜料）的制备与TiCl₄的水解。
- **考点**：
    1. **硫酸法**：FeTiO₃（钛铁矿）+2H₂SO₄→TiOSO₄+FeSO₄+2H₂O；加热水解：TiO²⁺+2H₂O→TiO(OH)₂↓+2H⁺（或写作H₂TiO₃），再灼烧得TiO₂。
    2. **氯化法**：TiO₂+2C+2Cl₂→TiCl₄+2CO（高温）；TiCl₄+2Mg→Ti（海绵钛）+2MgCl₂（Ar保护）；**TiCl₄强烈水解**：TiCl₄+3H₂O→H₂TiO₃↓+4HCl↑（冒白雾）。
    3. TiO₂光催化（降解有机物）。

### V（钒）：催化与合金

- **硫酸工业**：V₂O₅作SO₂氧化为SO₃的催化剂（钒触媒）。
- **溶液化学**：V有多种价态（+2、+3、+4、+5），颜色变化丰富（VO₂⁺黄色，VO²⁺蓝色），高考较少深究，知道其催化性能即可。

### Cr（铬）：变价与平衡移动

- **工业来源**：铬铁矿(FeO·Cr₂O₃)。
- **核心考点**：
    1. **Cr(III)与Cr(VI)的转化**：
        - 碱性氧化：2Cr(OH)₃+3NaClO+4NaOH→2Na₂CrO₄+3NaCl+5H₂O（CrO₄²⁻，黄色）
        - 酸性转化：2CrO₄²⁻+2H⁺⇌Cr₂O₇²⁻+H₂O（橙红色，**平衡移动**考点）
    2. **K₂Cr₂O₇的氧化性**：酸性条件下氧化Fe²⁺、SO₃²⁻、I⁻等，自身被还原为Cr³⁺（绿色）。
    3. **Cr(OH)₃两性**：溶于强酸得Cr³⁺，溶于强碱得CrO₂⁻（或[Cr(OH)₄]⁻）。
    4. **工业废水处理**：Cr(VI)剧毒，需还原为Cr(III)后调pH至8-9生成Cr(OH)₃沉淀除去。

### Mn（锰）：氧化态的舞蹈

- **高考高频**：MnO₂、KMnO₄、MnSO₄的转化。
- **核心考点**：
    1. **MnO₂**：
        - 干电池正极（去极化剂）。
        - 催化剂（KClO₃、H₂O₂分解）。
        - 制备KMnO₄原料：2MnO₂+4KOH+O₂→2K₂MnO₄+2H₂O（熔融）；3K₂MnO₄+2H₂O→2KMnO₄+MnO₂↓+4KOH（歧化）。
    2. **KMnO₄**：
        - **强氧化剂**，酸性介质还原产物为Mn²⁺（近乎无色），自身作指示剂（紫红色褪去）。
        - 氧化Fe²⁺、C₂O₄²⁻（草酸，标定KMnO₄浓度）、SO₂等。
    3. **Mn²⁺**：在碱性条件下易被氧化（如空气中的O₂氧化为MnO(OH)₂），是除去溶液中Fe²⁺、Mn²⁺时，先氧化Fe²⁺为Fe³⁺沉淀，再调pH使Mn²⁺生成MnO₂·xH₂O（或MnO(OH)₂）沉淀。
    4. **MnO₄⁻与MnO₂在酸性条件下的归中**：2MnO₄⁻+3Mn²⁺+2H₂O→5MnO₂↓+4H⁺。

### Fe、Co、Ni：铁系元素

- **Fe（绝对核心）**：
    - **冶炼**：高炉炼铁（C、CO还原）；直接还原法（H₂还原铁粉）。
    - **腐蚀与防护**：吸氧腐蚀（负极Fe-2e⁻=Fe²⁺，正极O₂+2H₂O+4e⁻=4OH⁻）；牺牲阳极保护法（Zn块）。
    - **离子性质**：Fe²⁺浅绿色（还原性，遇NO₃⁻/H⁺、ClO⁻、MnO₄⁻被氧化）；Fe³⁺黄色（氧化性，可氧化I⁻、S²⁻、SO₃²⁻）；**Fe³⁺水解**：净水剂（FeCl₃饱和溶液滴入沸水制Fe(OH)₃胶体）。
    - **沉淀**：Fe(OH)₂→Fe(OH)₃的白色→灰绿→红褐变色过程。
    - **工业流程**：Fe³⁺与Fe反应转化为Fe²⁺除杂；调节pH=4左右使Fe³⁺完全沉淀（Ksp[Fe(OH)₃]极小）。
- **Co、Ni**：
    - 常与Fe伴生，**Ni为不锈钢铁素**，Co用于锂电池正极(LiCoO₂)。
    - **Ni²⁺**：电镀（镀镍防腐），检验：与丁二酮肟生成鲜红色沉淀。
    - **Co²⁺/Co³⁺**：Co³⁺氧化性极强（酸性），Co(OH)₃氧化盐酸生成Cl₂。

### Cu、Zn：ds区代表

- **Cu**：
    - **冶炼**：
        - **火法**：黄铜矿(CuFeS₂)→冰铜(Cu₂S·FeS)→粗铜→电解精炼（阳极泥含Au、Ag、Pt）。
        - **湿法**：Fe+CuSO₄=FeSO₄+Cu（古代）；现代：CuS/ CuFeS₂→酸浸/细菌氧化→Cu²⁺溶液→萃取→电积。
    - **氧化还原**：Cu²⁺氧化性（与I⁻反应生成CuI↓+I₂）；Cu⁺歧化（2Cu⁺=Cu²⁺+Cu），固态Cu₂O稳定。
    - **配合物**：Cu(OH)₂溶于氨水生成[Cu(NH₃)₄]²⁺（深蓝色，检验醛基）；Cu²⁺+4SCN⁻=[Cu(SCN)₄]²⁻（用于Cu²⁺的萃取）。
    - **碱式碳酸铜**：Cu₂(OH)₂CO₃（铜绿）。
- **Zn**：
    - **两性**：Zn(OH)₂溶于强酸、强碱（生成ZnO₂²⁻或[Zn(OH)₄]²⁻）。
    - **工业**：闪锌矿(ZnS)→焙烧得ZnO→硫酸溶解→电解或热还原（C还原得粗Zn，精馏提纯）。
    - **原电池**：Zn作负极（牺牲阳极），Zn-MnO₂干电池。

### Ga、Ge、As：类金属与半导体

- **Ga**：与Al同族，**铝土矿中伴生**，回收Ga用于半导体(GaAs)。Ga(OH)₃为两性（酸性弱于Al(OH)₃）。
- **Ge**：半导体（光导纤维掺杂），GeO₂为两性。
- **As（砷，重要）**：与P同族。
    - **毒性**：As₂O₃（砒霜，白色粉末，微溶于水）。
    - **工业废水处理**：AsO₃³⁻/AsO₄³⁻，与Fe³⁺形成FeAsO₄沉淀除去；或硫化物沉淀(As₂S₃)。
    - **氧化性**：H₃AsO₄（砷酸）可氧化I⁻（H₃AsO₄+2HI→H₃AsO₃+I₂+H₂O，可逆），此反应用于分析化学。

### Se、Br：非金属深化

- **Se**：半导体，光敏材料（用于复印机鼓），与S性质相似，H₂Se有剧毒。
- **Br（溴，重要）**：
    - **海水提溴（空气吹出法）**：Cl₂氧化Br⁻→Br₂；热空气吹出；SO₂吸收（Br₂+SO₂+2H₂O=2HBr+H₂SO₄，富集）；再Cl₂氧化。
    - **有机溶剂萃取**：Br₂在CCl₄或苯中呈橙红色。
    - **AgBr**：感光材料（分解生成Ag）。

## 第五周期：贵金属、Sn与I（重点）

### Rb、Sr、Y、Zr、Nb、Mo、Tc：过渡金属简述

- **Sr**：焰色反应**洋红色**，用于信号弹；SrCO₃用于磁性材料。
- **Zr**：锆英石(ZrSiO₄)，耐腐蚀（核反应堆材料），与Hf性质极相似（镧系收缩导致分离困难）。
- **Mo**：辉钼矿(MoS₂)，催化剂（合成氨残余），MoO₃用于制特种钢。
- **Tc**：第一个人工合成元素，放射性，医用示踪剂。

### Ru、Rh、Pd、Ag、Cd：贵金属与Zn同族

- **Ru、Rh、Pd**：铂族金属，催化剂（汽车尾气三元催化，Pd/Rh催化CO/HC/NOx转化）。
- **Ag（银，重要）**：
    - **配合物提银**：Ag⁺+2CN⁻=[Ag(CN)₂]⁻（稳定常数极大），氰化法提银（剧毒，现多被硫脲法等替代）。
    - **卤化银**：AgCl（白，可溶于氨水），AgBr（浅黄，可溶于Na₂S₂O₃，照相底片），AgI（黄，感光性更强，人工降雨）。
    - **银镜反应**：[Ag(NH₃)₂]⁺被醛基还原。
    - **氧化性**：Ag⁺氧化性（与Fe²⁺反应生成Ag+Fe³⁺）。
- **Cd**：与Zn同族，**剧毒**（痛痛病），CdS（黄色颜料，镉黄），Cd用于镍镉电池。

### In、Sn、Sb、Te：主族金属与类金属

- **In**：铟锡氧化物(ITO)，液晶显示器透明电极，与Ga同族。
- **Sn（锡，重要）**：
    - **价态**：Sn²⁺（还原性，可被Fe³⁺、Hg²⁺氧化），Sn⁴⁺稳定。
    - **马口铁**：镀锡铁（Sn比Fe活泼，破损后Sn作阳极保护Fe？**错误**，Sn的活泼性弱于Fe，破损后Fe作阳极腐蚀更快，故马口铁一旦破损腐蚀加速）。
    - **水解**：SnCl₂+H₂O→Sn(OH)Cl↓+HCl（配制时需加HCl抑制水解）。
    - **反应**：Sn²⁺+2Hg²⁺→Sn⁴⁺+Hg₂²⁺（白色）→Sn⁴⁺+2Hg↓（黑色，检验Hg²⁺）。
- **Sb（锑）**：与As同族，H[Sb(OH)₆]（锑酸）用于检验Na⁺（白色晶状沉淀）；Sb₂S₃（橙色，用于火柴、安全火柴的摩擦剂）。
- **Te**：碲化镉(CdTe)太阳能电池。

### I（碘，重要）

- **海带提碘**：灼烧→溶解→氧化（Cl₂或H₂O₂）→萃取（CCl₄）→反萃取（NaOH溶液，I₂→I⁻+IO⁻）→酸化归中（5I⁻+IO⁻+6H⁺→3I₂）→过滤得粗碘。
- **特性**：升华（物理提纯），遇淀粉变蓝。
- **碘量法滴定**：I₂+2S₂O₃²⁻=2I⁻+S₄O₆²⁻（硫代硫酸根），I⁻被氧化剂（如O₃、ClO⁻、NO₂⁻）氧化生成I₂，再用Na₂S₂O₃滴定。

## 第六周期：重元素、镧系收缩与放射性（重点）

### Cs、Ba：重碱金属与碱土金属

- **Cs**：光电效应（光电管），比Na、K更活泼，**不能用于水溶液中的反应**。
- **Ba（钡，重要）**：
    - **BaSO₄**：重晶石，**钡餐**（X射线造影，不溶于水也不溶于酸，Ksp极小）；钡盐中毒用Na₂SO₄或MgSO₄（泻盐）解毒。
    - **BaCO₃**：剧毒（溶于胃酸生成Ba²⁺），用于陶瓷釉料。
    - **Ba²⁺检验**：焰色**黄绿色**，或加Na₂SO₄/稀H₂SO₄得白色沉淀（不溶于稀HNO₃，区别于CaSO₄微溶）。

### 镧系元素（稀土，重要）

- **总体性质**：镧系收缩（导致Zr≈Hf、Nb≈Ta、Mo≈W性质极相似，分离困难）；+3价最稳定。
- **Ce（铈）**：**Ce⁴⁺**有强氧化性（酸性介质，E°=1.61V），可氧化H₂O₂、Fe²⁺等；CeO₂用于抛光粉、汽车尾气催化。
- **工业提取**：**萃取法**（利用Ln³⁺在不同pH下与萃取剂结合能力差异）或**离子交换法**；**共沉淀法**（草酸盐、碳酸盐）。
- **应用**：永磁材料(Nd₂Fe₁₄B)、荧光粉(Eu、Y)、催化剂(La)。

### Hf、Ta、W、Re：高熔点金属

- **Hf**：与Zr几乎无法化学分离，需离子交换或溶剂萃取。
- **Ta**：钽电容（电子产品），耐腐蚀（与Pt相似）。
- **W**：钨丝（灯丝，熔点最高3422℃），硬质合金（WC）。
- **Re**：Re₂O₇易挥发，用于催化剂（汽油重整）。

### Os、Ir、Pt、Au、Hg：重贵金属

- **Pt**：惰性电极，催化氢化（铂黑），溶于王水(HCl+HNO₃)生成H₂PtCl₆。
- **Au（金，重要）**：
    - **惰性**：不溶于普通酸，溶于**王水**（Au+HNO₃+4HCl→HAuCl₄+NO↑+2H₂O）或**氰化法**（4Au+8NaCN+O₂+2H₂O→4Na[Au(CN)₂]+4NaOH，剧毒但高效，用于低品位金矿石）。
    - **配合物**：[Au(CN)₂]⁻（用于电镀金），AuCl₃（用于有机合成催化剂）。
- **Hg（汞，重要）**：
    - **毒性**：Hg²⁺、Hg₂²⁺、有机汞（甲基汞，水俣病）均有毒；Hg挥发（蒸气有毒）。
    - **汞齐**：Hg与Na、Ag、Au等形成合金（Na汞齐用于还原反应，安全处理Na）。
    - **甘汞**：Hg₂Cl₂（白色，微毒，用作泻药，现已少用）。
    - **反应**：Hg²⁺+S²⁻=HgS↓（黑色，极难溶，处理汞泄漏用硫磺粉）。

### Tl、Pb、Bi、Po：p区重元素

- **Tl（铊）**：与Al同族但**无两性**，Tl⁺（类似K⁺，剧毒，曾用于鼠药）。
- **Pb（铅，重要）**：
    - **蓄电池**：Pb+PbO₂+2H₂SO₄⇌2PbSO₄+2H₂O（放电/充电）。
    - **冶炼**：方铅矿(PbS)→焙烧得PbO→C还原。
    - **难溶盐**：PbSO₄、PbCO₃、PbCrO₄（黄色，检验CrO₄²⁻或Pb²⁺）；Pb(Ac)₂（醋酸铅，弱电解质，可溶，**唯一可溶铅盐**，有毒）。
    - **Pb²⁺/PbO₂**：PbO₂强氧化性（与Mn²⁺反应生成紫红色的MnO₄⁻）。
- **Bi（铋）**：Bi³⁺易水解生成BiO⁺（铋酰离子）或BiOCl（氯化氧铋，白色沉淀，用于化妆品）。
- **Po**：放射性，简单提及。

### At、Rn：放射性

- **At**：砹，类碘，极稀有，放射性。
- **Rn**：氡，放射性气体，来自镭衰变，室内污染物（肺癌风险）。

## 锕系元素（简介）

- **Ac、Th、Pa、U**：均为放射性元素。
- **U（铀）**：核燃料，U₃O₈（黄饼），UF₆（用于同位素分离，气体扩散法）。
- **Pu（钚）**：核武器原料，超铀元素多为人工合成。

## 工业流程大题中的元素思维导图

```
原料预处理 → 酸浸/碱浸 → 氧化/还原调价 → 除杂（调pH/沉淀/萃取）→ 产品制备
     ↓              ↓                ↓                    ↓
   粉碎/焙烧     Ti/Cr/Mn/Fe     MnO₂/H₂O₂/Cl₂      水解(Ti⁴⁺,Sn⁴⁺,Bi³⁺)
   去有机物      Cu/Zn/Ni溶解     (绿色氧化剂)        沉淀(Cr³⁺,Fe³⁺)
   活化          Al/Ga两性        SO₂/H₂S还原        结晶(MnSO₄,胆矾)
                 Au/Ag配合物      电解精炼
```

**核心规律总结**：

1. **变价元素**（Cr、Mn、Fe、Cu、N、S）常考氧化还原与pH控制。
2. **两性元素**（Al、Zn、Ga、Sn、Pb）常考酸碱分离与提纯。
3. **易水解元素**（Ti⁴⁺、Fe³⁺、Sn⁴⁺、Bi³⁺）常考加热水解或控制pH防水解。
4. **难溶物转化**（BaSO₄/BaCO₃、CaSO₄/CaCO₃、AgCl/AgBr/AgI）常考溶度积Ksp计算与沉淀溶解平衡移动。
5. **配合物**（Au、Ag、Cu）常考氰化法或氨络合分离。

掌握以上，足以应对高中化学工业流程大题中关于元素性质的所有深层考查。**一、概述**  
本文以《元素周期表》（前 6 周期 + 锕系）为框架，对每个元素进行“逐个”讲解。

- **重点**：高中课内所学的基本化学性质、常见化合物、典型的氧化‑还原行为以及在工业流程（如氯碱、炼铁、铝电解、硅提纯、铀燃料循环等）中的实际应用。
- **兼顾**：部分较深层次的化学现象（如配合物的形成、d‑电子的占位、f‑电子的收缩效应等）以及高考常考的“难点”。
- **取舍原则**：前 3 周期只给提纲式概述；第 4、5、6 周期及锕系是重点，重点阐述；对几乎不出现于工业流程的极稀有元素（如 Po、At、Rn 等）仅作“一句话”简介。

| 原子半径                 | **减小**                        | **增大**                       |
| 第一电离能               | **增大**                        | **减小**                       |
| 电子亲和能（除稀有气体） | **增大**（在卤素处峰值）        | **减小**                       |

> **锕系收缩**：第 6 周期出现 **4f‑收缩**（镧系），导致第 6 周期的 Hf‑Ta‑W‑Re‑Os‑Ir‑Pt‑Au‑Hg 与第 5 周期的同族元素相比半径更小、熔点更高。

| **镍氢/镍镉电池**                                  | Ni、Co（Ni‑MH），Cd（Ni‑Cd） | Ni(OH)₂ + MH ↔ NiOOH + M                     | 充放电的氧化‑还原、能量密度、环境问题        |
| **金、银的电解精炼**                               | Au、Ag、Cu（粗铜）           | Cu（粗） + CuSO₄ → Cu（纯）                  | 电解方程、阴极沉积、阳极溶解                 |
| **汞的氯碱电池（已淘汰）**                         | Hg、Na                       | Hg + NaCl → NaHg + Cl₂                       | 环境污染、汞的危害（现已禁用）               |

### 高考常见“难点”与“技巧”

| 难点                                           | 应对技巧                                                                                 |
| ---------------------------------------------- | ---------------------------------------------------------------------------------------- |
| **氧化态的判定**（如 Mn、Cr、V 多价）          | 记忆常见氧化态表；利用“氧化剂-还原剂”配平法（半反应法）                                  |
| **配位化合物的命名与几何**                     | 记住配位数与配位体（NH₃、Cl⁻、H₂O）对应的几何（四方/八面体）                             |
| **电子排布与磁性**                             | 用 **洪特规则**、**d‑电子数** 计算磁矩（μ = √(n(n+2))）                                  |
| **金属活动顺序与反应预测**                     | 熟记 **金属活动顺序表**（K>Na>Ca>…>Au），判断能否置换                                    |
| **周期表趋势的定量应用**（如离子半径、晶格能） | 记住 **原子半径≈0.037 nm·Z⁻¹/³**；利用 **库仑定律** 计算晶格能                           |
| **电化学电位与电池**                           | 记住标准电极电位表（E°），会写 **电极反应**、**电池符号**、**电动势计算**                |
| **热化学与平衡**                               | 熟悉 **勒沙特列原理**，能用 **ΔG° = –RT ln K** 判定反应方向                              |
| **工业流程中的产率、能耗、成本**               | 能根据 **反应方程式** 计算 **理论产量** 与 **实际产率**；了解 **能量回收**（如余热利用） |

## 十一、复习建议

1. **把元素当作“家族”来记忆**：
    - **碱金属**：焰色、+1、强还原、与水剧烈反应。
    - **碱土金属**：焰色、+2、形成氢氧化物（微溶）。
    - **卤素**：颜色、-1（+奇数氧化态）、强氧化性。
    - **过渡金属**：d‑电子可多配位、颜色、磁性、催化性。
    - **镧系**：+3 为主、镧系收缩、用于磁性材料。
    - **锕系**：放射性、+3‑+6 常见、铀、钚是核能核心。

2. **从“工业流程”逆向推导**：
    - **高炉** → 考察 **Fe₂O₃ + CO → Fe** 的还原；
    - **氯碱** → 考察 **电解** 与 **阳极/阴极** 的氧化‑还原；
    - **霍尔‑赫鲁兹** → 考察 **Al₂O₃** 的 **两性** 与 **电解耗能**。

3. **练习写方程式时，先写半反应，再合并**：
    - 例：**Cr₂O₇²⁻ + 14H⁺ + 6e⁻ → 2Cr³⁺ + 7H₂O**（酸性介质）
    - 再配合 **6Fe²⁺ → 6Fe³⁺ + 6e⁻** → 合并得到完整的 **氧化‑还原** 方程式。

4. **做高考真题时，务必弄清**
    - 题干给出的 **元素符号** 与 **常见化合物**；
    - **实验条件**（如 pH、温度、催化剂）对 **产物** 的影响。

5. **对照周期表** 记忆：
    - **左下 → 金属、还原剂**；
    - **右上 → 非金属、氧化剂**；
    - **中间 → 过渡金属**（兼具金属性与氧化性）。

### 📌 高考工业流程题的底层逻辑

1. **浸出**：酸溶/碱溶/氧化还原溶（如MnO₂+H₂O₂+H⁺→Mn²⁺；Al₂O₃+2OH⁻→2AlO₂⁻+H₂O）
2. **除杂**：调pH沉淀（Fe³⁺/Al³⁺/Ti⁴⁺易水解）、氧化还原除杂（Fe²⁺→Fe³⁺后沉淀）、络合掩蔽、萃取/离子交换
3. **转化**：结晶/煅烧/电解/置换（如Cu²⁺+Fe→Cu；2Al₂O₃(熔融)→4Al+3O₂↑）
4. **信息提示必抓**：题干给出的“某离子在pH>X沉淀”“+n价易水解”“可被H₂O₂/Cl₂氧化”“形成稳定络离子”是解题钥匙。

### 🔹 第1~3周期：提纲挈领（课内基础，流程题常作背景或对比）

| 周期 | 核心元素 | 高考/流程考点                                                                                                                                                     |
| ---- | -------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 1    | H, He    | H⁺/H₂还原性、H₂O₂氧化性/还原性、氢化物稳定性递变；He惰性不考                                                                                                      |
| 2    | Li~Ne    | C/N/O/F非金属性递变；NH₃/HNO₃/NOx循环；CO₃²⁻/HCO₃⁻转化；HF刻蚀SiO₂；O₃/H₂O₂绿色氧化剂；Li/Be（Be(OH)₂两性，课标外但可类比Al）                                     |
| 3    | Na~Ar    | Na/Mg/Al金属性递变及冶炼；Al₂O₃/Al(OH)₃两性（碱溶除铝/调pH沉铝）；Si/SiO₂/硅酸盐工业；P/S/Cl含氧酸酸性；Cl₂/次氯酸盐/氯碱工业；S/硫酸工业；侯氏制碱（NaHCO₃沉淀） |

> 💡 **流程题提示**：第3周期元素常作为“已知参照物”出现（如“某元素性质与Al相似”“某酸酸性强于H₃PO₄”），用于推断陌生元素行为。

### 🔸 第4周期：过渡金属主战场（K~Kr）

#### 🟢 主族（K, Ca, Ga, Ge, As, Se, Br, Kr）

- **K/Ca**：K⁺/Ca²⁺不水解；CaSO₄微溶/ CaCO₃沉淀用于硬水软化或除SO₄²⁻/CO₃²⁻。
- **Ga**：+3价为主，Ga(OH)₃两性（类似Al），GaAs半导体，流程中常与Al/In共存，靠pH或萃取分离。
- **Ge**：GeO₂两性偏酸，GeCl₄易水解，半导体材料，高中少考。
- **As**：As₂O₃两性，AsO₃³⁻/AsO₄³⁻酸碱转化，剧毒；流程中常用Fe³⁺/Ca²⁺沉淀除砷（生成FeAsO₄/Ca₃(AsO₄)₂）。
- **Se**：H₂SeO₃中强酸，H₂SeO₄强氧化性（类似H₂SO₄但氧化性更强），光电材料。
- **Br**：海水提溴经典流程：`Cl₂氧化Br⁻ → 空气吹出Br₂ → SO₂吸收生成HBr → Cl₂再氧化 → 蒸馏`。Br₂/Br⁻氧化还原对常考。
- **Kr**：惰性，不考。

#### 🔵 过渡金属（Sc~Zn）⚠️ 高考绝对重点

| 元素      | 常见价态   | 核心性质/流程考点                                                                                                  |
| --------- | ---------- | ------------------------------------------------------------------------------------------------------------------ |
| **Ti**    | +4         | TiO₂（钛白粉）；TiCl₄极易水解：`TiCl₄+(x+2)H₂O→TiO₂·xH₂O↓+4HCl`；Kroll法：`TiCl₄+2Mg→Ti+2MgCl₂`（Ar保护）          |
| **V**     | +2~+5      | V₂O₅催化SO₂→SO₃；钒电池：`VO₂⁺/VO²⁺`（+5/+4）与`V³⁺/V²⁺`（+3/+2）氧化还原对，pH影响价态稳定性                      |
| **Cr**    | +3, +6     | Cr³⁺两性（pH>5沉淀，pH>12溶解）；CrO₄²⁻（黄）⇌ Cr₂O₇²⁻（橙）酸碱转化；Cr(VI)剧毒，流程中用Fe²⁺/SO₂还原为Cr³⁺后沉淀 |
| **Mn**    | +2, +4, +7 | MnO₂催化/氧化剂；KMnO₄氧化产物随pH变：酸性→Mn²⁺，中性→MnO₂，碱性→MnO₄²⁻；Mn²⁺稳定，调pH>8沉淀Mn(OH)₂               |
| **Fe**    | +2, +3     | Fe²⁺易氧化，Fe³⁺易水解（pH≈3完全沉淀）；炼钢除Si/P/S（CaO造渣）；FeCl₃蚀刻Cu；Fe(OH)₂白色→灰绿→红褐                |
| **Co/Ni** | +2, +3     | 电池正极材料（LiCoO₂, Ni(OH)₂）；Co²⁺/Ni²⁺沉淀pH接近，流程中常加氧化剂将Co²⁺→Co³⁺或加络合剂分离                    |
| **Cu**    | +1, +2     | Cu²⁺水解显酸性；湿法炼铜：`Fe+Cu²⁺→Fe²⁺+Cu`；电解精炼；Cu⁺不稳定：`2Cu⁺→Cu+Cu²⁺`；[Cu(NH₃)₄]²⁺深蓝络离子           |
| **Zn**    | +2         | ZnO/Zn(OH)₂两性（溶于强碱）；镀锌防腐；Zn²⁺沉淀pH≈6~8，常与Fe³⁺/Al³⁺分步沉淀                                       |

> 💡 **流程题高频套路**：
>
> - “含Ti/V/Cr/Mn/Fe/Cu/Zn的矿渣/废液” → 酸浸 → 调pH除Fe/Al → 氧化还原调价 → 沉淀/萃取 → 煅烧得氧化物
> - 注意题干给的`Ksp`、`沉淀pH范围`、`氧化剂/还原剂选择`（如H₂O₂绿色、Cl₂引入Cl⁻、NaClO碱性氧化）

| 元素            | 高考/流程考点                                                                                                        |
| --------------- | -------------------------------------------------------------------------------------------------------------------- |
| **Sr**          | 焰色红；SrSO₄极难溶（除SO₄²⁻或提锶）；SrCO₃溶于酸                                                                    |
| **Y/Zr/Nb/Mo**  | Y常与镧系共生；ZrO₂耐高温/核包壳；MoO₃两性偏酸，钼酸铵沉淀除钼；MoS₂润滑剂                                           |

> 💡 **流程题提示**：第5周期元素多出现在“电子废弃物回收”“催化剂再生”“半导体废料处理”情境。重点抓**络合溶解**（如Ag⁺/Pd²⁺/Pt⁴⁺溶于氨水/王水/氰化物）和**分步沉淀**（如Sn²⁺/Sb³⁺/Bi³⁺水解pH差异）。

### 🔸 第6周期：难熔金属、重金属与镧系（Cs~Rn + 镧系）

| ----------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Ba**            | BaSO₄不溶（钡餐/除SO₄²⁻）；BaCO₃溶于酸；焰色绿                                                                                                                                           |
| **Hf/Ta/W/Re**    | 难熔金属：W（钨丝，`WO₃+3H₂→W+3H₂O`；Na₂WO₄酸化得H₂WO₄）；Ta（电容，Ta₂O₅耐蚀）；Re（高温合金，Re₂O₇易溶）                                                                               |
| **Os/Ir/Pt/Au**   | 贵金属：Pt/Au王水溶解：`3Pt+4HNO₃+18HCl→3H₂[PtCl₆]+4NO↑+8H₂O`；`Au+4HCl+HNO₃→HAuCl₄+NO↑+2H₂O`；催化/氰化法提金：`4Au+8NaCN+O₂+2H₂O→4Na[Au(CN)₂]+4NaOH`（Zn置换）                         |

> 💡 **流程题提示**：

> - 重金属（Pb/Hg/Bi/Cd）除杂必考**沉淀法**（S²⁻/OH⁻/CO₃²⁻）和**吸附法**
> - 镧系/锕系背景题重在**信息提取**：“某稀土离子在pH=6开始沉淀，pH=8完全沉淀”“Ce³⁺可被NaClO氧化为Ce⁴⁺后沉淀”
> - 王水溶解、氰化提金、铅蓄电池反应是经典方程式考点

### ☢️ 锕系（Ac~Lr）：核工业背景（高中极浅）

- **Th/U/Pu** 为主：U以`UO₂²⁺`（铀酰离子）形式存在于酸性溶液，流程中常用`NH₃·H₂O`沉淀为`(NH₄)₂U₂O₇`（重铀酸铵），或TBP萃取提铀。
- **高中定位**：仅作为“核能/放射性/防护”背景出现，不考具体反应。若流程题出现，必给足信息（如“UO₂²⁺在pH>4水解”“可被H₂O₂还原为U⁴⁺”）。
- **注意**：放射性元素分离涉及离子交换/溶剂萃取，原理与稀土类似，但高考不要求掌握核反应方程。

### 📝 高考实战策略：如何“破译”流程题中的元素信息

1. **看价态**：题干给“+n价” → 联想已知同族/同周期元素（如“+4价Ti易水解”类比Si⁴⁺；“+6价Cr强氧化”类比S⁶⁺但Cr(VI)毒性大）
2. **看pH**：给“沉淀pH范围” → 画数轴，分步沉淀是除杂核心（Fe³⁺≈3，Al³⁺≈5，Cu²⁺≈6，Zn²⁺≈7）
3. **看试剂**：
    - `H₂O₂`：绿色氧化剂（Fe²⁺→Fe³⁺，Cr³⁺→CrO₄²⁻碱性条件）
    - `NaClO/Cl₂`：氧化+引入Cl⁻（注意后续除Cl⁻）
    - `NH₃·H₂O`：调pH+络合（Cu²⁺/Ag⁺/Zn²⁺/Ni²⁺）
    - `Na₂S/NaHS`：沉淀重金属（Ksp极小，注意H₂S毒性）
4. **看产物**：最终产品是氧化物/盐/单质 → 反推最后一步是煅烧/结晶/电解/置换
5. **陌生元素处理**：题干必给1~2条性质提示，直接套用周期律（金属性/非金属性/最高价含氧酸酸性/氢化物稳定性/两性/水解倾向）
