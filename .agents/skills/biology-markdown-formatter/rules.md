# 生物 Markdown 排版规则详解

**详细的错误与正确写法对照表**

---

## 一、生化分子式基础规范

### 1.1 基本原则

**所有生化分子式必须用 `$\ce{...}$` 包裹**

生物文档中最常出现的分子式包括：

| 类别 | 常见分子 | 错误写法 | 正确写法 |
|------|----------|----------|----------|
| 能量货币 | ATP / ADP / AMP | `ATP`(纯文本)、`$ATP$`(数学模式) | `$\ce{ATP}$` / `$\ce{ADP}$` / `$\ce{AMP}$` |
| 辅酶因子 | NAD⁺ / NADH | `$NAD^+$` | `$\ce{NAD+}$` / `$\ce{NADH}$` |
| 辅酶因子 | FAD / FADH₂ | `FADH2`、`$FADH_2$` | `$\ce{FAD}$` / `$\ce{FADH2}$` |
| 辅酶因子 | 辅酶 A / CoA | `CoA` | `$\ce{CoA}$` / `$\ce{CoA-SH}$` |
| 高能磷酸 | GTP / GDP / Pi | `GTP`、`$Pi$` | `$\ce{GTP}$` / `$\ce{GDP}$` / `$\ce{Pi}$` |
| 电子载体 | NADP⁺ / NADPH | `$NADP^+$` | `$\ce{NADP+}$` / `$\ce{NADPH}$` |
| 信号分子 | cAMP | `cAMP` | `$\ce{cAMP}$` |
| 代谢中间物 | 乙酰辅酶 A | `乙酰CoA` | `乙酰 \ce{CoA}` |
| 化学物质 | CO₂ / H₂O / O₂ | `$CO_2$`、`$H_2O$` | `$\ce{CO2}$` / `$\ce{H2O}$` / `$\ce{O2}$` |

✅ 正确示例：

```markdown
有氧呼吸以葡萄糖为底物，产生 $\ce{ATP}$ 和 $\ce{NADH}$。
糖酵解中消耗 $\ce{2ATP}$，产生 $\ce{4ATP}$，净得 $\ce{2ATP}$。
脂肪酸在 $\beta-$氧化中生成 $\ce{NADH}$、$\ce{FADH2}$ 和 $\ce{CoA}$。
```

### 1.2 下标与电荷写法

**在 `\ce{}` 内部，数字自动下标，不需要手动加 `_`：**

| 错误（数学模式） | 正确（`\ce{}`内） | 说明 |
|------|------|------|
| `$NAD^+$` | `$\ce{NAD+}$` | 离子电荷自动处理 |
| `$NADH$` | `$\ce{NADH}$` | 分子式直接连写 |
| `$CO_2$` | `$\ce{CO2}$` | 数字自动下标 |
| `$H_2O$` | `$\ce{H2O}$` | 数字自动下标 |
| `$Ca^{2+}$` | `$\ce{Ca^{2+}}$` | 复杂电荷用花括号 |
| `$O^{2-}$` | `$\ce{O^{2-}}$` | 复杂电荷用花括号 |
| `$Fe^{3+}$` | `$\ce{Fe^{3+}}$` | 复杂电荷用花括号 |

### 1.3 多个分子并列时的格式

**分子式之间的空格和符号处理：**

| 错误 | 正确 |
|------|------|
| `ATP / ADP` | `$\ce{ATP} / \ce{ADP}$` 或 `$\ce{ATP/ADP}$` |
| `NADH和FADH2` | `$\ce{NADH}$ 和 $\ce{FADH2}$` |
| `ATP → ADP + Pi` | `$\ce{ATP -> ADP + Pi}$`（在 `\ce`内） |
| `ATP-ADP循环` | `$\ce{ATP}$-$\ce{ADP}$ 循环` |

## 二、希腊字母标记

### 2.1 基础规则

希腊字母在生物文档中常见于代谢中间物和结构名称中。**必须使用 LaTeX `\` 转义序列，并将连字符也包含在数学模式中：**

| 错误 | 正确 | 说明 |
|------|------|------|
| `α-酮戊二酸` | `$\alpha-$酮戊二酸` | α 在数学模式，连字符也在数学模式 |
| `β-氧化` | `$\beta-$氧化` | β 和连字符一起在数学模式 |
| `β-折叠` | `$\beta-$折叠` | 同前 |
| `γ-氨基丁酸` | `$\gamma-$氨基丁酸` | 同前 |
| `$\beta$-氧化` | `$\beta-$氧化` | 连字符必须在数学模式内 |
| `$\alpha$-酮戊二酸` | `$\alpha-$酮戊二酸` | 连字符必须在数学模式内 |
| `α螺旋` | `$\alpha$ 螺旋` | 无连字符时，数学模式后加空格 |

### 2.2 常见希腊字母对照

```markdown
$\alpha$   — α（阿尔法）— 最常见于 α-酮戊二酸、α-淀粉酶
$\beta$    — β（贝塔）  — 最常见于 β-氧化、β-折叠
$\gamma$   — γ（伽马）  — γ-氨基丁酸、γ-亚麻酸
$\delta$   — δ（德尔塔）— δ-氨基乙酰丙酸
$\epsilon$ — ε（埃普西隆）
$\theta$   — θ（西塔）
$\kappa$   — κ（卡帕）
$\mu$      — μ（缪）    — μm（微米）
$\sigma$   — σ（西格玛）
$\omega$   — ω（欧米伽）
```

✅ 完整示例：

```markdown
- $\alpha-$酮戊二酸脱氢酶复合体催化 $\alpha-$酮戊二酸转化为琥珀酰辅酶 A
- 脂肪酸的 $\beta-$氧化产生大量 $\ce{NADH}$ 和 $\ce{FADH2}$
- $\gamma-$氨基丁酸（$\ce{GABA}$）是重要的抑制性神经递质
```

## 三、`[H]` 标记规范

### 3.1 基本写法

高中生物教材中常用 `[H]` 简化表示还原型辅酶（NADH 和 FADH₂）。**必须使用 `$[\ce{H}]$` 格式：**

| 错误 | 正确 | 说明 |
|------|------|------|
| `[H]` | `$[\ce{H}]$` | 化学式必须在 `\ce{}` 内 |
| `产生少量的 [H]` | `产生少量的 $[\ce{H}]$` | 数学模式包裹整个标记 |
| `[H] 与氧结合` | `$[\ce{H}]$ 与氧结合` | 数学模式不影响周围文本 |
| `大量的 [H]` | `大量的 $[\ce{H}]$` | 统一格式 |
| `这里的[H]是一种` | `这里的 $[\ce{H}]$ 是一种` | 中文字符和数学模式之间保留空格 |

### 3.2 特殊说明

当需要明确区分 NADH、FADH₂ 和简化标记 `[H]` 时：

- **简化写法**（适用于高中教材）：`$[\ce{H}]$」
- **明确写法**（适用于大学/竞赛）：直接使用 `$\ce{NADH}$」或`$\ce{FADH2}$」

✅ 正确示例：

```markdown
糖酵解阶段产生少量的 $[\ce{H}]$，并释放出少量能量。
上述两个阶段产生的 $[\ce{H}]$，经过一系列化学反应与 $\ce{O2}$ 结合。
这里的 $[\ce{H}]$ 是一种简化的表示方式，主要是指 $\ce{NADH}$。
```

## 四、代谢途径与酶

### 4.1 化学反应式

**所有代谢途径中的化学反应必须使用 `\ce{}` 包裹：**

✅ 正确：

```markdown
$$
\ce{C6H12O6 + 2NAD+ + 2ADP + 2Pi -> 2CH3COCOOH + 2NADH + 2H+ + 2ATP + 2H2O}
$$
```

❌ 错误：

```markdown
$$
C6H12O6 + 2NAD+ + 2ADP + 2Pi -> 2CH3COCOOH + 2NADH + 2H+ + 2ATP + 2H2O
$$
```

### 4.2 反应条件与箭头

使用 `->[条件]` 语法：

| 类型 | 错误 | 正确 |
|------|------|------|
| 可逆反应 | `<=>` | `<=>` 在 `\ce` 内自动处理 |
| 单向反应 | `→`(Unicode) | `->` |
| 光照条件 | `\xrightarrow{光照}` | `\ce{->[光照]}` |
| 酶催化 | `\xrightarrow{酶}` | `\ce{->[酶]}` |

### 4.3 `\text{}` 禁止规则

**绝对禁止**在 `\ce{}` 内使用 `\text{}`：

| 错误 | 正确 |
|------|------|
| `\ce{C6H12O6 + \text {能量}}` | `\ce{C6H12O6 + 能量}` |
| `\ce{2C3H6O3 (\text {乳酸})}` | `\ce{2C3H6O3 (乳酸)}` |
| `\ce{\text{葡萄糖}}` | `\text{葡萄糖}`（放在 `\ce` 外） |

**注意**：如果需要在反应式中包含中文文字，直接写在 `\ce{}` 内容中即可，mhchem 会将其作为文本渲染。

### 4.4 酶的名称

酶的名称在正文中一般使用纯文本（不加 `\ce{}`），除非是酶-底物复合物中的化学式成分：

```markdown
- 丙酮酸脱氢酶复合体（PDC）催化丙酮酸氧化脱羧
- 磷酸果糖激酶（PFK）是糖酵解的关键调控酶
- 柠檬酸合酶催化 $\ce{CoA-S-COCH3}$ 与草酰乙酸的缩合
```

## 五、物理单位

### 5.1 生化常用单位

| 量 | 错误 | 正确 |
|-----|------|------|
| 能量 | `2870 kJ` | `$\pu{2870 kJ}$` |
| 能量 | `30.5 kJ/mol` | `$\pu{30.5 kJ/mol}$` |
| 温度 | `37℃`、`37°C` | `$\pu{37 ^oC}$` |
| 温度 | `100 °C` | `$\pu{100 ^oC}$` |
| 物质的量 | `1 mol` | `$\pu{1 mol}$` |
| 摩尔质量 | `180 g/mol` | `$\pu{180 g/mol}$` |
| 体积 | `2 mL` | `$\pu{2 mL}$` |
| 浓度 | `0.1 mol/L` | `$\pu{0.1 mol/L}$` |
| 压力 | `245 kPa` | `$\pu{245 kPa}$` |
| pH | `pH = 7` | `$\text{pH} = 7$` 或直接 `pH = 7` |
| 电子 | `e⁻` | `$\pu{e^-}$` |

### 5.2 温度特殊写法

**统一使用 `\pu{数值 ^oC}` 格式：**

```markdown
✅ 正确：
$\pu{37 ^oC}$（人体温度）
$\pu{60 ^oC}$（淀粉酶最适温度）
$\pu{90 ^oC}$（加热条件）
$\pu{35 \sim 40 ^oC}$（动物酶最适温度范围）

❌ 错误：
$37℃$
$37^\circ\text{C}$
$\pu{37°C}$
```

## 六、数值与范围

### 6.1 范围表示

**使用 `\sim` 表示数值范围：**

| 错误 | 正确 |
|------|------|
| `30-32 ATP` | `$30 \sim 32$ $\ce{ATP}$` |
| `10~15 分钟` | `$10 \sim 15$ 分钟` |
| `35 ~ 40 ℃` | `$\pu{35 \sim 40 ^oC}$` |
| `pH 6.5 ~ 8.0` | `$\text{pH}~6.5 \sim 8.0$` 或 `pH $6.5 \sim 8.0$` |

### 6.2 约数表示

```markdown
✅ 约 $10^7 \sim 10^{13}$ 倍
✅ 约 $30 \sim 32$ 个 $\ce{ATP}$
✅ $60\% \sim 69\%$ 以热能形式散失
```

## 七、生化表格规范

### 7.1 代谢核算表格式

生物文档中常见的代谢组分流向核算表，应遵循以下格式：

```markdown
| 组分 | 消耗 | 生成 | 备注 |
| :--- | :--- | :--- | :--- |
| **碳链上的碳原子** | $6$ | $0$ | 完全氧化 |
| **$\ce{CO2}$** | $0$ | $6$ | - |
| **$\ce{H2O}$** | $0$ | $12$ | 还原产物 |
| **电子（$\pu{e^-}$）** | $24$ | $0$ | 注：$20$ 个来自 $\ce{NADH}$ |
| **$\ce{NADH} / \ce{NAD+}$** | $\ce{10NADH}$ | $\ce{10NAD+}$ | 氧化过程 |
| **$\ce{ATP} / \ce{ADP}$** | $\ce{26 \sim 28[ADP + Pi]}$ | $\ce{26 \sim 28[ATP]}$ | 氧化磷酸化 |
```

- 表头左对齐（`:---`）或居中（`:---:`）均可，但全表需一致
- 数字使用数学模式 `$6$`
- 化学式使用 `\ce{}`
- 单位使用 `\pu{}`
- 备注栏中的说明保留普通文本

## 八、细胞结构名词

### 8.1 细胞器与结构命名

细胞结构名词在正文中一般保持纯文本：

```markdown
- 线粒体基质（mitochondrial matrix）
- 细胞质基质（cytosol）
- 线粒体内膜（inner mitochondrial membrane）
- 类囊体膜（thylakoid membrane）
- 叶绿体基质（stroma）
```

但化学式化的结构成分需要 `\ce{}`：

```markdown
- 丙酮酸通过线粒体内膜的 $\ce{MPC}$（丙酮酸载体）进入
- 电子传递链位于线粒体内膜的复合体 $\ce{I}$ ~ $\ce{IV}$
```

## 九、气体与能量表述

### 9.1 气体符号

在 `\ce{}` 内部的气体符号 `^` 前后必须有空格：

| 错误 | 正确 |
|------|------|
| `$\ce{O2^}$` | `$\ce{O2 ^}$` |
| `$\ce{CO2^}$` | `$\ce{CO2 ^}$` |
| `$\ce{O2}\uparrow$` | `$\ce{O2 ^}$` |

### 9.2 能量表述

```markdown
- ≈ $\pu{2870 kJ/mol}$（葡萄糖彻底氧化释放能量）
- 合成 $\pu{1 mol}$ $\ce{ATP}$ 需 $\pu{30.5 kJ}$（标准条件）
- $\ce{NADH} \approx \pu{2.5 ATP}$（当前主流教材修正值）
- $\ce{FADH2} \approx \pu{1.5 ATP}$
- 电子传递链泵出 $\pu{10 H+}$ 每 $\ce{NADH}$（复合体 I / III / IV）
```

## 十、绝对禁忌清单

### 10.1 禁止清单

1. **禁止 `\ce` 嵌套**：`$\ce{\ce{ATP}}$` → 应为 `$\ce{ATP}$`
2. **禁止在 `\ce` 内使用 `\text`**：`$\ce{\text{ATP}}$` → 应为 `$\ce{ATP}$`
3. **禁止 Unicode 化学式**：`CO₂`(带 Unicode 下标)、`H₂O` → 应为 `\ce{CO2}`、`\ce{H2O}`
4. **禁止普通数学模式替代 `\ce{}`**：`$NAD^+$` → 应为 `$\ce{NAD+}$`
5. **禁止 `\pH` 命令**：`\pH` 在 LaTeX 中未定义，直接用文本 `pH`
6. **禁止连续 `[H]`**：`[H]` → 应为 `$[\ce{H}]$`
7. **禁止希腊字母文本化**：`α-酮戊二酸` → 应为 `$\alpha-$酮戊二酸`
8. **禁止 `\text{...}` 在 `$$...$$` 公式块中作为单位容器**：使用 `\pu{...}` 替代
9. **禁止旧式温度符号**：`℃`、`°C`、`^\circ\text{C}` → 应为 `\pu{^oC}`

### 10.2 空格规范速查

| 场景 | 规则 |
|------|------|
| 纯文本 `ATP` → `$\ce{ATP}$` | 转换后，`$\ce{ATP}$` 前后不用加额外空格 |
| 数字 + 中文 | 加空格：`6 个 $\ce{NADH}$` |
| 数学模式 + 中文 | 加空格：`$\ce{ATP}$ 合酶` |
| 单位 + 数字 | `\pu{}` 内处理，不额外加空格 |

## 十二、完整示例文档

```markdown

### 糖酵解

糖酵解发生在**细胞质基质**中，该过程**不需要氧气**。

**准备阶段**（消耗能量）：投入 **2 分子 $\ce{ATP}$**，将葡萄糖转化为两分子甘油醛-$3-$磷酸。

**放能阶段**（产生能量）：每一分子 $\ce{P-C3}$ 被氧化，脱下两个质子和两个电子，转移给一分子 $\ce{NAD+}$ 生成一分子 $\ce{NADH}$。

**能量统计**：

- 投入：**2 $\ce{ATP}$**
- 产出：**4 $\ce{ATP}$** + **2 $\ce{NADH}$**
- 净收益：**2 $\ce{ATP}$** + **2 $\ce{NADH}$**

**核心化学反应**：

$$
\ce{C6H12O6 + 2NAD+ + 2ADP + 2Pi -> 2CH3COCOOH + 2NADH + 2H+ + 2ATP + 2H2O}
$$

### 丙酮酸的去向

- **有氧条件**：丙酮酸进入线粒体基质，氧化脱羧生成乙酰辅酶 A，进入三羧酸循环
- **无氧条件**：丙酮酸转化为**乳酸**（动物）或**酒精和 $\ce{CO2}$**（酵母）

### $\beta-$氧化

脂肪酸在**线粒体基质**中通过 $\beta-$氧化分解：

1. 脱氢（生成 $\ce{FADH2}$）
2. 水合
3. 再脱氢（生成 $\ce{NADH}$）
4. 硫解（释放乙酰辅酶 A）
