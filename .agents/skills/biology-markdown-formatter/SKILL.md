# Biology Markdown Formatter

**生物 Markdown 排版标准化技能**

- 核心规则: `.agents/skills/biology-markdown-formatter/rules.md`
- 项目排版规范: `.agents/rules/style.md`（数学公式、空格、标点章节）

---

## 使用场景

触发条件（任意一条）：

- 用户要求修正生物文档（细胞呼吸、光合作用、遗传、代谢等）的排版格式
- 文档中包含生化分子式（ATP、NADH、FADH2 等）但格式不规范
- 文档中的希腊字母标记（如 α-酮戊二酸、β-氧化）未正确使用 LaTeX
- 文档中的 `[H]` 标记未使用 `$[\ce{H}]$` 格式
- 用户明确要求用本技能格式化

---

## 执行策略：批量并行处理模式

**核心原则**：在一次回复中并行调用多个工具，每次读取50行，批量匹配修改，确保精确度和效率最大化。

### 零外部依赖原则

- **禁止任何 CLI 工具**：包括 `review.js`、`autocorrect`、`markdownlint`、`node`、`npm`、`python` 等
- **仅使用内置工具**：`read`、`write`、`edit`、`grep`、`glob`
- **不引用 review skill**：本技能是完全自包含的

### 批量分段并行处理流程

每次响应内按 **50行** 为一个处理单元并行执行：

1. **读取**（并行）：
   - `read(filePath, offset=N, limit=50)` — 读取当前区块
   - `read(.agents/skills/biology-markdown-formatter/rules.md)` — 读取规则文档

2. **分析**（内部）：
   - 对照 `rules.md` 中的规范表，识别当前区块所有格式问题
   - 将同类问题分组，准备批量 `edit` 调用

3. **修正**（并行）：
   - 对每一处问题，准备精确匹配上下文的 `oldString → newString`
   - 一次性发起多个 `edit` 调用（同一行内的多处修改合并为一个 edit）
   - **确保 uniqueness**：每个 `oldString` 必须包含足够上下文，避免误匹配

4. **验证**（并行）：
   - `grep` 搜索常见错误模式，确认当前区块已修正
   - 如有遗漏，下一轮继续

5. **推进**：偏移量 `offset += 50`，下一轮继续

---

## 优先级排序（由高到低）

1. **生化分子式**（`ATP` → `$\ce{ATP}$`，`NADH` → `$\ce{NADH}$` 等）— 最高频问题
2. **希腊字母标记**（`α-酮戊二酸` → `$\alpha-$酮戊二酸`，`β-氧化` → `$\beta-$氧化`）— 用户最敏感
3. **`[H]` 标记**（`[H]` → `$[\ce{H}]$`）— 用户明确要求
4. **独立公式中的 `\text{}`**（`\ce{... \text {中文} ...}` → 拆除 `\text`）— 渲染必须
5. **物理单位**（`℃` → `$\pu{^oC}$`，`kJ` → `$\pu{kJ}$`）— 与化学规范一致
6. **数值范围**（`35 ~ 40` → `$35 \sim 40$`）— 与项目规范一致
7. **空格与标点**（数字与中文之间的空格、全角标点）— 排版一致性

## 并行检查清单（每50行）

处理每50行区块时，按此清单并行执行：

### [ ] 检查 1: 生化分子式

- [ ] 搜索 `ATP`、`ADP`、`AMP`、`NAD+`、`NADH`、`FAD`、`FADH2`、`CoA`、`Pi`、`GTP`、`GDP`、`cAMP`
- [ ] 确认每个都被 `$\ce{...}$` 包裹
- [ ] 确认不在 `\ce{}` 内重复包裹

**批量处理**：同一行内所有分子式，一个 `edit` 完成

### [ ] 检查 2: 希腊字母

- [ ] 搜索 `α-`、`β-`、`γ-`、`δ-`、`ε-` 等希腊字母前缀
- [ ] 确认格式为 `$\alpha-$中文`、`$\beta-$中文`
- [ ] 确认连字符在数学模式内

**批量处理**：将 `α-` → `$\alpha-$`，同时修正后续文字

### [ ] 检查 3: `[H]` 标记

- [ ] 搜索 `[H]`（不在代码块内）
- [ ] 替换为 `$[\ce{H}]$`
- [ ] 保留前后空格：`...的 $[\ce{H}]$ 与...`

### [ ] 检查 4: 独立公式中的 `\text{}`

- [ ] 在 `$$...$$` 区块内搜索 `\text{`
- [ ] 若在 `\ce{}` 内，拆除 `\text{}`：直接保留中文
- [ ] 若在 `\ce{}` 外，评估是否应为 `\pu{}` 或直接文本

### [ ] 检查 5: 温度与单位

- [ ] 搜索 `℃`、`°C`、`^\circ` 等温度符号
- [ ] 替换为 `$\pu{数值 ^oC}`
- [ ] 搜索 `kJ`、`mol`、`g/mol` 等单位
- [ ] 替换为 `$\pu{...}`

### [ ] 检查 6: 数值范围

- [ ] 搜索 `30-32`、`~108`、`35 ~ 40` 等范围表达
- [ ] 确认使用 `$30 \sim 32$`、`$\pu{35 \sim 40 ^oC}$`

### [ ] 检查 7: 空格规范

- [ ] 数字与中文之间：`6 个` 而非 `6个`
- [ ] 数学模式与中文之间：`$\ce{ATP}$ 合酶` 而非 `$\ce{ATP}$合酶`
- [ ] 标点符号：全角标点，括号前后空格一致

## 禁止操作

| 禁止项 | 说明 |
|--------|------|
| ❌ 使用 CLI 命令 | 包括 `review.js`、`autocorrect`、`markdownlint-cli2` |
| ❌ 调用 review skill | 本技能自包含 |
| ❌ 批量 `replaceAll` | 每次 `edit` 必须精确定位，不盲替换 |
| ❌ 修改标题层级结构 | 只改格式，不动文档结构 |
| ❌ 修改列表缩进风格 | 保留原文 |
| ❌ 修改内容语义 | 只改排版 |
| ❌ 在 `\ce{}` 内使用 `\text{}` | 必须拆除 |
| ❌ 在 `\ce{}` 内使用下划线 `_` | 数字自动下标，不需要 `_` |

---

## 验证标准

处理完毕后，文档应符合：

1. [ ] 所有生化分子式（ATP, ADP, NAD+, NADH, FAD, FADH2, CoA, Pi, GTP, GDP, cAMP 等）均在 `$\ce{}` 内
2. [ ] 所有希腊字母标记使用 `$\alpha-$`、`$\beta-$` 等格式
3. [ ] 所有 `[H]` 均写作 `$[\ce{H}]$`
4. [ ] 所有独立公式中的 `\text{}` 已拆除
5. [ ] 所有温度使用 `$\pu{^oC}` 格式
6. [ ] 所有物理单位使用 `$\pu{}` 包裹
7. [ ] 中英文/数字之间有适当空格
8. [ ] 无任何 `\ce` 嵌套或 `\ce` 内使用 `\text`

## 工作流示例

**场景**：用户要求修正 `docs/science/cell/3.md`

**执行**（单次响应内）：

```
# 第一轮：offset=1, limit=50
1. read(3.md, offset=1, limit=50)      → 获取第1-50行
2. read(rules.md)                      → 获取规则对照
3. grep -n "ATP" 3.md                  → 确认位置
4. 分析：发现 "ATP" 未包裹（行3, 行12, 行45）
5. edit(行3, "产生 ATP 为细胞" → "产生 $\\ce{ATP}$ 为细胞")
6. edit(行12, "...", "...")
7. edit(行45, "...", "...")
8. grep -n "ATP" 3.md                  → 验证已无裸露 ATP

# 第二轮：offset=51, limit=50（下一轮响应）
9. read(3.md, offset=51, limit=50)     → 获取第51-100行
...重复流程...
```

## 快速修复对照表

| 主题 | 错误写法 | 正确写法 |
|------|----------|----------|
| ATP | `ATP`、`$ATP$` | `$\ce{ATP}$` |
| NAD+ | `$NAD^+$` | `$\ce{NAD+}$` |
| NADH | `$NADH$` | `$\ce{NADH}$` |
| FADH2 | `FADH2`、`$FADH_2$` | `$\ce{FADH2}$` |
| [H] | `[H]` | `$[\ce{H}]$` |
| α-酮戊二酸 | `α-酮戊二酸`、`$\alpha$-酮戊二酸` | `$\alpha-$酮戊二酸` |
| β-氧化 | `β-氧化`、`$\beta$-氧化` | `$\beta-$氧化` |
| 温度 | `37℃`、`37°C` | `$\pu{37 ^oC}$` |
| 能量 | `2870 kJ` | `$\pu{2870 kJ}$` |
| 范围 | `30-32 ATP` | `$30 \sim 32$ $\ce{ATP}$` |
