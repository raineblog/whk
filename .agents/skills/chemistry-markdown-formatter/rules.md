# 化学 Markdown 排版规则详解

**详细的错误与正确写法对照表**

---

## 一、化学式基础规范

### 1.1 基本原则

**所有化学式必须用 `\ce{}` 包裹**

❌ 错误示例：

```markdown
$Na_2O$           # 普通数学模式
$\text{NaOH}$     # 使用 \text
NaOH              # 没有任何包裹
```

✅ 正确示例：

```markdown
$\ce{Na2O}$
$\ce{NaOH}$
$\ce{H2O}$
```

### 1.2 下标写法

在 `\ce{}` 内部，数字自动下标，不需要手动加 `_`：

| 错误 | 正确 | 说明 |
|------|------|------|
| `$\ce{Na_2O}$` | `$\ce{Na2O}$` | 不需要下划线 |
| `$\ce{H_2O_2}$` | `$\ce{H2O2}$` | 数字自动下标 |
| `$\ce{CO_3^{2-}}$` | `$\ce{CO3^{2-}}$` | 下划线只用于分隔非数字内容 |

### 1.3 离子电荷

| 类型 | 错误 | 正确 | 规则 |
|------|------|------|------|
| 单一正电荷 | `$Na^+$` | `$\ce{Na+}$` | 直接写 `+` |
| 单一负电荷 | `$Cl^-$` | `$\ce{Cl-}$` | 直接写 `-` |
| 多件正电荷 | `$Ca^{2+}$` | `$\ce{Ca^{2+}}$` | 数字需花括号 |
| 多件负电荷 | `$O^{2-}$` | `$\ce{O^{2-}}$` | 数字需花括号 |
| 复杂离子 | `$SO_4^{2-}$` | `$\ce{SO4^{2-}}$` | 花括号保护数字电荷 |
| 复杂离子 | `$NH_4^+$` | `$\ce{NH4+}$` | 简单正电荷不需花括号 |

## 二、气体与沉淀符号

### 2.1 写法规范

**在 `\ce{}` 内部使用简写符号，且前后必须有空格：**

| 情况 | 错误 | 正确 | 说明 |
|------|------|------|------|
| 气体向上 | `$\uparrow$` | `$\ce{^}$` | 在\ce内用`^` |
| 气体向上 | `$\ce{O2^}$` | `$\ce{O2 ^}$` | `^`前要空格 |
| 气体向上 | `$\ce{CO2^}$` | `$\ce{CO2 ^}$` | `^`前要空格 |
| 沉淀向下 | `$\downarrow$` | `$\ce{v}$` | 在\ce内用`v` |
| 沉淀向下 | `$\ce{BaSO4v}$` | `$\ce{BaSO4 v}$` | `v`前要空格 |

### 2.2 完整示例

```markdown
✅ 正确：
$\ce{2Na2O2 + 2H2O = 4NaOH + O2 ^}$
$\ce{NaCl + NH3 + CO2 + H2O = NaHCO3 v + NH4Cl}$
$\ce{2NaHCO3 ->[\Delta] Na2CO3 + H2O + CO2 ^}$

❌ 错误：
$\ce{2Na2O2 + 2H2O = 4NaOH + O2^}$      # 缺少空格
$\ce{2Na2O2 + 2H2O = 4NaOH} + O2\uparrow$  # \uparrow在\ce外
```

## 三、反应条件与箭头

### 3.1 加热符号

**使用 `->[条件]` 语法，禁止 `\xlongequal` 和 `\xrightarrow` 等**

| 错误 | 正确 | 说明 |
|------|------|------|
| `$\xlongequal{\Delta}$` | `$\ce{->[\Delta]}$` | 用 -> 代替 |
| `$\xrightarrow{\Delta}$` | `$\ce{->[\Delta]}$` | 用 -> 代替 |
| `$\stackrel{\Delta}{=}$` | `$\ce{->[\Delta]}$` | 用 -> 代替 |

### 3.2 完整反应式示例

```markdown
❌ 错误：
$2Na_2O + O_2 \xlongequal{\Delta} 2Na_2O_2$
$2NaHCO_3 \xrightarrow{\Delta} Na_2CO_3 + H_2O + CO_2\uparrow$

✅ 正确：
$\ce{2Na2O + O2 ->[\Delta] 2Na2O2}$
$\ce{2NaHCO3 ->[\Delta] Na2CO3 + H2O + CO2 ^}$
```

## 四、物理单位规范

### 4.1 使用 `\pu{}` 宏包

**`\pu` = physics unit，专门用于物理单位**

| 类型 | 错误 | 正确 | 说明 |
|------|------|------|------|
| 密度 | `$0.97 \text{ g/cm}^3$` | `$\pu{0.97 g/cm^3}$` | 用 \pu |
| 温度 | `$180.5^\circ\text{C}$` | `$\pu{180.5 ^oC}$` | 用 ^oC 而非 ^\circ\text{C} |
| 温度 | `$97.81 \text{ ℃}$` | `$\pu{97.81 ^oC}$` | 用 ^oC 而非 ℃ |
| 物质的量 | `$1 \text{ mol}$` | `$\pu{1 mol}$` | 用 \pu |
| 质量 | `$19 \text{ g}$` | `$\pu{19 g}$` | 用 \pu |
| 浓度 | `$0.1 \text{ mol/L}$` | `$\pu{0.1 mol/L}$` | 用 \pu |

### 4.2 温度写法特殊说明

**使用 `^oC` 而非 `^\circ\text{C}` 或 `℃`：**

```markdown
✅ 正确：
$\pu{97.81 ^oC}$
$\pu{638 ^oC}$
$\pu{180.5 ^oC}$

❌ 错误：
$97.81^\circ\text{C}$
$97.81 \text{ ℃}$
$638^\circ C$
```

### 4.3 完整示例

```markdown
- **密度**：锂是**密度最小的金属**（约 $\pu{0.534 g/cm^3}$）。
- **熔点**：约为 $\pu{180.5 ^oC}$，沸点约为 $\pu{1342 ^oC}$。
- **电子转移**：每生成 $\pu{1 mol}$ $\ce{O2}$ 转移 $\pu{2 mol}$ 电子。
```

## 五、数学模式中的化学符号

### 5.1 离子半径比较

**即使在数学表达式中，化学符号也要用 `\ce{}`：**

| 错误 | 正确 |
|------|------|
| `$r(M^+) < r(M)$` | `$r(\ce{M+}) < r(\ce{M})$` |
| `$r(Li^+) < r(Na^+)$` | `$r(\ce{Li+}) < r(\ce{Na+})$` |
| `$r(Na^+) < r(K^+)$` | `$r(\ce{Na+}) < r(\ce{K+})$` |
| `$r(Li^+) < r(Na^+) < r(K^+) < r(Rb^+) < r(Cs^+)$` | `$r(\ce{Li+}) < r(\ce{Na+}) < r(\ce{K+}) < r(\ce{Rb+}) < r(\ce{Cs+})$` |

### 5.2 通式中的化学式

| 错误 | 正确 |
|------|------|
| `$(CO)_m(H_2)_n$` | `$(\ce{CO})_m(\ce{H2})_n$` |
| `$(H_2O)_n$` | `$(\ce{H2O})_n$` |

### 5.3 其他数学表达式

```markdown
❌ 错误：
$V_1 = V_2$ 时，溶质仅为 $Na_2CO_3$

✅ 正确：
$V_1 = V_2$ 时，溶质仅为 $\ce{Na2CO3}$

---

❌ 错误：
电子转移的电子数为 $2N_A$

✅ 正确：
电子转移的电子数为 $2\ce{N_A}$
```

## 六、化学术语与括号格式

### 6.1 化学名称与式的组合

**保持中文习惯，中文名称后跟括号内写化学式**

| 错误 | 正确 |
|------|------|
| `氢化钠 ($NaH$)` | `氢化钠（$\ce{NaH}$）` |
| `氧化钠 ($Na_2O$)：` | `氧化钠（$\ce{Na2O}$）：` |
| `过氧化钠 ($Na_2O_2$)` | `过氧化钠（$\ce{Na2O2}$）` |
| `sodium ($\ce{Na}$)` | `钠（$\ce{Na}$）` |

### 6.2 标点符号

**中文主体文字使用中文全角标点：**

| 错误 | 正确 |
|------|------|
| `氢化钠($NaH$):` | `氢化钠（$\ce{NaH}$）：` |
| `氧化钠（$\ce{Na2O}$）:` | `氢化钠（$\ce{Na2O}$）：` |
| `与水反应,生成...` | `与水反应，生成...` |
| `参数为:约...` | `参数为：约...` |

| 错误 | 正确 |

| `Li $\rightarrow$ Cs` | `$\ce{Li -> Cs}$` |
| `Na $\rightarrow$ K` | `$\ce{Na -> K}$` |

## 八、多行公式格式

### 8.1 空行规范

**多行显示公式 $$ 前后必须有空行：**

```markdown
✅ 正确：

$$
\ce{2Na(s) + H2(g) ->[\Delta] 2NaH(s)}
$$

❌ 错误：
$$
\ce{2Na(s) + H2(g) ->[\Delta] 2NaH(s)}
$$

❌ 错误（内部空行）：
$$

\ce{2Na(s) + H2(g) ->[\Delta] 2NaH(s)}

$$
```

### 8.2 行内公式

**行内公式 $...$ 不需要额外空行**

```markdown
✅ 正确：
氢化钠与水反应生成 $\ce{NaOH}$

❌ 不必要的空行：
氢化钠与水反应生成
$\ce{NaOH}$
```

## 九、绝对禁忌清单

### 9.1 禁止 `\ce` 嵌套

```markdown
❌ 绝对禁止：
$\ce{\ce{Na2O}}$     # \ce 内又套 \ce

✅ 正确：
$\ce{Na2O}$
```

### 9.2 禁止在 `\ce` 内使用 `\text`

```markdown
❌ 绝对禁止：
$\ce{\text{NaOH}}$   # \ce 内使用 \text

✅ 正确：
$\ce{NaOH}$
```

### 9.3 禁止气体符号在 `\ce` 外

```markdown
❌ 绝对禁止：
$\ce{O2}\uparrow$     # \uparrow 在 \ce 外
$\ce{H2} \uparrow$    # \uparrow 在 \ce 外

✅ 正确：
$\ce{O2 ^}$
$\ce{H2 ^}$
```

### 9.4 禁止旧版箭头命令

```markdown
❌ 绝对禁止：
$\xlongequal{\Delta}$   # 使用 ->[\Delta] 代替
$\xrightarrow{\Delta}$ # 使用 ->[\Delta] 代替
$\stackrel{\Delta}{=}$  # 使用 ->[\Delta] 代替
```

### 9.5 禁止复杂下划线在 \ce 内

```markdown
❌ 绝对禁止：
$\ce{Na_2O}$          # 不需要下划线
$\ce{H_2O_2}$         # 不需要下划线

✅ 正确：
$\ce{Na2O}$
$\ce{H2O2}$
```

## 十、完整示例文档

### 示例：钠元素化合物

```markdown
## 钠的氧化物

钠有两种重要的氧化物：**氧化钠（$\ce{Na2O}$）**和**过氧化钠（$\ce{Na2O2}$）**。

### 氧化钠（$\ce{Na2O}$）

1. **物理性质与结构**：
    - **外观**：白色固体。
    - **离子个数比**：阴、阳离子（$\ce{Na+} : \ce{O^{2-}}$）个数比为 **$2:1$**。

2. **化学性质**：
    - **与水反应**：$\ce{Na2O + H2O = 2NaOH}$
    - **与酸反应**：$\ce{Na2O + 2HCl = 2NaCl + H2O}$

### 过氧化钠（$\ce{Na2O2}$）

1. **物理性质**：
    - **外观**：**淡黄色**固体或粉末。
    - **结构**：氧元素的化合价为 **$-1$ 价**。

2. **化学性质**：
    - **与水反应**：$\ce{2Na2O2 + 2H2O = 4NaOH + O2 ^}$
    - **与二氧化碳反应**：$\ce{2Na2O2 + 2CO2 = 2Na2CO3 + O2}$
    - **加热分解**：$\ce{2NaHCO3 ->[\Delta] Na2CO3 + H2O + CO2 ^}$

### 核心考点

1. **电子转移规律**：每生成 $\pu{1 mol}$ $\ce{O2}$，转移的电子数为 $\pu{2 mol}$。

2. **离子半径比较**：
    $r(\ce{Li+}) < r(\ce{Na+}) < r(\ce{K+}) < r(\ce{Rb+}) < r(\ce{Cs+})$

3. **递变关系**：$\ce{Li -> Cs}$
```
