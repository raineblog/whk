# 化学 Markdown 排版检查清单

**系统性检查步骤**

---

## 检查前准备

- [ ] 确保理解本清单的所有规则
- [ ] 准备好 `rules.md` 作为参考
- [ ] 开启文档的 KaTeX 渲染预览

---

## 第一轮：化学式基础检查

### 1.1 检查 `\ce{}` 使用

- [ ] 检索所有 `$...$` 内的化学式
- [ ] 确保每个化学式都用 `\ce{}` 包裹
- [ ] 确认没有孤立的 `$Na_2O$` 等未包裹形式

**搜索模式：**

- `\[A-Z][a-z]?_\d` —— 查找未使用 `\ce` 的化学式
- `\b(Na|K|Ca|Mg|Fe|Cu|Zn|Cl|O|H|C|N|S|P)[\d_]` —— 常见元素

### 1.2 检查下标写法

- [ ] 在 `\ce{}` 内部，确保没有多余的 `_`
- [ ] 数字直接连写，如 `Na2O` 而非 `Na_2O`
- [ ] 花括号内的复杂下标保留，如 `CO3^{2-}`

**搜索模式：**

- `\ce{[A-Z][a-z]?_\d` —— 查找 `\ce` 内错误使用下划线

### 1.3 检查离子电荷

- [ ] 简单电荷直接写：`Na+`, `Cl-`
- [ ] 复杂电荷用花括号：`Ca^{2+}`, `O^{2-}`
- [ ] 确保离子符号在 `\ce{}` 内

**搜索模式：**

- `[\^\-]_\{` —— 确保电荷在 `\ce` 内

---

## 第二轮：气体与沉淀符号

### 2.1 检查气体符号 `^`

- [ ] 确保气体符号在 `\ce{}` 内部
- [ ] 确保 `^` 前后都有空格

**搜索模式：**

- `\uparrow` —— 应替换为 `^`
- `\\ce{[^}]+[^^ ]\^[^ }]"` —— 查找 `^` 前缺少空格
- `\\ce{[^}]+\^[^{}]+[^ }]` —— 查找 `^` 后缺少空格

### 2.2 检查沉淀符号 `v`

- [ ] 确保沉淀符号在 `\ce{}` 内部
- [ ] 确保 `v` 前后都有空格

**搜索模式：**

- `\downarrow` —— 应替换为 `v`
- `\\ce{[^}]+[^ ]v` —— 查找 `v` 前缺少空格
- `\\ce{[^}]+v[^ }]` —— 查找 `v` 后缺少空格

---

## 第三轮：反应条件与箭头

### 3.1 检查加热符号

- [ ] 将 `\xlongequal{\Delta}` 替换为 `\ce{->[\Delta]}`
- [ ] 将 `\xrightarrow{\Delta}` 替换为 `\ce{->[\Delta]}`
- [ ] 将 `\stackrel{\Delta}{=}` 替换为 `\ce{->[\Delta]}`

**搜索模式：**

- `\\xlongequal` —— 查找旧版加热符号
- `\\xrightarrow` —— 查找箭头命令

### 3.2 检查反应箭头

- [ ] 确保所有反应箭头在 `\ce{}` 内
- [ ] 确保使用 `->` 而非 `\to` 或 `\rightarrow`

---

## 第四轮：物理单位

### 4.1 检查 `\pu{}` 使用

- [ ] 将 `数值 \text{ 单位}` 替换为 `\pu{数值 单位}`
- [ ] 确保温度使用 `^oC` 格式

**搜索模式：**

- `\d+\.?\d*\s*\\text{` —— 查找未使用 \pu 的单位
- `\^\\circ` —— 查找旧版温度格式

### 4.2 常见物理量检查

| 量 | 正确格式 | 检查点 |
|-----|----------|---------|
| 密度 | `\pu{0.97 g/cm^3}` | 单位用 `\pu` |
| 温度 | `\pu{97.81 ^oC}` | 用 `^oC` 而非 `^\circ\text{C}` |
| 物质的量 | `\pu{1 mol}` | 用 `\pu` |
| 质量 | `\pu{19 g}` | 用 `\pu` |
| 浓度 | `\pu{0.1 mol/L}` | 用 `\pu` |
| 体积 | `\pu{22.4 L/mol}` | 用 `\pu` |

---

## 第五轮：数学模式中的化学符号

### 5.1 检查离子半径表达式

- [ ] 确保离子半径表达式中的化学符号使用 `\ce{}`

**正确示例：**

```latex
$r(\ce{M+}) < r(\ce{M})$
$r(\ce{Li+}) < r(\ce{Na+}) < r(\ce{K+})$
```

**搜索模式：**

- `r\([A-Z][a-z]?[\^\+]\)` —— 查找离子半径表达式

### 5.2 检查通式中的化学式

**正确示例：**

```latex
$(\ce{CO})_m(\ce{H2})_n$
$(\ce{H2O})_n$
```

---

## 第六轮：标点与格式

### 6.1 检查括号内的化学式

- [ ] 确保化学名称后的化学式使用 `\ce{}`

**正确示例：**

```markdown
氢化钠（$\ce{NaH}$）
氧化钠（$\ce{Na2O}$）：
过氧化钠（$\ce{Na2O2}$）
```

### 6.2 检查标点符号

- [ ] 确保使用中文全角标点：`：`、`，`、`。`
- [ ] 确保括号内外的空格正确

---

## 第七轮：禁忌项检查

### 7.1 绝对禁止清单

- [ ] 检查没有 `\ce{\ce{...}}` 嵌套
- [ ] 检查没有 `\ce{\text{...}}` 混用
- [ ] 检查没有 `\ce{...}\uparrow` 或 `\ce{...}\downarrow`

**搜索模式：**

```
\\ce\\{\\s*\\ce\\{  # \ce 嵌套
\\ce\\{[^}]*\\text\\{  # \ce 内使用 \text
\\}\\s*\\\\uparrow   # \uparrow 在 \ce 外
\\}\\s*\\\\downarrow # \downarrow 在 \ce 外
```

### 7.2 检查旧版命令

- [ ] 确保没有 `\xlongequal`
- [ ] 确保没有 `\xrightarrow`
- [ ] 确保没有 `\stackrel`
- [ ] 确保没有在 `\ce` 内的 `_` 下划线

---

## 第八轮：多行公式检查

### 8.1 检查空行

- [ ] 确保 `$$...$$` 前后都有空行
- [ ] 确保 `$$` 内部没有额外空行

**正确示例：**

```markdown

$$
\ce{2Na(s) + H2(g) ->[\Delta] 2NaH(s)}
$$

```

### 8.2 检查单行公式

- [ ] 行内公式 `$...$` 不需要额外空行
- [ ] 确保没有连续的空行打断文本

---

## 验证阶段

### 9.1 KaTeX 渲染验证

- [ ] 在预览中检查所有公式
- [ ] 确保没有红色错误提示
- [ ] 确保化学式渲染正确

### 9.2 文本搜索验证

- [ ] 搜索 `$` 确保化学式都在 `\ce` 内
- [ ] 搜索 `\text{` 确保化学式没有使用 `\text`
- [ ] 搜索 `\uparrow` 和 `\downarrow` 确保已替换

### 9.3 视觉检查

- [ ] 确认气体符号 `↑` 正确显示
- [ ] 确认沉淀符号 `↓` 正确显示
- [ ] 确认上下标位置正确

---

## 最终确认

- [ ] 所有化学式都使用 `\ce{}`
- [ ] 气体/沉淀符号在 `\ce` 内部且前后有空格
- [ ] 加热条件使用 `->[\Delta]` 语法
- [ ] 物理单位使用 `\pu{}`
- [ ] 温度使用 `^oC` 格式
- [ ] 数学模式中的化学符号也使用 `\ce{}`
- [ ] 没有 `\ce` 嵌套
- [ ] 没有 `\ce` 内使用 `\text`
- [ ] 没有 `\xlongequal` 或 `\xrightarrow`
- [ ] 多行公式空行正确

---

## 注意事项

1. **逐行检查**：不要进行大规模批量替换，逐行审查每个化学式
2. **先易后难**：先修正明显的 `\ce` 缺失，再处理复杂的反应条件
3. **双重验证**：每次修改后都用 KaTeX 渲染验证
4. **保留原文结构**：不要改变列表缩进、段落结构等格式
5. **注意空格**：`中文（$\ce{...}$）中文` 中的括号前后空格
