```markdown
#### 辨析：概率为 0 与不可能事件，概率为 1 与必然事件

这是概率论中极易混淆的核心概念。必须严格区分**集合论层面**和**概率论层面**的概念：

**互斥与对立的集合定义**：
- 互斥事件：$A \cap B = \varnothing$（绝对没有交集）
- 对立事件：$A \cap B = \varnothing$ 且 $A \cup B = \Omega$（互斥且完备）

**常见误解陷阱**：

| 误解 | 正确理解 |
|------|----------|
| $P(A \cap B) = 0 \implies A \cap B = \varnothing$ | $P(A \cap B) = 0$ 只说明交集是**零概率事件**，但交集可能**非空** |
| $P(A \cup B) = 1 \implies A \cup B = \Omega$ | $P(A \cup B) = 1$ 只说明并集是**几乎必然事件**，但并集可能**不等于全集** |

**关键区分**：
- **不可能事件**：集合为空集 $\varnothing$，逻辑上**绝对不发生**
- **零概率事件**：集合**非空**但测度为 0，称为**几乎不可能**（Almost Never）
- **必然事件**：集合为全集 $\Omega$，逻辑上**绝对发生**
- **概率为 1 的事件**：集合**不等于** $\Omega$ 但测度为 1，称为**几乎必然**（Almost Surely）

**经典反例（连续型几何概型）**：
设样本空间 $\Omega = [0,1]$（均匀分布），令：
- $A = [0, 0.5]$
- $B = [0.5, 1]$

则：
- $P(A) = 0.5,\ P(B) = 0.5,\ P(A)+P(B)=1$
- $P(A\cup B)=1$
- 但 $A \cap B = \{0.5\} \neq \varnothing$，故 $A,B$ 不是对立事件

**根本原因**：在连续空间中，**单点集的概率为 0 但非空**。概率为 0 不等于空集，概率为 1 不等于全集。

**适用边界**：仅在**古典概型**（样本点有限且概率均大于 0）中，概率极值才等价于集合极值。
```
