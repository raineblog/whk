# 有机反应记忆 {#_1}

## 烃 {#_2}

### 烷烃（甲烷） {#_3}

烷烃通式：[]{.arithmatex}$\ce{C_nH_{2n+2}}$

1.  氧化反应：

    甲烷**不能使**酸性高锰酸钾溶液及溴水褪色。

    甲烷的燃烧：

    :::: {.arithmatex}
    ::: {.MathJax_Preview}
    \ce{CH4 + 2O2 -\>\[\text{点燃}\] CO2 + 2H2O}
    :::

    $$\ce{CH4 + 2O2 ->[\text{点燃}] CO2 + 2H2O}$$
    ::::

2.  取代反应：

    :::: {.arithmatex}
    ::: {.MathJax_Preview}
    \ce{CH4 + Cl2 -\>\[\text{光照}\] CH3Cl + HCl}
    :::

    $$\ce{CH4 + Cl2 ->[\text{光照}] CH3Cl + HCl}$$
    ::::

### 烯烃（乙烯） {#_4}

烯烃通式：[]{.arithmatex}$\ce{C_nH_{2n}}$

1.  乙烯的制取：

    :::: {.arithmatex}
    ::: {.MathJax_Preview}
    \ce{CH3CH2OH -\>\[\text{浓}H2SO4\]\[\pu{170\^oC}\] CH2=CH2 \^ + H2O}
    :::

    $$\ce{CH3CH2OH ->[\text{浓}H2SO4][\pu{170^oC}] CH2=CH2 ^ + H2O}$$
    ::::

2.  氧化反应：

    乙烯**能使**酸性高锰酸钾溶液褪色，发生氧化反应。

    乙烯的燃烧：

    :::: {.arithmatex}
    ::: {.MathJax_Preview}
    \ce{C2H4 + 3O2 -\>\[\text{点燃}\] 2CO2 + 2H2O}
    :::

    $$\ce{C2H4 + 3O2 ->[\text{点燃}] 2CO2 + 2H2O}$$
    ::::

3.  加成反应：

    乙烯与溴水加成：

    :::: {.arithmatex}
    ::: {.MathJax_Preview}
    \ce{CH2=CH2 + Br2 -\> CH2Br-CH2Br}
    :::

    $$\ce{CH2=CH2 + Br2 -> CH2Br-CH2Br}$$
    ::::

    乙烯与氢气加成：

    :::: {.arithmatex}
    ::: {.MathJax_Preview}
    \ce{CH2=CH2 + H2 -\>\[\text{催化剂}\] CH3-CH3}
    :::

    $$\ce{CH2=CH2 + H2 ->[\text{催化剂}] CH3-CH3}$$
    ::::

    乙烯与水加成：

    :::: {.arithmatex}
    ::: {.MathJax_Preview}
    \ce{CH2=CH2 + H2O -\>\[\text{催化剂}\]\[\text{高温高压}\] CH3CH2OH}
    :::

    $$\ce{CH2=CH2 + H2O ->[\text{催化剂}][\text{高温高压}] CH3CH2OH}$$
    ::::

4.  加聚反应：

    乙烯加聚反应：

    :::: {.arithmatex}
    ::: {.MathJax_Preview}
    \ce{nCH2=CH2 -\>\[\text{引发剂}\] -\[CH2-CH2\]\_n-}
    :::

    $$\ce{nCH2=CH2 ->[\text{引发剂}] -[CH2-CH2]_n-}$$
    ::::

    1,3-丁二烯的加成（1,4 加成）：

    :::: {.arithmatex}
    ::: {.MathJax_Preview}
    \ce{CH2=CH-CH=CH2 + H2 -\>\[\text{1,4加成}\] CH3-CH=CH-CH3}
    :::

    $$\ce{CH2=CH-CH=CH2 + H2 ->[\text{1,4加成}] CH3-CH=CH-CH3}$$
    ::::

    1,3-丁二烯的加聚：

    :::: {.arithmatex}
    ::: {.MathJax_Preview}
    \ce{nCH2=CH-CH=CH2 -\>\[\text{催化剂}\] -\[CH2-CH=CH-CH2\]\_n-}
    :::

    $$\ce{nCH2=CH-CH=CH2 ->[\text{催化剂}] -[CH2-CH=CH-CH2]_n-}$$
    ::::

### 炔烃（乙炔） {#_5}

炔烃的通式：[]{.arithmatex}$\ce{C_nH_{2n-2}}$

1.  乙炔的制取：

    :::: {.arithmatex}
    ::: {.MathJax_Preview}
    \ce{CaC2 + 2H2O -\> Ca(OH)2 + CH#CH \^}
    :::

    $$\ce{CaC2 + 2H2O -> Ca(OH)2 + CH#CH ^}$$
    ::::

2.  氧化反应：

    乙炔**能使**酸性高锰酸钾溶液褪色，发生氧化反应。

    乙炔的燃烧：

    :::: {.arithmatex}
    ::: {.MathJax_Preview}
    \ce{2C2H2 + 5O2 -\>\[\text{点燃}\] 4CO2 + 2H2O}
    :::

    $$\ce{2C2H2 + 5O2 ->[\text{点燃}] 4CO2 + 2H2O}$$
    ::::

3.  加成反应：

    与溴水加成：

    :::: {.arithmatex}
    ::: {.MathJax_Preview}
    \ce{CH#CH + 2Br2 -\> CHBr2-CHBr2}
    :::

    $$\ce{CH#CH + 2Br2 -> CHBr2-CHBr2}$$
    ::::

    与氢气加成：

    :::: {.arithmatex}
    ::: {.MathJax_Preview}
    \ce{CH#CH + 2H2 -\>\[\text{催化剂}\] CH3-CH3}
    :::

    $$\ce{CH#CH + 2H2 ->[\text{催化剂}] CH3-CH3}$$
    ::::

    与氯化氢加成（制聚氯乙烯单体）：

    :::: {.arithmatex}
    ::: {.MathJax_Preview}
    \ce{CH#CH + HCl -\>\[\text{催化剂}\]\[\triangle\] CH2=CHCl}
    :::

    $$\ce{CH#CH + HCl ->[\text{催化剂}][\triangle] CH2=CHCl}$$
    ::::

4.  聚合反应：

    氯乙烯加聚，得到聚氯乙烯：

    :::: {.arithmatex}
    ::: {.MathJax_Preview}
    \ce{nCH2=CHCl -\>\[\text{催化剂}\] -\[CH2-CHCl\]\_n-}
    :::

    $$\ce{nCH2=CHCl ->[\text{催化剂}] -[CH2-CHCl]_n-}$$
    ::::

    乙炔加聚，得到聚乙炔：

    :::: {.arithmatex}
    ::: {.MathJax_Preview}
    \ce{nCH#CH -\>\[\text{催化剂}\] -\[CH=CH\]\_n-}
    :::

    $$\ce{nCH#CH ->[\text{催化剂}] -[CH=CH]_n-}$$
    ::::

### 苯 {#_6}

苯的同系物通式：[]{.arithmatex}$\ce{C_nH_{2n-6}}$

1.  氧化反应：

    苯**不能使**溴水和酸性高锰酸钾溶液褪色。

    苯的燃烧：

    :::: {.arithmatex}
    ::: {.MathJax_Preview}
    \ce{2C6H6 + 15O2 -\>\[\text{点燃}\] 12CO2 + 6H2O}
    :::

    $$\ce{2C6H6 + 15O2 ->[\text{点燃}] 12CO2 + 6H2O}$$
    ::::

2.  取代反应：

    苯与溴反应：

    :::: {.arithmatex}
    ::: {.MathJax_Preview}
    \ce{C6H6 + Br2(\text{液}) -\>\[FeBr3\] C6H5Br + HBr}
    :::

    $$\ce{C6H6 + Br2(\text{液}) ->[FeBr3] C6H5Br + HBr}$$
    ::::

    硝化反应：

    :::: {.arithmatex}
    ::: {.MathJax_Preview}
    \ce{C6H6 + HNO3(\text{浓}) -\>\[\text{浓}H2SO4\]\[50\sim60\pu{\^oC}\] C6H5NO2 + H2O}
    :::

    $$\ce{C6H6 + HNO3(\text{浓}) ->[\text{浓}H2SO4][50\sim60\pu{^oC}] C6H5NO2 + H2O}$$
    ::::

3.  苯与氢气加成反应：

    :::: {.arithmatex}
    ::: {.MathJax_Preview}
    \ce{C6H6 + 3H2 -\>\[\text{催化剂}\] C6H12}
    :::

    $$\ce{C6H6 + 3H2 ->[\text{催化剂}] C6H12}$$
    ::::

    产物为环己烷。

### 甲苯 {#_7}

甲苯**不能使**溴水褪色。

1.  氧化反应：**能使**酸性高锰酸钾溶液褪色。

    :::: {.arithmatex}
    ::: {.MathJax_Preview}
    \ce{5C6H5CH3 + 6KMnO4 + 9H2SO4 -\> 5C6H5COOH + 3K2SO4 + 6MnSO4 + 14H2O}
    :::

    $$\ce{5C6H5CH3 + 6KMnO4 + 9H2SO4 -> 5C6H5COOH + 3K2SO4 + 6MnSO4 + 14H2O}$$
    ::::

    注：氧化成苯甲酸。

2.  取代反应（制 TNT）：

    :::: {.arithmatex}
    ::: {.MathJax_Preview}
    \ce{C6H5CH3 + 3HNO3(\text{浓}) -\>\[\text{浓}H2SO4\]\[\triangle\] C6H2(NO2)3CH3 + 3H2O}
    :::

    $$\ce{C6H5CH3 + 3HNO3(\text{浓}) ->[\text{浓}H2SO4][\triangle] C6H2(NO2)3CH3 + 3H2O}$$
    ::::

    注意：甲苯在光照条件下发生侧链的取代，而在催化剂条件下发生苯环上的取代。

3.  与氢气加成反应：

    :::: {.arithmatex}
    ::: {.MathJax_Preview}
    \ce{C6H5CH3 + 3H2 -\>\[\text{催化剂}\] C6H11CH3}
    :::

    $$\ce{C6H5CH3 + 3H2 ->[\text{催化剂}] C6H11CH3}$$
    ::::

    产物为甲基环己烷。

## 烃的衍生物 {#_8}

### 溴乙烷 {#_9}

纯净的溴乙烷是无色液体，沸点 []{.arithmatex}$\pu{38.4 ^\circ C}$，密度比水大。

1.  取代反应：溴乙烷的水解反应

    :::: {.arithmatex}
    ::: {.MathJax_Preview}
    \ce{CH3CH2Br + NaOH -\>\[H2O\]\[\triangle\] CH3CH2OH + NaBr}
    :::

    $$\ce{CH3CH2Br + NaOH ->[H2O][\triangle] CH3CH2OH + NaBr}$$
    ::::

2.  消去反应：溴乙烷的消去反应

    :::: {.arithmatex}
    ::: {.MathJax_Preview}
    \ce{CH3CH2Br + NaOH -\>\[\text{醇}\]\[\triangle\] CH2=CH2 \^ + NaBr + H2O}
    :::

    $$\ce{CH3CH2Br + NaOH ->[\text{醇}][\triangle] CH2=CH2 ^ + NaBr + H2O}$$
    ::::

### 乙醇 {#_10}

1.  与钠反应：

    :::: {.arithmatex}
    ::: {.MathJax_Preview}
    \ce{2CH3CH2OH + 2Na -\> 2CH3CH2ONa + H2 \^}
    :::

    $$\ce{2CH3CH2OH + 2Na -> 2CH3CH2ONa + H2 ^}$$
    ::::

2.  催化氧化：

    :::: {.arithmatex}
    ::: {.MathJax_Preview}
    \ce{2CH3CH2OH + O2 -\>\[Cu\]\[\triangle\] 2CH3CHO + 2H2O}
    :::

    $$\ce{2CH3CH2OH + O2 ->[Cu][\triangle] 2CH3CHO + 2H2O}$$
    ::::

3.  消去反应：

    :::: {.arithmatex}
    ::: {.MathJax_Preview}
    \ce{CH3CH2OH -\>\[\text{浓}H2SO4\]\[\pu{170\^oC}\] CH2=CH2 \^ + H2O}
    :::

    $$\ce{CH3CH2OH ->[\text{浓}H2SO4][\pu{170^oC}] CH2=CH2 ^ + H2O}$$
    ::::

4.  取代反应：

    与浓 []{.arithmatex}$\ce{HBr}$ 反应：

    :::: {.arithmatex}
    ::: {.MathJax_Preview}
    \ce{CH3CH2OH + HBr -\>\[\triangle\] CH3CH2Br + H2O}
    :::

    $$\ce{CH3CH2OH + HBr ->[\triangle] CH3CH2Br + H2O}$$
    ::::

    酯化反应：

    :::: {.arithmatex}
    ::: {.MathJax_Preview}
    \ce{CH3COOH + C2H5OH \<=\>\[\text{浓}H2SO4\]\[\triangle\] CH3COOC2H5 + H2O}
    :::

    $$\ce{CH3COOH + C2H5OH <=>[\text{浓}H2SO4][\triangle] CH3COOC2H5 + H2O}$$
    ::::

    分子间脱水生成乙醚：

    :::: {.arithmatex}
    ::: {.MathJax_Preview}
    \ce{2C2H5OH -\>\[\text{浓}H2SO4\]\[\pu{140\^oC}\] C2H5-O-C2H5 + H2O}
    :::

    $$\ce{2C2H5OH ->[\text{浓}H2SO4][\pu{140^oC}] C2H5-O-C2H5 + H2O}$$
    ::::

总结乙醇的氧化反应有哪些？燃烧；催化氧化；被酸性 []{.arithmatex}$\ce{KMnO4}$ 氧化；被 []{.arithmatex}$\ce{K2Cr2O7}$ 氧化。

### 苯酚 {#_11}

苯酚是无色晶体，露置在空气中会因氧化显粉红色。苯酚具有特殊的气味，熔点 []{.arithmatex}$\pu{43 ^\circ C}$，水中溶解度不大，易溶于有机溶剂。苯酚有毒，是一种重要的化工原料。

1.  苯酚的酸性：[]{.arithmatex}$\ce{H2CO3}$ \> []{.arithmatex}$\ce{C6H5OH}$ \> []{.arithmatex}$\ce{HCO3^-}$

    苯酚（俗称石炭酸）的电离：

    :::: {.arithmatex}
    ::: {.MathJax_Preview}
    \ce{C6H5OH \<=\> C6H5O\^- + H\^+}
    :::

    $$\ce{C6H5OH <=> C6H5O^- + H^+}$$
    ::::

    苯酚与 []{.arithmatex}$\ce{NaOH}$ 的反应：

    :::: {.arithmatex}
    ::: {.MathJax_Preview}
    \ce{C6H5OH + NaOH -\> C6H5ONa + H2O}
    :::

    $$\ce{C6H5OH + NaOH -> C6H5ONa + H2O}$$
    ::::

    苯酚钠与 []{.arithmatex}$\ce{CO2}$ 反应：

    :::: {.arithmatex}
    ::: {.MathJax_Preview}
    \ce{C6H5ONa + CO2 + H2O -\> C6H5OH + NaHCO3}
    :::

    $$\ce{C6H5ONa + CO2 + H2O -> C6H5OH + NaHCO3}$$
    ::::

    苯酚钠与盐酸反应：

    :::: {.arithmatex}
    ::: {.MathJax_Preview}
    \ce{C6H5ONa + HCl -\> C6H5OH + NaCl}
    :::

    $$\ce{C6H5ONa + HCl -> C6H5OH + NaCl}$$
    ::::

2.  取代反应（与浓溴水反应）：

    :::: {.arithmatex}
    ::: {.MathJax_Preview}
    \ce{C6H5OH + 3Br2 -\> C6H2Br3OH v + 3HBr}
    :::

    $$\ce{C6H5OH + 3Br2 -> C6H2Br3OH v + 3HBr}$$
    ::::

    生成三溴苯酚白色沉淀。

3.  显色反应：苯酚能和 []{.arithmatex}$\ce{FeCl3}$ 溶液反应，使溶液**显紫色**。

### 乙醛 {#_12}

乙醛是无色无味，具有刺激性气味的液体，沸点 []{.arithmatex}$\pu{20.8 ^\circ C}$，密度比水小，易挥发。

1.  加成反应（乙醛与氢气反应）：

    :::: {.arithmatex}
    ::: {.MathJax_Preview}
    \ce{CH3CHO + H2 -\>\[\text{催化剂}\] CH3CH2OH}
    :::

    $$\ce{CH3CHO + H2 ->[\text{催化剂}] CH3CH2OH}$$
    ::::

2.  氧化反应：

    乙醛的催化氧化：

    :::: {.arithmatex}
    ::: {.MathJax_Preview}
    \ce{2CH3CHO + O2 -\>\[\text{催化剂}\]\[\triangle\] 2CH3COOH}
    :::

    $$\ce{2CH3CHO + O2 ->[\text{催化剂}][\triangle] 2CH3COOH}$$
    ::::

    乙醛的银镜反应：

    :::: {.arithmatex}
    ::: {.MathJax_Preview}
    \ce{CH3CHO + 2Ag(NH3)2OH -\>\[\triangle\] CH3COONH4 + 2Ag v + 3NH3 + H2O}
    :::

    $$\ce{CH3CHO + 2Ag(NH3)2OH ->[\triangle] CH3COONH4 + 2Ag v + 3NH3 + H2O}$$
    ::::

    有关制备银氨溶液的方程式：

    :::: {.arithmatex}
    ::: {.MathJax_Preview}
    \ce{AgNO3 + NH3.H2O -\> AgOH v + NH4NO3}
    :::

    $$\ce{AgNO3 + NH3.H2O -> AgOH v + NH4NO3}$$
    ::::

    :::: {.arithmatex}
    ::: {.MathJax_Preview}
    \ce{AgOH + 2NH3.H2O -\> \[Ag(NH3)2\]OH + 2H2O}
    :::

    $$\ce{AgOH + 2NH3.H2O -> [Ag(NH3)2]OH + 2H2O}$$
    ::::

    乙醛与新制的氢氧化铜反应：

    :::: {.arithmatex}
    ::: {.MathJax_Preview}
    \ce{CH3CHO + 2Cu(OH)2 + NaOH -\>\[\triangle\] CH3COONa + Cu2O v + 3H2O}
    :::

    $$\ce{CH3CHO + 2Cu(OH)2 + NaOH ->[\triangle] CH3COONa + Cu2O v + 3H2O}$$
    ::::

    乙二醛氧化为乙二酸：

    :::: {.arithmatex}
    ::: {.MathJax_Preview}
    \ce{OHC-CHO + O2 -\>\[\text{催化剂}\] HOOC-COOH}
    :::

    $$\ce{OHC-CHO + O2 ->[\text{催化剂}] HOOC-COOH}$$
    ::::

### 乳酸的自聚 {#_18}

乳酸：[]{.arithmatex}$\ce{CH3CH(OH)COOH}$

1.  一分子内成环酯：生成丙交酯（Lactide）。

2.  两分子间成环酯：

    :::: {.arithmatex}
    ::: {.MathJax_Preview}
    \ce{2CH3CH(OH)COOH -\>\[\text{浓}H2SO4\]\[\triangle\] \text{Cyclic Diester} + 2H2O}
    :::

    $$\ce{2CH3CH(OH)COOH ->[\text{浓}H2SO4][\triangle] \text{Cyclic Diester} + 2H2O}$$
    ::::

3.  聚酯：

    :::: {.arithmatex}
    ::: {.MathJax_Preview}
    \ce{nCH3CH(OH)COOH -\>\[\text{催化剂}\] -\[O-CH(CH3)-CO\]\_n- + nH2O}
    :::

    $$\ce{nCH3CH(OH)COOH ->[\text{催化剂}] -[O-CH(CH3)-CO]_n- + nH2O}$$
    ::::

### 葡萄糖 {#_19}

葡萄糖的结构简式：[]{.arithmatex}$\ce{CH2OH(CHOH)4CHO}$

1.  与银氨溶液的反应：

    :::: {.arithmatex}
    ::: {.MathJax_Preview}
    \ce{CH2OH(CHOH)4CHO + 2Ag(NH3)2OH -\>\[\triangle\] CH2OH(CHOH)4COONH4 + 2Ag v + 3NH3 + H2O}
    :::

    $$\ce{CH2OH(CHOH)4CHO + 2Ag(NH3)2OH ->[\triangle] CH2OH(CHOH)4COONH4 + 2Ag v + 3NH3 + H2O}$$
    ::::

2.  与新制 []{.arithmatex}$\ce{Cu(OH)2}$ 溶液的反应：

    :::: {.arithmatex}
    ::: {.MathJax_Preview}
    \ce{CH2OH(CHOH)4CHO + 2Cu(OH)2 + NaOH -\>\[\triangle\] CH2OH(CHOH)4COONa + Cu2O v + 3H2O}
    :::

    $$\ce{CH2OH(CHOH)4CHO + 2Cu(OH)2 + NaOH ->[\triangle] CH2OH(CHOH)4COONa + Cu2O v + 3H2O}$$
    ::::

### 有机反应条件要记清 {#_21}

请填写下列由已知有机物生成产物的反应条件。

1.  反应物 []{.arithmatex}$\ce{BrCH2CH2CH2OH}$：

    生成 []{.arithmatex}$\ce{CH2=CHCH2OH}$：条件为 **[]{.arithmatex}**$\ce{NaOH}$ 醇溶液，加热。

    生成 []{.arithmatex}$\ce{HOCH2CH2CH2OH}$：条件为 **[]{.arithmatex}**$\ce{NaOH}$ 水溶液，加热。

    生成 []{.arithmatex}$\ce{BrCH2CH=CH2}$：条件为**浓硫酸，[]{.arithmatex}**$\pu{170 ^\circ C}$。

2.  苯环侧链与光照 / 催化剂的区别：

    []{.arithmatex}$\ce{C6H5CH3 + Cl2 ->[\text{光照}] C6H5CH2Cl}$（侧链取代）

    []{.arithmatex}$\ce{C6H5CH3 + Cl2 ->[Fe] Cl-C6H4-CH3}$（苯环取代，邻 / 对位）

3.  反应物 []{.arithmatex}$\ce{CH2=CHCH2OH}$：

    生成 []{.arithmatex}$\ce{BrCH2CHBrCH2OH}$：条件为**溴水或溴的 []{.arithmatex}**$\ce{CCl4}$ 溶液。

    生成 []{.arithmatex}$\ce{CH2=CHCHO}$：条件为 **[]{.arithmatex}**$\ce{Cu/O2}$，加热。

    生成 []{.arithmatex}$\ce{CH2=CHCH2OOCCH3}$：条件为**浓硫酸，加热（与乙酸酯化）**。

4.  反应物 []{.arithmatex}$\ce{CH3CHBrCOOCH3}$：

    生成 []{.arithmatex}$\ce{CH3CH(OH)COOH}$：条件为 **[]{.arithmatex}**$\ce{NaOH}$ 水溶液，加热；稀硫酸。

    生成 []{.arithmatex}$\ce{CH3CHBrCOOH}$：条件为**稀硫酸，加热**。
