烃

烷烃（甲烷）

烷烃通式：CXnHX2n+2\ce{C_nH_{2n+2}}CXn​HX2n+2​​

  1. 氧化反应：

    甲烷不能使酸性高锰酸钾溶液及溴水褪色。

    甲烷的燃烧：

    CHX4+2 OX2→点燃COX2+2 HX2O \ce{CH4 + 2O2 ->[\text{点燃}] CO2 + 2H2O} CHX4​+2OX2​点燃​COX2​+2HX2​O

  2. 取代反应：

    CHX4+ClX2→光照CHX3Cl+HCl \ce{CH4 + Cl2 ->[\text{光照}] CH3Cl + HCl} CHX4​+ClX2​光照​CHX3​Cl+HCl

烯烃（乙烯）

烯烃通式：CXnHX2n\ce{C_nH_{2n}}CXn​HX2n​

  1. 乙烯的制取：

    CHX3CHX2OH→170 ∘C浓HX2SOX4CHX2=CHX2↑+HX2O \ce{CH3CH2OH ->[\text{浓}H2SO4][\pu{170^oC}] CH2=CH2 ^ + H2O} CHX3​CHX2​OH浓HX2​SOX4​170 ∘C​CHX2​=CHX2​↑+HX2​O

  2. 氧化反应：

    乙烯能使酸性高锰酸钾溶液褪色，发生氧化反应。

    乙烯的燃烧：

    CX2HX4+3 OX2→点燃2 COX2+2 HX2O \ce{C2H4 + 3O2 ->[\text{点燃}] 2CO2 + 2H2O} CX2​HX4​+3OX2​点燃​2COX2​+2HX2​O

  3. 加成反应：

    乙烯与溴水加成：

    CHX2=CHX2+BrX2→CHX2Br−CHX2Br \ce{CH2=CH2 + Br2 -> CH2Br-CH2Br} CHX2​=CHX2​+BrX2​​CHX2​Br−CHX2​Br

    乙烯与氢气加成：

    CHX2=CHX2+HX2→催化剂CHX3−CHX3 \ce{CH2=CH2 + H2 ->[\text{催化剂}] CH3-CH3} CHX2​=CHX2​+HX2​催化剂​CHX3​−CHX3​

    乙烯与水加成：

    CHX2=CHX2+HX2O→高温高压催化剂CHX3CHX2OH \ce{CH2=CH2 + H2O ->[\text{催化剂}][\text{高温高压}] CH3CH2OH} CHX2​=CHX2​+HX2​O催化剂高温高压​CHX3​CHX2​OH

  4. 加聚反应：

    乙烯加聚反应：

    n CHX2=CHX2→引发剂−[CHX2−CHX2]XnX− \ce{nCH2=CH2 ->[\text{引发剂}] -[CH2-CH2]_n-} nCHX2​=CHX2​引发剂​−[CHX2​−CHX2​]Xn​X−

    1,3-丁二烯的加成（1,4 加成）：

    CHX2=CH−CH=CHX2+HX2→1,4加成CHX3−CH=CH−CHX3 \ce{CH2=CH-CH=CH2 + H2 ->[\text{1,4加成}] CH3-CH=CH-CH3} CHX2​=CH−CH=CHX2​+HX2​1,4加成​CHX3​−CH=CH−CHX3​

    1,3-丁二烯的加聚：

    n CHX2=CH−CH=CHX2→催化剂−[CHX2−CH=CH−CHX2]XnX− \ce{nCH2=CH-CH=CH2 ->[\text{催化剂}] -[CH2-CH=CH-CH2]_n-} nCHX2​=CH−CH=CHX2​催化剂​−[CHX2​−CH=CH−CHX2​]Xn​X−

炔烃（乙炔）

炔烃的通式：CXnHX2n−2\ce{C_nH_{2n-2}}CXn​HX2n−2​​

  1. 乙炔的制取：

    CaCX2+2 HX2O→Ca(OH)X2+CH≡CH↑ \ce{CaC2 + 2H2O -> Ca(OH)2 + CH#CH ^} CaCX2​+2HX2​O​Ca(OH)X2​+CH≡CH↑

  2. 氧化反应：

    乙炔能使酸性高锰酸钾溶液褪色，发生氧化反应。

    乙炔的燃烧：

    2 CX2HX2+5 OX2→点燃4 COX2+2 HX2O \ce{2C2H2 + 5O2 ->[\text{点燃}] 4CO2 + 2H2O} 2CX2​HX2​+5OX2​点燃​4COX2​+2HX2​O

  3. 加成反应：

    与溴水加成：

    CH≡CH+2 BrX2→CHBrX2−CHBrX2 \ce{CH#CH + 2Br2 -> CHBr2-CHBr2} CH≡CH+2BrX2​​CHBrX2​−CHBrX2​

    与氢气加成：

    CH≡CH+2 HX2→催化剂CHX3−CHX3 \ce{CH#CH + 2H2 ->[\text{催化剂}] CH3-CH3} CH≡CH+2HX2​催化剂​CHX3​−CHX3​

    与氯化氢加成（制聚氯乙烯单体）：

    CH≡CH+HCl→△催化剂CHX2=CHCl \ce{CH#CH + HCl ->[\text{催化剂}][\triangle] CH2=CHCl} CH≡CH+HCl催化剂△​CHX2​=CHCl

  4. 聚合反应：

    氯乙烯加聚，得到聚氯乙烯：

    n CHX2=CHCl→催化剂−[CHX2−CHCl]XnX− \ce{nCH2=CHCl ->[\text{催化剂}] -[CH2-CHCl]_n-} nCHX2​=CHCl催化剂​−[CHX2​−CHCl]Xn​X−

    乙炔加聚，得到聚乙炔：

    n CH≡CH→催化剂−[CH=CH]XnX− \ce{nCH#CH ->[\text{催化剂}] -[CH=CH]_n-} nCH≡CH催化剂​−[CH=CH]Xn​X−

苯

苯的同系物通式：CXnHX2n−6\ce{C_nH_{2n-6}}CXn​HX2n−6​​

  1. 氧化反应：

    苯不能使溴水和酸性高锰酸钾溶液褪色。

    苯的燃烧：

    2 CX6HX6+15 OX2→点燃12 COX2+6 HX2O \ce{2C6H6 + 15O2 ->[\text{点燃}] 12CO2 + 6H2O} 2CX6​HX6​+15OX2​点燃​12COX2​+6HX2​O

  2. 取代反应：

    苯与溴反应：

    CX6HX6+BrX2(液)→FeBrX3CX6HX5Br+HBr \ce{C6H6 + Br2(\text{液}) ->[FeBr3] C6H5Br + HBr} CX6​HX6​+BrX2​(液)FeBrX3​​CX6​HX5​Br+HBr

    硝化反应：

    CX6HX6+HNOX3(浓)→50 ∼60 ∘C浓HX2SOX4CX6HX5NOX2+HX2O \ce{C6H6 + HNO3(\text{浓}) ->[\text{浓}H2SO4][50\sim60\pu{^oC}] C6H5NO2 + H2O} CX6​HX6​+HNOX3​(浓)浓HX2​SOX4​50∼60∘C​CX6​HX5​NOX2​+HX2​O

  3. 苯与氢气加成反应：

    CX6HX6+3 HX2→催化剂CX6HX12 \ce{C6H6 + 3H2 ->[\text{催化剂}] C6H12} CX6​HX6​+3HX2​催化剂​CX6​HX12​

    产物为环己烷。

甲苯

甲苯不能使溴水褪色。

  1. 氧化反应：能使酸性高锰酸钾溶液褪色。

    5 CX6HX5CHX3+6 KMnOX4+9 HX2SOX4→5 CX6HX5COOH+3 KX2SOX4+6 MnSOX4+14 HX2O \ce{5C6H5CH3 + 6KMnO4 + 9H2SO4 -> 5C6H5COOH + 3K2SO4 + 6MnSO4 + 14H2O} 5CX6​HX5​CHX3​+6KMnOX4​+9HX2​SOX4​​5CX6​HX5​COOH+3KX2​SOX4​+6MnSOX4​+14HX2​O

    注：氧化成苯甲酸。

  2. 取代反应（制 TNT）：

    CX6HX5CHX3+3 HNOX3(浓)→△浓HX2SOX4CX6HX2(NOX2)X3CHX3+3 HX2O \ce{C6H5CH3 + 3HNO3(\text{浓}) ->[\text{浓}H2SO4][\triangle] C6H2(NO2)3CH3 + 3H2O} CX6​HX5​CHX3​+3HNOX3​(浓)浓HX2​SOX4​△​CX6​HX2​(NOX2​)X3​CHX3​+3HX2​O

    注意：甲苯在光照条件下发生侧链的取代，而在催化剂条件下发生苯环上的取代。

  3. 与氢气加成反应：

    CX6HX5CHX3+3 HX2→催化剂CX6HX11CHX3 \ce{C6H5CH3 + 3H2 ->[\text{催化剂}] C6H11CH3} CX6​HX5​CHX3​+3HX2​催化剂​CX6​HX11​CHX3​

    产物为甲基环己烷。

纯净的溴乙烷是无色液体，沸点 38.4 ∘C\pu{38.4 ^\circ C}38.4 ∘C，密度比水大。

  1. 取代反应：溴乙烷的水解反应

    CHX3CHX2Br+NaOH→△HX2OCHX3CHX2OH+NaBr \ce{CH3CH2Br + NaOH ->[H2O][\triangle] CH3CH2OH + NaBr} CHX3​CHX2​Br+NaOHHX2​O△​CHX3​CHX2​OH+NaBr

  2. 消去反应：溴乙烷的消去反应

    CHX3CHX2Br+NaOH→△醇CHX2=CHX2↑+NaBr+HX2O \ce{CH3CH2Br + NaOH ->[\text{醇}][\triangle] CH2=CH2 ^ + NaBr + H2O} CHX3​CHX2​Br+NaOH醇△​CHX2​=CHX2​↑+NaBr+HX2​O

乙醇

  1. 与钠反应：

    2 CHX3CHX2OH+2 Na→2 CHX3CHX2ONa+HX2↑ \ce{2CH3CH2OH + 2Na -> 2CH3CH2ONa + H2 ^} 2CHX3​CHX2​OH+2Na​2CHX3​CHX2​ONa+HX2​↑

  2. 催化氧化：

    2 CHX3CHX2OH+OX2→△Cu2 CHX3CHO+2 HX2O \ce{2CH3CH2OH + O2 ->[Cu][\triangle] 2CH3CHO + 2H2O} 2CHX3​CHX2​OH+OX2​Cu△​2CHX3​CHO+2HX2​O

  3. 消去反应：

    CHX3CHX2OH→170 ∘C浓HX2SOX4CHX2=CHX2↑+HX2O \ce{CH3CH2OH ->[\text{浓}H2SO4][\pu{170^oC}] CH2=CH2 ^ + H2O} CHX3​CHX2​OH浓HX2​SOX4​170 ∘C​CHX2​=CHX2​↑+HX2​O

  4. 取代反应：

    与浓 HBr\ce{HBr}HBr 反应：

    CHX3CHX2OH+HBr→△CHX3CHX2Br+HX2O \ce{CH3CH2OH + HBr ->[\triangle] CH3CH2Br + H2O} CHX3​CHX2​OH+HBr△​CHX3​CHX2​Br+HX2​O

    酯化反应：

    分子间脱水生成乙醚：

    2 CX2HX5OH→140 ∘C浓HX2SOX4CX2HX5−O−CX2HX5+HX2O \ce{2C2H5OH ->[\text{浓}H2SO4][\pu{140^oC}] C2H5-O-C2H5 + H2O} 2CX2​HX5​OH浓HX2​SOX4​140 ∘C​CX2​HX5​−O−CX2​HX5​+HX2​O

总结乙醇的氧化反应有哪些？燃烧；催化氧化；被酸性 KMnOX4\ce{KMnO4}KMnOX4​ 氧化；被 KX2CrX2OX7\ce{K2Cr2O7}KX2​CrX2​OX7​ 氧化。

苯酚

苯酚是无色晶体，露置在空气中会因氧化显粉红色。苯酚具有特殊的气味，熔点 43 ∘C\pu{43 ^\circ C}43 ∘C，水中溶解度不大，易溶于有机溶剂。苯酚有毒，是一种重要的化工原料。

  1. 苯酚的酸性：HX2COX3\ce{H2CO3}HX2​COX3​ > CX6HX5OH\ce{C6H5OH}CX6​HX5​OH > HCOX3X−\ce{HCO3^-}HCOX3​X−

    苯酚（俗称石炭酸）的电离：

    CX6HX5OH⇌CX6HX5OX−+HX+ \ce{C6H5OH <=> C6H5O^- + H^+} CX6​HX5​OH​CX6​HX5​OX−+HX+

    苯酚与 NaOH\ce{NaOH}NaOH 的反应：

    CX6HX5OH+NaOH→CX6HX5ONa+HX2O \ce{C6H5OH + NaOH -> C6H5ONa + H2O} CX6​HX5​OH+NaOH​CX6​HX5​ONa+HX2​O

    苯酚钠与 COX2\ce{CO2}COX2​ 反应：

    CX6HX5ONa+COX2+HX2O→CX6HX5OH+NaHCOX3 \ce{C6H5ONa + CO2 + H2O -> C6H5OH + NaHCO3} CX6​HX5​ONa+COX2​+HX2​O​CX6​HX5​OH+NaHCOX3​

    苯酚钠与盐酸反应：

    CX6HX5ONa+HCl→CX6HX5OH+NaCl \ce{C6H5ONa + HCl -> C6H5OH + NaCl} CX6​HX5​ONa+HCl​CX6​HX5​OH+NaCl

  2. 取代反应（与浓溴水反应）：

    CX6HX5OH+3 BrX2→CX6HX2BrX3OH↓+3 HBr \ce{C6H5OH + 3Br2 -> C6H2Br3OH v + 3HBr} CX6​HX5​OH+3BrX2​​CX6​HX2​BrX3​OH↓+3HBr

    生成三溴苯酚白色沉淀。

  3. 显色反应：苯酚能和 FeClX3\ce{FeCl3}FeClX3​ 溶液反应，使溶液显紫色。

乙醛

乙醛是无色无味，具有刺激性气味的液体，沸点 20.8 ∘C\pu{20.8 ^\circ C}20.8 ∘C，密度比水小，易挥发。

  1. 加成反应（乙醛与氢气反应）：

    CHX3CHO+HX2→催化剂CHX3CHX2OH \ce{CH3CHO + H2 ->[\text{催化剂}] CH3CH2OH} CHX3​CHO+HX2​催化剂​CHX3​CHX2​OH

  2. 氧化反应：

    乙醛的催化氧化：

    2 CHX3CHO+OX2→△催化剂2 CHX3COOH \ce{2CH3CHO + O2 ->[\text{催化剂}][\triangle] 2CH3COOH} 2CHX3​CHO+OX2​催化剂△​2CHX3​COOH

    乙醛的银镜反应：

    CHX3CHO+2 Ag(NHX3)X2OH→△CHX3COONHX4+2 Ag↓+3 NHX3+HX2O \ce{CH3CHO + 2Ag(NH3)2OH ->[\triangle] CH3COONH4 + 2Ag v + 3NH3 + H2O} CHX3​CHO+2Ag(NHX3​)X2​OH△​CHX3​COONHX4​+2Ag↓+3NHX3​+HX2​O

    有关制备银氨溶液的方程式：

    AgNOX3+NHX3 ⋅ HX2O→AgOH↓+NHX4NOX3 \ce{AgNO3 + NH3.H2O -> AgOH v + NH4NO3} AgNOX3​+NHX3​⋅HX2​O​AgOH↓+NHX4​NOX3​
    AgOH+2 NHX3 ⋅ HX2O→[Ag(NHX3)X2]OH+2 HX2O \ce{AgOH + 2NH3.H2O -> [Ag(NH3)2]OH + 2H2O} AgOH+2NHX3​⋅HX2​O​[Ag(NHX3​)X2​]OH+2HX2​O

    乙醛与新制的氢氧化铜反应：

    CHX3CHO+2 Cu(OH)X2+NaOH→△CHX3COONa+CuX2O↓+3 HX2O \ce{CH3CHO + 2Cu(OH)2 + NaOH ->[\triangle] CH3COONa + Cu2O v + 3H2O} CHX3​CHO+2Cu(OH)X2​+NaOH△​CHX3​COONa+CuX2​O↓+3HX2​O
