# 电池概述 {#_1}

## 一次电池 {#_2}

### 锂离子电池 {#_3}

正极反应：[\$\ce{Li\_{1-x}CoO2 + xe- + x Li+ = LiCoO2}\$]{.arithmatex}。

负极反应：[\$\ce{Li_xC6 - xe- = x Li+ + C6}\$]{.arithmatex}。

总反应：[\$\ce{Li\_{1-x}CoO2 + Li_xC6 = LiCoO2 + C6}\$]{.arithmatex}。

## 充电电池 {#_4}

### 铅酸蓄电池 {#_5}

放电：

- 正极反应：[\$\ce{PbO2 + 4H+ + SO4\^2- + 2e- = PbSO4 + 2H2O}\$]{.arithmatex}。

- 负极反应：[\$\ce{Pb + SO4\^2- - 2e- = PbSO4}\$]{.arithmatex}。

- 总反应：[\$\ce{Pb + PbO2 + 2H2SO4 = 2PbSO4 + 2H2O}\$]{.arithmatex}。

充电：

- 正极反应：[\$\ce{PbSO4 + 2H2O - 2e- -\> PbO2 + 4H+ + SO4\^2-}\$]{.arithmatex}。

- 负极反应：[\$\ce{PbSO4 + 2e- -\> Pb + SO4\^2-}\$]{.arithmatex}。

- 总反应：[\$\ce{2PbSO4 + 2H2O -\> Pb + PbO2 + 2H2SO4}\$]{.arithmatex}。

## 燃料电池 {#_6}

酸性氢氧燃料电池：

- 正极反应：[\$\ce{O2 + 4H+ + 4e- = 2H2O}\$]{.arithmatex}。

- 负极反应：[\$\ce{2H2 - 4e- = 4H+}\$]{.arithmatex}。

碱性氢氧燃料电池：

- 正极反应：[\$\ce{O2 + 2H2O + 4e- = 4OH-}\$]{.arithmatex}。

- 负极反应：[\$\ce{2H2 - 4e- + 4OH- = 4H2O}\$]{.arithmatex}。

甲醇酸性燃料电池：

- 正极反应：[\$\ce{O2 + 4H+ + 4e- = 2H2O}\$]{.arithmatex}。

- 负极反应：[\$\ce{CH3OH + H2O - 6e- = CO2 + 6H+}\$]{.arithmatex}。

甲醇碱性燃料电池：

- 正极反应：[\$\ce{O2 + 2H2O + 4e- = 4OH-}\$]{.arithmatex}。

- 负极反应：[\$\ce{CH3OH + 8OH- - 6e- = CO3\^2- + 6H2O}\$]{.arithmatex}。

甲烷酸性燃料电池：

- 正极反应：[\$\ce{O2 + 4H+ + 4e- = 2H2O}\$]{.arithmatex}。

- 负极反应：[\$\ce{CH4 + 2H2O - 8e- = CO2 + 8H+}\$]{.arithmatex}。

甲烷碱性燃料电池：

- 正极反应：[\$\ce{O2 + 2H2O + 4e- = 4OH-}\$]{.arithmatex}。

- 负极反应：[\$\ce{CH4 + 10OH- - 8e- = CO3\^2- + 7H2O}\$]{.arithmatex}。

甲烷碳酸盐燃料电池：

- 正极反应：[\$\ce{O2 + 2CO2 + 4e- = 2CO3\^2-}\$]{.arithmatex}。

- 负极反应：[\$\ce{CH4 + 4CO3\^2- - 8e- = 5CO2 + 2H2O}\$]{.arithmatex}。

甲醇碳酸盐燃料电池：

- 正极反应：[\$\ce{O2 + 2CO2 + 4e- = 2CO3\^2-}\$]{.arithmatex}。

- 负极反应：[\$\ce{CH3OH + 3CO3\^2- - 6e- = 4CO2 + 2H2O}\$]{.arithmatex}。
