    #铅酸蓄电池

    放电：

    • 正极反应：\cePbO2+4H++SO42−+2e−=PbSO4+2H2O\ce{PbO2 + 4H+ + SO4^2- + 2e- = PbSO4 + 2H2O}\cePbO2+4H++SO42−+2e−=PbSO4+2H2O。

    • 负极反应：\cePb+SO42−−2e−=PbSO4\ce{Pb + SO4^2- - 2e- = PbSO4}\cePb+SO42−−2e−=PbSO4。

    • 总反应：\cePb+PbO2+2H2SO4=2PbSO4+2H2O\ce{Pb + PbO2 + 2H2SO4 = 2PbSO4 + 2H2O}\cePb+PbO2+2H2SO4=2PbSO4+2H2O。

    充电：

    • 正极反应：\cePbSO4+2H2O−2e−−>PbO2+4H++SO42−\ce{PbSO4 + 2H2O - 2e- -> PbO2 + 4H+ + SO4^2-}\cePbSO4+2H2O−2e−−>PbO2+4H++SO42−。

    • 负极反应：\cePbSO4+2e−−>Pb+SO42−\ce{PbSO4 + 2e- -> Pb + SO4^2-}\cePbSO4+2e−−>Pb+SO42−。

    • 总反应：\ce2PbSO4+2H2O−>Pb+PbO2+2H2SO4\ce{2PbSO4 + 2H2O -> Pb + PbO2 + 2H2SO4}\ce2PbSO4+2H2O−>Pb+PbO2+2H2SO4。

    #燃料电池

    酸性氢氧燃料电池：

    • 正极反应：\ceO2+4H++4e−=2H2O\ce{O2 + 4H+ + 4e- = 2H2O}\ceO2+4H++4e−=2H2O。

    • 负极反应：\ce2H2−4e−=4H+\ce{2H2 - 4e- = 4H+}\ce2H2−4e−=4H+。

    碱性氢氧燃料电池：

    • 正极反应：\ceO2+2H2O+4e−=4OH−\ce{O2 + 2H2O + 4e- = 4OH-}\ceO2+2H2O+4e−=4OH−。

    • 负极反应：\ce2H2−4e−+4OH−=4H2O\ce{2H2 - 4e- + 4OH- = 4H2O}\ce2H2−4e−+4OH−=4H2O。

    甲醇酸性燃料电池：

    • 正极反应：\ceO2+4H++4e−=2H2O\ce{O2 + 4H+ + 4e- = 2H2O}\ceO2+4H++4e−=2H2O。

    • 负极反应：\ceCH3OH+H2O−6e−=CO2+6H+\ce{CH3OH + H2O - 6e- = CO2 + 6H+}\ceCH3OH+H2O−6e−=CO2+6H+。

    甲醇碱性燃料电池：

    • 正极反应：\ceO2+2H2O+4e−=4OH−\ce{O2 + 2H2O + 4e- = 4OH-}\ceO2+2H2O+4e−=4OH−。

    • 负极反应：\ceCH3OH+8OH−−6e−=CO32−+6H2O\ce{CH3OH + 8OH- - 6e- = CO3^2- + 6H2O}\ceCH3OH+8OH−−6e−=CO32−+6H2O。

    甲烷酸性燃料电池：

    • 正极反应：\ceO2+4H++4e−=2H2O\ce{O2 + 4H+ + 4e- = 2H2O}\ceO2+4H++4e−=2H2O。

    • 负极反应：\ceCH4+2H2O−8e−=CO2+8H+\ce{CH4 + 2H2O - 8e- = CO2 + 8H+}\ceCH4+2H2O−8e−=CO2+8H+。

    甲烷碱性燃料电池：

    • 正极反应：\ceO2+2H2O+4e−=4OH−\ce{O2 + 2H2O + 4e- = 4OH-}\ceO2+2H2O+4e−=4OH−。

    • 负极反应：\ceCH4+10OH−−8e−=CO32−+7H2O\ce{CH4 + 10OH- - 8e- = CO3^2- + 7H2O}\ceCH4+10OH−−8e−=CO32−+7H2O。

    甲烷碳酸盐燃料电池：

    • 正极反应：\ceO2+2CO2+4e−=2CO32−\ce{O2 + 2CO2 + 4e- = 2CO3^2-}\ceO2+2CO2+4e−=2CO32−。

    • 负极反应：\ceCH4+4CO32−−8e−=5CO2+2H2O\ce{CH4 + 4CO3^2- - 8e- = 5CO2 + 2H2O}\ceCH4+4CO32−−8e−=5CO2+2H2O。

    甲醇碳酸盐燃料电池：

    • 正极反应：\ceO2+2CO2+4e−=2CO32−\ce{O2 + 2CO2 + 4e- = 2CO3^2-}\ceO2+2CO2+4e−=2CO32−。

    • 负极反应：\ceCH3OH+3CO32−−6e−=4CO2+2H2O\ce{CH3OH + 3CO3^2- - 6e- = 4CO2 + 2H2O}\ceCH3OH+3CO32−−6e−=4CO2+2H2O。
