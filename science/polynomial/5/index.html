二项式定理

二项式定理

基本形式

二项式定理（Binomial Theorem）是组合数学中最核心的定理之一，它建立了多项式展开与组合数之间的桥梁。其基本形式为：

(a+b)n=i=0∑n​(in​)an−ibi​

令 b=1，我们可以得出最能体现性质的形式：

(a+1)n=i=0∑n​(in​)ai​

二项式定理也可以很容易地扩展为多项式的形式：

(x1​+⋯+xk​)n=n1​+⋯+nk​=n∑​(n1​,…,nk​n​)x1n1​​…xknk​​​

由此可得一个重要的恒等式：

n1​+⋯+nk​=n∑​(n1​,…,nk​n​)=kn​

这个恒等式有什么用？

我们可以用它来检验展开是否正确。例如，对 (x1​+x2​+x3​)n 的展开式，令所有变量取 1，则左边为 3n，右边为所有多项式系数之和。如果不等，说明展开有误。

杨辉三角

杨辉三角（Pascal’s Triangle）是二项式系数的几何化表示，其递推式为：

ai,1​=ai,i​=1ai,j​=ai−1,j−1​+ai−1,j​

你可能会注意到，杨辉三角中的数恰好就是组合数。具体来说，第 i 行第 j 个数，表示的是 (a+1)i−1 的 aj−1 次项系数，也就是 (j−1i−1​)。下面是杨辉三角的前 11 行：

1234567891011​111121133114641151010511615201561172135352171182856705628811936841261268436911104512021025221012045101​​

二项式系数的单调性

我们来讨论二项式系数：

(0n​),(1n​),…,(nn​)​

注意区分「二项式系数」与「系数」

二项式系数就是组合数本身，但在具体的展开式中，系数还需要乘上各项本身的数字系数。例如 (3a+2)2 的第一项，二项式系数是 C(2,0)=1，但实际系数是 32×1=9。

我们来研究这个序列的单调性。设

f(n,k)=(kn​)

考虑相邻两项的比值：

q=f(n,k)f(n,k+1)​

化简，得：

q=f(n,k)f(n,k+1)​=(k+1)!(n−k−1)!n!​×n!k!(n−k)!​=k+1n−k​

现在我们分情况讨论：

  • 函数单增：q>1⇒k<2n−1​；

  • 函数单减：q<1⇒k>2n−1​。

据此，我们可以得出二项式系数的单增和单减区间为：

[0,2n−1​],(2n−1​,n]

应用向下取整函数和向上取整函数化简：

[0,⌊2n​⌋],[⌈2n​⌉,n]​

由此我们得到一个重要结论：

  • 若 n 是偶数，那么存在唯一的极大值点 n/2；

  • 若 n 是奇数，那么存在 (n+1)/2 和 (n−1)/2 两个极大值点。

应用例题

下面我们来看一个利用二项式定理展开的例题。

在

(x+y)(x−y)5

的展开式中，求 xny6−n 的系数。

解：首先，我们注意到原式可以拆分为：

(x+y)(x−y)5=x(x−y)5+y(x−y)5

那么要求 xny6−n 的系数，其实就是前一项中 xn−1(−y)6−n 的系数，加上后一项中 xn(−y)5−n 的系数，即：

(−1)6−n(6−n5​)+(−1)5−n(5−n5​)=(−1)6−n[(6−n5​)−(5−n5​)]​

多项式展开的一般套路

掌握了二项式定理后，我们来总结一下多项式展开题的一般方法。

对于 (x1​+x2​+x3​)n 的展开，相当于考虑每一组满足 k1​+k2​+k3​=n 的非负整数解，找出对应的贡献：

(k1​,k2​,k3​n​)x1k1​​x2k2​​x3k3​​

下面我们通过一道例题来体会这个方法。

在

(x−1+px2)6

的展开式中，求使 x5 项系数最大时 p 的值。

解：我们需要确定 x5 的系数。根据多项式展开的规则，我们需要找出所有满足 k1​+k2​+k3​=6 且 k1​+2k3​=5（x 的次数为 5）的非负整数组合：

(1)(2)(3)​​x135​−1321​+px2210​​

上面列出了所有能配出 x5 的组合方式。我们可以把上标看作一共选几个物品，把下面的数字看作每类物品提供了几个因子。

这些组合分别对答案的贡献为：

(1)(2)(3)​:−60p2:60p:−6​

合并后，x5 的系数为：

−60p2+60p−6

这是一个关于 p 的二次函数，开口向下。取最大值时：

p=21​

这是由二次函数的顶点公式直接得到的。

用二项式定理推导组合恒等式

二项式定理的一个重要应用是推导组合恒等式。我们以如下形式为基础：

(a+1)n=i=0∑n​(in​)ai​

通过给 a 赋不同的值，我们可以得到一系列优美的组合恒等式。

令 a = 1，得：

i=0∑n​(in​)=2n​

令 a = 2，得：

i=0∑n​(in​)2i=3n​

令 a = −1，得：

i=0∑n​(−1)i(in​)=[n=0]​

上式中，当 n=0 时：

i=0∑n​(−1)i(in​)=0

利用这个结论，我们可以进一步推导出偶数项和奇数项组合数之和的关系。

假定 n 是偶数，将正负项分别移到等号两边：

(0n​)+(2n​)+⋯+(nn​)=(1n​)+(3n​)+⋯+(n−1n​)

又因为左右两边之和等于 2n，所以每一边都等于 2n−1，即：

(0n​)+(2n​)+⋯+(nn​)=2n−1

(1n​)+(3n​)+⋯+(n−1n​)=2n−1

以上结论对 n 为正偶数成立。

类似地，当 n 为正奇数时：

(0n​)+(2n​)+⋯+(n−1n​)=(1n​)+(3n​)+⋯+(nn​)

同理可得：

(0n​)+(2n​)+⋯+(n−1n​)=2n−1

(1n​)+(3n​)+⋯+(nn​)=2n−1 综上所述，我们得到如下两个重要的恒等式：

(0n​)+(2n​)+⋯=2n−1​

(1n​)+(3n​)+⋯=2n−1​

其中省略号表示加到下标不超过 n 为止。根据组合数的定义，当 k>n 时 (kn​)=0，因此这些求和式也可以等价地理解为无穷求和，只不过超过 n 的项均为零。

广义二项式定理

当指数 n 为实数时，二项式定理可以推广为如下形式：

(a+b)n=i∑​k!nk​​xn−kyk

这一推广在概率论和组合计数中的应用场景相对有限，了解即可。

成套思想

成套思想的精髓在于：我们不一定需要求出展开式中的每一个系数，而是通过构造方程组，将问题转化为求几个关键量的和与差。

例题 设展开式：

(3x−1)8=a8​x8+a7​x7+⋯+a1​x+a0​

求 a8​+a6​+a4​+a2​+a0​。

我们不妨设：

A=a8​+a6​+a4​+a2​+a0​

B=a7​+a5​+a3​+a1​

这样，我们只需要求出 A+B 和 A−B 即可。

系数 a0​∼a8​ 都是常数（与 x 无关），且对所有有意义的 x 均成立，因此我们可以代入特定的 x 值来建立方程。

代入 x=1：

28=a8​+a7​+⋯+a0​

即：

A+B=256

代入 x=−1：

48=a8​−a7​+⋯+a0​

即：

A−B=216=65536

由此易得：

{AB​=32896=−32640​

证明整除问题

二项式定理在整除性证明中有非常广泛的应用。基本思路是：将一个数写成 a=qm+r 的形式（其中 r∈[0,m)），然后展开为二项式，展开后的每一项都含有模数 m 的倍数，消去后即可得到结论。

例题一 证明 (n+1)n−1 可被 n2 整除。

(n+1)n−1​=(1n​)n+(2n​)n2+⋯+(nn​)nn=n2+(2n​)n2+⋯+(nn​)nn​

可以看出，右边每一项都含有因子 n2，因此原式可被 n2 整除。

例题二 证明 9910−1 可被 100 整除。

利用同余关系：

9910−1≡(−1)10−1≡1−1≡0(mod100)

例题三 证明 5555+9 可被 8 整除。

5555+9≡755+1≡(−1)55+1≡−1+1≡0(mod8)

上述两个例子都利用了「将底数转化为接近模数的数，再通过二项式展开或同余化简」的思路。

伯努利不等式

伯努利不等式（Bernoulli’s Inequality）是二项式定理的一个经典推论。

简单形式 对于实数 x≥−1：

  • 若 n ≥ 1，则 (1 + x)n ≥ 1 + nx；
  • 若 0 ≤ n ≤ 1，则 (1 + x)n ≤ 1 + nx。

等号成立当且仅当 n=0，n=1 或 x=0 时。

证明思路很直接：通过二项式定理展开后丢弃后面的非负项即可。
