**条件概率与“状态转移”**（前后局相关）：当“前一局胜负影响下一局”时，题目本质上考察的是**全概率公式**与**数列递推（马尔可夫链模型）**。利用全概率公式，将第 n 局的概率 P(A_n) 拆分为基于第 n-1 局结果的条件概率之和，从而构造数列递推式。

解：设事件 A_n 为“甲在第 n 局获胜”，令 P(A_n) = p_n。由题意可知，若上一局甲胜，则本局甲胜的概率为 P(A_n | A\_{n-1}) = \\alpha；若上一局甲负，则本局甲胜的概率为 P(A_n | \\bar{A}\_{n-1}) = \\beta。

由全概率公式 P(A_n) = P(A\_{n-1})P(A_n | A\_{n-1}) + P(\\bar{A}_{n-1})P(A_n | \\bar{A}_{n-1}) 即 p_n = \\alpha \\cdot p\_{n-1} + \\beta \\cdot (1 - p\_{n-1}) 整理得：p_n = (\\alpha - \\beta)p\_{n-1} + \\beta。

令 p_n - \\dfrac{\\beta}{1 - (\\alpha - \\beta)} = (\\alpha - \\beta)\\left[ p\_{n-1} - \\dfrac{\\beta}{1 - (\\alpha - \\beta)} \\right] 且 p_1 = x （初始概率），故数列 {p_n - \\text{常数}} 是以 \\dots 为首项，(\\alpha - \\beta) 为公比的等比数列。从而求得 P(A_n) = p_n = \\dots。
