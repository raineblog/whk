过渡元素概述

铁单质

物理性质

固体，铁块为银白色，铁粉为黑色。

常见形式为化合态，游离态存在于陨铁中。

有矿石：黄铁矿（FeSX2\ce{FeS2}FeSX2​）、赤铁矿（FeX2OX3\ce{Fe2O3}FeX2​OX3​）、磁铁矿（FeX3OX4\ce{Fe3O4}FeX3​OX4​）、黄铜矿（FeCuSX2\ce{FeCuS2}FeCuSX2​）。

不锈钢的合金元素主要是铬（Cr\ce{Cr}Cr）和镍（Ni\ce{Ni}Ni）。

碳素钢的分类：

  1. 低碳钢（含碳量少于 0.3%0.3\%0.3%）：韧性好，焊接性好，强度低。用于制造钢丝、钢管、钢板等。

  2. 中碳钢（含碳量在 0.3%∼0.6%0.3\%\sim0.6\%0.3%∼0.6%）：强度高，韧性和加工性好。用于制造钢轨、车轮和建材等。

  3. 高碳钢（含碳量大于 0.6%0.6\%0.6%）：硬而脆。用于制造器械、弹簧和刀具等。

化学性质

  • 与氧气反应：

    • 常温：Fe+OX2+HX2O→FeX2OX3 ⋅ x HX2O\ce{Fe + O2 + H2O -> Fe2O3*xH2O}Fe+OX2​+HX2​O​FeX2​OX3​⋅xHX2​O。

    • 点燃：3 Fe+2 OX2→点燃FeX3OX4\ce{3Fe + 2O2 ->[点燃] Fe3O4}3Fe+2OX2​点燃​FeX3​OX4​，火星四射。

  • 与酸反应：

    • 与非氧化性酸反应：Fe+2 HX+→FeX2++HX2↑\ce{Fe + 2H+ -> Fe^2+ + H2 ^}Fe+2HX+​FeX2++HX2​↑。

    • 与浓硫酸、浓硝酸：钝化（迅速生成 FeX3OX4\ce{Fe3O4}FeX3​OX4​ 致密氧化膜，阻止反应继续进行）。

  • 和水反应：

    • 3 Fe+4 HX2O(g)→高温FeX3OX4+4 HX2\ce{3Fe + 4H2O(g) ->[高温] Fe3O4 + 4H2}3Fe+4HX2​O(g)高温​FeX3​OX4​+4HX2​。

    • 现象：点燃肥皂泡，发出爆鸣声。

  • 与酸反应。

  • 铝热反应：详见铝部分。

  • 碱性氧化物：FeO+2 HX+→FeX2++HX2O\ce{FeO + 2H+ -> Fe^2+ + H2O}FeO+2HX+​FeX2++HX2​O。

  • 还原性：6 FeO+OX2→高温2 FeX3OX4\ce{6FeO + O2 ->[高温] 2Fe3O4}6FeO+OX2​高温​2FeX3​OX4​。

氧化铁：

  • 碱性氧化物：FeX2OX3+6 HX+→2 FeX3++3 HX2O\ce{Fe2O3 + 6H+ -> 2Fe^3+ + 3H2O}FeX2​OX3​+6HX+​2FeX3++3HX2​O。

  • 氧化性：2 Al+FeX2OX3→高温2 Fe+AlX2OX3\ce{2Al + Fe2O3 ->[高温] 2Fe + Al2O3}2Al+FeX2​OX3​高温​2Fe+AlX2​OX3​。

四氧化三铁：

  • 特殊氧化物：FeX3OX4+8 HX+→FeX2++2 FeX3++4 HX2O\ce{Fe3O4 + 8H+ -> Fe^2+ + 2Fe^3+ + 4H2O}FeX3​OX4​+8HX+​FeX2++2FeX3++4HX2​O。

  • 氧化性：8 Al+3 FeX3OX4→高温9 Fe+4 AlX2OX3\ce{8Al + 3Fe3O4 ->[高温] 9Fe + 4Al2O3}8Al+3FeX3​OX4​高温​9Fe+4AlX2​OX3​。

  • 还原性。

高炉炼铁

碳、铁矿石、空气、碳酸钙。

  • 碳提供还原性，同时燃烧提供热量。

  • 空气用于支持燃烧。

  • 碳酸钙用于除去铁矿石中的二氧化硅。

铁盐和亚铁盐

铁盐的物理性质

  • FeClX3 ⋅ 6 HX2O\ce{FeCl3*6H2O}FeClX3​⋅6HX2​O：黄色。
  • FeSOX4 ⋅ 7 HX2O\ce{FeSO4*7H2O}FeSOX4​⋅7HX2​O（绿矾）：浅绿色。

铁盐的化学性质

  1. 与碱反应：

    • FeX3++3 OHX−→Fe(OH)X3\ce{Fe^3+ + 3OH- -> Fe(OH)3}FeX3++3OHX−​Fe(OH)X3​。

    • 迅速产生红褐色沉淀，可用于检验铁离子。

  2. 氧化性：

    • 2 FeX3++Fe→3 FeX2+\ce{2Fe^3+ + Fe -> 3Fe^2+}2FeX3++Fe​3FeX2+。

      加入 KSCN\ce{KSCN}KSCN 后，溶液呈血红色。

      加入足量铁粉后，溶液褪色。

    • 2 FeX3++Cu→2 FeX2++CuX2+\ce{2Fe^3+ + Cu -> 2Fe^2+ + Cu^2+}2FeX3++Cu​2FeX2++CuX2+。

      溶液由黄变蓝。

      应用：蚀刻铜电路板。

    • 2 FeX3++2 IX−→2 FeX2++IX2\ce{2Fe^3+ + 2I- -> 2Fe^2+ + I2}2FeX3++2IX−​2FeX2++IX2​。

      溶液由黄变棕黄。

      加入淀粉后溶液变蓝。

亚铁盐的化学性质

  • 与碱反应：详见亚铁的氢氧化物。

  • 氧化性：Zh+FeX2+→Fe+ZnX2+\ce{Zh + Fe^2+ -> Fe + Zn^2+}Zh+FeX2+​Fe+ZnX2+。

  • 还原性：

    • 2 FeX2++HX2OX2+2 HX+→2 FeX3++2 HX2O\ce{2Fe^2+ + H2O2 + 2H+ -> 2Fe^3+ + 2H2O}2FeX2++HX2​OX2​+2HX+​2FeX3++2HX2​O。

      加入 KSCN\ce{KSCN}KSCN 无色。

      逐滴加入酸性 HX2OX2\ce{H2O2}HX2​OX2​，呈血红色。

    • 被 ClX2, OX2, KMnOX4\ce{Cl2,O2,KMnO4}ClX2​,OX2​,KMnOX4​ 等氧化。

  • 亚铁盐的保存：（以硫酸亚铁为例）加入铁粉防氧化，加入适量稀硫酸防水解。

检验和除杂

  • 硫氰化钾 KSCN\ce{KSCN}KSCN 法：

    • 检验亚铁离子：加入 KSCN\ce{KSCN}KSCN 无明显现象，加入适量氧化剂（ClX2\ce{Cl2}ClX2​）变血红色。

    • 检验铁离子：加入 KSCN\ce{KSCN}KSCN 变血红色，加入适量还原剂（Fe\ce{Fe}Fe）变无色。

    • 注意：FeX3++3 SCNX−→Fe(SCN)X3\ce{Fe^3+ + 3SCN- -> Fe(SCN)3}FeX3++3SCNX−​Fe(SCN)X3​ 为配合反应，故即使 Fe(SCN)X3\ce{Fe(SCN)3}Fe(SCN)X3​ 不是沉淀也不能大量共存。

  • 铁氰化钾（KX3[Fe(CN)X6]\ce{K3[Fe(CN)6]}KX3​[Fe(CN)X6​]）、亚铁氰化钾（KX4[Fe(CN)X6]\ce{K4[Fe(CN)6]}KX4​[Fe(CN)X6​]）法：

    • 检验亚铁离子：加入铁氰化钾，产生蓝色沉淀（铁氰化亚铁）。

    • 检验铁离子：加入亚铁氰化钾，产生蓝色沉淀（亚铁氰化铁）。

  • 邻二氮菲在 pH 为 2∼92\sim92∼9 时，会与亚铁离子（FeX2+\ce{Fe^2+}FeX2+）形成稳定的橙红色邻二氮菲亚铁离子。

  • 除杂：

    • FeClX2(FeClX3)\ce{FeCl2(FeCl3)}FeClX2​(FeClX3​)：2 FeX3++Fe→3 FeX2+\ce{2Fe^3+ + Fe -> 3Fe^2+}2FeX3++Fe​3FeX2+。

    • FeClX3(FeClX2)\ce{FeCl3(FeCl2)}FeClX3​(FeClX2​)：ClX2, HX2OX2(HX+), KMnOX4\ce{Cl2,H2O2(H+),KMnO4}ClX2​,HX2​OX2​(HX+),KMnOX4​。

碳酸亚铁的制备

如果用铁离子和碳酸根制备的话，会因为碳酸根碱性过强导致生成氢氧化亚铁沉淀杂质。

因此，需要用碱性更弱的碳酸氢根来制备硫酸亚铁，同时需要向碳酸氢根溶液中逐滴滴加亚铁离子溶液。如果反过来也会导致碳酸氢根碱性强生成氢氧化亚铁沉淀。

补充：高铁酸

制备原理：FeX3++OHX−+ClX2 / ClOX−→FeOX4X2−\ce{Fe^3+ + OH- + Cl2 / ClO- -> FeO4^2-}FeX3++OHX−+ClX2​ / ClOX−​FeOX4​X2−。

正六价的铁是铁的最高价态，而高铁酸钾 KX2FeOX4\ce{K2FeO4}KX2​FeOX4​ 也只在碱性环境中稳定存在。

高铁酸钾在受潮、受热，酸性、中性环境中会分解产生氧气。

因为其强氧化性，且还原产物三价铁离子可以起到吸附的作用，常用作杀菌净水剂。

氢氧化物的化学性质

  • 碱性（弱碱性）：

    • Fe(OH)X3+3 HX+→FeX3++3 HX2O\ce{Fe(OH)3 + 3H+ -> Fe^3+ + 3H2O}Fe(OH)X3​+3HX+​FeX3++3HX2​O。

    • Fe(OH)X2+2 HX+→FeX2++2 HX2O\ce{Fe(OH)2 + 2H+ -> Fe^2+ + 2H2O}Fe(OH)X2​+2HX+​FeX2++2HX2​O。

  • 不稳定性（易氧化）：

    1. 生成白色絮状沉淀：FeX2++2 OHX−→Fe(OH)X2\ce{Fe^2+ + 2OH- -> Fe(OH)2}FeX2++2OHX−​Fe(OH)X2​。

    2. 迅速变为灰绿色：不稳定，易被氧化。

    3. 最终变为红褐色：4 Fe(OH)X2+OX2+2 HX2O→4 Fe(OH)X3\ce{4Fe(OH)2 + O2 + 2H2O -> 4Fe(OH)3}4Fe(OH)X2​+OX2​+2HX2​O​4Fe(OH)X3​。

  • 三价铁离子的水解：

    FeX3++3 HX2O⇌Fe(OH)X3+3 HX+\ce{Fe^3+ + 3H2O <=> Fe(OH)3 + 3H+}FeX3++3HX2​O​Fe(OH)X3​+3HX+

  • 热稳定性：受热产生三氧化二铁，以及水。

氢氧化物的制备

  • 氢氧化铁胶体的制备：

    • 向沸腾的水中逐滴滴加饱和 FeClX3\ce{FeCl3}FeClX3​ 溶液。

    • FeClX3+3 HX2O→△Fe(OH)X3(胶体)+3 HCl\ce{FeCl3 + 3H2O ->[\triangle] Fe(OH)3(胶体) + 3HCl}FeClX3​+3HX2​O△​Fe(OH)X3​(胶体)+3HCl。

    • 待溶液呈红褐色，停止加热。

  • 氢氧化亚铁固体的制备：

    • 液面用煤油或苯液封，不能用 CClX4\ce{CCl4}CClX4​ 因为其密度大于水。

    • 胶头滴管伸入液面以下。

汞元素概述

汞齐

汞齐为汞与其他金属的合金；根据汞的比例，可能是固态，膏状或液态。

这些合金通过金属键形成，几乎所有金属都可溶于汞，除了铁、铂、钨和钽，中国古代道教认为汞齐是长生不老药的成分。

  • 锡汞齐：锡汞齐在 19 世纪中叶用作反射镜涂层。

  • 锌汞齐：可用于有机合成；以前干电池的锌板用少量汞混合，防止储存变质。

  • 钾汞齐：将碱金属溶于汞会放热，碱金属汞齐对空气和水非常敏感，遇氧氧化、遇水放氢，应在干燥氮气环境使用。

  • 钠汞齐：常用作强还原剂，在水中可以形成悬浊液，比直接使用钠要安全的多。

  • 铝汞齐：汞防止铝表形成坚固氧化层而让铝展现其真正活度，此时的铝不仅可以与水剧烈地发生反应，在空气中还可以被不断氧化，形成长条状的氧化铝。铝汞齐可在汞中研磨铝粒或铝丝来制备，或让铝丝、铝箔与氯化汞溶液反应制备。用作还原剂，反应本身及产生的废物都有汞，需要特殊的安全预防措施和处置方法。

  • 金银铜汞齐：金汞齐常用于精制金，蒸发掉汞令溶解其中的金析出。纯银、纯铜和汞形成的汞齐比较少见，但是它们与其他金属混合后与汞形成汞齐却被广泛运用于牙科医学。经过汞齐化处理的铜对腐蚀抗性有着明显的提升。

在自然界中，存在有金、银和铜与汞形成的天然矿物，如围山矿、汞银矿等。

稀有金属

稀土元素（“冶金工业的维生素”）：元素周期表中原子序数从57-71（从镧至镥，称为镧系元素）的15种元素以及钪和钇，共17种元素。

稀土金属广泛应用在冶金、石油化工、材料工业（电子材料、荧光材料、发光材料、永磁材料、超导材料、染色材料、纳米材料、引火合金和催化剂等）、医药及农业等领域。

我国的稀土储量、稀土产量、稀土用量和稀土出口量均居世界第一位。我国化学家徐光宪院士与其研究团队在稀土元素的分离及应用中作出了重要贡献。
