碳族元素概述

硅单质

硅的物理性质

硅有同素异形体：晶体硅、无定形硅。下文除非特殊说明，默认指晶体硅。

硅为灰黑色固体，熔沸点高，自然界中主要存在形式为硅酸盐和二氧化硅。

硅是现在使用最多的半导体材料，曾经最早使用的是锗 Ge\ce{Ge}Ge。

硅的化学性质

与强碱反应：

Si+2 NaOH+HX2O→NaX2SiOX3+2 HX2↑ \ce{Si + 2NaOH + H2O -> Na2SiO3 + 2H2 ^} Si+2NaOH+HX2​O​NaX2​SiOX3​+2HX2​↑

与 HF\ce{HF}HF（唯一可以溶解硅的酸）：

Si+4 HF→SiFX4↑+2 HX2↑ \ce{Si + 4HF -> SiF4 ^ + 2H2 ^} Si+4HF​SiFX4​↑+2HX2​↑

硅的工业制取

粗硅的制备：

  • SiOX2+2 C→高温Si(粗硅)+2 CO\ce{SiO2 + 2C ->[高温] Si(粗硅) + 2CO}SiOX2​+2C高温​Si(粗硅)+2CO。

  • 注意：此处碳做还原剂，不能说明碳的非金属性比硅强。

纯硅的制备：

  • Si+2 ClX2→高温SiClX4\ce{Si + 2Cl2 ->[高温] SiCl4}Si+2ClX2​高温​SiClX4​。
  • SiClX4+2 HX2→高温Si(粗硅)+4 HCl\ce{SiCl4 + 2H2 ->[高温] Si(粗硅) + 4HCl}SiClX4​+2HX2​高温​Si(粗硅)+4HCl。

总结如下：

SiOX2→高温CSi(粗硅)→ClX2SiClX4→HX2Si(纯硅) \ce{SiO2 ->[C][高温] Si(粗硅) ->[Cl2] SiCl4 ->[H2] Si(纯硅)} SiOX2​C高温​Si(粗硅)ClX2​​SiClX4​HX2​​Si(纯硅)

也存在四氯氢硅用氢气还原：

SiHClX3+HX2→Si+3 HCl \ce{SiHCl3 + H2 -> Si + 3HCl} SiHClX3​+HX2​​Si+3HCl

二氧化硅经碳还原所得为晶体硅，其性质稳定。

但是，用镁还原二氧化硅，则可以得到活泼的无定形硅，这种硅可以在空气中燃烧。

硅的化合物

二氧化硅

自然界中的存在形式：石英、沙子。

属于酸性氧化物，对应的酸为硅酸，可以发生反应：

SiOX2+2 NaOH→NaX2SiOX3+HX2O \ce{SiO2 + 2NaOH -> Na2SiO3 + H2O} SiOX2​+2NaOH​NaX2​SiOX3​+HX2​O
SiOX2+CaO→高温CaSiOX3 \ce{SiO2 + CaO ->[高温] CaSiO3} SiOX2​+CaO高温​CaSiOX3​

同样可以和 HF\ce{HF}HF 反应（仅和氢氟酸这一种酸反应）：

SiOX2+4 HF→SiFX4↑+2 HX2O \ce{SiO2 + 4HF -> SiF4 ^ + 2H2O} SiOX2​+4HF​SiFX4​↑+2HX2​O

用途为蚀刻玻璃。

硅酸

难溶于水的白色胶状沉淀，酸性比碳酸弱。

NaX2SiOX3+HX2O+COX2→NaX2COX3+HX2SiOX3↓ \ce{Na2SiO3 + H2O + CO2 -> Na2CO3 + H2SiO3 v} NaX2​SiOX3​+HX2​O+COX2​​NaX2​COX3​+HX2​SiOX3​↓

可以理解为碳的非金属性比硅强。

HX2SiOX3+HX2O→HX4SiOX4 \ce{H2SiO3 + H2O -> H4SiO4} HX2​SiOX3​+HX2​O​HX4​SiOX4​

可以与水结合，因此可以作为干燥剂。

硅酸加热易分解：

HX2SiOX3→△SiOX2+HX2O \ce{H2SiO3 ->[\triangle] SiO2 + H2O} HX2​SiOX3​△​SiOX2​+HX2​O

硅的应用

物质应用
Si\ce{Si}Si半导体（芯片、太阳能电池）
SiOX2\ce{SiO2}SiOX2​光导纤维、水晶、石英、玛瑙
HX2SiOX3\ce{H2SiO3}HX2​SiOX3​干燥剂、吸附剂

三大硅酸盐产品：

  • 陶瓷：黏土。

  • 水泥：CaSiOX3, CaO\ce{CaSiO3,CaO}CaSiOX3​,CaO 混合。

  • 玻璃：NaX2COX3+CaCOX3+SiOX2→NaX2SiOX3+CaSiOX3+COX2\ce{Na2CO3 + CaCO3 + SiO2 -> Na2SiO3 + CaSiO3 + CO2}NaX2​COX3​+CaCOX3​+SiOX2​​NaX2​SiOX3​+CaSiOX3​+COX2​。

本质上是两个反应：

NaX2COX3+SiOX2→NaX2SiOX3+COX2 \ce{Na2CO3 + SiO2 -> Na2SiO3 + CO2} NaX2​COX3​+SiOX2​​NaX2​SiOX3​+COX2​
CaCOX3+SiOX2→CaSiOX3+COX2 \ce{CaCO3 + SiO2 -> CaSiO3 + CO2} CaCOX3​+SiOX2​​CaSiOX3​+COX2​

玻璃的主要成分是 NaX2SiOX3\ce{Na2SiO3}NaX2​SiOX3​ 和 CaSiOX3\ce{CaSiO3}CaSiOX3​，以及没有反应完全的 SiOX2\ce{SiO2}SiOX2​。人们经常向玻璃内添加 PbO\ce{PbO}PbO，以增加玻璃的折射率和密度，制成水晶玻璃；还也可以添加 BX2OX3\ce{B2O3}BX2​OX3​，以增加玻璃的耐热性，制成硼硅玻璃。添加 FeX2OX3\ce{Fe2O3}FeX2​OX3​ 可以使玻璃呈绿色，添加 CoO\ce{CoO}CoO 可以使玻璃呈蓝色，添加 CuX2O\ce{Cu2O}CuX2​O 可以使玻璃呈红色。

钢化玻璃：将玻璃加热到软化点以上，然后迅速冷却，形成表面压应力，增加强度和韧性。
