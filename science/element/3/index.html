贫金属与类金属

铝及其化合物

铝单质

铝是一种银白色的轻金属，位于元素周期表第 3 周期、第 \ce{IIIA} 族（第 13 族），原子序数为 13。

  • 核外电子排布：基态铝原子的核外电子排布式为 \ce{1s^2 2s^2 2p^6 3s^2 3p^1}，也可简写为 \ce{[Ne] 3s^2 3p^1}。
  • 价电子特征：铝原子的价电子排布为 \ce{3s^2 3p^1}，最外层有 3 个电子。在化学反应中，铝原子极易失去这 3 个电子形成带有 3 个单位正电荷的铝离子（\ce{Al^3+}），因此常表现出 +3 价。

铝是地壳中含量最丰富的金属元素，丰度约为 8.23\% 至 8.8\%，在所有元素中居第三位（仅次于氧和硅）。

  1. 外观与质地：铝具有银白色金属光泽，质地较软，具有良好的延展性。1 \pu{g} 铝可以拉成约 \pu{37 m} 的细丝，或展成厚度仅为 \pu{0.00008 mm} 的铝箔。
  2. 密度与熔沸点：其密度约为 \pu{2.70 g/cm^3}，约为铁的 1/3，属于典型的轻金属。熔点为 \pu{660.32 °C}，沸点为 \pu{2519 °C}。
  3. 导电与导热性：铝具有优良的导电性和导热性。虽然导电能力略低于铜，但由于其质量轻且价格更低，常被用于制造高压输电线（钢芯铝绞线）。

铝是一种活泼金属，但在常温下具有极强的抗腐蚀性。这是因为铝在空气中能迅速与 \ce{O2} 反应，表面生成一层厚度约为 \pu{10 nm} 的致密、连续且无孔的氧化铝（\ce{Al2O3}）保护膜，保护内部金属不再被氧化。

  1. 与非金属单质反应

    • 与氧气反应：在常温下形成氧化膜；在点燃或加热条件下，铝粉或铝箔在空气/氧气中剧烈燃烧，发出耀眼的白光并放出大量的热： \ce{4Al + 3O2 \xlongequal{点燃} 2Al2O3}
    • 与氯气反应：\ce{2Al + 3Cl2 \xlongequal{\Delta} 2AlCl3}（产生白烟）。
    • 与硫反应：\ce{2Al + 3S \xlongequal{\Delta} Al2S3}。

  2. 两性特征：与酸、强碱的反应 铝是中学阶段唯一既能与酸反应产生 \ce{H2}，又能与强碱溶液反应产生 \ce{H2} 的金属。

    • 与酸反应：铝能溶于非氧化性强酸（如盐酸、稀硫酸），放出氢气： \ce{2Al + 6H+ = 2Al^3+ + 3H2 ^} 注意： 在常温下，铝遇冷的浓硫酸或浓硝酸会发生钝化现象。这是因为浓酸的强氧化性使铝表面生成致密的氧化膜，阻止了反应的进一步进行，因此工业上常用铝制容器（铝槽车）盛装或运输冷浓硝酸/浓硫酸。
    • 与强碱反应：铝能与 \ce{NaOH} 等强碱溶液反应产生氢气： \ce{2Al + 2OH- + 6H2O = 2[Al(OH)4]- + 3H2 ^} （注：产生的配离子也可写为偏铝酸根 \ce{AlO2-}，方程式相应调整为 \ce{2Al + 2OH- + 2H2O = 2AlO2- + 3H2 ^}）。该反应的实质是铝先与水反应生成 \ce{Al(OH)3}，生成的 \ce{Al(OH)3} 随即被强碱溶解，反应中氧化剂是水。

  3. 铝热反应 在高温下，铝能与许多金属氧化物（如 \ce{Fe2O3, Fe3O4, Cr2O3, V2O5} 等）发生剧烈的置换反应，并放出巨大的热量。反应生成的金属通常呈熔融态： \ce{2Al + Fe2O3 \xlongequal{高温} Al2O3 + 2Fe} 应用：铝热反应常用于焊接钢轨及冶炼某些难熔金属。

铝的工业制取

常见的工业制铝为霍尔-埃鲁法，现在常与与拜耳法联用，产铝量大。

铝的工业冶炼是一个复杂的多阶段过程，主要包括从铝土矿中提取纯净氧化铝的拜耳法（Bayer process）以及通过电解熔融氧化铝制取铝单质的霍尔-埃鲁法（Hall-Héroult process）。

以下结合您的笔记与来源资料，详细展开讲解铝的工业冶炼流程：

  1. 铝土矿的预处理与粉碎：铝在自然界中主要以铝土矿（主要成分为 \ce{Al2O3.nH2O}，含 \ce{Fe2O3}、\ce{SiO2}、\ce{TiO2} 等杂质）形式存在。

    • 物理除杂与粉碎：首先用水冲洗掉颗粒表面的粘土。随后将铝土矿与低浓度的氢氧化钠溶液（拜耳法余液）混合，利用球磨机进行研磨形成悬浊液。
    • 意义：研磨使矿石粒径变小，极大地增加了铝土矿的比表面积，从而加快后续化学反应的速率并提高原料利用率。

  2. 溶出阶段（拜耳法核心）：将悬浮液送入反应釜，在高温（约 \pu{140 °C} 至 \pu{150 °C}）和高压下，用高浓度 \ce{NaOH} 溶液处理。

    1. 氧化铝的溶解：\ce{Al2O3} 与 \ce{NaOH} 反应生成可溶的偏铝酸钠（四羟基合铝酸钠）：

      \ce{Al2O3 + 2NaOH + 3H2O = 2Na[Al(OH)4]}

    2. 杂质的分离（赤泥）：铝土矿中的 \ce{Fe2O3}、\ce{CaO} 和 \ce{TiO2} 基本不与 \ce{NaOH} 反应，形成固体沉淀。这些沉淀因含有大量的氧化铁而呈红色，被称为赤泥 [笔记, 225]。赤泥具有强腐蚀性（\pu{pH} 达 11\sim 12），处理不当会带来严重的环保问题。

    3. 脱硅反应：\ce{SiO2} 杂质会先生成硅酸钠，随后与偏铝酸钠反应生成方钠石结构的水合铝硅酸钠沉淀： \ce{SiO2 + 2NaOH = Na2SiO3 + H2O} 产生的硅酸钠沉淀被过滤除去，确保只有铝酸钠留在上清液中。

  3. 析出阶段：为了从溶液中析出氢氧化铝，需要改变溶液的平衡状态。

    1. 稀释与冷却：将热的铝酸钠溶液进入冷却装置并加水稀释，溶液的碱度降低，促使铝酸钠发生水解。
    2. 晶种分解：加入纯净的氧化铝粉末作为晶种，诱导白色的 \ce{Al(OH)3} 固体析出。
    3. 改进工艺：部分厂家通入过量的 \ce{CO2} 以酸化溶液，提高 \ce{Al(OH)3} 的沉淀效率： \ce{Na[Al(OH)4] + CO2 = Al(OH)3 v + NaHCO3}
    4. 碱液循环：过滤出的溶液仍含有较高浓度的 \ce{NaOH}，可返回流程第一步循环利用。

  4. 分解阶段（煅烧）：将过滤出的 \ce{Al(OH)3} 沉淀在 \pu{1000 °C} 以上的高温下进行煅烧，使其发生热分解。

    \ce{2Al(OH)3 \xlongequal{\Delta} Al2O3 + 3H2O}

    煅烧产生的纯净 \ce{Al2O3} 即为下一步电解的原料。

  5. 电解阶段（霍尔-埃鲁法）：由于 \ce{AlCl3} 是共价化合物，其熔融状态不导电，因此工业上必须电解熔融的 \ce{Al2O3}。

    1. 助熔剂的作用：\ce{Al2O3} 的熔点极高（约 \pu{2054 °C}），直接熔化能耗巨大。加入冰晶石（\ce{Na3AlF6}，六氟合铝酸钠）后，混合物的熔点可降至约 \pu{950 °C} 至 \pu{1000 °C}。
    2. 电解原理：
      • 阳极（石墨）：\ce{2O^2- - 4e- = O2 ^}
      • 阴极（石墨）：\ce{Al^3+ + 3e- = Al}
      • 总反应：\ce{2Al2O3 (熔融) \xrightarrow[Na3AlF6]{通电} 4Al + 3O2 ^}
    3. 电极消耗：阳极产生的 \ce{O2} 在高温下会与石墨电极反应生成 \ce{CO} 或 \ce{CO2}（此反应为放热反应，可补充熔融能量），因此石墨阳极需要定期更换。
    4. 产物收集：在约 \pu{1000 °C} 时，液态铝的密度（约 \pu{2.3 g/cm^3}）大于熔融电解质的密度（约 \pu{2.1 g/cm^3}），因此铝会沉积在电解池底部。工业上利用虹吸原理将铝液移出，并不断加入 \ce{Al2O3} 以维持生产。
    5. 能耗与环保：电解铝是高能耗行业，制取 1 \pu{t} 铝约需消耗 1.7 \times 10^4 至 2 \times 10^4 度电。再生铝（回收废铝）所需的能量仅为原生铝的 5 \% 左右，具有极高的回收价值。

铝的化学反应核心

铝及其化合物的化学性质极具特色，其核心在于铝元素的两性（既能与酸反应，又能与强碱反应）。这种特性在水溶液中集中体现为“铝三角”，即铝离子（\ce{Al^3+}）、氢氧化铝（\ce{Al(OH)3}）与四羟基合铝酸根（\ce{[Al(OH)4]-}，中学阶段常简写为偏铝酸根 \ce{AlO2-}）之间的相互转化。

铝是第 3 周期第 \ce{IIIA} 族的轻金属，原子序数为 13，最外层有 3 个电子，极易失去电子表现出强还原性。

  1. 与非金属反应：
    • 氧化反应：常温下表面生成致密氧化膜（厚度约 10 \pu{nm}），具有抗腐蚀性。点燃时剧烈燃烧：\ce{4Al + 3O2 \xlongequal{点燃} 2Al2O3}。
    • 与卤素/硫反应：\ce{2Al + 3Cl2 \xlongequal{\Delta} 2AlCl3}；\ce{2Al + 3S \xlongequal{\Delta} Al2S3}。
  2. 两性反应（生成“铝三角”的两端）：
    • 与酸反应：铝能溶于盐酸或稀硫酸，置换出 \ce{H2}：\ce{2Al + 6H+ = 2Al^3+ + 3H2 ^}。
      • 注意：常温下，铝遇冷的浓硫酸或浓硝酸会发生钝化，表面生成致密的氧化膜阻止反应继续。
    • 与强碱反应：铝是少数能与强碱溶液反应产生 \ce{H2} 的金属：\ce{2Al + 2OH- + 6H2O = 2[Al(OH)4]- + 3H2 ^}。此反应中，水是氧化剂，铝先与水反应生成 \ce{Al(OH)3}，后者再溶于碱。
  3. 铝热反应：在高温下，铝能将某些金属氧化物（如 \ce{Fe2O3, Cr2O3, MnO2, V2O5}）还原为金属单质，并放出大量的热： \ce{2Al + Fe2O3 \xlongequal{高温} Al2O3 + 2Fe}。

氧化铝是白色难溶固体，熔点高达 \pu{2054 °C}，硬度极大（刚玉）。

  1. 两性氧化物性质：
    • 与酸反应：\ce{Al2O3 + 6H+ = 2Al^3+ + 3H2O}。
    • 与强碱反应：\ce{Al2O3 + 2OH- + 3H2O = 2[Al(OH)4]-}。
  2. 冶炼制铝：工业上采用电解熔融氧化铝的方法。加入冰晶石（\ce{Na3AlF6}）作为助熔剂可将熔点降至约 \pu{950 °C} 至 \pu{1000 °C}： \ce{2Al2O3 (熔融) \xrightarrow[\ce{Na3AlF6}]{\text{通电}} 4Al + 3O2 ^}。

氢氧化铝是白色胶状沉淀，具有两性，不溶于弱酸（如碳酸）和弱碱（如氨水）。

  1. 两性电离平衡： \ce{Al(OH)3} 在水溶液中存在以下电离平衡： \ce{H+ + AlO2- + H2O}（酸式电离） \ce{<=> Al(OH)3 <=> Al^3+ + 3OH-}（碱式电离）
  2. 转化为铝离子（碱性表现）：与强酸反应：\ce{Al(OH)3 + 3H+ = Al^3+ + 3H2O}。
  3. 转化为四羟基合铝酸根（酸性表现）：与强碱反应：\ce{Al(OH)3 + OH- = [Al(OH)4]-}。
  4. 受热分解：\ce{2Al(OH)3 \xlongequal{\Delta} Al2O3 + 3H2O}。
  5. 实验室制备：常用铝盐与氨水反应，以防止过量的强碱使沉淀溶解： \ce{Al^3+ + 3NH3.H2O = Al(OH)3 v + 3NH4+}。

铝三角的核心转化（\ce{Al^3+}、\ce{Al(OH)3}、\ce{[Al(OH)4]-}）：

  1. \ce{Al^3+ \rightarrow Al(OH)3}：加入少量 \ce{OH-} 或氨水。
  2. \ce{Al(OH)3 \rightarrow Al^3+}：加入强酸（\ce{H+}）。
  3. \ce{Al(OH)3 \rightarrow [Al(OH)4]-}：加入强碱（\ce{OH-}）。
  4. \ce{[Al(OH)4]- \rightarrow Al(OH)3}：加入少量强酸（\ce{H+}）或通入 \ce{CO2}：\ce{[Al(OH)4]- + CO2 = Al(OH)3 v + HCO3-}。
  5. \ce{Al^3+ \rightarrow [Al(OH)4]-}（极限跨越）：加入过量强碱：\ce{Al^3+ + 4OH- = [Al(OH)4]- + 2H2O}。
  6. \ce{[Al(OH)4]- \rightarrow Al^3+}（极限跨越）：加入过量强酸：\ce{[Al(OH)4]- + 4H+ = Al^3+ + 4H2O}。
  7. 相互促进的水解反应（双水解）：当 \ce{Al^3+} 与 \ce{[Al(OH)4]-} 混合时，会发生彻底的双水解生成 \ce{Al(OH)3}：\ce{Al^3+ + 3[Al(OH)4]- = 4Al(OH)3 v}。

编号反应方程式
①\ce{Al^3+ + 3OH- -> Al(OH)3 v}
②\ce{Al^3+ + 3NH3*H2O -> Al(OH)3 v + 3NH4+}
③\ce{Al(OH)3 + 3H+ -> Al^3+ + 3H2O}
④\ce{Al(OH)3 + OH- -> [Al(OH)4]-}
⑤\ce{[Al(OH)4]- + H+ -> Al(OH)3 v + H2O}
⑥\ce{[Al(OH)4]- + CO2 -> Al(OH)3 v + HCO3-}
⑦\ce{[Al(OH)4]- + 4H+ -> Al^3+ + 4H2O}
⑧\ce{Al^3+ + 4OH- -> [Al(OH)4]-}

核心计算规律与应用：

  1. 等量铝消耗酸碱及产氢量：
    • 等质量的铝分别与足量的盐酸和 \ce{NaOH} 反应，产生的 \ce{H2} 体积比为 1 : 1。
    • 消耗 \ce{H+} 与 \ce{OH-} 的物质的量之比为 3 : 1。
  2. 等摩尔酸碱与足量铝反应：
    • 足量的铝与等物质的量的 \ce{HCl} 和 \ce{NaOH} 反应，产氢体积比为 1 : 3。
  3. 滴加顺序鉴别：
    • 向 \ce{AlCl3} 溶液滴加 \ce{NaOH}：先出现白色沉淀，后溶解。
    • 向 \ce{NaOH} 溶液滴加 \ce{AlCl3}：开始无明显现象，后产生白色沉淀。
  4. 除杂应用：
    • 除去 \ce{Fe2O3} 中的 \ce{Al2O3}，可用过量 \ce{NaOH} 溶液处理后过滤。
    • 除去 \ce{Mg} 中的 \ce{Al}，可用足量 \ce{NaOH} 溶液溶解铝。

对于 \pu{1mol} 铝单质，我们用 \pu{3mol} 氢离子将其转化为 \pu{1mol} 铝离子，然后用 \pu{3mol} 氢氧根将其转化为氢氧化铝。但是这很费原料，不妨进行改进：先用水和 \pu{1mol} 氢氧根转化为四羟基合铝离子，然后用 \pu{1mol} 氢离子转化为氢氧化铝。但是我们还有更好的方法：将 \pu{4mol} 铝单质分为两份，一份 \pu{3mol} 用 \pu{3mol} 氢氧根处理为四羟基合铝离子，一份 \pu{1mol} 用 \pu{3mol} 氢离子处理为铝离子，两份混合得到 \pu{4mol} 氢氧化铝。相当于 \pu{1mol} 铝消耗 \pu{3/4mol} 氢离子和氢氧根。

铝的重要化合物

铝的重要化合物中，盐类和类盐化合物占据了核心地位，其化学行为深受铝离子（\ce{Al^3+}）强极化力和缺电子结构的影响。以下详细展开讲解这些化合物的性质与转化。

氯化铝是铝最常见的卤化物之一，呈现出明显的共价性。

  1. 结构特征：
    • 二聚现象：在气相或非极性溶剂中，\ce{AlCl3} 以二聚分子 \ce{Al2Cl6} 形式存在。这是由于铝原子的缺电子结构，两个 \ce{AlCl3} 分子通过氯桥键结合，铝原子呈 sp^3 杂化，整体为无极性分子。
    • 物理性质：无水 \ce{AlCl3} 为白色固体，熔、沸点较低，且易升华（升华点约 \pu{177.8 °C}）。
  2. 化学性质与水解：
    • 强烈水解：\ce{AlCl3} 遇水发生剧烈水解并放热，在潮湿空气中会产生白雾（\ce{HCl}）。水解反应分步进行，最终生成 \ce{Al(OH)3} 沉淀：\ce{AlCl3 + 3H2O \xlongequal{\Delta} Al(OH)3 v + 3HCl ^}。
    • 电离：虽然熔融状态不导电（分子晶体），但在水溶液中它是强电解质，完全电离出 \ce{Al^3+} 和 \ce{Cl-}。
  3. 应用：常用作有机合成（如石油工业）的催化剂。

硫酸铝与明矾：复盐与净水核心。

  1. 硫酸铝 (\ce{Al2(SO4)3})：
    • 白色粉末，易溶于水。它是由强酸和弱碱形成的盐，水溶液呈酸性。
    • 由于 \ce{Al^3+} 的水解生成具有强吸附能力的 \ce{Al(OH)3} 胶体，它被广泛用作水的净化剂。
  2. 明矾 (\ce{KAl(SO4)2.12H2O})：
    • 复盐定义：明矾是典型的复盐，在水中完全电离为 \ce{K+}、\ce{Al^3+} 和 \ce{SO4^2-}。
    • 净水原理：\ce{Al^3+} 在水中发生水解平衡：\ce{Al^3+ + 3H2O <=> Al(OH)3 (胶体) + 3H+}。生成的氢氧化铝胶体带正电荷，能吸附天然水中带负电的悬浮微粒并聚沉。
    • 注意：明矾净水只能除去悬浮杂质，不具备杀菌消毒作用。

偏铝酸盐（四羟基合铝酸盐）：在强碱性环境下，铝元素以阴离子形式存在。

  1. 固体状态常写为偏铝酸钠 (\ce{NaAlO2})，而在水溶液中实际以四羟基合铝酸钠 (\ce{Na[Al(OH)4]}) 形式存在。
  2. 与酸反应：滴加少量强酸产生 \ce{Al(OH)3} 沉淀，过量则溶解生成 \ce{Al^3+}。
  3. 通入 \ce{CO2}：无论 \ce{CO2} 是否过量，均产生 \ce{Al(OH)3} 沉淀且不溶解。
    • \ce{CO2} 过量：\ce{[Al(OH)4]- + CO2 = Al(OH)3 v + HCO3-}。
    • \ce{CO2} 少量：\ce{2[Al(OH)4]- + CO2 = 2Al(OH)3 v + CO3^2- + H2O}。

类盐化合物，通常表现出共价特性，且易发生双水解。

  1. 氮化铝 (\ce{AlN})：
    • 共价晶体，具有高强度和优良的导热性。
    • 在约 \pu{90 °C} 以上遇水会发生水解，生成 \ce{Al(OH)3} 和 \ce{NH3}：\ce{AlN + 3H2O \xlongequal{\Delta} Al(OH)3 + NH3 ^}。
  2. 硫化铝 (\ce{Al2S3})：
    • 完全双水解：\ce{Al2S3} 在水中极不稳定，会发生彻底的双水解反应：\ce{Al2S3 + 6H2O = 2Al(OH)3 v + 3H2S ^}。
    • 因此，\ce{Al2S3} 不能在水溶液中制备，只能通过铝粉与硫粉在高温下直接化合制得。

铝盐反应的定量规律与图像：

  1. 铝离子与强碱的滴加顺序：
    • 酸入碱（向 \ce{NaOH} 中滴加 \ce{AlCl3}）：开始无现象（生成 \ce{AlO2-}），后产生沉淀。
    • 碱入酸（向 \ce{AlCl3} 中滴加 \ce{NaOH}）：立即产生白色沉淀，后沉淀溶解。
  2. 等量关系：
    • 等物质的量的铝单质、氧化铝和氢氧化铝在与酸或碱反应时，消耗 \ce{H+} 与 \ce{OH-} 的物质的量之比始终为 3 : 1。
    • 等质量的铝分别与足量盐酸和 \ce{NaOH} 溶液反应，生成的 \ce{H2} 体积比为 1 : 1。
    • 足量的铝分别与等物质的量的 \ce{HCl} 和 \ce{NaOH} 反应，生成的 \ce{H2} 体积比为 1 : 3。

氧化铝和铝热反应

氧化铝是铝最稳定的氧化物，主要以铝土矿（又称铝矾土）的形式存在于自然界中。它是工业冶炼铝的主要原料，也是一种功能多样的无机材料。

  • 物理常数：氧化铝是一种白色、难溶于水的固体粉末，具有极高的熔点（约 2050 \pu{°C} 至 2054 \pu{°C}）和沸点（约 2977 \pu{°C}）。其莫氏硬度高达 9，仅次于金刚石。
  • 同质多级态：氧化铝存在多种晶型，其中最常见的是 \alpha-\ce{Al2O3} 和 \gamma-\ce{Al2O3}。
    • \alpha-\ce{Al2O3}（刚玉）：晶体结构极其致密（氧离子呈六方紧密堆积），性质稳定，不溶于酸或碱。天然产出的透明晶体根据杂质颜色不同被称为红宝石（含微量 Cr）或蓝宝石（含微量 Fe 和 Ti）。由于硬度大、耐磨，常用于制造精密仪表轴承、研磨材料和耐火材料。
    • \gamma-\ce{Al2O3}（活性氧化铝）：由氢氧化铝在较低温度（约 300 \pu{°C} 至 500 \pu{°C}）下脱水制得。它具有极大的比表面积（1 \pu{g} 样品的表面积可达 200 ～ 600 \pu{m^2}）和较强的吸附能力，常作为吸附剂、脱水剂或催化剂载体。

氧化铝是典型的两性氧化物，既能与强酸反应生成铝盐，又能与强碱反应生成铝酸盐。

  • 与强酸反应：\ce{Al2O3 + 6H+ = 2Al^3+ + 3H2O}。
  • 与强碱反应：氧化铝溶于强碱溶液生成四羟基合铝酸根（中学阶段常简写为偏铝酸根）： \ce{Al2O3 + 2OH- + 3H2O = 2[Al(OH)4]-}（或 \ce{Al2O3 + 2OH- = 2AlO2- + H2O}）。
  • 注意：由于氧化铝能被酸、碱和盐（如 \ce{NaCl}）侵蚀，因此铝制餐具不宜长时间存放酸性、碱性或过咸的食物。

工业上主要采用电解熔融氧化铝的方法制取铝单质。由于其熔点极高，生产中需加入冰晶石（\ce{Na3AlF6}）作为助熔剂，使混合物的熔化温度降低至约 950 \pu{°C} 左右，显著降低能耗。

铝热反应是指利用铝粉作为强还原剂，在高温条件下还原金属氧化物（通常是比铝不活泼的金属氧化物）并释放出巨大热量的氧化还原反应。

  • 铝热剂：铝粉与金属氧化物（如 \ce{Fe2O3}、\ce{Fe3O4}、\ce{Cr2O3}、\ce{MnO2}、\ce{V2O5} 等）按一定比例混合形成的混合物。
  • 典型化学方程式：
    • 与氧化铁：\ce{2Al + Fe2O3 \xlongequal{高温} Al2O3 + 2Fe}。
    • 与四氧化三铁：\ce{8Al + 3Fe3O4 \xlongequal{高温} 4Al2O3 + 9Fe}。
    • 与三氧化二铬：\ce{2Al + Cr2O3 \xlongequal{高温} Al2O3 + 2Cr}。
  • 能量特征：铝具有极强的还原性和很大的标准生成焓，反应产生的温度可高达 2000 \pu{°C} 至 3000 \pu{°C} 以上，使还原出的金属呈熔融态析出。

实验过程与现象：

  • 实验装置：通常在纸漏斗（通常多层叠加且底部剪孔）中进行，下方盛放细沙以承接熔融金属，防止损坏实验台。
  • 引发操作：在铝热剂上方平铺少量氯酸钾（\ce{KClO3}），并插入一根打磨过的镁条。点燃镁条后，镁燃烧释放热量使氯酸钾分解产生氧气，进一步加剧燃烧并产生极高温度，从而引发铝热反应。
  • 实验现象：镁条剧烈燃烧，反应混合物发出耀眼的白光，产生大量的烟，纸漏斗下部被烧穿，红热的液滴（熔融金属铁）落入沙中，冷却后变为黑色固体珠。

实际应用：

  • 冶炼难熔金属：工业上常用铝热法冶炼熔点较高且活动性比铝弱的金属，如铬（Cr）、锰（Mn）、钒（V）、钨（W）等，生产出的金属质量较高且易于分离。
  • 焊接钢轨：利用反应产生的高温液态铁水灌入钢轨缝隙，待其冷却凝固后将钢轨焊接在一起，该工艺设备简单，常用于野外无缝线路建设。
  • 军事用途：由于反应伴随剧烈发光发热，可用于制造铝热弹、信号弹或烟火剂。

以铝为核心的用量实验

铝（\ce{Al}）及其化合物的化学性质研究，重点在于其独特的两性特征。这种特征使得铝在与不同强度的酸碱反应时，呈现出明显的阶段性与竞争性，其转化的逻辑中枢即为“铝三角”［\ce{Al^3+}、\ce{Al(OH)3}、\ce{[Al(OH)4]-}］。

铝是中学阶段唯一既能与酸反应产生 \ce{H2}，又能与强碱反应产生 \ce{H2} 的金属。当铝与酸、碱反应时，产氢量与反应物用量之间存在如下规律：

  • 等质量的铝：与足量酸、碱分别反应时，产生 \ce{H2} 的物质的量相等（比值为 1 : 1），但消耗的 \ce{n(H+)} 与 \ce{n(OH-)} 之比为 3 : 1。
  • 等物质的量的酸与碱：若与足量的铝反应，由于生成 1 \pu{mol} \ce{H2} 分别需要 2 \pu{mol} \ce{H+} 或 2/3 \pu{mol} \ce{OH-}，其产氢体积比 V(\ce{H2, 酸}) : V(\ce{H2, 碱}) = 1 : 3。
  • 实验判据：若产生 \ce{H2} 的体积比在 1/3 与 1 之间，说明铝对酸过量，对碱不足。

铝及其氧化物、氢氧化物在不同酸碱介质中的存在形态，取决于溶液的 \pu{pH} 环境。

  1. 强碱方向：分步反应与滴加顺序。当向 \ce{Al^3+} 溶液中加入强碱（如 \ce{NaOH}）时，反应具有阶段性：

    • 第一步（碱不足）：\ce{Al^3+ + 3OH- = Al(OH)3 v}（生成白色胶状沉淀）。
    • 第二步（碱过量）：\ce{Al(OH)3 + OH- = [Al(OH)4]-}（沉淀溶解）。
    • 实验探究：
      • 碱入盐：现象为“先产生白色沉淀，后沉淀溶解”。
      • 盐入碱：现象为“开始无明显现象，后产生沉淀且不溶解”。
        • 原理：开始时碱过量，直接生成 \ce{[Al(OH)4]-}；随着 \ce{Al^3+} 增多，发生双水解：\ce{Al^3+ + 3[Al(OH)4]- = 4Al(OH)3 v}。

  2. 弱碱方向：制备氢氧化铝的理想路径。\ce{Al(OH)3} 能溶于强碱，但不溶于弱碱（如氨水）。

    • 反应方程式：\ce{Al^3+ + 3NH3 \cdot H2O = Al(OH)3 v + 3NH4+}。
    • 实验意义：实验室制取 \ce{Al(OH)3} 沉淀通常选用氨水，因为弱碱过量也不会导致沉淀溶解，便于控制用量。

  3. 强酸方向：偏铝酸盐的逆向转化。

    • 少量强酸：\ce{[Al(OH)4]- + H+ = Al(OH)3 v + H2O}。
    • 过量强酸：\ce{[Al(OH)4]- + 4H+ = Al^3+ + 4H2O}。

  4. 弱酸方向：二氧化碳的特殊作用，\ce{Al(OH)3} 不溶于弱酸（如碳酸）。

    • 过量 \ce{CO2}：\ce{[Al(OH)4]- + CO2 = Al(OH)3 v + HCO3-}。
    • 结论：无论 \ce{CO2} 是否过量，产物均为 \ce{Al(OH)3}。这在工业流程中常用于从偏铝酸钠溶液中沉淀出铝元素。

在含有多种离子的混合溶液中，加入强碱时遵循“强者优先”的竞争反应原则。

  1. 混合阳离子与 \ce{OH-} 的反应顺序：向含有 \ce{H+}、\ce{Al^3+}、\ce{Mg^2+}、\ce{NH4+} 的溶液中逐滴加入 \ce{NaOH}，反应次序如下：

    1. 中和反应：\ce{H+ + OH- = H2O}（现象：无沉淀）。
    2. 沉淀反应：\ce{Al^3+ + 3OH- = Al(OH)3 v}、\ce{Mg^2+ + 2OH- = Mg(OH)2 v}（现象：生成沉淀）。
    3. 弱碱生成：\ce{NH4+ + OH- = NH3 \cdot H2O}（现象：沉淀质量不变）。
    4. 沉淀溶解：\ce{Al(OH)3 + OH- = [Al(OH)4]-}（现象：沉淀部分溶解）。

  2. 明矾与 \ce{Ba(OH)2} 的配比反应：明矾［\ce{KAl(SO4)2 \cdot 12H2O}］中 \ce{Al^3+} 与 \ce{SO4^2-} 的比例为 1 : 2。加入 \ce{Ba(OH)2} 时，沉淀情况随用量变化：

    • \ce{Al^3+} 恰好完全沉淀时：n(\ce{Alum}) : n(\ce{Ba(OH)2}) = 2 : 3。 \ce{2Al^3+ + 3SO4^2- + 3Ba^2+ + 6OH- = 2Al(OH)3 v + 3BaSO4 v}。
    • 沉淀质量最大时（\ce{SO4^2-} 恰好完全沉淀）：n(\ce{Alum}) : n(\ce{Ba(OH)2}) = 1 : 2。此时 \ce{Al(OH)3} 会部分溶解转化为 \ce{[Al(OH)4]-}，反应为： \ce{Al^3+ + 2SO4^2- + 2Ba^2+ + 4OH- = [Al(OH)4]- + 2BaSO4 v}。

实验探究与应用：双水解与净水。

  1. 彻底双水解的实验探究：当铝盐与弱酸根（\ce{CO3^2-}、\ce{HCO3-}、\ce{S^2-}、\ce{AlO2-}）混合时，会发生完全的双水解反应，生成 \ce{Al(OH)3} 沉淀。

    • 泡沫灭火器：利用 \ce{Al^3+ + 3HCO3- = Al(OH)3 v + 3CO2 ^}。
    • 硫化铝的制备：\ce{Al2S3} 不能通过溶液反应制取，因为在水中会剧烈水解产生 \ce{H2S}：\ce{Al2S3 + 6H2O = 2Al(OH)3 v + 3H2S ^}。

  2. 实验探究：分离 \ce{Fe^3+} 与 \ce{Al^3+}**。利用 \ce{Al(OH)3} 的两性而 \ce{Fe(OH)3} 只有碱性，可实现二者的分离：

    1. 加入过量 \ce{NaOH}：\ce{Fe^3+} 转化为 \ce{Fe(OH)3} 沉淀，\ce{Al^3+} 转化为 \ce{[Al(OH)4]-} 溶液。
    2. 过滤：得到 \ce{Fe(OH)3} 滤渣和含铝滤液。
    3. 酸化再沉淀：向滤液中通入过量 \ce{CO2}，重新获得纯净的 \ce{Al(OH)3} 沉淀。

通过上述反应规律的掌握，可以精确控制铝元素在化学分析、工业冶炼及废水处理中的转化过程。

实验：探究铝及其化合物的性质

针对铝及其化合物的性质，可以通过以下几个核心实验进行详细的探究与分析：

铝是活泼金属，但在空气中表现出极佳的抗腐蚀性，这主要归功于其表面的致密氧化膜。

  1. 氧化膜的保护作用实验：

    • 操作：取一片薄铝箔，用坩埚钳夹持并在酒精灯火焰上加热。
    • 现象：铝片表面失去光泽并变暗，铝片熔化但并不滴落，仿佛被一层薄膜包裹着。
    • 结论：铝在空气中加热，表面会迅速生成熔点高达 \pu{2054 °C} 的 \ce{Al2O3} 保护膜，其熔点远高于铝单质（\pu{660 °C}），从而包裹住熔融的液态铝。

  2. 氧化膜的破坏实验（铝汞齐实验）：

    • 操作：将铝片浸入 \ce{Hg(NO3)2} 溶液中约 2 分钟，取出擦干。
    • 现象：铝表面很快长出白色的“毛刷”状物质（氧化铝细丝），并放出大量热。
    • 原理：铝置换出的汞与铝形成铝汞合金（铝汞齐），破坏了氧化膜的连续性，使内部的铝不断与空气中的氧气反应。

  3. 常温下的钝化实验：

    • 操作：将打磨过的铝条分别放入盛有稀盐酸、浓硝酸（常温）的试管中。
    • 现象：铝在稀盐酸中剧烈反应产生气泡；但在浓硝酸中几乎无明显现象。
    • 结论：常温下，铝遇冷的浓硝酸或浓硫酸会发生钝化，表面生成致密的氧化膜阻止反应进一步进行。

铝及其氧化物、氢氧化物均具有两性，即既能与强酸反应，又能与强碱反应。

  1. 铝单质的两性反应：

    • 反应方程式：
      • 与酸：\ce{2Al + 6H+ = 2Al^3+ + 3H2 ^}。
      • 与碱：\ce{2Al + 2OH- + 6H2O = 2[Al(OH)4]- + 3H2 ^}。
    • 实验结论：铝是中学阶段唯一既能与酸反应又能与强碱溶液反应产生 \ce{H2} 的金属。

  2. 氢氧化铝的制备与两性实验：

    • 制备操作：向 \ce{AlCl3} 溶液中滴加氨水，直至不再产生白色沉淀。
    • 两性验证：将所得 \ce{Al(OH)3} 沉淀分装于两支试管，分别加入盐酸和 \ce{NaOH} 溶液，沉淀均溶解。
    • 反应方程式：
      • \ce{Al(OH)3 + 3H+ = Al^3+ + 3H2O}。
      • \ce{Al(OH)3 + OH- = [Al(OH)4]-}。
    • 注意事项：实验室制取 \ce{Al(OH)3} 必须使用弱碱（如氨水），因为 \ce{Al(OH)3} 会溶于过量的强碱（如 \ce{NaOH}），导致沉淀消失。

铝热反应利用了铝的强还原性及反应放出的巨大热量。

  • 实验装置：使用两张圆形滤纸做成的纸漏斗，底层剪孔，架在盛有细沙的蒸发皿上方。
  • 反应物：铝粉与氧化铁（\ce{Fe2O3}）按比例混合（铝热剂），上方堆放少量 \ce{KClO3} 并插上镁条。
  • 现象：镁条燃烧放出热量引发反应，反应剧烈进行，发出耀眼白光，纸漏斗被烧穿，红热的液态铁水落入沙中凝固成铁珠。
  • 化学方程式：\ce{2Al + Fe2O3 \xlongequal{高温} Al2O3 + 2Fe}。
  • 应用：该反应常用于野外焊接钢轨以及冶炼铬、锰、钒等难熔金属。

铝离子在水溶液中的水解及与不同试剂的竞争反应是探究的重点。

  1. 明矾的净水实验：

    • 操作：向浑浊的河水中加入少量明矾粉末，搅拌并静置。
    • 现象：河水逐渐变得澄清。
    • 原理：\ce{Al^3+} 在水中发生水解生成 \ce{Al(OH)3} 胶体：\ce{Al^3+ + 3H2O <=> Al(OH)3 （胶体） + 3H+}。胶体具有较强的吸附性，能凝聚水中的悬浮物并沉降。

  2. 双水解反应探究：

    • 操作：向饱和明矾溶液中加入饱和 \ce{NaHCO3} 溶液。
    • 现象：立即产生大量白色气泡和白色胶状沉淀。
    • 化学方程式：\ce{Al^3+ + 3HCO3- = Al(OH)3 v + 3CO2 ^}。
    • 应用：该反应是泡沫灭火器的主要原理。

  3. 滴加顺序对实验现象的影响：

    • 向 \ce{AlCl3} 溶液中滴加 \ce{NaOH} 溶液：先产生白色沉淀，后沉淀逐渐溶解并消失。
    • 向 \ce{NaOH} 溶液中滴加 \ce{AlCl3} 溶液：开始时无明显现象（因为碱过量生成了 \ce{[Al(OH)4]-}），待碱消耗完后开始产生白色沉淀且不再溶解。

在实验探究中，产氢量的定量关系常用于物质组成的推断。

  1. 等质量的铝分别与足量的盐酸和 \ce{NaOH} 溶液反应，产生的 \ce{H2} 在同温同压下体积比为 1 : 1。
  2. 足量的铝分别与等物质的量的 \ce{HCl} 和 \ce{NaOH} 溶液反应，产生的 \ce{H2} 体积比为 1 : 3。
  3. 计算依据：根据电子守恒，每生成 \pu{1 mol} \ce{H2} 需转移 \pu{2 mol} 电子，而 \pu{1 mol} \ce{Al} 完全反应无论在酸中还是碱中均转移 \pu{3 mol} 电子。

硅及其化合物

硅单质

硅的物理性质

硅有同素异形体：晶体硅、无定形硅。下文除非特殊说明，默认指晶体硅。

硅为灰黑色固体，熔沸点高，自然界中主要存在形式为硅酸盐和二氧化硅。

硅是现在使用最多的半导体材料，曾经最早使用的是锗 \ce{Ge}。

硅的化学性质

与强碱反应：

\ce{Si + 2NaOH + H2O -> Na2SiO3 + 2H2 ^}

与 \ce{HF}（唯一可以溶解硅的酸）：

\ce{Si + 4HF -> SiF4 ^ + 2H2 ^}

硅的工业制取

粗硅的制备：

  • \ce{SiO2 + 2C ->[高温] Si(粗硅) + 2CO}。

  • 注意：此处碳做还原剂，不能说明碳的非金属性比硅强。

纯硅的制备：

  • \ce{Si + 2Cl2 ->[高温] SiCl4}。

  • \ce{SiCl4 + 2H2 ->[高温] Si(粗硅) + 4HCl}。

总结如下：

\ce{SiO2 ->[C][高温] Si(粗硅) ->[Cl2] SiCl4 ->[H2] Si(纯硅)}

也存在四氯氢硅用氢气还原：

\ce{SiHCl3 + H2 -> Si + 3HCl}

二氧化硅经碳还原所得为晶体硅，其性质稳定。

但是，用镁还原二氧化硅，则可以得到活泼的无定形硅，这种硅可以在空气中燃烧。

硅的化合物

二氧化硅

自然界中的存在形式：石英、沙子。

属于酸性氧化物，对应的酸为硅酸，可以发生反应：

\ce{SiO2 + 2NaOH -> Na2SiO3 + H2O}

\ce{SiO2 + CaO ->[高温] CaSiO3}

同样可以和 \ce{HF} 反应（仅和氢氟酸这一种酸反应）：

\ce{SiO2 + 4HF -> SiF4 ^ + 2H2O}

用途为蚀刻玻璃。

硅酸

难溶于水的白色胶状沉淀，酸性比碳酸弱。

\ce{Na2SiO3 + H2O + CO2 -> Na2CO3 + H2SiO3 v}

可以理解为碳的非金属性比硅强。

\ce{H2SiO3 + H2O -> H4SiO4}

可以与水结合，因此可以作为干燥剂。

硅酸加热易分解：

\ce{H2SiO3 ->[\triangle] SiO2 + H2O}

硅的应用

物质应用
\ce{Si}半导体（芯片、太阳能电池）
\ce{SiO2}光导纤维、水晶、石英、玛瑙
\ce{H2SiO3}干燥剂、吸附剂
\ce{Na2SiO3}水玻璃、粘合剂、防火剂
\ce{SiC}（碳化硅）金刚砂，熔点高、硬度大、强度高，新型材料

三大硅酸盐产品：

  • 陶瓷：黏土。

  • 水泥：\ce{CaSiO3,CaO} 混合。

  • 玻璃：\ce{Na2CO3 + CaCO3 + SiO2 -> Na2SiO3 + CaSiO3 + CO2}。

本质上是两个反应：

\ce{Na2CO3 + SiO2 -> Na2SiO3 + CO2}

\ce{CaCO3 + SiO2 -> CaSiO3 + CO2}

玻璃的主要成分是 \ce{Na2SiO3} 和 \ce{CaSiO3}，以及没有反应完全的 \ce{SiO2}。人们经常向玻璃内添加 \ce{PbO}，以增加玻璃的折射率和密度，制成水晶玻璃；还也可以添加 \ce{B2O3}，以增加玻璃的耐热性，制成硼硅玻璃。添加 \ce{Fe2O3} 可以使玻璃呈绿色，添加 \ce{CoO} 可以使玻璃呈蓝色，添加 \ce{Cu2O} 可以使玻璃呈红色。

钢化玻璃：将玻璃加热到软化点以上，然后迅速冷却，形成表面压应力，增加强度和韧性。

工业应用与扩展

  • 光导纤维：高纯度的 \ce{SiO2} 是制造光导纤维的主要材料。

  • 氢氟酸刻蚀：\ce{SiO2} 性质稳定，不与除氢氟酸以外的其他酸反应：\ce{SiO2 + 4HF -> SiF4 ^ + 2H2O}，常用于玻璃刻蚀。

  • 硅胶：由硅酸缩水得到，具有多孔结构，常用作干燥剂及催化剂载体。

  • 锡与铅：\ce{Sn^2+} 具有强还原性；\ce{PbO2} 具有强氧化性，广泛用于铅酸蓄电池的电极材料。

贫金属、类金属与半金属

贫金属的相似性与特性

贫金属（Poor metals），又称后过渡金属（Post-transition metals），是指元素周期表中位于第13族至第17族的金属元素。它们处于金属与非金属的分界线（类金属区）左侧，其性质在宏观和微观上均表现出从典型金属向非金属过渡的特征。

相比过渡金属，贫金属的电负性较高，熔点和沸点较低，机械强度较差，并且也更软。贫金属不是一个严格的IUPAC承认的命名法，然而贫金属约定俗成地包括铝、镓、铟、锡、铊、铅和铋，有时也包括通常认为是类金属或“半金属”的锗、锑和钋，以及性质和其他过渡金属差异较大的12族元素（锌、镉、汞）。此处，我们将前十个，视为“贫金属”进行讲解（其实作者很懒，把这一部分钅子旁的都放到贫金属，其他都放到类金属了）。

贫金属的微观性质由其原子结构决定，反映了元素周期律的演变：

  1. 价电子构型： 贫金属的价层电子排布在 ns 和 np 轨道上（构型为 ns^2 np^{1-5}），且其内层已填满 (n-1)d 或 (n-2)f 轨道。由于其原子核对最外层电子的吸引力较强，它们的电离能和电负性显著高于主族活泼金属（如碱金属和碱土金属）。
  2. 原子半径递变：
    • 同周期： 从左向右，核电荷数增加，原子核对电子的引力增强，导致原子半径逐渐减小。
    • 同主族： 自上而下，电子层数增多，原子半径逐渐增大。
  3. 惰性电子对效应（Inert-pair effect）： 这是第六周期贫金属（如 Tl、Pb、Bi）最显著的微观特性。受相对论效应影响，6s 轨道发生显著收缩并能量降低，使得 6s² 电子变得“惰性”，难以参与成键。这导致这些元素低价态比高价态更稳定，如 Pb 通常显 +2 价而非 +4 价，Bi 显 +3 价而非 +5 价。
  4. 电负性： 贫金属的电负性通常在 1.5 至 2.0 之间。其电负性在同周期内随原子序数递增而增大，在同主族内随原子序数递增而减小。

贫金属在宏观上表现出与过渡金属不同的物理和化学特质：

  1. 物理性质的相似性：
    • 状态与硬度： 多数贫金属具有较低的熔点和较小的硬度。例如，镓（Ga）的熔点极低（29.8 °C），在手心中即可熔化；铅（Pb）和锡（Sn）质地较软。
    • 导电与延展性： 它们具有金属光泽、良好的导电性和延展性，但其导电性通常优于类金属而逊于贵金属。
  2. 化学性质的相似性——两性（Amphoterism）：
    • 许多贫金属（如 Al、Zn、Sn、Pb）及其氧化物、氢氧化物表现出显著的两性，既能溶于强酸又能溶于强碱。
    • 例如，铝（Al）及其氢氧化物既能产生 Al^{3+} 盐，也能生成 [Al(OH)_4]^-（四羟基合铝酸盐）。
  3. 合金化倾向： 贫金属常作为低熔点合金的主要组分，如焊锡（Pb-Sn 合金）和伍德合金（Bi-Pb-Sn-Cd 合金）。

贫金属的宏观性质随其在周期表中的位置呈现规律性变化：

  1. 金属性与非金属性的蜕变：
    • 在碳族中表现最为典型：C(非金属) \rightarrow Si、Ge(类金属) \rightarrow Sn、Pb(贫金属)。随着原子序数增加，金属性逐渐增强，非金属性逐渐减弱。
  2. 最高价氧化物水化物的酸碱性：
    • 同周期（自左向右）： 碱性减弱，酸性增强。如第三周期：NaOH(强碱) \rightarrow Mg(OH)_2(中强碱) \rightarrow Al(OH)_3(两性) \rightarrow H_4SiO_4(极弱酸) \rightarrow \cdots \rightarrow HClO_4(最强含氧酸)。
    • 同主族（自上而下）： 酸性减弱，碱性增强。如第14族氧化物：CO_2（酸性）\rightarrow GeO_2, SnO_2（两性）\rightarrow PbO_2（碱性为主，两性）。
  3. 熔沸点趋势：
    • 通常在金属晶体中，金属阳离子半径越小、电荷数越多，金属键越强，熔沸点越高。
    • 在第4、5、6周期中，贫金属的熔点随原子序数增加先升后降，钨（W）达到极大，至汞（Hg）降至最低。
  4. 氧化还原性：
    • 金属性越强，单质的还原性越强。
    • 由于镧系收缩及相对论效应，第六周期贫金属的高价态具有极强的强氧化性（如 PbO_2 和 NaBiO_3），而其对应的低价态则是更稳定的存在形式。

贫金属是元素周期表中从金属性向非金属性转变的桥梁。在微观上，它们受核电荷数增加和相对论效应（特别是惰性电子对效应）的共同影响；在宏观上，它们展现出低熔点、软质感以及显著的两性特征。理解其递变性有助于掌握复杂元素化学的系统规律。

镓、锗及其化合物

第四周期贫金属（后过渡金属）主要包括镓（Ga）和锗（Ge）。它们位于金属与非金属的分界线附近，展现出显著的从金属性向非金属性过渡的特征。

镓（Gallium, Ga）及其化合物：

  1. 原子结构与周期表位置：镓的原子序数为31，位于第四周期第13族（ⅢA族），价电子排布式为 4s^2 4p^1。它是铝（Al）下方的同族元素，由于其内层填满了3d^{10}电子，其原子半径缩小（受到d区收缩影响），导致其电离能和电负性与铝相比有细微变化。

  2. 单质的物理特性：神奇的“低熔点”。

    • 状态与熔点： 镓是一种银白色的软金属。它最显著的特性是极低的熔点（29.76 °C）和极高的沸点（2676 K），液态温度区间极大。
    • 结构解释： 固态镓中存在原子对结构，对内结合牢固而对外结合力较弱，熔化时仅需破坏较弱的力，故熔点低；但要变为气态原子则需破坏牢固的内部结合，故沸点高。

  3. 化学性质：两性与还原性。镓的化学性质与铝极为相似，具有显著的两性：

    • 与酸碱反应： 镓既能溶于稀盐酸生成 Ga^{3+}，也能溶于强碱溶液生成 [Ga(OH)_4]^-（或 GaO_2^-）并放出氢气。
    • 钝化： 常温下金属表面会形成致密的氧化层，使其在空气中相对稳定。

  4. 重要化合物及其特性。

    • 氧化镓（Ga_2O_3）与氢氧化镓（Ga(OH)_3）： 均为两性化合物。
      • 特性： 与铝不同，Ga(OH)_3 能溶于氨水，而 Al(OH)_3 不行，这说明 Ga(OH)_3 的酸性略强于 Al(OH)_3。
    • 氮化镓（GaN）与砷化镓（GaAs）：
      • GaAs： 是极其重要的半导体材料，晶体结构类似于金刚石，常通过单质在高压下化合制得。
      • GaN： 作为第三代半导体材料，具有高热容、高热导率和半导体性质，广泛用于LED和5G通讯。
    • 卤化物： 镓能形成 GaX_3。有趣的是，GaCl_2 并非真正的+2价，而是由 Ga^+ 和 [GaCl_4]^- 组成的混合价态离子化合物。
    • 有机镓化合物： 如三甲基镓 [Ga(CH_3)_3]。它是分子晶体，中心原子为 sp^2 杂化；当它与乙醚形成配合物时，杂化方式变为 sp^3。

锗（Germanium, Ge）及其化合物：

  1. 原子结构与周期表位置：锗的原子序数为32，位于第四周期第14族（ⅣA族），价电子排布式为 4s^2 4p^2。它处于金属与非金属分界线的正上方，常被归类为类金属（准金属）。

  2. 物理特性与晶体结构。

    • 外观： 灰白色有光泽的硬脆金属，晶体结构为金刚石型。
    • 半导体性质： 锗是第一代半导体材料，其导电性介于导体与绝缘体之间。通过掺杂（如掺入Ga形成p型，或掺入P形成n型）可制成p-n结，是微电子技术的基础。

  3. 化学性质：惰性与变价。

    • 稳定性： 室温下对水、氧稳定，加热到700 °C以上才与氧反应生成 GeO_2。
    • 溶解性： 不溶于稀酸，能溶于热浓硫酸、王水以及在氧化剂存在下的碱溶液。
    • 变价： 主要氧化态为 +4（较稳定）和 +2。

  4. 重要化合物及其特性

    • 二氧化锗（GeO_2）： 具有两性但酸性胜过碱性。它有两种晶型：六方晶体（可溶于水生成锗酸）和四方晶体（不溶于水）。
    • 四氯化锗（GeCl_4）： 无色液体，分子晶体。它在水中极易水解生成 GeO_2 \cdot nH_2O 沉淀。工业上利用其易挥发且易提纯的特点，通过精馏和还原反应生产高纯锗。
    • 锗酸盐： 在碱性条件下，锗能形成锗酸盐，如 Mg_2GeO_4 和 Zn_2GeO_4（后者是优良的光催化剂）。

从第四周期的镓、锗可以看出贫金属明显的递变性：

  1. 从左向右（同周期）：

    • 原子结构： 核电荷数增加，原子半径减小，得电子能力增强。
    • 金属性： 镓具有明显的金属性，而锗已呈现类金属特征，非金属性显著增强。
    • 酸碱性： 最高价氧化物水化物的碱性减弱，酸性增强。Ga(OH)_3 是典型的两性，而 Ge(OH)_4 已表现出酸性强于碱性的特征。

  2. 自上而下（同主族）：

    • 金属性： 相比上一周期的铝和硅，镓和锗的金属性更强。
    • 价态稳定性： 随着周期数增大，低价态（+1或+2）的稳定性逐渐上升，这源于惰性电子对效应的影响。

  3. 对角线规则：

    • 铍（Be）与铝（Al）、**铝（Al）与锗（Ge）**之间在半径和电负性上的相似性，使得它们在两性和共价性上具有一定的相似性。

镓和锗作为第四周期贫金属，是半导体工业的基石。镓以其极低熔点和两性著称，主要用于GaN和GaAs等先进电子材料；锗则以类金属特性和易水解的氯化物为特征，在红外光学和高纯半导体领域占有重要地位。

铟、锡、锑及其化合物

第五周期贫金属（后过渡金属）主要包括铟（In）、锡（Sn）和锑（Sb）。它们位于元素周期表 p 区左下方，处于金属与非金属的分界线附近，其物理和化学性质表现出明显的从典型金属性向类金属性过渡的特征。

铟（Indium, In）：

  1. 原子结构与物理特性。

    • 原子结构： 原子序数为 49，位于第 13 族（ⅢA 族），价电子构型为 5s^2 5p^1。
    • 物理性质： 铟是一种银白色的液态温度区间很大的金属，质地比铝还软，具有极佳的延展性。其熔点较低（156.6 °C），沸点却高达 2072 °C，这种差异与其固态中存在紧密结合的原子对结构有关。

  2. 化学性质与重要化合物。

    • 氧化态： 主要氧化态为 +3。虽然理论上存在 +1 价，但在水溶液中极不稳定。
    • 两性特征： 铟具有一定的两性，但金属性强于同族的铝。其氢氧化物 In(OH)_3 难溶于水，在 pH 约为 4.7 时完全沉淀。
    • 重要物质：
      • 氧化铟锡（ITO）： 是铟最重要的应用领域，作为透明导电膜广泛用于液晶显示器（LCD）和触摸屏。
      • 磷化铟（InP）： 一种高性能半导体材料，广泛应用于光纤通信技术中，其晶体结构类似于闪锌矿。
      • 卤化物： InCl_3 易水解；有趣的 InCl_2 实际上是混合价态化合物 In[InCl_4]，表现为反磁性。

锡（Tin, Sn）：

  1. 同素异形体与“锡瘟”。

    • 白锡（\beta-Sn）： 常温下稳定的银白色金属，属四方晶系，具有良好的延展性。
    • 灰锡（\alpha-Sn）： 当温度低于 13.2 °C 时，白锡会缓慢转变为粉末状的灰锡（属于共价晶体，金刚石型结构）。这种转变会导致体积膨胀，使锡制品损毁，现象被称为**“锡瘟”**。

  2. 核心化学性质：强还原性。

    • 锡有 +2 和 +4 两种氧化态。在溶液中，Sn^{2+} 具有强还原性，易被空气或氧化剂氧化为稳定的 Sn^{4+}。
    • 检验反应： 利用 Sn^{2+} 还原 HgCl_2 生成白色的 Hg_2Cl_2 沉淀，过量时进一步还原为黑色的金属汞（Hg），此反应常用于检验 Sn^{2+} 或 Hg^{2+}。

  3. 重要化合物及其应用。

    • 氯化锡： SnCl_2 是实验室常用还原剂。SnCl_4 在常温下是无色液体（分子晶体），在潮湿空气中强烈水解发烟，军事上可与氨气配合制造烟幕。
    • 氧化物： SnO_2（锡石）是自然界中锡的主要矿石，具有两性，可用作玻璃抛光剂。
    • 合金应用： 锡广泛用于制造焊锡（Sn-Pb 合金）、青铜（Cu-Sn 合金）以及马口铁（镀锡薄钢板）。

锑（Antimony, Sb）：

  1. 物理特性：金属与类金属的桥梁。

    • 原子序数为 51，位于第 15 族（ⅤA 族）。
    • 单质呈银白色，质脆，无延展性。锑具有**“热缩冷胀”**的独特性质，因此常用于铸造铅字（活字金），使字迹清晰。
    • 虽然具有金属光泽，但由于导电导热性较差且性质脆，常被归类为类金属（半金属）。

  2. 化学性质与化合物特征。

    • 氧化态： 常见为 +3 和 +5。
    • 两性氧化物： Sb_2O_3 是两性氧化物（碱性略强于酸性），工业上常用作阻燃剂。
    • 易水解性： SbCl_3 极易水解，生成白色的碱式氯化锑（SbOCl）沉淀。配制溶液时需加入浓盐酸以抑制水解。
    • 超强酸： 锑的五氟化物 SbF_5 是极强的路易斯酸，与 HF 结合可形成氟锑酸（HSbF_6）。这是目前已知酸性最强的超强酸之一，能质子化许多极弱的碱。

在第五周期中，从铟到锑，性质呈现明显的规律性变化：

  1. 金属性逐渐减弱，非金属性增强： 铟是典型金属，锡表现出明显的两性，而锑已展现出类金属特性。
  2. 氢化物稳定性下降： InH_3（极不稳定） \rightarrow SnH_4 \rightarrow SbH_3。随着原子序数增加，气态氢化物的热稳定性依次减弱。
  3. 氧化物酸碱性递变：
    • 同周期（左向右）： 碱性减弱，酸性增强。In(OH)_3（弱碱性/两性） \rightarrow Sn(OH)_2/Sn(OH)_4（典型两性） \rightarrow Sb(OH)_3/HSb(OH)_6（两性偏酸）。
    • 惰性电子对效应（Inert-pair effect）： 在第五周期开始显现。相比于上一周期的元素（如 Ga、Ge），第五周期的低价态（Sn^{2+}、Sb^{3+}）稳定性有所提升，但仍具有较强的还原性。这一趋势在第六周期（Pb^{2+}、Bi^{3+}）将变得更加显著，届时低价态将成为最稳定状态。

铊、铅、铋、钋及其化合物

第六周期贫金属（后过渡金属）主要包括铊（Tl）、铅（Pb）、铋（Bi）和钋（Po）。它们位于周期表 p 区的底部，其性质受镧系收缩和相对论效应（特别是惰性电子对效应）的强烈影响，展现出显著的特殊性。

铊（Thallium, Tl）：

  1. 原子结构与氧化态。

    • 位置与价电子： 原子序数 81，位于第 13 族（ⅢA 族），价电子构型为 6s^2 6p^1。
    • 惰性电子对效应： 受相对论效应影响，6s 轨道显著收缩且能量降低，导致 6s² 电子难以参与成键。因此，铊的 +1 氧化态比 +3 氧化态更稳定，其性质在某些方面类似于碱金属。

  2. 物理与化学特性。

    • 单质： 银白色软金属，在空气中表面易变暗。
    • 氢氧化物： TlOH 是强碱，极易溶解于水，这与同族上方 Al(OH)_3 的两性及难溶性形成鲜明对比。
    • 化合物： TlCl 等卤化物难溶于水（类似银盐）；Tl_2O_3 具有强氧化性且受热易分解。
    • 毒性： 铊及其化合物具有剧毒，曾被用作杀鼠剂。

铅（Lead, Pb）：

  1. 氧化态的稳定性递变。

    • 价态特征： 价电子构型为 6s^2 6p^2。主要氧化态为 +2（稳定）和 +4（强氧化性）。
    • 强氧化性： PbO_2 是强氧化剂，在酸性介质中能将 Mn^{2+} 氧化为紫红色的 MnO_4^-。

  2. 重要化合物及其转化。

    • 氧化物：
      • PbO（一氧化铅，俗称密陀僧/黄丹）：有红色和黄色两种晶型。
      • Pb_3O_4（四氧化三铅，俗称铅丹/红丹）：视作 2PbO \cdot PbO_2，常用作防锈底漆。
    • 难溶盐与应用：
      • PbSO_4（硫酸铅）：白色固体，难溶于水。在废铅酸蓄电池回收中，常通过加入 Na_2CO_3 将其转化为 PbCO_3。
      • Pb(CH_3COO)_2（醋酸铅，俗称铅糖）：是少数易溶于水的弱电解质铅盐，有毒。
    • 典型应用： 铅酸蓄电池，其正极为 PbO_2，负极为 Pb，电解质为 H_2SO_4。

铋（Bismuth, Bi）：

  1. 金属性的增强。

    • 位置： 原子序数 83，位于第 15 族（ⅤA 族），价电子构型为 6s^2 6p^3。
    • 物理特性： 银白色或微红色金属，质脆，导电导热性差。铋具有独特的热缩冷胀性。

  2. 化学性质与化合物

    • 极易水解： Bi^{3+} 在水中极易水解生成氧基铋离子（BiO^+），进而形成白色的 BiOCl（氯化氧铋）沉淀。配制铋盐溶液需加入强酸抑制水解。
    • 强氧化剂： **铋酸钠（NaBiO_3）**是极强的氧化剂（强于 PbO_2），常用于检验 Mn^{2+}。
    • 合金应用： 用于制造低熔点合金（如伍德合金，熔点仅 65-70 °C），常用作保险丝材料。

钋（Polonium, Po）放射性与基本特性：

  • 特性： 原子序数 84，位于第 16 族（ⅥA 族），电子构型为 6s^2 6p^4。由居里夫人发现，具有强放射性。
  • 金属性： 在氧族元素中，钋的金属性最强，能与水和稀酸反应生成氢气和 Po^{2+}。

第六周期贫金属的递变性总结：

  1. 微观递变性（相对论效应）： 随着原子序数从 Tl 增加到 Po，6s 轨道的“惰性”愈发显著。这导致该周期元素的特征氧化态比族序数低 2。例如，Tl 倾向于 +1 价而非 +3 价，Pb 倾向于 +2 价而非 +4 价。
  2. 宏观金属性递变：
    • 同周期（左向右）： 金属性逐渐减弱，非金属性增强。Tl 是典型金属，Pb 是重金属，Bi 已表现出某些脆性，而 Po 是强放射性金属。
    • 同主族（上向下）： 第六周期元素是各族中金属性最强的。例如 Bi(OH)_3 呈碱性，而同族的 As(OH)_3、Sb(OH)_3 具有显著两性。
  3. 高价态氧化性： 由于低价态的极端稳定，第六周期的最高价态化合物（如 PbO_2、NaBiO_3）均表现出极强的氧化性。
  4. 物理性质： 这些元素通常具有较大的密度和较低的熔点，易于形成低熔点合金。

类金属的相似性与特性

类金属（Metalloid），又称准金属或半金属，是指在元素周期表中位于金属与非金属分界线附近的元素。它们在微观结构和宏观性质上均处于过渡地带，兼具两者的特征。已确定为类金属的元素通常包括**硼（B）、硅（Si）、锗（Ge）、砷（As）、锑（Sb）、碲（Te）**等，有时也将硒（Se）、钋（Po）和砹（At）纳入其中。

本文粗略的将这些中石字旁的归为类金属，具体的说法见贫金属部分定义和说明。

类金属的微观本质由其在周期表中的位置决定，反映了原子结构从金属向非金属的量变到质变。

  1. 价电子构型： 类金属主要分布在 p 区，其价电子排布在 ns 和 np 轨道上。它们的价电子数通常介于典型金属（较少）与典型非金属（较多）之间。
  2. 电负性： 类金属的电负性通常在 2.0 左右。
    • 递变性： 同一周期内从左向右电负性增大，同一主族内自上而下电负性减小。
    • 意义： 电负性在 2.0 附近意味着这些原子对键合电子的吸引力中等，既不像活泼金属那样容易完全失去电子，也不像强非金属那样容易完全夺取电子，因此常形成共价键。
  3. 原子半径：
    • 趋势： 随原子序数增加，同周期内原子半径逐渐减小（核对电子引力增强），同主族内逐渐增大（电子层数增多）。
    • 影响： 中等的原子半径和核电荷数决定了类金属原子的第一电离能也处于中间水平，反映了其得失电子能力的“中庸”特性。
  4. 微观对称性与成键： 许多类金属（如 Si、Ge）遵循 8-N 规则，每个原子形成 8-N 个共价键以填满外壳层（N 为族序数）。这导致它们常形成复杂的一维、二维或三维大分子结构（如金刚石型结构的 Si、Ge）。

类金属在宏观物理和化学行为上展现出高度的相似性：

  1. 物理状态： 类金属在常温下均为固体，且大多具有金属光泽，但在机械性质上表现为硬而脆，缺乏金属的延展性。
  2. 半导体性质： 这是类金属最核心的物理特性。它们的导电性介于金属（导体）与非金属（绝缘体）之间。
    • 特性： 纯质导电性不强，但导电性随温度升高而上升（与金属相反），且通过 p型或 n型掺杂后导电度会显著增加。这使其成为微电子工业和计算机芯片的基础材料。
  3. 两性特征： 许多类金属及其氧化物、氢氧化物展现出两性。
    • 例如，硼和硅的最高价氧化物水化物（H_3BO_3、H_4SiO_4）虽然显弱酸性，但在特定条件下也能表现出极弱的碱性倾向。
    • 锗、锑等元素的氧化物具有显著的两性，既能溶于强酸也能溶于强碱。

类金属的性质随其在周期表中的位置移动呈现出显著的趋势：

  1. 金属性与非金属性的蜕变：
    • 同周期（左→右）： 金属性逐渐减弱，非金属性逐渐增强。例如在第三周期，Na（强金属）\rightarrow Mg（金属）\rightarrow Al（金属/两性）\rightarrow Si（类金属） \rightarrow P、S、Cl（非金属）。
    • 同主族（上→下）： 金属性逐渐增强，非金属性减弱。在第14族（碳族）表现最为典型：C（典型非金属）\rightarrow Si、Ge（类金属） \rightarrow Sn、Pb（金属）。
  2. 最高价氧化物对应水化物的酸碱性：
    • 规律： 同周期自左向右，酸性增强，碱性减弱；同主族自上而下，酸性减弱，碱性增强。
    • 过渡： 类金属处于酸碱性转变的转折点，如 Al(OH)_3 是典型的两性氢氧化物，而紧邻的 H_4SiO_4 则是弱酸。
  3. 氢化物的稳定性： 类金属的简单气态氢化物稳定性随原子序数增大（同周期向右、同主族向下）呈现递减趋势。例如，SiH_4 比 CH_4 不稳定，且类金属氢化物通常具有较强的还原性。
  4. 对角线相似规律： 位于周期表左上右下对角线位置的元素性质相似。
    • 硼（B）与硅（Si）： 它们均为类金属，单质均具有高熔点和硬度，氢化物均极易被氧化，且最高价氧化物水化物均为弱酸。

类金属是物质世界的“边际元素”，在微观上通过中等的电负性和共价成键方式，在宏观上通过半导体特性和脆性固体形态，完美连接了金属与非金属两个领域。它们的递变规律生动地诠释了元素周期律中**“结构决定性质”**的核心科学观念。

砷、碲、碲及其化合物

砷（As）、碲（Te）和砹（At）在元素周期表中均位于金属与非金属的分界线附近，常被称为类金属或半金属，在半导体材料、工业生产及医药领域具有重要地位。

砷（Arsenic, As）：砷位于元素周期表第四周期、第VA族（氮族），原子序数为33。

  1. 单质性质与同素异形体：

    • 物理性质：砷有三种主要的同素异形体：灰砷、黑砷和黄砷。
      • 灰砷是最稳定的形态，具有灰白色金属光泽，质脆，能传热导电。
      • 黄砷结构与白磷相似，表现出非金属性，易溶于 CS_2。
    • 化学活性：在干燥空气中稳定，但在潮湿空气中表面会生成黑色氧化膜。点燃或灼烧时能与氧气反应生成三价氧化物。

  2. 重要化合物：

    • 氧化物（砒霜）：
      • 三氧化二砷（As_2O_3）：俗称砒霜，白色霜状粉末，剧毒。它是两性氧化物但以酸性为主，溶于水生成亚砷酸（H_3AsO_3），溶于碱生成亚砷酸盐。
      • 五氧化二砷（As_2O_5）：砷的最高价氧化物，为酸性氧化物，其对应水化物为砷酸。
    • 含氧酸：
      • 砷酸（H_3AsO_4）：三元中强酸，酸性与磷酸相似。它具有氧化性，可将 I^- 氧化为 I_2。
    • 氢化物（胂）：
      • 砷化氢（AsH_3）：无色、有大蒜味的极毒气体。它极不稳定，受热易分解为单质砷，在玻璃管上形成亮黑色的“砷镜”，这是马氏（Marsh）试砷法的原理。
    • 硫化物：自然界中常以硫化物存在，如雌黄（As_2S_3，黄色）和雄黄（As_4S_4，红色）。

  3. 应用与检测：

    • 应用：用于制造合金（提高铅的硬度）、木材防腐剂和农药。**砷化镓（GaAs）**是极其重要的半导体材料。
    • 检测：除了马氏试砷法，还可以利用古氏（Gutzeit）试砷法（与 AgNO_3 反应）进行痕量检测。

碲（Tellurium, Te）：碲位于元素周期表第五周期、第VIA族（氧族），原子序数为52。

  1. 单质性质：

    • 物理性质：银白色固体，带有金属光泽，质脆。
    • 电学性质：碲是典型的半导体材料，且具有导电性。
    • 化学活性：不溶于水，但溶于浓硫酸、硝酸和王水。加热时能与氢气反应生成不稳定的碲化氢。

  2. 重要化合物：

    • 氢化物：碲化氢（H_2Te）。无色、有臭味且剧毒的气体。在氧族元素氢化物中，其还原性最强、热稳定性最差、水溶液酸性最强。
    • 氧化物与酸：
      • 二氧化碲（TeO_2）和三氧化碲（TeO_3）。
      • 碲酸通常以 H_6TeO_6 形式存在，是一种弱酸。
    • 盐类：碲化镉（CdTe）常用于制造薄膜太阳能电池。

  3. 工业用途：碲加入钢或铅中可显著提高合金的延展性和硬度。它也广泛用于彩色玻璃的着色剂（蓝、棕、红色）和橡胶的硫化剂。

砹（Astatine, At）：砹位于元素周期表第六周期、第VIIA族（卤族），原子序数为85。

  1. 基本特性：

    • 放射性与稀有性：砹是一种人工合成的放射性元素，地壳总量极少（不足1克），极不稳定。
    • 金属性：作为卤素中原子序数最大的元素，砹表现出明显的金属性。

  2. 性质预测（基于元素周期律）：由于其极高的放射性和稀缺性，中学阶段常通过周期律推测其性质。

    • 单质颜色：预测为深紫黑色或黑色固体。
    • 溶解性：砹难溶于水，易溶于四氯化碳等有机溶剂。
    • 化合物特性：
      • HAt：极不稳定，易分解，水溶液呈酸性。
      • AgAt：预测为不溶于水的固体，且具有极强的感光性，见光易分解。
    • 氧化态：已知存在 0, \pm1, +3, +5, +7 等多种价态。

  3. 研究与应用：由于半衰期极短（如 ^{211}At 仅为7.2小时），砹主要用于基础科学研究。

递变规律与共性：

  1. 金属性增强：在各自的族中，随着原子序数增加（N\rightarrow P\rightarrow As\rightarrow Sb\rightarrow Bi 以及 O\rightarrow S\rightarrow Se\rightarrow Te\rightarrow Po），元素的金属性逐渐增强，非金属性逐渐减弱。
  2. 氢化物趋势：AsH_3 和 H_2Te 等重元素氢化物的热稳定性依次减弱，还原性依次增强。
  3. 半导体特性：砷和碲由于处于金属/非金属交界处，其单质及化合物（如GaAs、CdTe）是现代电子工业的核心材料。

半金属与类金属的区别

在化学与材料科学中，中文术语“半金属”是一个极具争议且容易混淆的概念，因为它在不同的学术语境下分别对应着三个完全不同的英文术语：Metalloid（类金属）、Semimetal（物理学意义上的半金属）以及 Half-metal（自旋电子学中的半金属）。

类金属（Metalloid / 准金属 / 常用作“半金属”）是中学及大学基础化学中最常见的定义。它指代在元素周期表中处于金属与非金属分界线（粗折线）附近，兼具两类元素性质的元素。

  • 确定的成员：硼（B）、硅（Si）、锗（Ge）、砷（As）、锑（Sb）、碲（Te）。
  • 争议成员：硒（Se）、钋（Po）、砹（At）有时也被归入此类。
  • 特征：外观具有金属光泽，但质地较脆；化学性质上既能表现出一定的金属性（如形成某些阳离子），也具有明显的非金属性（如形成共价网状晶体）。

半金属（Semimetal / 物理定义）是凝聚态物理学根据能带理论给出的定义。它描述的是一种特殊的能带结构：导带底和价带顶在能量上发生微小重叠。

  • 典型成员：砷（As）、锑（Sb）、铋（Bi）、石墨（C的一种同素异形体）、灰锡（α-Sn）。
  • 特征：由于能带重叠，它们在绝对零度时也有载流子，表现出弱导电性，但这与真正的金属（能带大幅重叠）或半导体（具有能隙/禁带）有本质区别。

半极性金属（Half-metal / 自旋电子学定义）是一个更前沿的概念。它指代一类磁性材料，其对不同自旋方向的电子展现出截然不同的传导性。

  • 特征：对于一种自旋方向的电子，它表现为导电的“金属”；而对于另一种自旋方向的电子，它表现为不导电的“半导体或绝缘体”。这使得电流具有100%的自旋极化率。为了避免与上述两个概念混淆，学术界常建议译为“半极性金属”。

类金属（Metalloid）与 半金属（Semimetal）的区别：虽然砷（As）和锑（Sb）同时属于这两个分类，但“类金属”是一个化学/周期律分类，而“物理半金属”是一个能带结构分类。

维度类金属 (Metalloid)半金属 (Semimetal)
侧重点化学性质（两性、电负性、成键方式）导电物理机制（导带与价带重叠程度）
判定标准周期表位置、电负性约在2.0左右费米能阶处的电子态密度（DOS）是否非零
半导体特性通常是半导体（具有禁带 E_g）不是半导体（禁带为负值或重叠）
典型代表B, Si, Ge, As, Sb, TeAs, Sb, Bi, 石墨, 灰锡
  • 区别实例：
    • 硅（Si）和锗（Ge）：它们是典型的类金属，在化学上处于交界处；但在物理上它们是半导体，因为它们具有明确的禁带（禁带宽度 E_g > 0）。
    • 铋（Bi）和石墨：物理上它们是标准的半金属（能带微重叠）；但在化学分类中，铋通常被视为典型的金属，石墨则被视为非金属。

结合砷、碲、砹的具体分析：

  1. 砷（As）与锑（Sb）：双重身份。砷和锑既是化学上的类金属，也是物理上的半金属。

    • 在中学化学中，它们被归为“类金属”，强调其具有金属光泽但质脆、其氧化物显两性等化学特征。
    • 在物理实验中，它们的电子结构显示导带和价带在动量空间的不同位置发生了重叠，因此是研究“半金属”性质的理想材料。

  2. 碲（Te）：典型的类金属半导体。碲在所有来源中都被坚定地划分为类金属。

    • 碲表现出明显的半导体性质（如光电导效应），而非“物理半金属”的带隙重叠。
    • 碲化汞（HgTe）等化合物在特定条件下可展现出更复杂的能带性质，常被用于研究拓扑绝缘体等前沿物理。

  3. 砹（At）：被“预言”的类金属。砹由于其极高的放射性，在地壳中总量极少，难以获得宏观单质进行能带测量。

    • 周期律预测：它是卤族元素（非金属）中原子序数最大的一员，基于其位置，它应该表现出最强的金属性。
    • 分类尴尬：部分来源将其标为非金属，但在研究“类金属三角区”时，砹往往被认为位于分界线上，是潜在的类金属。

为了精确表达，在讨论化学周期律和半导体材料（如Si, Ge, Te）时，应优先使用**“类金属” (Metalloid)；在讨论特定能带结构（如石墨, Bi）时，使用物理定义的“半金属” (Semimetal)；而涉及电子自旋存储等尖端技术时，则应指明为“半极性金属” (Half-metal)**。
