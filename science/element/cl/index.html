卤素性质概述

卤素物理性质

卤族元素普遍在 CClX4\ce{CCl4}CClX4​ 中溶解度大于水，相似相容。

氯气的物理性质：

  • 黄绿色气体。
  • 有刺激性气味。
  • 密度大于水。
  • 可溶于水。
  • 有毒。

卤素化学性质

能够在水溶液中发生置换反应：

ClX2+2 KI→IX2+2 KCl \ce{Cl2 + 2KI -> I2 + 2KCl} ClX2​+2KI​IX2​+2KCl

特殊的，氟气能氧化水：

2 FX2+2 HX2O→4 HF+OX2 \ce{2F2 + 2H2O -> 4HF + O2} 2FX2​+2HX2​O​4HF+OX2​

最高价含氧酸酸性递减，高氯酸 HClOX4\ce{HClO4}HClOX4​ 为最强含氧酸。

氯气的化学性质

与金属单质反应

  • 氯气和钠单质：

    • 剧烈燃烧，产生黄色火焰，有白烟生成。

    • 2 Na+ClX2→点燃2 NaCl\ce{2Na + Cl2 ->[点燃] 2NaCl}2Na+ClX2​点燃​2NaCl。

  • 氯气和铁单质：

    • 剧烈燃烧，产生棕红色烟，水溶液呈黄色。

    • 2 Fe+3 ClX2→点燃2 FeClX3\ce{2Fe + 3Cl2 ->[点燃] 2FeCl3}2Fe+3ClX2​点燃​2FeClX3​。

  • 氯气和铜单质：

    • 剧烈燃烧，产生棕黄色烟，加水后溶液逐渐由黄色变为黄绿色、蓝绿色、蓝色。

    • Cu+ClX2→点燃CuClX2\ce{Cu + Cl2 ->[点燃] CuCl2}Cu+ClX2​点燃​CuClX2​。

与非金属单质反应

  • 氯气和氢气：

    • 安静燃烧，产生苍白色火焰，瓶口有白雾生成。

    • HX2+ClX2→点燃2 HCl\ce{H2 + Cl2 ->[点燃] 2HCl}HX2​+ClX2​点燃​2HCl。

    • 氢气与氯气混合后，强光或加热，引起爆炸。

氯气的反应及制取

  • 氯气和水：

    • 产生盐酸和次氯酸，其中次氯酸为弱酸。

    • ClX2+HX2O⇌HCl+HClO\ce{Cl2 + H2O <=> HCl + HClO}ClX2​+HX2​O​HCl+HClO。

  • 氯气和碱的反应：

    • 常温或冰水浴中，产生次氯酸；热水浴产生氯酸根。

    • 冰水：ClX2+2 OHX−→ClX−+ClOX−+HX2O\ce{Cl2 + 2OH- -> Cl- + ClO- + H2O}ClX2​+2OHX−​ClX−+ClOX−+HX2​O。

    • 热水：3 ClX2+6 OHX−→5 ClX−+ClOX3X−+3 HX2O\ce{3Cl2 + 6OH- -> 5Cl- + ClO3- + 3H2O}3ClX2​+6OHX−​5ClX−+ClOX3​X−+3HX2​O。

    • 注意：溴单质常温即生成溴酸根。

  • 制取：

    • 原理：氧化 ClX−\ce{Cl-}ClX− 离子。

    • 常用氧化剂：MnOX2, KMnOX4, KClOX3, NaClO\ce{MnO2,KMnO4,KClO3,NaClO}MnOX2​,KMnOX4​,KClOX3​,NaClO 或通电。

  • 验满：

    • 将湿润的淀粉碘化钾试纸放在瓶口，变蓝。

    • ClX2+2 IX−→IX2+2 ClX−\ce{Cl2 + 2I- -> I2 + 2Cl-}ClX2​+2IX−​IX2​+2ClX−。

  • 尾气处理：NaOH\ce{NaOH}NaOH 溶液。

氯水和漂白剂

氯水的组成

一般认为，氯水由三种分子、四种离子组成：

  • 三分：ClX2, HX2O, HClO\ce{Cl2,H2O,HClO}ClX2​,HX2​O,HClO。

  • 四离：HX+, ClX−, OHX−, ClOX−\ce{H+,Cl-,OH-,ClO-}HX+,ClX−,OHX−,ClOX−。

氯水成分的检验：

  • ClX2\ce{Cl2}ClX2​：观察颜色。

  • ClX−\ce{Cl-}ClX−：加入硝酸酸化的硝酸银溶液，产生白色沉淀。

  • HX+\ce{H+}HX+：加入 NaHCOX3\ce{NaHCO3}NaHCOX3​ 固体，产生使澄清石灰水变浑浊的气体。

  • HClO\ce{HClO}HClO：使有色布条褪色，加热产生气泡。

  • 将新制氯水加入石蕊试液，石蕊先变红后褪色（酸性、漂白性）。

在参与反应的时候，认为是氯气分子 ClX2\ce{Cl2}ClX2​ 参与即可。

漂白粉的原理

成分：

  • 主要成分：CaClX2, Ca(ClO)X2\ce{CaCl2,Ca(ClO)2}CaClX2​,Ca(ClO)X2​。
  • 有效成分：Ca(ClO)X2\ce{Ca(ClO)2}Ca(ClO)X2​。

制取方法：

  • 将氯气通入石灰乳中。
  • 2 Ca(OH)X2+2 ClX2→CaClX2+Ca(ClO)X2+2 HX2O\ce{2Ca(OH)2 + 2Cl2 -> CaCl2 + Ca(ClO)2 + 2H2O}2Ca(OH)X2​+2ClX2​​CaClX2​+Ca(ClO)X2​+2HX2​O。

漂白原理：

  • 生效原理：

    • Ca(ClO)X2+HX2O+COX2→2 HClO+CaCOX3↓\ce{Ca(ClO)2 + H2O + CO2 -> 2HClO + CaCO3 v}Ca(ClO)X2​+HX2​O+COX2​​2HClO+CaCOX3​↓。

  • 失效原理：

    • Ca(ClO)X2+HX2O+COX2→2 HClO+CaCOX3↓\ce{Ca(ClO)2 + H2O + CO2 -> 2HClO + CaCO3 v}Ca(ClO)X2​+HX2​O+COX2​​2HClO+CaCOX3​↓。

    • 2 HClO→光2 HCl+OX2↑\ce{2HClO ->[光] 2HCl + O2 ^}2HClO光​2HCl+OX2​↑。

漂白液的原理

成分：

  • 主要成分：NaCl, NaClO\ce{NaCl,NaClO}NaCl,NaClO。
  • 有效成分：NaClO\ce{NaClO}NaClO。

制取方法：

  • 将氯气通入氢氧化钠溶液中。
  • 2 NaOH+ClX2→NaCl+NaClO+HX2O\ce{2NaOH + Cl2 -> NaCl + NaClO + H2O}2NaOH+ClX2​​NaCl+NaClO+HX2​O。

漂白原理：

  • 生效原理：

    • NaClO+HX2O+COX2→HClO+NaHCOX3\ce{NaClO + H2O + CO2 -> HClO + NaHCO3}NaClO+HX2​O+COX2​​HClO+NaHCOX3​。

  • 失效原理：

    • NaClO+HX2O+COX2→HClO+NaHCOX3\ce{NaClO + H2O + CO2 -> HClO + NaHCO3}NaClO+HX2​O+COX2​​HClO+NaHCOX3​。

    • 2 HClO→光2 HCl+OX2↑\ce{2HClO ->[光] 2HCl + O2 ^}2HClO光​2HCl+OX2​↑。

次氯酸概述

  1. 弱酸性

    • HClO⇌HX++ClOX−\ce{HClO <=> H+ + ClO-}HClO​HX++ClOX−。

    • 其酸性弱于碳酸。

  2. 强氧化性

    • 含 +1+1+1 价的氯。

    • 用途：杀菌消毒、漂白。

  3. 不稳定性

    • 2 HClO→△/光2 HCl+OX2↑\ce{2HClO ->[\triangle/光] 2HCl + O2 ^}2HClO△/光​2HCl+OX2​↑。

    • 因此新制氯水久置会变为稀盐酸，应当放置在棕色瓶内、遮光阴凉处。

一般情况下，不需要考虑次氯酸氧化性对氯气氧化性检验的干扰。

溴和碘的概述

碘的化学性质

海带提碘

海带自带富集碘的作用，海带中碘含量是海水的 10410^4104 倍作用。

  1. 将坩埚放在泥三角上，用酒精灯灼烧海带，待冷却后加水，过滤，可以得到含有 IX−\ce{I-}IX− 的溶液。

  2. 用氯气氧化碘离子，得到碘单质。注意，氯气过量会将碘单质继续氧化为碘酸根，但是因为氯气价格低廉、无需酸性环境即可氧化，更常用氯气作为氧化剂。

  3. 使用四氯化碳作为萃取剂，萃取碘单质，分液得到碘单质的四氯化碳溶液。此时若直接蒸馏，分离效果不好，如果一定要蒸馏的话，可以使用减压蒸馏或用乙醚作为萃取剂。这里采用反萃取的方式。

  4. 将碘的四氯化碳溶液加入氢氧化钠溶液中，充分震荡、静置、分液，得到碘离子和次碘酸根、碘酸根等。在加入足量氯化氢，归中产生碘单质，过滤。这个步骤称为反萃取。

海水提溴

海水中也有不少的溴，我们同样先用氯气氧化，然后用热空气吹出得到含有溴单质（液态溴易挥发）的空气，与二氧化硫混合通入水中，得到富含溴离子和硫酸根的水，然后继续用氯气氧化、热空气吹出，就可以得到富含溴单质的蒸汽，冷却后即可得到液态溴。
