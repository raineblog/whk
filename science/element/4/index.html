氮族活泼非金属

氮的基本性质

氮气及其性质

氮气（\text{N}_2）是大气中含量最丰富的气体，约占空气总体积的 78%。作为氮族元素（第 15 族）的首位元素，氮气在自然界循环和工业生产中具有举足轻重的地位。

氮气分子的结构特征是其化学性质高度稳定的根本原因：

  • 成键方式：氮原子（1s^2 2s^2 2p^3）有 3 个未成对电子。两个氮原子通过共用三对电子结合，形成 共价三键（\text{N}\equiv\text{N}）。
  • 轨道重叠：三键由一个 \sigma 键（原子轨道“头碰头”重叠）和两个 \pi 键（原子轨道“肩并肩”重叠）组成。
  • 高键能与稳定性：氮氮三键的键能极高（约为 941~946 kJ/mol），断开此键需要巨大的能量。因此，氮气在常温下表现出极强的 化学惰性，难以与其他物质反应。

物理性质：

  • 基本特征：纯净的氮气是 无色、无味、无嗅 的气体。
  • 密度与溶解性：密度（1.25 g/L）略小于空气，难溶于水。
  • 相变点：在常压下，沸点约为 -196^\circ\text{C}，熔点约为 -210^\circ\text{C}。液氮常作为低温实验或医学手术的冷冻剂。

尽管氮气性质稳定，但在高温、高压、放电或催化剂等特定条件下，仍可发生反应：

  1. 与非金属反应
    • 还原性（与氧气反应）：在 放电或高温 条件下，氮气能与氧气化合生成一氧化氮。
      • \text{N}_2 + \text{O}_2 \xrightarrow{\text{放电/高温}} 2\text{NO}
      • 此反应是“雷雨发庄稼”过程的第一步。
    • 氧化性（与氢气反应）：即工业合成氨（哈伯法），是人工固氮的核心。
      • \text{N}_2 + 3\text{H}_2 \xrightleftharpoons[\text{催化剂}]{\text{高温、高压}} 2\text{NH}_3
  2. 与金属反应（表现氧化性）
    • 与镁反应：点燃的镁条能在氮气中燃烧，生成 淡黄色（或黄绿色）的氮化镁固体。
      • 3\text{Mg} + \text{N}_2 \xrightarrow{\text{点燃}} \text{Mg}_3\text{N}_2
      • \text{Mg}_3\text{N}_2 遇水会发生剧烈水解，产生氨气：\text{Mg}_3\text{N}_2 + 6\text{H}_2\text{O} = 3\text{Mg}(\text{OH})_2 + 2\text{NH}_3 \uparrow。
    • 与锂反应：锂是极少数在 常温 下能与氮气缓慢反应的金属。
      • 6\text{Li} + \text{N}_2 = 2\text{Li}_3\text{N}

实验室制备方法：

  1. 最常用法：加热 饱和氯化铵和亚硝酸钠 溶液。
    • \text{NH}_4\text{Cl} + \text{NaNO}_2 \xrightarrow{\Delta} \text{NaCl} + \text{N}_2 \uparrow + 2\text{H}_2\text{O}
  2. 超纯氮制备：加热 叠氮化钠（\text{NaN}_3）分解（此原理亦用于汽车安全气囊）。
    • 2\text{NaN}_3 \xrightarrow{\Delta} 2\text{Na} + 3\text{N}_2 \uparrow
  3. 氨气还原法：用氨气还原灼热的氧化铜。
    • 2\text{NH}_3 + 3\text{CuO} \xrightarrow{\Delta} 3\text{Cu} + \text{N}_2 + 3\text{H}_2\text{O}

主要用途：

  • 保护气：利用其化学不活泼性，用于金属焊接保护、食品充氮防腐、灯泡填充气防止钨丝氧化。
  • 工业原料：制造氨气、硝酸、化肥、炸药及合成材料。
  • 科研医疗：液氮作为超导材料的低温环境及医疗冷冻麻醉。

氮的氧化物及其性质

氮的氧化物种类繁多，涵盖了从 +1 到 +5 的所有正价态。这些化合物在工业生产、环境保护以及生命科学中都具有极其重要的地位。

氮的主要氧化物包括以下六种，其中氮元素的化合价随氧化程度增加而升高：

  1. 一氧化二氮 (\text{N}_2\text{O})：+1 价
  2. 一氧化氮 (\text{NO})：+2 价
  3. 三氧化二氮 (\text{N}_2\text{O}_3)：+3 价（亚硝酸酐）
  4. 二氧化氮 (\text{NO}_2)：+4 价
  5. 四氧化二氮 (\text{N}_2\text{O}_4)：+4 价（\text{NO}_2 的二聚体）
  6. 五氧化二氮 (\text{N}_2\text{O}_5)：+5 价（硝酸酐）

重点氧化物详解：

  1. 一氧化氮 (\text{NO})
    • 物理性质：无色、无味、有毒的气体，密度比空气略大，不溶于水。
    • 化学性质：
      • 极易氧化：在常温下即可与空气中的氧气反应生成红棕色的二氧化氮：2\text{NO} + \text{O}_2 = 2\text{NO}_2。
      • 毒性原理：\text{NO} 与血红蛋白的结合能力比 \text{CO} 还强，容易引起人体缺氧窒息。
      • 生物学作用：在人体内，极少量的 \text{NO} 是重要的“信使分子”，具有扩张血管、提高免疫力和增强记忆力的功能。
    • 实验室制法：通过铜与稀硝酸反应制取：3\text{Cu} + 8\text{HNO}_3(稀) = 3\text{Cu}(\text{NO}_3)_2 + 2\text{NO}\uparrow + 4\text{H}_2\text{O}。收集时必须使用排水法。
  2. 二氧化氮 (\text{NO}_2)
    • 物理性质：红棕色、有刺激性气味、有毒的气体，密度比空气大，易液化，易溶于水。
    • 化学性质：
      • 与水反应：\text{NO}_2 溶于水发生歧化反应，生成硝酸和一氧化氮：3\text{NO}_2 + \text{H}_2\text{O} = 2\text{HNO}_3 + \text{NO}。
      • 与碱反应：可被烧碱溶液吸收生成硝酸盐和亚硝酸盐：2\text{NO}_2 + 2\text{NaOH} = \text{NaNO}_3 + \text{NaNO}_2 + \text{H}_2\text{O}。
      • 可逆平衡：\text{NO}_2 存在二聚反应平衡，2\text{NO}_2 \rightleftharpoons \text{N}_2\text{O}_4（\text{NO}_2 为红棕色，\text{N}_2\text{O}_4 为无色，该反应为放热反应）。升高温度或降低压强，颜色会加深。
    • 实验室制法：通过铜与浓硝酸反应制取：\text{Cu} + 4\text{HNO}_3(浓) = \text{Cu}(\text{NO}_3)_2 + 2\text{NO}_2\uparrow + 2\text{H}_2\text{O}。由于其易溶于水且与水反应，只能使用向上排空气法收集。
  3. 其他氧化物
    • 一氧化二氮 (\text{N}_2\text{O})：俗称“笑气”，有甜味，可用作麻醉剂。常温下较稳定，但在高温下可助燃。
    • 三氧化二氮 (\text{N}_2\text{O}_3)：亚硝酸的酸酐，在低温下为深蓝色液体或固体，常温下极易分解为 \text{NO} 和 \text{NO}_2。
    • 五氧化二氮 (\text{N}_2\text{O}_5)：硝酸的酸酐，无色固体，极易潮解，不稳定，受热易分解放出氧气。在固态时其结构为离子晶体 [\text{NO}_2]^+[\text{NO}_3]^-。

氮氧化物（通常统称为 \text{NO}_x）是主要的大气污染物之一。

  • 危害：
    • 酸雨：形成硝酸型酸雨。
    • 光化学烟雾：\text{NO}_x 与碳氢化合物在紫外线照射下发生反应，产生臭氧和 \text{PAN} 等有毒物质。
    • 臭氧层破坏：\text{NO}_2 可与平流层中的臭氧发生作用，导致臭氧层损耗。
  • 尾气处理方法：碱液吸收法，利用 \text{NaOH} 溶液吸收，通常需保证 n(\text{NO}_2) \geq n(\text{NO}) 才能被完全吸收。

氮氧化物的实验室制备

硝酸与硝酸盐

硝酸（\text{HNO}_3）是氮元素最高价（+5价）含氧酸，具有酸的通性，更以其突出的不稳定性和强氧化性著称。以下结合来源，对硝酸及其盐的性质、与金属反应的现象及规律进行详细展开。

  1. 物理性质：纯硝酸是无色、易挥发、有刺激性气味的液体。市售浓硝酸质量分数通常约为 68%～70%。质量分数在 95% 以上的硝酸在空气中挥发出蒸气，与水蒸气结合成微小液滴，产生“发烟”现象，称为发烟硝酸。
  2. 酸的通性：硝酸是一元强酸，在水中完全电离。它能使紫色石蕊试液变红（浓硝酸会使其先变红后褪色），并能与碱性氧化物、碱及弱酸盐发生复分解反应。

硝酸的化学特性：

  1. 不稳定性：硝酸见光或受热易分解，生成 \text{NO}_2、\text{O}_2 和 \text{H}_2\text{O}。
    • 化学方程式：4\text{HNO}_3 \xlongequal{光照或\Delta} 4\text{NO}_2\uparrow + \text{O}_2\uparrow + 2\text{H}_2\text{O}。
    • 现象：分解生成的 \text{NO}_2 溶于硝酸中，使久置的浓硝酸显黄色。实验室需用棕色瓶密封并置于阴凉处保存。
  2. 强氧化性：硝酸无论浓、稀均具有强氧化性，其氧化性随浓度增大而增强。
    • 非金属反应：加热条件下，浓硝酸能氧化木炭、硫、磷等非金属单质至最高价态，自身通常被还原为红棕色的 \text{NO}_2。
      • 例（与木炭）：\text{C} + 4\text{HNO}_3(浓) \xlongequal{\Delta} \text{CO}_2\uparrow + 4\text{NO}_2\uparrow + 2\text{H}_2\text{O}。

硝酸与金属反应时，既表现出强氧化性又表现出酸性。其核心特征是：反应通常不产生氢气。

  1. 还原产物的浓度规律：硝酸的还原产物十分复杂，主要取决于酸的浓度。
    • 浓硝酸：还原产物主要为红棕色的 \text{NO}_2。
      • Cu 与浓硝酸：反应剧烈，产生红棕色气体，溶液变蓝。
    • 稀硝酸：还原产物主要为无色的 \text{NO}。
      • Cu 与稀硝酸：反应较缓慢，产生无色气泡（在瓶口遇空气变为红棕色），溶液变蓝。
    • 极稀硝酸：与活泼金属（如 Mg、Zn）反应时，产物可能是 \text{N}_2\text{O}、\text{N}_2 甚至 \text{NH}_4^+。
  2. 特殊的“钝化”现象：常温下，铁（Fe）、铝（Al）和铬（Cr）遇到浓硝酸（或浓硫酸）时，表面会生成一层致密的氧化物保护膜。
    • 实质：这是一种化学变化，生成的膜阻止了内部金属与酸的进一步反应。
    • 应用：常温下可用铁质或铝质容器盛装、运输浓硝酸。但在加热条件下，钝化膜会被破坏，反应将剧烈进行。
  3. 与不同金属的反应性：
    • 不活泼金属：Cu、Ag、Hg 等能溶于硝酸，产生相应的硝酸盐和氮氧化物。
    • 贵金属（Au、Pt）：不溶于单一的硝酸，但可溶于王水（浓硝酸与浓盐酸体积比 1:3 的混合液）。
    • 反应计算模型：
      • N 原子守恒：n(\text{HNO}_3)_{参加} = n(\text{还原产物}) + n(\text{NO}_3^-)_{盐}。
      • 得失电子守恒：金属失去电子总数等于硝酸中氮原子得到电子总数。

硝酸盐的性质与热分解规律：

  1. 物理性质：硝酸盐大多数为无色晶体，均易溶于水。
  2. 热稳定性与分解规律：硝酸盐在高温下不稳定，受热分解产物随金属阳离子的活动性不同而异：
    • 活泼金属（如 K、Na、Ca 之前）：分解生成亚硝酸盐和氧气。
      • 例：2\text{NaNO}_3 \xlongequal{\Delta} 2\text{NaNO}_2 + \text{O}_2\uparrow。
    • 较活泼金属（如 Mg 到 Cu 之间）：分解生成金属氧化物、二氧化氮和氧气。
      • 例：2\text{Cu(NO}_3)_2 \xlongequal{\Delta} 2\text{CuO} + 4\text{NO}_2\uparrow + \text{O}_2\uparrow。
    • 不活泼金属（如 Hg、Ag 之后）：分解生成金属单质、二氧化氮和氧气。
      • 例：2\text{AgNO}_3 \xlongequal{\Delta} 2\text{Ag} + 2\text{NO}_2\uparrow + \text{O}_2\uparrow。
  3. 氧化性表现：固态硝酸盐受热是强氧化剂（如黑火药成分之一）。在水溶液中，\text{NO}_3^- 本身几乎无氧化性，但若加入强酸使溶液显酸性，则表现出与硝酸类似的强氧化性。

亚硝酸与亚硝酸盐

亚硝酸（\text{HNO}_2）与亚硝酸盐（\text{NO}_2^-）是氮元素处于中间价态（+3价）的重要化合物，在化学合成、食品工业及医学领域均有广泛应用。

亚硝酸（\text{HNO}_2）：

  1. 物理性质与稳定性
    • 存在形式：亚硝酸是一种弱酸（K_a \approx 5 \times 10^{-4}），它极不稳定，仅存在于稀的水溶液中。
    • 不稳定性：亚硝酸在冷水中即可发生歧化反应，生成硝酸、一氧化氮和水：3\text{HNO}_2 = \text{HNO}_3 + 2\text{NO}\uparrow + \text{H}_2\text{O}。微热时分解更剧烈。
    • 酸酐：亚硝酸的酸酐是三氧化二氮（\text{N}_2\text{O}_3），一种在低温下呈深蓝色的液体。
  2. 氧化性与还原性：亚硝酸中氮元素处于中间价态，既有氧化性又有还原性，但以氧化性为主。
    • 氧化性：在酸性条件下，它能将碘离子（\text{I}^-）氧化为单质碘（用于检验）：2\text{HNO}_2 + 2\text{I}^- + 2\text{H}^+ = \text{I}_2 + 2\text{NO}\uparrow + 2\text{H}_2\text{O}。也能将 \text{Fe}^{2+} 氧化为 \text{Fe}^{3+}。
    • 还原性：遇到强氧化剂（如 \text{KMnO}_4、\text{O}_2）时表现出还原性，被氧化为硝酸。
  3. 制备方法：
    • 混合气体法：将 \text{NO} 和 \text{NO}_2 混合物溶解在接近零度的水中：\text{NO} + \text{NO}_2 + \text{H}_2\text{O} = 2\text{HNO}_2。
    • 酸化法：向亚硝酸盐溶液中加入强酸：\text{NaNO}_2 + \text{HCl} = \text{NaCl} + \text{HNO}_2。

亚硝酸盐（以亚硝酸钠 \text{NaNO}_2 为代表）：

  1. 物理性质与结构
    • 外观：亚硝酸钠为无色或微黄色晶体，外观与口感均与食盐（\text{NaCl}）相似，因此常发生误食中毒事故。
    • 溶解性：易溶于水，水溶液呈碱性（因 \text{NO}_2^- 水解）。
    • 微观结构：\text{NO}_2^- 离子中心氮原子采取 sp^2 杂化，空间构型为 V形。
  2. 化学性质
    • 热稳定性：比亚硝酸稳定。碱金属的亚硝酸盐加热至熔融不分解。通常由活泼金属硝酸盐加热分解制得：2\text{NaNO}_3 \xrightarrow{\Delta} 2\text{NaNO}_2 + \text{O}_2\uparrow。
    • 氧化还原反应：
      • 氧化性：酸性条件下氧化 \text{KI}（淀粉变蓝）和 \text{FeCl}_2（溶液变黄）。
      • 还原性：能使酸性 \text{KMnO}_4 溶液或 \text{K_2Cr_2O_7} 溶液褪色/变色：\text{Cr}_2\text{O}_7^{2-} + 3\text{NO}_2^- + 8\text{H}^+ = 2\text{Cr}^{3+} + 3\text{NO}_3^- + 4\text{H}_2\text{O}。
  3. 鉴别 \text{NaNO}_2 与 \text{NaCl}
    • 溶液酸碱性：\text{NaNO}_2 溶液呈碱性，\text{NaCl} 呈中性。
    • 沉淀法：加入 \text{AgNO}_3 溶液，均产生沉淀，但 \text{AgNO}_2 沉淀可溶于稀硝酸，\text{AgCl} 不溶。
    • 氧化还原法：加入酸化的 \text{KI} 溶液，使淀粉变蓝的是 \text{NaNO}_2。

有机化学中的转化：

  1. 重氮化反应（Diazotization）：这是芳香族合成的核心反应。在 0 \sim 5^\circ\text{C} 下，芳香族伯胺与亚硝酸（由 \text{NaNO}_2 + \text{HCl} 现场产生）反应生成重氮盐。重氮盐（\text{Ar-N}_2^+）是极佳的合成中间体，可转化为酚（水解）、卤代烃（Sandmeyer 反应）或偶氮染料。

  2. 亚硝胺的形成：仲胺与亚硝酸反应生成 N-亚硝基化合物（亚硝胺，R_2\text{N-NO}），此类物质是世界公认的强致癌物。

毒性、用途与环境保护：亚硝酸盐有毒，它能将人体血红蛋白中的 \text{Fe}^{2+} 氧化为 \text{Fe}^{3+}，使其失去与氧气结合的能力，导致组织缺氧（高铁血红蛋白症）。

临床解毒：

  • 维生素 C：作为强还原剂，可直接还原高铁血红蛋白，缓解中毒。
  • 亚甲蓝（美蓝）：作为电子传递体，在体内酶的作用下加速血红蛋白的还原。

用途：

  • 食品添加剂：作为抗氧化剂、防腐剂（抑制细菌生长）和增色剂（使肉类颜色鲜红），常用于腊肉、火腿等，但须严格控制用量。
  • 工业用途：用于织物染色（偶氮染料合成）、电镀及金属热处理剂。

尾气处理：工业生产硝酸排放的尾气（\text{NO}_x）可用碱液吸收产生亚硝酸钠，达到“变废为宝”的目的：\text{NO} + \text{NO}_2 + 2\text{NaOH} = 2\text{NaNO}_2 + \text{H}_2\text{O}。

金属氮化物及其性质

金属氮化物是由金属元素与氮元素形成的化合物。根据其成键性质和结构特征，金属氮化物主要分为类盐氮化物和间充型氮化物两大类。

类盐氮化物通常由活泼金属（如碱金属和碱土金属）与氮气在高温下直接反应制得。

  1. 成键与结构：
    • 它们是离子化合物，氮元素以 N^{3-}（氮根离子） 形式存在，表现出 -3 价。
    • 典型代表：氮化锂（Li_3N）、氮化镁（Mg_3N_2）、氮化钙（Ca_3N_2）等。其中 Li_3N 是极少数能在室温下直接与氮气化合生成的产物。
  2. 物理性质：
    • 多为固体。例如，Mg_3N_2 是淡黄色或黄绿色的固体。
  3. 化学性质：剧烈的水解性：
    • 类盐氮化物是强碱或强亲核试剂，遇水会发生剧烈且不可逆的“双水解”反应，生成对应的金属氢氧化物沉淀并放出氨气（NH_3）。
    • 反应方程式（以 Mg_3N_2 为例）： Mg_3N_2 + 6H_2O = 3Mg(OH)_2\downarrow + 2NH_3\uparrow。
    • 现象：产生的氨气具有刺激性气味，能使湿润的红色石蕊试纸变蓝。这是检验氮气（通过生成氮化物再水解）的间接方法。
    • 与酸反应：会生成两种盐（金属盐和铵盐）。 Mg_3N_2 + 8HCl = 3MgCl_2 + 2NH_4Cl。

间充型氮化物主要由过渡金属（如 Ti, V, Zr, Nb, W, Fe 等）形成。

  1. 结构特征：
    • 原子半径较小的氮原子填充在金属晶格的间隙位置中，称为金属间隙化合物。
    • 这种结构保留了金属键的性质，因此它们通常具有金属光泽和良好的导电、导热性。
  2. 卓越的物理性能：
    • 超高硬度：其硬度通常极大，例如碳化钛（TiC）和某些氮化物的硬度等级可大于 9。
    • 极高熔点：它们的熔点往往比原来的纯金属更高。例如，TiN 等化合物具有高熔点、耐磨损和耐腐蚀的特性。
  3. 化学稳定性：
    • 与类盐氮化物不同，间充型氮化物通常不发生水解反应，化学性质非常稳定。

其他重要的金属氮化物：

  1. 氮化铝 (AlN)：
    • 属于共价化合物（共价晶体）。
    • 性质：具有高强度、高导热性、优异的绝缘性和耐热冲击性。
    • 用途：常用于制造熔铸纯铁、铝及其合金的理想坩埚材料，以及新型陶瓷材料。
  2. 钢中的氮化物：
    • 在冶金中，氮常与钢中的 Al, V, Nb, Ti 等元素结合，形成弥散分布的微小氮化物（如 AlN, VN, TiN），起到细化晶粒、强化基体的作用，从而提高钢的强度和韧性。

氮气及固氮

固氮概述

固氮的定义：从氮气 \ce{N2} 到各种含氮化合物。

\ce{N2 ->[O2，闪电] NO ->[O2] NO2 ->[H2O] HNO3 -> 降雨}

酸雨的形成过程

  • 自然固氮：

    • 高能固氮：闪电。

    • 生物固氮：豆科植物的根瘤菌。

  • 人工固氮：\ce{N2 + 3H2 <=>[高温高压][催化剂] 2NH3}（工业合成氨）。

工业合成氨

反应原理：

\ce{N2(g) + 3H2(g) <=> 2NH3(g)}\quad\Delta H = \pu{-91.5kJ/mol}

  • 压强效应：增大压强，反应向右移动。

  • 温度效应：降低温度，反应向右移动。

哈伯法制氨的过程，即使高温会降低产率，温度仍被设定为较高值以保证反应的速率快速。对于这类放热并且反应速率随温度升高而升高的反应，一般都会取一个折中的反应温度。

  • 催化剂：铁触媒。

  • 温度：一般取催化剂的适宜温度，即约 \pu{700K} 或约 \pu{500^oC}。

  • 压强：\pu{40\sim60 MPa} 左右，实际生产中又分为高压、中压、低压等，但整体压强都较高。

  • 因为催化机理的问题，为了保证氮气占有一定份额的催化剂活性中心，并提高吸附速率，应当适当提高氮气浓度，一般取 n(\ce{N2}):n(\ce{H2})=1:2.8。

  • 实际生产中，一定要注意气体净化，避免杂质使得催化剂中度，活性降低；过程中应当液化分离氨气，增加整体的产率。

注意：加入催化剂平衡不移动，催化剂只改变反应速率，但不影响平衡。

氮氧化物综合

  • \ce{NO2} 可以支持燃烧，证明如下：

    • \ce{4N2,O2} 木条不复燃。

    • \ce{4NO2,O2} 木条复燃。

  • \ce{NO} 可以作为传递神经信息的信使分子。

  • 二氧化氮若直接溶于碱溶液，发生反应：

    \ce{2NO2 + 2OH- -> NO3- + NO2- + H2O}

    生成硝酸根和亚硝酸根。而若二氧化氮和一氧化氮 1:1 通入碱溶液，则只生成亚硝酸根（亚硝酸钠的制备）。

  • 一氧化氮的制备：密封环境中，铜丝和稀硝酸反应。

氨气及其性质

氨气的物理性质

有刺激性气味气体，密度小于空气，极易溶于水（1:700）。

氨气极易溶于水的原因：与水形成氢键，增加了溶解度。

喷泉实验：

  • 气体极易溶于水，或与液体反应。

  • 短时间内形成压强差。

沸点 \pu{-33.5^oC} 较高，易液化放热、汽化吸热，可作为制冷剂。

氨气的制取与检验

工业制取：人工固氮。

实验室制取，常用五种方法：

  1. \ce{NH4HCO3(s)} 受热分解：

    • \ce{NH4HCO3 ->[\triangle] NH3 ^ + CO2 ^ + H2O}。

    • 用碱石灰除杂即可。

  2. \ce{NH4Cl(s),Ca(OH)2(s)} 共热：

    • \ce{2NH4Cl + Ca(OH)2 ->[\triangle] CaCl2 + 2H2O + 2NH3 ^}。

    • 用碱石灰干燥即可。

  3. \ce{NH4+(aq),OH-(aq)} 共热：

    • \ce{NH4+ + OH- -> NH3 ^ + H2O}。

    • 用碱石灰干燥即可。

  4. \ce{NH3(aq)} 加热：

    • \ce{NH3*H2O ->[\triangle] NH3 ^ + H2O}。

    • 用碱石灰干燥即可。

  5. 浓氨水滴到碱石灰上。

    • 生石灰与水反应放热。

    • 氢氧化钠与氨水反应。

注意：

  • 稀溶液必须加热，浓溶液无需加热。

  • 不可使用 \ce{NH4Cl} 加热，因为产物 \ce{NH3,HCl} 会迅速反应生成固体。

氨气的检验：

  • 酸碱指示剂，湿润的红色石蕊试纸。

  • 用玻璃棒蘸取浓硝酸或浓盐酸等挥发性酸，产生白烟。

氨气的化学性质

  • 弱碱性：

    • \ce{NH3 + HCl -> NH4Cl}，有白烟生成。

    • \ce{NH3 + H2O <=> NH3*H2O <=> NH4+ + OH-}。

    • 三分三离：\ce{NH3*H2O,H2O,NH3}；\ce{NH4+,H+,OH-}。

  • 还原性：

    • 氨气与氯气的反应较为复杂，可能会生成各种氮的价态产物。

    • \ce{4NH3 + 5O2 ->[催化剂][\triangle] 4NO + 6H2O}。

    • 氨的催化氧化，常用铂、铑作为催化剂。

  • 不稳定性：受热分解。

硝酸

硝酸的物理性质

硝酸，无色透明，有刺激性气味，易挥发。

浓硝酸的质量分数为 70\% 以上，发烟硝酸质量分数 \ge95\%。

因为浓硝酸中的硝酸分子分解产生 \ce{NO2} 而呈浅黄色，通入氧气后可以消除。

注意：硝酸铵不稳定，受热和剧烈撞击会发生爆炸。

稀硝酸的化学性质

氧化性和酸性：

  • 非氧化性酸由氢离子 \ce{H+} 体现氧化性。

  • 通常情况下，稀硝酸为 \ce N 体现氧化性，故其为氧化性酸。

  • 在与金属的反应中，还原产物通常为 \ce{NO}，浓度不同也可能被还原为氮气、氨气或亚硝酸。

与铜的反应：

  • \ce{3Cu + 8HNO3(稀) -> 3Cu(NO3)2 + 2NO + 4H2O}。

  • 现象：铜丝溶解，溶液变蓝，有无色气体产生，接触空气后变为红棕色。

与铁的反应：

  • 少量稀硝酸：\ce{3Fe + 8HNO3 -> 3Fe(NO3)2 + 2NO + 4H2O}。

  • 稀硝酸也可以氧化二价铁离子。

  • 过量稀硝酸：\ce{Fe + 4HNO3 -> Fe(NO3)3 + NO + 2H2O}。

浓硝酸的化学性质

氧化性和酸性：

  • 强氧化性漂白，使酸碱指示剂褪色。

  • 在与金属、非金属的反应中，通常被还原为 \ce{NO2}。

与铜的反应：

  • \ce{Cu + 4HNO3(浓) -> Cu(NO3)2 + 2NO2 + 2H2O}。

  • 产生红棕色气体，溶液变绿。

与红热的碳的反应：

  • \ce{C + 4HNO3(浓) ->[\triangle] CO2 ^ + 4NO2 ^ + 2H2O}。

  • 同时发生浓硝酸的受热分解，硝酸只体现氧化性。

与铁和铝钝化，加热进行。

王水

浓硝酸和浓盐酸按照体积比 1:3 混合，得到的强氧化性物质。

王水通常会缓慢分解出氯气和二氧化氮，这是浓硝酸氧化氯离子的结果。

王水甚至可以溶解金这种非常稳定的金属。

硝酸的工业制取

  2. 降温冷却后溶于水：\ce{4NO + O2 +2H2O -> 4HNO3}。

氰酸和异氰酸是同分异构体，而氢氰酸结构完全不同。两者能互相转化（互变异构），能量差约为 \pu{10kJ/mol}，绝大多数实验表明 \ce{HNCO} 更为稳定。

化学性质差异

酸性强度：

\ce{HCN} \ (\mathrm{p}K_\mathrm{a} \approx 9.2) \ < \ \ce{HOCN} \ (\mathrm{p}K_\mathrm{a} \approx 3.7) \approx \ce{HNCO} \ (\mathrm{p}K_\mathrm{a} \approx 3.7)

氢氰酸 (\ce{HCN})：

  • 容易发生聚合反应（储存时需加稳定剂）

  • 与醛、酮发生加成反应（氰醇合成）

  • 易与金属形成稳定配合物

  • 与 \ce{Ag+} 生成 \ce{AgCN} 沉淀（检测氰离子）

  • 与 \ce{Fe^3+} → 普鲁士蓝 \ce{Fe4[Fe(CN)6]3}

氰酸 (\ce{HOCN})：

  • 极不稳定，室温下迅速三聚：3\ce{HOCN} \rightarrow \ce{C3H3N3O3}（氰尿酸）

  • 通常以盐的形式（如 \ce{KOCN}）稳定存在

异氰酸 (\ce{HNCO})：

  • 相对稳定，但仍会缓慢聚合

  • 亲电性强，易与含活泼氢的化合物反应

  • 与醇反应生成碳酰胺（氨基甲酸酯）

  • 与胺反应生成脲

所有涉及氰化物的实验或工业操作均应严格遵守当地法规，配备相应的安全设施与培训。若无专业防护经验，请务必在合格的实验室或工厂环境下进行。

工业应用与扩展

  • 叠氮化钠（\ce{NaN3}）：在电点火产生的高温下迅速分解并释放大量 \ce{N2}，曾用于汽车安全气囊产气剂。

  • 五氧化二磷（\ce{P4O10}）：极强的吸水性，是常用的酸性干燥剂，但不能干燥 \ce{NH3}。

  • 砒霜（\ce{As2O3}）：剧毒。在工业流程中，若涉及砷元素的除去，通常利用其在不同 pH 下的沉淀规律或通过生成 \ce{FeAsO4} 沉淀。
