氮族元素概述

氮气及固氮

固氮概述

固氮的定义：从氮气 NX2\ce{N2}NX2​ 到各种含氮化合物。

  • 自然固氮：

    • 高能固氮：闪电。

    • 生物固氮：豆科植物的根瘤菌。

  • 人工固氮：NX2+3 HX2⇌催化剂高温高压2 NHX3\ce{N2 + 3H2 <=>[高温高压][催化剂] 2NH3}NX2​+3HX2​高温高压催化剂​2NHX3​（工业合成氨）。

工业合成氨

反应原理：

NX2(g)+3 HX2(g)⇌2 NHX3(g)ΔH=−91.5 kJ/mol \ce{N2(g) + 3H2(g) <=> 2NH3(g)}\quad\Delta H = \pu{-91.5kJ/mol} NX2​(g)+3HX2​(g)​2NHX3​(g)ΔH=−91.5 kJ/mol

  • 压强效应：增大压强，反应向右移动。

  • 温度效应：降低温度，反应向右移动。

哈伯法制氨的过程，即使高温会降低产率，温度仍被设定为较高值以保证反应的速率快速。对于这类放热并且反应速率随温度升高而升高的反应，一般都会取一个折中的反应温度。

  • 催化剂：铁触媒。

  • 温度：一般取催化剂的适宜温度，即约 700 K\pu{700K}700 K 或约 500 ∘C\pu{500^oC}500 ∘C。

  • 压强：40 ∼60 MPa\pu{40\sim60 MPa}40 ∼60MPa 左右，实际生产中又分为高压、中压、低压等，但整体压强都较高。

  • 因为催化机理的问题，为了保证氮气占有一定份额的催化剂活性中心，并提高吸附速率，应当适当提高氮气浓度，一般取 n(NX2):n(HX2)=1:2.8n(\ce{N2}):n(\ce{H2})=1:2.8n(NX2​):n(HX2​)=1:2.8。

  • 实际生产中，一定要注意气体净化，避免杂质使得催化剂中度，活性降低；过程中应当液化分离氨气，增加整体的产率。

注意：加入催化剂平衡不移动，催化剂只改变反应速率，但不影响平衡。

氮氧化物综合

  • NOX2\ce{NO2}NOX2​ 可以支持燃烧，证明如下：

    • 4 NX2, OX2\ce{4N2,O2}4NX2​,OX2​ 木条不复燃。

    • 4 NOX2, OX2\ce{4NO2,O2}4NOX2​,OX2​ 木条复燃。

  • NO\ce{NO}NO 可以作为传递神经信息的信使分子。

  • 二氧化氮若直接溶于碱溶液，发生反应：

    NOX2+6 OHX−→NOX3X−+NOX2X−+3 HX2O \ce{NO2 + 6OH- -> NO3- + NO2- + 3H2O} NOX2​+6OHX−​NOX3​X−+NOX2​X−+3HX2​O

    生成硝酸根和亚硝酸根。而若二氧化氮和一氧化氮 1:11:11:1 通入碱溶液，则只生成亚硝酸根（亚硝酸钠的制备）。

  • 一氧化氮的制备：密封环境中，铜丝和稀硝酸反应。

氨气及其性质

氨气的物理性质

有刺激性气味气体，密度小于空气，极易溶于水（1:7001:7001:700）。

氨气极易溶于水的原因：与水形成氢键，增加了溶解度。

喷泉实验：

  • 气体极易溶于水，或与液体反应。

  • 短时间内形成压强差。

沸点 −33.5 ∘C\pu{-33.5^oC}−33.5 ∘C 较高，易液化放热、汽化吸热，可作为制冷剂。

氨气的制取与检验

工业制取：人工固氮。

实验室制取，常用五种方法：

  1. NHX4HCOX3(s)\ce{NH4HCO3(s)}NHX4​HCOX3​(s) 受热分解：

    • NHX4HCOX3→△NHX3↑+COX2↑+HX2O\ce{NH4HCO3 ->[\triangle] NH3 ^ + CO2 ^ + H2O}NHX4​HCOX3​△​NHX3​↑+COX2​↑+HX2​O。

    • 用碱石灰除杂即可。

  2. NHX4Cl(s), Ca(OH)X2(s)\ce{NH4Cl(s),Ca(OH)2(s)}NHX4​Cl(s),Ca(OH)X2​(s) 共热：

    • 2 NHX4Cl+Ca(OH)X2→△CaClX2+2 HX2O+2 NHX3↑\ce{2NH4Cl + Ca(OH)2 ->[\triangle] CaCl2 + 2H2O + 2NH3 ^}2NHX4​Cl+Ca(OH)X2​△​CaClX2​+2HX2​O+2NHX3​↑。

    • 用碱石灰干燥即可。

  3. NHX4X+(aq), OHX−(aq)\ce{NH4+(aq),OH-(aq)}NHX4​X+(aq),OHX−(aq) 共热：

    • NHX4X++OHX−→NHX3↑+HX2O\ce{NH4+ + OH- -> NH3 ^ + H2O}NHX4​X++OHX−​NHX3​↑+HX2​O。

    • 用碱石灰干燥即可。

  4. NHX3(aq)\ce{NH3(aq)}NHX3​(aq) 加热：

    • NHX3 ⋅ HX2O→△NHX3↑+HX2O\ce{NH3*H2O ->[\triangle] NH3 ^ + H2O}NHX3​⋅HX2​O△​NHX3​↑+HX2​O。

    • 用碱石灰干燥即可。

  5. 浓氨水滴到碱石灰上。

    • 生石灰与水反应放热。

    • 氢氧化钠与氨水反应。

注意：

  • 稀溶液必须加热，浓溶液无需加热。

  • 不可使用 NHX4Cl\ce{NH4Cl}NHX4​Cl 加热，因为产物 NHX3, HCl\ce{NH3,HCl}NHX3​,HCl 会迅速反应生成固体。

氨气的检验：

  • 酸碱指示剂，湿润的红色石蕊试纸。

  • 用玻璃棒蘸取浓硝酸或浓盐酸等挥发性酸，产生白烟。

氨气的化学性质

  • 弱碱性：

    • NHX3+HCl→NHX4Cl\ce{NH3 + HCl -> NH4Cl}NHX3​+HCl​NHX4​Cl，有白烟生成。

    • NHX3+HX2O⇌NHX3 ⋅ HX2O⇌NHX4X++OHX−\ce{NH3 + H2O <=> NH3*H2O <=> NH4+ + OH-}NHX3​+HX2​O​NHX3​⋅HX2​O​NHX4​X++OHX−。

    • 三分三离：NHX3 ⋅ HX2O, HX2O, NHX3\ce{NH3*H2O,H2O,NH3}NHX3​⋅HX2​O,HX2​O,NHX3​；NHX4X+, HX+, OHX−\ce{NH4+,H+,OH-}NHX4​X+,HX+,OHX−。

  • 还原性：

    • 氨气与氯气的反应较为复杂，可能会生成各种氮的价态产物。

    • 4 NHX3+5 OX2→△催化剂4 NO+6 HX2O\ce{4NH3 + 5O2 ->[催化剂][\triangle] 4NO + 6H2O}4NHX3​+5OX2​催化剂△​4NO+6HX2​O。

    • 氨的催化氧化，常用铂、铑作为催化剂。

  • 不稳定性：受热分解。

硝酸

硝酸的物理性质

硝酸，无色透明，有刺激性气味，易挥发。

浓硝酸的质量分数为 70%70\%70% 以上，发烟硝酸质量分数 ≥95%\ge95\%≥95%。

因为浓硝酸中的硝酸分子分解产生 NOX2\ce{NO2}NOX2​ 而呈浅黄色，通入氧气后可以消除。

注意：硝酸铵不稳定，受热和剧烈撞击会发生爆炸。

稀硝酸的化学性质

氧化性和酸性：

  • 非氧化性酸由氢离子 HX+\ce{H+}HX+ 体现氧化性。

  • 通常情况下，稀硝酸为 N\ce NN 体现氧化性，故其为氧化性酸。

  • 在与金属的反应中，还原产物通常为 NO\ce{NO}NO，浓度不同也可能被还原为氮气、氨气或亚硝酸。

与铜的反应：

  • 3 Cu+8 HNOX3(稀)→3 Cu(NOX3)X2+2 NO+4 HX2O\ce{3Cu + 8HNO3(稀) -> 3Cu(NO3)2 + 2NO + 4H2O}3Cu+8HNOX3​(稀)​3Cu(NOX3​)X2​+2NO+4HX2​O。

  • 现象：铜丝溶解，溶液变蓝，有无色气体产生，接触空气后变为红棕色。

与铁的反应：

  • 少量稀硝酸：3 Fe+8 HNOX3→3 Fe(NOX3)X2+2 NO+4 HX2O\ce{3Fe + 8HNO3 -> 3Fe(NO3)2 + 2NO + 4H2O}3Fe+8HNOX3​​3Fe(NOX3​)X2​+2NO+4HX2​O。

  • 稀硝酸也可以氧化二价铁离子。

  • 过量稀硝酸：Fe+4 HNOX3→Fe(NOX3)X3+NO+2 HX2O\ce{Fe + 4HNO3 -> Fe(NO3)3 + NO + 2H2O}Fe+4HNOX3​​Fe(NOX3​)X3​+NO+2HX2​O。

浓硝酸的化学性质

氧化性和酸性：

  • 强氧化性漂白，使酸碱指示剂褪色。

  • 在与金属、非金属的反应中，通常被还原为 NOX2\ce{NO2}NOX2​。

与铜的反应：

  • Cu+4 HNOX3(浓)→Cu(NOX3)X2+2 NOX2+2 HX2O\ce{Cu + 4HNO3(浓) -> Cu(NO3)2 + 2NO2 + 2H2O}Cu+4HNOX3​(浓)​Cu(NOX3​)X2​+2NOX2​+2HX2​O。

  • 产生红棕色气体，溶液变绿。

与红热的碳的反应：

  • C+4 HNOX3(浓)→△COX2↑+4 NOX2↑+2 HX2O\ce{C + 4HNO3(浓) ->[\triangle] CO2 ^ + 4NO2 ^ + 2H2O}C+4HNOX3​(浓)△​COX2​↑+4NOX2​↑+2HX2​O。

  • 同时发生浓硝酸的受热分解，硝酸只体现氧化性。

与铁和铝钝化，加热进行。

王水

浓硝酸和浓盐酸按照体积比 1:31:31:3 混合，得到的强氧化性物质。

王水通常会缓慢分解出氯气和二氧化氮，这是浓硝酸氧化氯离子的结果。

王水甚至可以溶解金这种非常稳定的金属。

硝酸的工业制取

  1. 氨的催化氧化：4 NHX3+5 OX2→△催化剂4 NO+6 HX2O\ce{4NH3 + 5O2 ->[催化剂][\triangle] 4NO + 6H2O}4NHX3​+5OX2​催化剂△​4NO+6HX2​O。

  2. 降温冷却后溶于水：4 NO+OX2+2 HX2O→4 HNOX3\ce{4NO + O2 +2H2O -> 4HNO3}4NO+OX2​+2HX2​O​4HNOX3​。
