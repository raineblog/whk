碱金属概述

碱金属的性质

碱金属性质概述

  • 碱金属大多为银白色固体（铯 Cs 略带金色）、密度小柔软、熔沸点低。

  • 按照从上到下，Li, Na, K, Rb, Cs\ce{Li,Na,K,Rb,Cs}Li,Na,K,Rb,Cs，熔沸点降低、密度增大（钾反常小），与氧气反应更加复杂。

    Li+OX2→LiOX2\ce{Li + O2 -> LiO2}Li+OX2​​LiOX2​
    Na+OX2→NaX2O,NaX2OX2\ce{Na + O2 -> Na2O, Na2O2}Na+OX2​​NaX2​O,NaX2​OX2​
    K+OX2→KX2O,KX2OX2,KOX2,KOX3\ce{K + O2 -> K2O, K2O2, KO2, KO3}K+OX2​​KX2​O,KX2​OX2​,KOX2​,KOX3​

  • 对应的碱，除了氢氧化锂外均为强碱，且碱性依次增强。

钠单质的物理性质

  1. 质软。
  2. 银白色金属光泽。
  3. 常温下是固态。
  4. 密度比水小，比煤油大。
  5. 熔点低。

钠单质的化学性质

钠与氧气的反应

  • 常温：迅速被氧化为 NaX2O\ce{Na2O}NaX2​O，失去金属光泽。

    4 Na+OX2→2 NaX2O\ce{4Na + O2 -> 2Na2O}4Na+OX2​​2NaX2​O

  • 加热：先熔化后燃烧，发出黄色火焰，生成淡黄色固体。

    2 Na+OX2→NaX2OX2\ce{2Na + O2 -> Na2O2}2Na+OX2​​NaX2​OX2​

  • 钠着火后用沙土盖灭。

钠与水的反应

Na+2 HX2O→2 NaOH+HX2↑\ce{Na + 2H2O -> 2NaOH + H2 ^}Na+2HX2​O​2NaOH+HX2​↑
现象解释
钠浮在水面上钠密度比水小
熔化成光亮的小球反应放热，钠熔点低
四处游动反应生成气体
发出嘶嘶响声反应剧烈，产生气体
产生使酚酞变红的物质生成碱性物质

钠与酸的反应

与非氧化性酸（e.g. HCl）发生反应：

  1. 2 Na+2 HCl→2 NaCl+HX2↑\ce{2Na + 2HCl -> 2NaCl + H2 ^}2Na+2HCl​2NaCl+HX2​↑。
  2. 2 Na+HX2O→2 NaOH+HX2↑\ce{2Na + H2O -> 2NaOH + H2 ^}2Na+HX2​O​2NaOH+HX2​↑。

其中与强酸的反应一般比与水的更剧烈。

钠与盐的反应

与盐溶液反应：

  1. 先和水生成碱。

  2. 后发生碱和盐的反应。

    2 Na+2 HX2O+CuSOX4→HX2↑+NaX2SOX4+Cu(OH)X2↓\ce{2Na + 2H2O + CuSO4 -> H2 ^ + Na2SO4 + Cu(OH)2 v}2Na+2HX2​O+CuSOX4​​HX2​↑+NaX2​SOX4​+Cu(OH)X2​↓

  3. 现象：同时有气体和沉淀产生。

与熔融盐盐发生置换反应：

2 Na+CuSOX4→Cu+NaX2SOX4\ce{2Na + CuSO4 -> Cu + Na2SO4}2Na+CuSOX4​​Cu+NaX2​SOX4​

钠的应用

  1. 高压钠灯。
  2. 钠钾合金用作原子反应堆的导热剂。

钠冶炼金属：

  • 通常用于冶炼制取纯度高的金属，尤其是熔融态还原金属氯化物。

  • 钠还原四氯化钛（TiCl₄）是工业制钛的重要方法（亨特法）：

    此反应在惰性气氛中进行，生成的海绵钛经熔炼后用于航空、医疗等领域。

    亨特法逐渐被用镁还原的克劳尔法所替代。

  • 此外，还可以用于冶炼锆（Zr）、铌（Nb）、钽（Ta）。

钠的氧化物

氧化钠的性质

作为碱性氧化物，与酸、水、酸性氧化物的反应：

同时存在 2 NaX2O+OX2→2 NaX2OX2\ce{2Na2O + O2 -> 2Na2O2}2NaX2​O+OX2​​2NaX2​OX2​ 在加热条件下生成。

过氧化钠的性质

类似氧化钠的：

2 NaX2OX2+4 HCl→4 NaCl+2 HX2O+OX22 NaX2OX2+2 HX2O→4 NaOH+OX22 NaX2OX2+2 COX2→2 NaX2COX3+OX2\begin{aligned} \ce{2Na2O2 + 4HCl &-> 4NaCl + 2H2O + O2}\\ \ce{2Na2O2 + 2H2O &-> 4NaOH + O2}\\ \ce{2Na2O2 + 2CO2 &-> 2Na2CO3 + O2} \end{aligned}2NaX2​OX2​+4HCl2NaX2​OX2​+2HX2​O2NaX2​OX2​+2COX2​​​4NaCl+2HX2​O+OX2​​4NaOH+OX2​​2NaX2​COX3​+OX2​​

可以进行氧气供给。

NaX2OX2+SOX2→NaSOX4NaX2OX2+SOX3→NaSOX4+OX2\begin{aligned} \ce{Na2O2 + SO2 &-> NaSO4}\\ \ce{Na2O2 + SO3 &-> NaSO4 + O2} \end{aligned}NaX2​OX2​+SOX2​NaX2​OX2​+SOX3​​​NaSOX4​​NaSOX4​+OX2​​

这是因为其具有强氧化性，另外其还具有漂白性，可以用于杀菌消毒。

钠的氧化物题型

关于 NaX2OX2\ce{Na2O2}NaX2​OX2​ 的增重：

  • NaX2OX2\ce{Na2O2}NaX2​OX2​ 与 HX2O\ce{H2O}HX2​O 反应，相当于增加了 HX2\ce{H2}HX2​ 的重量。
  • NaX2OX2\ce{Na2O2}NaX2​OX2​ 与 COX2\ce{CO2}COX2​ 反应，相当于增加了 CO\ce{CO}CO 的重量。

因此：aga\pu gag 的 HX2, CO\ce{H2,CO}HX2​,CO 混合气体与足量氧气点燃，生成的气体通过足量过氧化钠，增重 aga\pu gag。

钠的碳酸（氢）盐

套管实验

理论：NaX2COX3\ce{Na2CO3}NaX2​COX3​ 比 NaHCOX3\ce{NaHCO3}NaHCOX3​ 更碱、更溶、更稳定。

  • 外层：NaX2COX3\ce{Na2CO3}NaX2​COX3​，无明显现象。
  • 内层：NaHCOX3\ce{NaHCO3}NaHCOX3​，塞蘸有无水硫酸铜的棉花，插入澄清石灰水中。

类似的结论有 OHX−+HCOX3X+→HX2O+COX3X2−\ce{OH- + HCO3+ -> H2O + CO3^2-}OHX−+HCOX3​X+​HX2​O+COX3​X2−。

半径法

  1. 向 NaOH\ce{NaOH}NaOH 溶液中通入二氧化碳。
  2. 向得到的溶液中加入稀盐酸，记录二氧化碳产生量与加入盐酸的量的折线图。

从折点开始画圆，半径为左右较小的长度，圆内平阶段发生碳酸钠的反应，前发生氢氧化钠的反应。

海水制碱

氨碱法（索尔维制碱法）：

  • 原料：食盐 NaCl\ce{NaCl}NaCl、石灰石 CaCOX3\ce{CaCO3}CaCOX3​。

  • 媒介：NHX3\ce{NH3}NHX3​。

  • 流程：

    饱和食盐水→NHX3饱和氨盐水→COX2碳酸氢钠→过滤→加热碳酸钠\ce{饱和食盐水 ->[NH3] 饱和氨盐水 ->[CO2] 碳酸氢钠 ->[过滤] ->[加热] 碳酸钠}饱和食盐水NHX3​​饱和氨盐水COX2​​碳酸氢钠过滤​加热​碳酸钠

  • 化学方程式：

    NaCl+NHX3+COX2+HX2O→NHX4Cl+NaHCOX3\ce{NaCl + NH3 + CO2 + H2O -> NH4Cl + NaHCO3}NaCl+NHX3​+COX2​+HX2​O​NHX4​Cl+NaHCOX3​

经典问题：

  • 氨气使溶液具有碱性，可以吸收更多的二氧化碳。

  • 相同温度下，NaHCOX3\ce{NaHCO3}NaHCOX3​ 溶解度小于 NHX4Cl\ce{NH4Cl}NHX4​Cl，先析出。

  • NaHCOX3\ce{NaHCO3}NaHCOX3​ 的相对分子质量大于 NHX4Cl\ce{NH4Cl}NHX4​Cl，先达到饱和。

工业流程：

  • 循环使用 NaHCOX3\ce{NaHCO3}NaHCOX3​ 分解产生的氨气。

  • 使用碳酸钙制取 Ca(OH)X2\ce{Ca(OH)2}Ca(OH)X2​ 与 NHX4Cl\ce{NH4Cl}NHX4​Cl 反应，循环利用氨气。

  • 缺点：产生 CaClX2\ce{CaCl2}CaClX2​ 无用物质，NaCl\ce{NaCl}NaCl 利用率低。

联合制减法（侯氏制碱法）：

  • 氨厂与碱厂联合：碱厂直接利用氨厂产出的氨气和二氧化碳。

  • 中国农业化肥需求量大，NHX4Cl\ce{NH4Cl}NHX4​Cl 直接作为化肥生产，无 CaClX2\ce{CaCl2}CaClX2​ 污染。
