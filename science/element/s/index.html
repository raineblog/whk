    #氧族元素概述

    预计阅读时间: 7 分钟

    #硫单质

    #硫单质的物理性质

    游离态主要存在于火山口，化合态包括硫酸盐和硫化物。

    常见的含硫矿石：黄铁矿（FeSX2\ce{FeS2}FeSX2​）、黄铜矿（FeCuSX2\ce{FeCuS2}FeCuSX2​）、辰砂朱砂（HgS\ce{HgS}HgS）、雄黄（AsX4SX4\ce{As4S4}AsX4​SX4​）、雌黄（AsX2SX3\ce{As2S3}AsX2​SX3​）、芒硝（NaX2SOX4 ⋅ 10 HX2O\ce{Na2SO4*10H2O}NaX2​SOX4​⋅10HX2​O）、生石膏（2 CaSOX4 ⋅ HX2O\ce{2CaSO4*H2O}2CaSOX4​⋅HX2​O）、熟石膏（CaSOX4 ⋅ 2 HX2O\ce{CaSO4*2H2O}CaSOX4​⋅2HX2​O）。

    硫（硫磺）为黄色到浅黄色固体，存在同素异形体（正交硫、单斜硫）：SX2, SX4, SX6, SX8\ce{S2,S4,S6,S8}SX2​,SX4​,SX6​,SX8​。

    不溶于水、微溶于酒精、易溶于二硫化碳（CSX2\ce{CS2}CSX2​），因此常用二硫化碳洗涤试管上的硫磺，硫单质在水溶液中的反应通常先用酒精浸泡硫磺。

    #硫的化学性质

    硫单质的氧化性：

    • 与金属反应：生成较低价硫化物。

      • 2 Na+S→研磨NaX2S\ce{2Na + S ->[研磨] Na2S}2Na+S研磨​NaX2​S。

      • Fe+S→△FeS\ce{Fe + S ->[\triangle] FeS}Fe+S△​FeS（黑色固体）。

      • Cu+S→△CuX2S\ce{Cu + S ->[\triangle] Cu2S}Cu+S△​CuX2​S（黑色固体）。

      • Hg+S→HgS\ce{Hg + S -> HgS}Hg+S​HgS（黑色固体，用于处理汞的少量泄露）

      • 与氢气反应：HX2+S→点燃HX2S\ce{H2 + S ->[点燃] H2S}HX2​+S点燃​HX2​S。

    • 黑火药：S+2 KNOX3+3 S→点燃NX2+3 COX2+KX2S\ce{S + 2KNO3 + 3S ->[点燃] N2 + 3CO2 + K2S}S+2KNOX3​+3S点燃​NX2​+3COX2​+KX2​S（一硫二硝三木炭）。

    #硫的氧化物

    #二氧化硫

    无色有刺激性气味的气体，有毒，易溶于水（1:401:401:40）。

    作为酸性氧化物，可以发生反应：

    • 与碱反应：注意亚硫酸根显碱性、亚硫酸氢根显酸性。

    • 和水反应：SOX2+HX2O⇌HX2SOX3\ce{SO2 + H2O <=> H2SO3}SOX2​+HX2​O​HX2​SOX3​。

    • 和碱性氧化物反应：SOX2+CaO→CaSOX3\ce{SO2 + CaO -> CaSO3}SOX2​+CaO​CaSOX3​。

    作为硫的中间价态，同时具有氧化性和还原性：

    • 氧化性：

      • SOX2+2 SX2−+2 HX2O→3 S↓+4 OHX−\ce{SO2 + 2S^2- + 2H2O -> 3S v + 4OH-}SOX2​+2SX2−+2HX2​O​3S↓+4OHX−。

      • 3 SOX2+2 SX2−→3 S↓+2 SOX3X2−\ce{3SO2 + 2S^2- -> 3S v + 2SO3^2-}3SOX2​+2SX2−​3S↓+2SOX3​X2−。

      • 5 SOX2+2 SX2−+2 HX2O→3 S↓+4 NaHSOX3\ce{5SO2 + 2S^2- + 2H2O -> 3S v + 4NaHSO3}5SOX2​+2SX2−+2HX2​O​3S↓+4NaHSOX3​。

      • 另外，SOX2+2 HX2S→3 S↓+2 HX2O\ce{SO2 + 2H2S -> 3S v + 2H2O}SOX2​+2HX2​S​3S↓+2HX2​O。

    • 还原性：

      • 2 SOX2+OX2⇌△催化剂SOX3\ce{2SO2 + O2 <=>[催化剂][\triangle] SO3}2SOX2​+OX2​催化剂△​SOX3​。

      • XX2+SOX2+2 HX2O→2 HX+HX2SOX4\ce{X_2 + SO2 + 2H2O -> 2HX + H2SO4}XX2​+SOX2​+2HX2​O​2HX+HX2​SOX4​（X=ClX2 / BrX2 / IX2\ce{X = Cl2 / Br2 / I2}X=ClX2​ / BrX2​ / IX2​）。

      • 5 SOX2+2 MnOX4X−+2 HX2O→2 MnX2++5 SOX4X2−+4 HX+\ce{5SO2 + 2MnO4- + 2H2O -> 2Mn^2+ + 5SO4^2- + 4H+}5SOX2​+2MnOX4​X−+2HX2​O​2MnX2++5SOX4​X2−+4HX+。

      • SOX2+NaX2OX2→NaX2SOX4\ce{SO2 + Na2O2 -> Na2SO4}SOX2​+NaX2​OX2​​NaX2​SOX4​。

      • 另外，FeX3++SOX2→FeX2++SOX4X2−\ce{Fe^3+ + SO2 -> Fe^2+ + SO4^2-}FeX3++SOX2​​FeX2++SOX4​X2−。

    另外，二氧化硫具有漂白性，加热或久置后可恢复原色。

    • 鉴别 COX2, SOX2\ce{CO2,SO2}COX2​,SOX2​：品红溶液，酸性高锰酸钾溶液，淀粉碘化钾试液。

    • SOX2(SOX3)\ce{SO2(SO3)}SOX2​(SOX3​) 除杂：浓硫酸，饱和 NaHSOX3\ce{NaHSO3}NaHSOX3​ 溶液、冰水。

    #三氧化硫

    熔点 16.8 ∘C\pu{16.8^oC}16.8 ∘C，沸点 44.5 ∘C\pu{44.5^oC}44.5 ∘C，加热后为气体，常温下为液态，标况下为固体（易升华）。

    作为酸性氧化物，可以和碱、水、碱性氧化物反应，在某些时候（如和 −2-2−2 价的硫）显示氧化性。

    #硫酸

    #浓硫酸

    浓硫酸：质量分数 ≥70%\ge70\%≥70% 的硫酸水溶液，俗称坏水，为无色无味的油状液体（难挥发）。

    对于 98.3%98.3\%98.3% 的浓硫酸，其浓度为 18.4mol/L18.4\pu{mol/L}18.4mol/L，密度为 1.84g/cm31.84\pu{g/cm^3}1.84g/cm3。

    其中有大量未电离的硫酸分子，一般以硫酸分子的性质参与氧化还原反应。

    • 强氧化性：

      • 与 Fe, Al\ce{Fe,Al}Fe,Al 常温下钝化，加热反应。

      • 与 Cu\ce{Cu}Cu 反应生成 CuSOX4\ce{CuSO4}CuSOX4​ 和 SOX2\ce{SO2}SOX2​，实际情况下可能会产生硫化铜（CuS\ce{CuS}CuS）、硫化亚铜（CuX2S\ce{Cu2S}CuX2​S）、硫单质、无水硫酸铜、五水合硫酸铜等复杂的产物。

      • 与 C\ce CC 反应：C+2 HX2SOX4(浓)→△COX2↑+2 SOX2↑+2 HX2O\ce{C + 2H2SO4(浓) ->[\triangle] CO2 ^ + 2SO2 ^ + 2H2O}C+2HX2​SOX4​(浓)△​COX2​↑+2SOX2​↑+2HX2​O，只体现氧化性。

    • 吸水性和脱水性：

      • 吸水性：作为常见的酸性干燥剂，不能干燥碱性气体（NHX3\ce{NH3}NHX3​）和还原性气体（HX2S, HBr, HI\ce{H2S,HBr,HI}HX2​S,HBr,HI 等）。

      • 脱水性：按照 H:O=2:1\ce{H}:\ce{O}=2:1H:O=2:1 的比例脱去有机物中的氢氧元素，使其碳化。黑面包实验 CX12HX22OX11→浓硫酸12 C+11 HX2O\ce{C12H22O11 ->[浓硫酸] 12C + 11H2O}CX12​HX22​OX11​浓硫酸​12C+11HX2​O。

      • 补充：PX2OX5\ce{P2O5}PX2​OX5​ 同为酸性干燥剂。

    若浓硫酸沾到皮肤上，应立即用大量清水冲洗，然后涂上 3%∼5%3\%\sim5\%3%∼5% 的碳酸氢钠溶液，以防灼烧皮肤。

    注意，真实情况下，铜和浓硫酸反应可以理解为，主反应浓硫酸可能先将铜氧化生成氧化铜 CuO\ce{CuO}CuO 然后与硫酸反应生成硫酸铜 CuSOX4\ce{CuSO4}CuSOX4​，而因为浓硫酸中水少，可能有部分硫酸铜以五水合硫酸铜（CuSOX4 ⋅ HX2O\ce{CuSO4 * H2O}CuSOX4​⋅HX2​O）的形式析出；副反应可能报考的产物可能包括硫化铜（CuS\ce{CuS}CuS）、硫化亚铜（CuX2S\ce{Cu2S}CuX2​S）、硫单质（S\ce{S}S）。再加上可能未完全反应的氧化铜（CuO\ce{CuO}CuO）和生成的气体、反应放热产生的水蒸气等，现象较为复杂。另外铜与稀硫酸在加热条件下与氧气也可以反应生成硫酸铜 CuSOX4\ce{CuSO4}CuSOX4​），这个通常是由铜制备硫酸铜的较好方法。

    #硫代硫酸钠

    硫代硫酸钠（NaX2SX2OX3\ce{Na2S2O3}NaX2​SX2​OX3​）。

    形式上如以一硫代氧，硫的化合价为 +2+2+2 价，实际为 0,+40,+40,+4 平均。

    易被氧化，常用在滴定反应中。

    通常用 98.3%98.3\%98.3% 的浓硫酸吸收，放置形成酸雾腐蚀设备。

    #酸雨

    定义 pH<5.6\pu{pH}<5.6pH<5.6 的雨水为酸雨。

    酸雨的来源常为硫氧化物、氮氧化物，最终形成 HX2SOX4, HNOX3\ce{H2SO4,HNO3}HX2​SOX4​,HNOX3​ 等酸性雨水。
