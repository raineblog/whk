    #氧族元素概述

    预计阅读时间: 7 分钟

    #硫单质

    #硫单质的物理性质

    游离态主要存在于火山口，化合态包括硫酸盐和硫化物。

    常见的含硫矿石：黄铁矿（\ceFeS2\ce{FeS2}\ceFeS2）、黄铜矿（\ceFeCuS2\ce{FeCuS2}\ceFeCuS2）、辰砂朱砂（\ceHgS\ce{HgS}\ceHgS）、雄黄（\ceAs4S4\ce{As4S4}\ceAs4S4）、雌黄（\ceAs2S3\ce{As2S3}\ceAs2S3）、芒硝（\ceNa2SO4∗10H2O\ce{Na2SO4*10H2O}\ceNa2SO4∗10H2O）、生石膏（\ce2CaSO4∗H2O\ce{2CaSO4*H2O}\ce2CaSO4∗H2O）、熟石膏（\ceCaSO4∗2H2O\ce{CaSO4*2H2O}\ceCaSO4∗2H2O）。

    硫（硫磺）为黄色到浅黄色固体，存在同素异形体（正交硫、单斜硫）：\ceS2,S4,S6,S8\ce{S2,S4,S6,S8}\ceS2,S4,S6,S8。

    不溶于水、微溶于酒精、易溶于二硫化碳（\ceCS2\ce{CS2}\ceCS2），因此常用二硫化碳洗涤试管上的硫磺，硫单质在水溶液中的反应通常先用酒精浸泡硫磺。

    #硫的化学性质

    硫单质的氧化性：

    • 与金属反应：生成较低价硫化物。

      • \ce2Na+S−>[研磨]Na2S\ce{2Na + S ->[研磨] Na2S}\ce2Na+S−>[研磨]Na2S。

      • \ceFe+S−>[△]FeS\ce{Fe + S ->[\triangle] FeS}\ceFe+S−>[△]FeS（黑色固体）。

      • \ceCu+S−>[△]Cu2S\ce{Cu + S ->[\triangle] Cu2S}\ceCu+S−>[△]Cu2S（黑色固体）。

      • \ceHg+S−>HgS\ce{Hg + S -> HgS}\ceHg+S−>HgS（黑色固体，用于处理汞的少量泄露）

      • 与氢气反应：\ceH2+S−>[点燃]H2S\ce{H2 + S ->[点燃] H2S}\ceH2+S−>[点燃]H2S。

    • 黑火药：\ceS+2KNO3+3S−>[点燃]N2+3CO2+K2S\ce{S + 2KNO3 + 3S ->[点燃] N2 + 3CO2 + K2S}\ceS+2KNO3+3S−>[点燃]N2+3CO2+K2S（一硫二硝三木炭）。

    #硫的氧化物

    #二氧化硫

    无色有刺激性气味的气体，有毒，易溶于水（1:401:401:40）。

    作为酸性氧化物，可以发生反应：

    • 与碱反应：注意亚硫酸根显碱性、亚硫酸氢根显酸性。

    • 和水反应：\ceSO2+H2O<=>H2SO3\ce{SO2 + H2O <=> H2SO3}\ceSO2+H2O<=>H2SO3。

    • 和碱性氧化物反应：\ceSO2+CaO−>CaSO3\ce{SO2 + CaO -> CaSO3}\ceSO2+CaO−>CaSO3。

    作为硫的中间价态，同时具有氧化性和还原性：

    • 氧化性：

      • \ceSO2+2S2−+2H2O−>3Sv+4OH−\ce{SO2 + 2S^2- + 2H2O -> 3S v + 4OH-}\ceSO2+2S2−+2H2O−>3Sv+4OH−。

      • \ce3SO2+2S2−−>3Sv+2SO32−\ce{3SO2 + 2S^2- -> 3S v + 2SO3^2-}\ce3SO2+2S2−−>3Sv+2SO32−。

      • \ce5SO2+2S2−+2H2O−>3Sv+4NaHSO3\ce{5SO2 + 2S^2- + 2H2O -> 3S v + 4NaHSO3}\ce5SO2+2S2−+2H2O−>3Sv+4NaHSO3。

      • 另外，\ceSO2+2H2S−>3Sv+2H2O\ce{SO2 + 2H2S -> 3S v + 2H2O}\ceSO2+2H2S−>3Sv+2H2O。

    • 还原性：

      • \ce2SO2+O2<=>[催化剂][△]SO3\ce{2SO2 + O2 <=>[催化剂][\triangle] SO3}\ce2SO2+O2<=>[催化剂][△]SO3。

      • \ceX2+SO2+2H2O−>2HX+H2SO4\ce{X_2 + SO2 + 2H2O -> 2HX + H2SO4}\ceX2​+SO2+2H2O−>2HX+H2SO4（\ceX=Cl2/Br2/I2\ce{X = Cl2 / Br2 / I2}\ceX=Cl2/Br2/I2）。

      • \ce5SO2+2MnO4−+2H2O−>2Mn2++5SO42−+4H+\ce{5SO2 + 2MnO4- + 2H2O -> 2Mn^2+ + 5SO4^2- + 4H+}\ce5SO2+2MnO4−+2H2O−>2Mn2++5SO42−+4H+。

      • \ceSO2+Na2O2−>Na2SO4\ce{SO2 + Na2O2 -> Na2SO4}\ceSO2+Na2O2−>Na2SO4。

      • 另外，\ceFe3++SO2−>Fe2++SO42−\ce{Fe^3+ + SO2 -> Fe^2+ + SO4^2-}\ceFe3++SO2−>Fe2++SO42−。

    另外，二氧化硫具有漂白性，加热或久置后可恢复原色。

    • 鉴别 \ceCO2,SO2\ce{CO2,SO2}\ceCO2,SO2：品红溶液，酸性高锰酸钾溶液，淀粉碘化钾试液。

    • \ceSO2(SO3)\ce{SO2(SO3)}\ceSO2(SO3) 除杂：浓硫酸，饱和 \ceNaHSO3\ce{NaHSO3}\ceNaHSO3 溶液、冰水。

    #三氧化硫

    熔点 \pu16.8oC\pu{16.8^oC}\pu16.8oC，沸点 \pu44.5oC\pu{44.5^oC}\pu44.5oC，加热后为气体，常温下为液态，标况下为固体（易升华）。

    作为酸性氧化物，可以和碱、水、碱性氧化物反应，在某些时候（如和 −2-2−2 价的硫）显示氧化性。

    #硫酸

    #浓硫酸

    浓硫酸：质量分数 ≥70%\ge70\%≥70% 的硫酸水溶液，俗称坏水，为无色无味的油状液体（难挥发）。

    对于 98.3%98.3\%98.3% 的浓硫酸，其浓度为 18.4\pumol/L18.4\pu{mol/L}18.4\pumol/L，密度为 1.84\pug/cm31.84\pu{g/cm^3}1.84\pug/cm3。

    其中有大量未电离的硫酸分子，一般以硫酸分子的性质参与氧化还原反应。

    • 强氧化性：

      • 与 \ceFe,Al\ce{Fe,Al}\ceFe,Al 常温下钝化，加热反应。

      • 与 \ceCu\ce{Cu}\ceCu 反应生成 \ceCuSO4\ce{CuSO4}\ceCuSO4 和 \ceSO2\ce{SO2}\ceSO2，实际情况下可能会产生硫化铜（\ceCuS\ce{CuS}\ceCuS）、硫化亚铜（\ceCu2S\ce{Cu2S}\ceCu2S）、硫单质、无水硫酸铜、五水合硫酸铜等复杂的产物。

      • 与 \ceC\ce C\ceC 反应：\ceC+2H2SO4(浓)−>[△]CO2+2SO2+2H2O\ce{C + 2H2SO4(浓) ->[\triangle] CO2 ^ + 2SO2 ^ + 2H2O}\ceC+2H2SO4(浓)−>[△]CO2+2SO2+2H2O，只体现氧化性。

    • 吸水性和脱水性：

      • 吸水性：作为常见的酸性干燥剂，不能干燥碱性气体（\ceNH3\ce{NH3}\ceNH3）和还原性气体（\ceH2S,HBr,HI\ce{H2S,HBr,HI}\ceH2S,HBr,HI 等）。

      • 脱水性：按照 \ceH:\ceO=2:1\ce{H}:\ce{O}=2:1\ceH:\ceO=2:1 的比例脱去有机物中的氢氧元素，使其碳化。黑面包实验 \ceC12H22O11−>[浓硫酸]12C+11H2O\ce{C12H22O11 ->[浓硫酸] 12C + 11H2O}\ceC12H22O11−>[浓硫酸]12C+11H2O。

      • 补充：\ceP2O5\ce{P2O5}\ceP2O5 同为酸性干燥剂。

    若浓硫酸沾到皮肤上，应立即用大量清水冲洗，然后涂上 3%∼5%3\%\sim5\%3%∼5% 的碳酸氢钠溶液，以防灼烧皮肤。

    注意，真实情况下，铜和浓硫酸反应可以理解为，主反应浓硫酸可能先将铜氧化生成氧化铜 \ceCuO\ce{CuO}\ceCuO 然后与硫酸反应生成硫酸铜 \ceCuSO4\ce{CuSO4}\ceCuSO4，而因为浓硫酸中水少，可能有部分硫酸铜以五水合硫酸铜（\ceCuSO4∗H2O\ce{CuSO4 * H2O}\ceCuSO4∗H2O）的形式析出；副反应可能报考的产物可能包括硫化铜（\ceCuS\ce{CuS}\ceCuS）、硫化亚铜（\ceCu2S\ce{Cu2S}\ceCu2S）、硫单质（\ceS\ce{S}\ceS）。再加上可能未完全反应的氧化铜（\ceCuO\ce{CuO}\ceCuO）和生成的气体、反应放热产生的水蒸气等，现象较为复杂。另外铜与稀硫酸在加热条件下与氧气也可以反应生成硫酸铜 \ceCuSO4\ce{CuSO4}\ceCuSO4），这个通常是由铜制备硫酸铜的较好方法。

    #硫代硫酸钠

    硫代硫酸钠（\ceNa2S2O3\ce{Na2S2O3}\ceNa2S2O3）。

    形式上如以一硫代氧，硫的化合价为 +2+2+2 价，实际为 0,+40,+40,+4 平均。

    易被氧化，常用在滴定反应中。

    #硫酸的工业制取

    整体的流程是：

    \ceS/FeS2−>[O2][(1)]SO2−>[O2][(2)]SO3−>[H2O][(3)]H2SO4\ce{S / FeS2 ->[O2][(1)] SO2 ->[O2][(2)] SO3 ->[H2O][(3)] H2SO4}\ceS/FeS2−>[O2][(1)]SO2−>[O2][(2)]SO3−>[H2O][(3)]H2SO4
    \left\{\begin{aligned} \ce{S + O2 &->[点燃] SO2}\\ \ce{4FeS2 + 11O2 &->[高温] 2Fe2O3 + 8SO2} \end{aligned}\right.\tag1 \ce2SO2+O2<=>[\pu400∼500oC][V2O5]2SO3(2)\ce{2SO2 + O2 <=>[\pu{400\sim500^oC}][V2O5] 2SO3}\tag2\ce2SO2+O2<=>[\pu400∼500oC][V2O5]2SO3(2) \ceH2O+SO3−>H2SO4(3)\ce{H2O + SO3 -> H2SO4}\tag3\ceH2O+SO3−>H2SO4(3)

    通常用 98.3%98.3\%98.3% 的浓硫酸吸收，放置形成酸雾腐蚀设备。

    #酸雨

    定义 \pupH<5.6\pu{pH}<5.6\pupH<5.6 的雨水为酸雨。

    酸雨的来源常为硫氧化物、氮氧化物，最终形成 \ceH2SO4,HNO3\ce{H2SO4,HNO3}\ceH2SO4,HNO3 等酸性雨水。
