函数探索技能库 (Function Exploration Skills)

1. 核心解题技能与路径规约

基础理论与核心模型

  • 反函数性质：单调函数必有反函数，原反函数单调性一致且图像关于 y = x 对称，导数互为倒数 f′(x) ⋅ (f−1)′(y) = 1。

  • Lambert W 函数：定义为 xex 的反函数，指对混含方程通解路径 xex = a ⇔ x = W(a)，辅助判定根的存在性趋势。

  • 周期与对称推导：双对称轴 x = a, x = b 产生周期 T = 2|a − b|；轴心各一 f(a + x) = f(a − x), f(b + x) + f(b − x) = 2c 后周期 T = 4|a − b|。

  • 三次函数模型：必关于拐点 (xw, f(xw)) 中心轴对称，过外点切线条数由拐点切线划分区域判定 (Y0 − f(X0))(Y0 − kX0) < 0 ⟹ 3 条。

  • 严格单调性判定：充要条件为 f′(x) ≥ 0 且在子区间不恒为 0，若函数连续则单调区间包含端点且无视孤立不可导点。

  • 凹凸性放缩：二阶导 f″(x) > 0 表示下凸，满足割线放缩 $\dfrac{f(x_1)+f(x_2)}{2} > f\left(\dfrac{x_1+x_2}{2}\right)$ 辅助极值点偏移必要性探路。

参变分离与边界探测

  • 分离与最值：将参数孤立转化为函数最值判定路径 a ≥ g(x) ⟹ a ≥ g(x)max，适用于解析式导数计算相对简单的常规题型。

  • 端点效应法：利用 f(x0) = 0 且 f′(x0) = 0 寻找参数临界值，通过必要性探路结合单调性验证确认参数最终范围。

  • 半分离/切线放缩：当分离后导数计算复杂时，保留部分项并利用经典切线 ex ≥ x + 1, ln x ≤ x − 1 确认解析式临界边界。

同构变换路径

  • 指对同构母函数：识别 $xe^x, \dfrac{\ln x}{x}, x \pm \ln x$ 等特征，将不等式两侧化为同一结构 f(x1) ≥ f(x2) 进行单调性降维。

  • 复合同构变换：利用 eln x = x 进行结构对等，例如 aex ≥ ln x + b 转化为 ex + ln a ≥ ln x + b 后构造辅助函数 h(t) = et − t。

  • 混合结构同构：针对 x1ex1 = ln x2 + x2 类型，利用 x2 = eln x2 转化为 x1ex1 = ln x2eln x2 后直接对应 x1 = ln x2。

极值点偏移与核心不等式

  • 对称构造路径：设 x1 < x0 < x2，构造 F(x) = f(x) − f(2x0 − x)，利用 F(x) 在极值点附近单调性判定 x1 + x2 与 2x0 偏离方向。

  • 比值增量法：引入变元 $t = \dfrac{x_2}{x_1} > 1$，将目标不等式转化为单变量函数 g(t) 的符号判定模型，适用于齐次结构。

  • 对数均值不等式：处理对数结构偏移的专项工具，利用 $\dfrac{x_1-x_2}{\ln x_1 - \ln x_2} < \dfrac{x_1+x_2}{2}$ 直接实现双主元向单主元的放缩转化。

齐次化与变量消元

  • 齐次化构造：针对 f(x1) = f(x2) 的多元关系题，通过引入比值 k 或和积关系 S, P 将双变量耦合降维为单一变量分析。

  • 消元主元法：利用已知约束（如韦达定理 x1 + x2 = −b/a）彻底消除冗余变量，将多元目标函数强制转化为标准一维单调性问题。

辅助函数与算子理论

  • 导数指纹识别：见指纹 xf′ ± f 构造 [xf]′/[f/x]′；见 f′ ± f 构造 [exf]′/[f/ex]′；见 f′ln x ± f/x 构造 [f/ln x]′。

  • 积分因子万能公式：对于 f′ + Pf > 0，统一乘 $\mu(x) = e^{\int P \d x}$ 凑出全导数 [μ(x)f(x)]′ 判定复杂不等式的单调递增属性。

  • 算子移位恒等式：利用线性算子分解 D + P = e−∫PDe∫P 将复杂导数关系剥离，建立关于整体结构的递延判定逻辑。

极限论与数列放缩

  • 等价无穷小替换：在局部阶数分析中将 sin x, ex − 1, ln (1 + x) 等效为 x，快速简化分式求极限的预研路径。

  • 洛必达局限与变形：非标准型应先变形为 0/0 或 ∞/∞，并在每轮求导后校验条件，避免在非变号点误用。

  • 积分夹逼数列放缩：利用单调函数的面积定界 $\int_1^{n+1} f(x) \d x < \sum_{k=1}^n f(k) < \int_0^n f(x) \d x \ (f \downarrow)$ 解决神秘求和放缩。

  • 微分方程近似法：将递推式 an + 1 = an + g(an) 映射为微分方程 y′ = g(y)，通过解析解 y(n) 快速锁定数列通项的整数部分及阶数。
