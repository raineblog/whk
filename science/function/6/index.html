凹凸性与 Hermite-Hadamard 不等式

函数的凹凸性（Convexity）是导数考察中用于寻找“界限”的核心工具。函数的凹凸性是导数压轴题中用于寻找“界限”的核心几何工具。为了避免国内教材中“凹”与“凸”定义的语义冲突，本文统一采用直观的下凸与上凸进行描述。

下凸函数 (Convex)

若函数 f(x)f(x)f(x) 的图像呈现“向上开口”的 ∪\cup∪ 型形态（如 y=x2,y=ex,y=1/xy = x^2, y = e^x, y = 1/xy=x2,y=ex,y=1/x），则称其为下凸函数。

  • 几何本质：割线始终在弧线上方，切线始终在弧线下方。
  • 判据：f′′(x)>0f''(x) > 0f′′(x)>0。
  • Jensen 不等式：f(x1+x22)≤f(x1)+f(x2)2f\left( \dfrac{x_1 + x_2}{2} \right) \le \dfrac{f(x_1) + f(x_2)}{2}f(2x1​+x2​​)≤2f(x1​)+f(x2​)​。

上凸函数 (Concave)

若函数 f(x)f(x)f(x) 的图像呈现“向下开口”的 ∩\cap∩ 型形态（如 y=ln⁡x,y=x,y=sin⁡xy = \ln x, y = \sqrt{x}, y = \sin xy=lnx,y=x​,y=sinx），则称其为上凸函数。

  • 几何本质：割线始终在弧线下方，切线始终在弧线上方。
  • 判据：f′′(x)<0f''(x) < 0f′′(x)<0。
  • Jensen 不等式：f(x1+x22)≥f(x1)+f(x2)2f\left( \dfrac{x_1 + x_2}{2} \right) \ge \dfrac{f(x_1) + f(x_2)}{2}f(2x1​+x2​​)≥2f(x1​)+f(x2​)​。

基于凹凸性的切线放缩

切线放缩的核心思想是：利用线性函数（一次函数）去局部逼近复杂的非线性函数。这种方法在证明“函数恒大于 …\dots…”或“存在性证明”中具有极高的实战价值。

设函数 f(x)f(x)f(x) 在点 x0x_0x0​ 处可导，其切线方程为 y=f(x0)+f′(x0)(x−x0)y = f(x_0) + f'(x_0)(x - x_0)y=f(x0​)+f′(x0​)(x−x0​)。

  • 下凸平移性：若 f(x)f(x)f(x) 是下凸的（f′′>0f'' > 0f′′>0），则曲线始终位于切线的上方：

    f(x)≥f(x0)+f′(x0)(x−x0)f(x) \ge f(x_0) + f'(x_0)(x - x_0)f(x)≥f(x0​)+f′(x0​)(x−x0​)

  • 上凸限制性：若 f(x)f(x)f(x) 是上凸的（f′′<0f'' < 0f′′<0），则曲线始终位于切线的下方：

    f(x)≤f(x0)+f′(x0)(x−x0)f(x) \le f(x_0) + f'(x_0)(x - x_0)f(x)≤f(x0​)+f′(x0​)(x−x0​)

等号成立的充要条件通常为 x=x0x = x_0x=x0​。

一、 核心认知透视：为什么要把三角函数放进导数里？

在导数压轴题中，三角函数（sin⁡x,cos⁡x\sin x, \cos xsinx,cosx）本质上扮演着"扰动项"或"界限控制项"的角色。

  1. 多阶同构性（求导的轮回）：sin⁡x\sin xsinx 和 cos⁡x\cos xcosx 的导数以 444 为周期循环。这意味着，一阶导数无法判断符号时，我们可以肆无忌惮地求二阶、三阶甚至四阶导，直至超越项（如 ex,x2e^x, x^2ex,x2）的符号彻底压制住三角函数的震荡（如课件中 f′′′(x)=ex−sin⁡x>0f'''(x) = e^x - \sin x > 0f′′′(x)=ex−sinx>0）。
  2. 泰勒展开（降维打击）：超越函数在零点附近的本质是多项式。课件中大量引用的 1−x22≤cos⁡x≤1−x22+x4241 - \frac{x^2}{2} \le \cos x \le 1 - \frac{x^2}{2} + \frac{x^4}{24}1−2x2​≤cosx≤1−2x2​+24x4​，其实质是用代数多项式去逼近、替换不可解的三角函数，从而实现"非代数问题代数化"。
